　　在比川域基地更向西的地方，这里人迹罕至，末世爆发时也是难得丧尸最少的一片地带，在经过了三年之后，部分地区也被川域基地派出新人类队伍扫荡过，让在这片地带生存的幸存者避免丧尸的威胁。
<br/>　　一个个小型的基地也由此出现，或是简陋的部落接连在一起，相互依赖的生存。
<br/>　　只不过川域基地的扫荡只解决了丧尸问题，这里还存在着狂化动物的巨大威胁，尤其在当下的末世三年，动物近乎都强大到了一种恐怖的程度。
<br/>　　小型基地并没有抵挡这些动物的能力，一批批相聚在一起的人类，只能将生存地往高原迁移。
<br/>　　他们不是没有想过往东面走，去寻找大基地的庇护，但不是高阶新人类根本没有能力穿过充满狂化动物的丛林，更无法对抗那些不知道会从哪里冒出来的丧尸，更别说东方动不动就爆发战争，人类与丧尸，人类与异种，甚至人类与人类。
<br/>　　求生欲望驱使着这些人越走越高，最后安居在以前他们都难以想象的高海拔。
<br/>　　而这些人之中，就有着从遥远东方搬迁而来的叶永阳一家，这个有着好几代族人的家族，在长途跋涉之中遇到过多次危机，当他们到达西面的人类聚集地时，一家只剩下了星零几个人。
<br/>　　叶永阳最为叶家当下的一家之主，提出搬迁的决定并强行带着一家老小上下几十号人口徒步跋涉，一路上的大量家人惨死，让他在所难免的受到了家人愤怒的质疑。
<br/>　　但叶永阳不给任何人商量的机会，年迈的他在这段路途中展现出了惊人的毅力，也爆发出了难以想象的能量，硬是带着最后几个叶家人来到了这处海拔极高的小型基地。
<br/>　　小基地的首领名叫杨飞杰，年级很轻但却已经是个六阶新人类，是整个西部地带等阶最高的人。
<br/>　　并非在此之前没有出现过高阶新人类，只是那些人没有再留在这里，毕竟等阶一旦超过五阶，完全有能力在东部谋生，过上普通人难以想象的锦衣玉食生活。
<br/>　　杨飞杰却选择留下，在这里照应大批手无缚鸡之力的幸存者。
<br/>　　“你们想在这里安居？”杨飞杰望着新来到这个小基地的叶永阳等人，目光不禁露出一抹惊疑。
<br/>　　这些叶家人各个衣衫褴褛如同乞丐，一看就经历过难以想象的长途跋涉，甚至恐怖至极的危险，而他们的眼睛却明亮的过分耀眼，让人不得不心生怀疑。
<br/>　　忽然冒出来的几个流民，准确的找到这个并不容易发现的基地，顺利通过了大门处门卫的盘问调查，甚至来到了他杨飞杰本人的面前。
<br/>　　怎么看都怎么不对劲！
<br/>　　“是，请首领收留。”叶永阳不卑不亢的与之直视。
<br/>　　杨飞杰的目光扫过叶永阳身后的几个人，而后皱眉问道：“听说，你们是从东边来的？”
<br/>　　叶永阳点头，好似松了一口气般，苍老的面容上露出一个笑容：“是，从最最东边而来，横穿大半个华夏。”
<br/>　　杨飞杰的心中涌·出了些许异样的情绪：“你们，走过来的？”
<br/>　　华夏有多大人尽皆知，从东到西的横穿，那是多少公里？
<br/>　　叶永阳继续点头，眼神有些辛酸：“走了两年多，一家老小几十号人口，也就剩下了我身后的这么几个。”
<br/>　　杨飞杰没想到真是如此，顿时再看杨飞杰等人的目光就已经大变，这种骇人听闻的事情还真存在？
<br/>　　“你们，这到底是为了啥啊？”杨飞杰不能理解：“值得吗？”
<br/>　　以他的感知力，能够很清晰的感受到眼前的这几个流民，都只是最为普通的幸存者，连一阶新人类都不是，这种毫无战力的普通人，到底是怎么做到横穿大半个华夏的？
<br/>　　叶永阳没有过多解释，只是无奈道：“当整个家族都即将面临灭顶之灾，哪怕再艰难也要尝试守护家族传承，失去了大部分，总比留在东边一下子全死·光了好。”
<br/>　　随着话音的落下，杨飞杰不禁震撼和佩服，叶氏老人身后的几名年轻男女，也顿时震惊不已。
<br/>　　一路上叶永阳从未对他们说过原因，哪怕他们再如何愤怒和反对，这个家族老人也不过是沉默。
<br/>　　可是当下，当叶永阳说出这一层危机之后，这些叶家最后得以保存的年轻人，也纷纷坐不住了。
<br/>　　杨飞杰没有再为难叶永阳，略微交代了这个小基地的几个注意事项，便将基地角落里的一间简陋空屋腾出来让叶家几个人暂时居住。
<br/>　　面对这只有二十几平米的窄小屋子，叶永阳感激道谢，毕竟他们什么都拿不出来，能有一间屋子已经意出望外，哪怕是几个人挤在一起也没什么。
<br/>　　在屋内只剩下叶家人后，叶家的一名年轻男丁忍不住开始发问：“爷爷，叶家的灭顶之灾到底是什么？”
<br/>　　叶永阳伸出手，摸了摸暗藏在胸口的一条挂坠，眼神有些悠远：“你们的堂叔叶子博，接触到了一个姓白的富家少爷，这件事你们都知道吧？”
<br/>　　“知道。”一人回答，不解道：“那时候子博叔叔还说这个白家少爷可以让我们一家都避免危险，还能住进大基地享受贵族般的待遇，可爷爷您听闻后就勃然大怒，直接把子博叔叔逐出家族了。”
<br/>　　叶永阳苦笑：“那个白家少爷所在的白家，就是我们叶家在千年前被牵制的几大家族之一。”
<br/>　　“千年？”叶家的年轻人都惊呆原地。
<br/>　　叶永阳知道这些辛秘说起来是一段极长的历史，于是摆摆手简化道：“你们只要知道，那个白家想要我们全家人死就好。”
<br/>　　“可是，听子博叔叔说，那个白少爷很诚恳的想要见您啊？”也有叶家的年轻人有些怀疑。
<br/>　　叶永阳冷笑：“那是他想要我身上的一样东西，或者说想要我给他们当导游。”
<br/>　　“导游？”叶家年轻人发现越来越理解不了爷爷的话。
<br/>　　叶永阳双眸闪着寒光，声音更是带着一股冷气：“千年墓穴的导游。”
