        &lt;h3&gt;第三卷第一百六十四章  郝景文甜甜的恋爱&lt;/h3&gt;
<br/>
<br/>        郝景文开着车，陈思宇坐在车上一言不发。
<br/>
<br/>        她本来想坐陈子昂的车来着，但是寒伯安的气场太强，再加上大家都坐了陈子昂的车，郝景文一个人，显得有点孤单。
<br/>
<br/>        “怎么不说话？”郝景文打破沉默。
<br/>
<br/>        “不知道和郝总说啥。”陈思宇说，其实有些女孩子的心思很奇怪，明明是喜欢的要命，想见的很，就是装着不在乎，比喻说陈思宇对郝景文就是这样的。
<br/>
<br/>        从那次郝景文单约陈思宇之后，陈思宇见了郝景文心跳都会加快。
<br/>
<br/>        这一会她觉得她的心都要跳出来了，陈思宇捂着自己的心口，在心里对自己说：“陈思宇，镇定，陈思宇、镇定。”越是这样说她的心里越是慌乱。
<br/>
<br/>        陈思宇家是扬州的，苏州扬州都是出美女的地方。
<br/>
<br/>        陈思宇有着江南小女子的温婉和灵秀，在众多美女云集的地方，她不算是最起眼的，但是一定是很明亮的。
<br/>
<br/>        她有着时下流行的弯月眉，双眼皮，长长的睫毛盖在灵秀的大眼睛上，高挺的鼻梁，皮肤白皙光亮、身高1.60左右，身形纤细、开口说话时嗓音有点沙沙甜腻腻的感觉。
<br/>
<br/>        “思宇，是不是对我有什么意见？”郝景文会时不时的想起陈思宇，就连晚上陈思宇都会跑到他的梦里，可是现实中她却不想见他，郝景文对着倒车镜看了一下自己，嗯，感觉很帅啊。
<br/>
<br/>        像他这种有钱又帅的男生没有几个吧？怎么，旁边坐着的妹子一副拒人千里之外？嗨，还是上学的时候好。不过，工作太忙了，连谈恋爱的时间都没有，他也想和大哥二哥一样，谈甜甜的恋爱，郝景文看了一眼旁边的陈思宇，他最近一定是狗粮吃多了，被影响的，话说，他以后要多带陈思宇出去，看大哥他们撒狗粮，说不定陈思宇就开窍了。
<br/>
<br/>        上一次表白之后，陈思宇直接就是不见他，不回信息。
<br/>
<br/>        “我，我哪里敢对郝总有意见啊。”陈思宇心跳加速，又甜又苦，能和郝景文单独在一起，心里甜滋滋的，可是她又不确定郝景文是什么想法，她一个女孩子，心里想着一个男孩子，而且做梦都在想，说出来该有多丢人？
<br/>
<br/>        “不要叫我郝总、郝总的，叫我文文。”郝景文心里有些不舒服，老是这么生疏的怎么行？
<br/>
<br/>        “啊？”
<br/>
<br/>        “啊什么啊，叫我文文，快点！”郝景文直接命令陈思宇。
<br/>
<br/>        “文......文。”陈思宇有点结结巴巴。
<br/>
<br/>        “这样就对了，记住啊，以后就这样叫。”虽然陈思宇有点结巴，但是总归是改过来了，陈思宇叫他文文的时候，郝景文身上划过一丝奇异的暖流，还真是甜甜的感觉。
<br/>
<br/>        嘿，难怪大哥总是爱显摆。
<br/>
<br/>        到达酒店的时候，寒伯安看了一下手表：“你们比我们迟了20分钟。”
<br/>
<br/>        “路上有点堵车，你不信问思宇。”郝景文说，他就是想多和陈思宇单独呆一会怎么了？
<br/>
<br/>        “思宇？这么亲密了？”江俞轩笑着问。
<br/>
<br/>        “郝......”陈思宇开口。
<br/>
<br/>        “嗯哼？”郝景文看着陈思宇。
<br/>
<br/>        “文.......文开得慢，路上、路上确实有点堵车。”陈思宇脸红了。
<br/>
<br/>        郝景文满意的拉着陈思宇的手坐到座位上。
<br/>
<br/>        “不是说路上堵车吗？这么快？”寒伯安看着郝景文，不可思议。
<br/>
<br/>        “刚开始有点堵，以后全程高速。”郝景文看着他的两个哥哥嘚瑟的说。
<br/>
<br/>        吃饭的时候，郝景文一会给陈思宇夹这菜，一会给陈思宇夹那个菜，陈思宇一直红着脸默默的吃着，桌上坐的可都是她的老板啊。
<br/>
<br/>        吃了一会，三个人实在看不下去了，齐齐放下筷子，看着郝景文。
<br/>
<br/>        郝景文手中的筷子还夹着菜正准备往陈思宇的碗里放，发现大家看着他，奇怪的问：“你们怎么都不吃了？”
<br/>
<br/>        “吃饱了，没兴趣。”三个人齐声说。
<br/>
<br/>        “哦，那我吃。”郝景文将要夹给陈思宇的菜放进了自己的碗里，他好像惹众怒了，话说，他们秀恩爱的时候，他郝景文还不是好好的看着？
<br/>
<br/>        “文文，子昂有孩子的事情一定不要透露出去，尤其是张函的事情。”江俞轩这才说到正题上。
<br/>
<br/>        “我没有透露给任何人啊，思宇，你也没有透露给任何人吧？”郝景文问陈思宇，郝景文本来就知道陈子昂的事情，他们是不会说出去的。
<br/>
<br/>        “嗯，没有的。我们公司里就我一个人知道。”陈思宇也赶紧回答，陈思宇也是个聪明的姑娘，他们陈总一直单身，那一次不小心知道了秘密，她是无论如何都不会讲给别人听的。
<br/>
<br/>        “思宇，真乖，可以当一个守口如瓶的老板娘了。”郝景文点点头，一本正经的说。
<br/>
<br/>        “这可是你说的啊。”陈思宇的心一横，都已经说到这个份上了，她陈思宇又不是叫之及来，挥之既走的小姑娘。
<br/>
<br/>        大家被陈思宇的认真吓了一跳。
<br/>
<br/>        “是啊，我说的。”郝景文放下筷子，很认真的看着陈思宇，难道他一直都不认真吗？
<br/>
<br/>        “老板娘是个什么鬼。”陈思宇看见大家看着她，一瞬间，蔫了。
<br/>
<br/>        对于面前的四个精英，陈思宇感觉自己怎么都无法企及，郝景文光在颜值上甩她几条街，上次陈子昂说的话她还记得，虽然她的家庭条件也不错，但是对于郝景文还是差了很多。
<br/>
<br/>        “就是公司还缺一个老板娘。”郝景文耐心的解释。
<br/>
<br/>        “文文，这种话私下说就好了，你这样说会吓坏人家小姑娘的。”江俞轩笑着打断郝景文，这家伙做生意是一把好手，追小姑娘的本事一点也没有。
<br/>
<br/>        “思宇，我有吓坏你吗？”郝景文问急切的问陈思宇。
<br/>
<br/>        陈思宇点点头又摇摇头，她现在都不知道该怎么面对这几个人了。
<br/>
<br/>        郝景文觉得有些无力，别人的队友都是神助攻，他的队友好像是泼冷水的。
<br/>
<br/>        “思宇，最近你多和郝总联系一下，公司的财务明细也要让郝总知道，工作上的事情多请教郝总，以后呢你就归郝总管理了。”陈子昂开口，当媒婆的最高境界呢就是要神不知鬼不觉的将好事促成了。
<br/>
<br/>        之前陈思宇有问过她，但是那个时候郝景文的意思不太明确，今天的郝景文好像不是开玩笑，不如鼓励鼓励陈思宇。
<br/>
<br/>        寒伯安和江俞轩相互看了一眼。
<br/>
<br/>        陈思宇毕竟是陈子昂公司重要部门的人，一个公司的财务就是核心，虽然他们是股东，核心部门他们并没有接触，当然江俞轩除外，如果让陈子昂误会了就会产生隔阂，后期合作就不会太顺利。
<br/>
<br/>        一个企业中的高层如果产生隔阂和不信任，合作的时间不会长久的，会影响企业的发展。所以他们俩个对郝景文的意思装聋作哑，但是陈子昂倒是鼓励陈思宇和郝景文来往，看来陈子昂的心，怀还是很宽广。
<br/>
<br/>        陈思宇看着陈子昂，点点头，脸一红“嗯”了一声便没有了下文。
<br/>
<br/>        郝景文：“我好难！”
