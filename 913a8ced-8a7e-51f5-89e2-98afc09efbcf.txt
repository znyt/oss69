　　“怎么回事？古恒？你怎么受重伤了？”
<br/>　　这男子，也算是古家的一位小天才了，古锦到来，看到这伤势，眼神也是极为凝重，问着旁边的一名武者开口道。
<br/>　　“发生了什么事。”
<br/>　　“禀告二小姐，古恒公子之前离开宁州古城，被九魔城的魔头袭击了，伤势久久不愈，已经是极为虚弱了，我们利用了不少的丹药，都无法恢复古恒公子的伤势。”
<br/>　　旁边一名奴仆状的武者，急忙开口道。
<br/>　　“无法恢复伤势？”
<br/>　　古锦灵识扫了一眼，也是同样发现这古恒身上的问题。
<br/>　　这一股力量，已经是融入到了古恒的伤口之中去，令古恒根本无法恢复伤势。
<br/>　　在这样下去，就算是一尊生死境武者，也距离陨落不远了。
<br/>　　“这一股灵力，我也无法驱散。”
<br/>　　古锦打量了几眼，便是清楚，如果受到重伤的人，是她的话，那她也是毫无办法。
<br/>　　这种灵力，不是她足以化解的了的。
<br/>　　“我去请父亲前来。”
<br/>　　古锦凝聚灵力，在她的面前，化为了一道玉符，这玉符刚刚打出的一刻。
<br/>　　突然之间，一股寒意，便是涌上楚风眠的心头。
<br/>　　就在这古锦凝聚灵力的一刻。
<br/>　　这躺在地上的古恒，突然之间便是一跃而起，从他的手心之中，握着一把黑色的匕首，猛然之间向着古锦刺了过去。
<br/>　　“刺客！”
<br/>　　楚风眠脑海之中猛然冒出这两个词。
<br/>　　这古恒，之前伪装倒在地上，现在正是等着古锦稍微分心的一刻，突然出手。
<br/>　　在这古恒出手的瞬间，他身上的精血都燃烧了起来，燃烧精血，凝聚所有灵力，都是融入到了这一次的刺杀之中，要这一招之内，将古锦直接刺杀。
<br/>　　“给我破！”
<br/>　　在这瞬息之间，古锦甚至都没有来得及反应过来，她的注意力都在刚刚凝聚玉符的一刻。
<br/>　　却没有想到，在这个时候，这刚刚倒在地上还是半死不活的古恒，现在便是突然向着他刺杀而来，而且这古恒的实力，也是瞬息之间达到了生死境巅峰的实力。
<br/>　　不，这已经不是古恒了，而是已经换了一个人。
<br/>　　一个刺客，伪装成这古恒的样子，混进到了古家来，将古锦吸引过来，要这一招之内，刺杀古锦。
<br/>　　“暗杀之道，有点意思。”
<br/>　　这刺客出手的一刻，楚风眠的目光都带有着几分惊讶，这刺客出手的瞬间，动用的居然是暗杀之道。
<br/>　　暗杀，对大部分武者来说，都是曾经有过的行为，不过这远远无法称之为暗杀之道。
<br/>　　在远古时代，有着一群人，他们专修暗杀，开创出了暗杀之道来，相传这修炼暗杀之道的无上强者，足以跨越一个大境界，刺杀敌人。
<br/>　　这边是相当于一名圣者巅峰的人物，足以刺杀生死境巅峰的武者。
<br/>　　这简直是一件无法想象的事情，而是在那个时代之中，却真真正正的发生过。
<br/>　　不过这暗杀之道，在数万年前便是已经消失了，楚风眠都只是从古籍记载之中看到过，还没有亲眼见到过，如今亲眼见到，楚风眠才反应过来，这居然是一名精通暗杀之道的刺客。
<br/>　　在九域之中，暗杀之道居然再次出现了。
<br/>　　“死吧！”
<br/>　　刺客面目狰狞，看着古锦，眼神中尽是杀意，疯狂，他潜入到这古家之中，暗杀古锦，已经是不惧生死，简直是如同疯子一般的人物。
<br/>　　只不过，还没有人可以在楚风眠的面前杀人，尤其是这古锦，还算是间接帮助过了楚风眠，楚风眠当然不会让她死。
<br/>　　“你的暗杀之道很不错，不过我很想要知道，你这暗杀之道，到底是从何而来？”
<br/>　　楚风眠突然之间身躯一动，直接挡在了那古锦的面前，他大手一挥，这刺客手中的匕首，便是被直接震飞出去。
<br/>　　“可恶！”
<br/>　　这刺客显然都没有想到，在这个时候，居然还会突然杀出来了一个人。
<br/>　　如今趁着古家家主不在，正是刺杀古锦最好的机会，可楚风眠的出现，却是打破了他的一切计划。
<br/>　　“天魔解体大法！走！”
<br/>　　这刺客的身躯，猛然爆炸开来，突然一道黑色遁光，便是从他的身躯之中飞了出去，直接向着古家之外，飞走。
<br/>　　“想逃？”
<br/>　　楚风眠冷冷一笑，他的大手一挥，这一道黑色遁光，便是直接是被楚风眠抓在了手心之中，这黑色身影，正是那刺客，不过这刺客也不在是古恒的样子了，而是一名瘦小的男子。

<br/>　　古家家主无比愤怒的声音，突然响起，古家上方的天空都被撕裂开来，古家家主的身形，将那从中出现，他的脸色简直是愤怒到了极点。
<br/>　　上一次古锦离开宁州古城，都是差一点的死在了血剑魔尊手中，这已经是让古家家主无比懊恼了。
<br/>　　可现在，在这古家之中，居然都有人敢刺杀他的女儿，让他如何不愤怒。

<br/>　　楚风眠平静的开口道。

<br/>　　古家家主看着那刺客，愤怒的吼道。

<br/>　　这刺客落入到楚风眠的手中，脸上毫无惧意，反而是极为疯狂，大声的吼了起来。
<br/>　　“哼！什么九魔城！告诉你们，我是六道魔门的人！不久的将来，我们六道魔门，将会占据整个中域，你们宁州古城，也会落入到我们六道魔门的手中！”
