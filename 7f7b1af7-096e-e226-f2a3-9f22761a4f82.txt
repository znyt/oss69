　　拍卖会结束了，所有武者纷纷离场。
<br/>　　十号包厢中，杨天威靠在兽皮椅上，面带微笑，心满意足。
<br/>　　这源石虽然只是劣质源石，真正的作用和价值不值真正源石的十分之一，但对于他来说，都有巨大的作用。
<br/>　　他修炼的武学，和别人有所不同，乃是土金双属性武学，这源石真是他需要的宝物。
<br/>　　“天威师兄，我们借了这么多灵晶，短时间内怕是还不上了！”绿衣女子叹道。
<br/>　　扬天成从十几名天虚宗弟子的手中，可是借了很多的灵晶，这可不是小数目
<br/>　　“呵呵！”
<br/>　　杨天威闻言一笑，摇头道：“高师妹，谁说我要还了，无需还他们！”
<br/>　　“啊！不还了？”绿衣女子闻言一怔。
<br/>　　“当然，就他们这些人，谁敢找我索要？”杨天威不屑的笑道，他压根就没想过要归还。
<br/>　　“也是！”绿衣女子点了点头，面上也露出了一丝笑容，的确不用归还，那些人也不敢说什么！
<br/>　　“族兄，我们要不要先出去，万一那苏莫跑了怎么办？”
<br/>　　此时，扬天成站了起来，面上带着一丝焦急之色，在他心中，擒住苏莫才是第一要务。
<br/>　　“你急什么？”
<br/>　　杨天威呵斥一声，摆了摆手道：“先把源石拿到手再说，而且，苏莫拍卖了不少宝物，总要核算收益吧！”
<br/>　　杨天成闻言无奈的点了点头，对方说得也是，还是先拿到源石再说吧。
<br/>　　不过，扬天成也没有大意，通过水幕墙壁，一直在注意着二十九号包厢，以防苏莫溜了。
<br/>　　没等多久，源石被人送进了包厢中，扬天成支付了灵晶，小心翼翼的收取了源石。
<br/>　　“走，我们去大门外堵苏莫！”杨天威大手一挥，朗声说道。
<br/>　　言罢，他便带领着杨天成和绿衣女子，向外走去。
<br/>　　想要擒拿苏莫，显然不能再拍卖场中进行，只能到大门口去堵了。
<br/>　　……
<br/>　　二十九号包厢中。
<br/>　　苏莫和吕罡静静的等待，等待拍卖所得的灵晶被送上来。
<br/>　　“苏莫，等会我们问问，这拍卖会有没有后门，免得被扬天成和杨天威他们截住！”吕罡面带一丝忧色的说道，虽然苏莫很自信，但他还是有些担心。
<br/>　　“嗯！”苏莫点了点头，虽然他不惧杨天威等人，但若是能不起冲突，他也尽量会避免。
<br/>　　现在对于他来说，也是尽量少惹麻烦。
<br/>　　咚咚咚！
<br/>　　两人没有等待多久，包厢外响起了低沉的敲门声。
<br/>　　“请进！”苏莫朗声说道。
<br/>　　随即，包厢的门被推开，龚忠满面笑意的走了进来，道：“两位，让你们久等了！”
<br/>　　“龚长老客气了！”苏莫长身而起，微笑着摇了摇头。
<br/>　　“小友，这是你的拍卖所得，共计一万二千三百三十六块中品灵晶，你可以清点一番。”龚忠手中出现了一枚储物戒，递向苏莫。
<br/>　　苏莫接过储物戒，意念一探，稍稍清点了一番，储物戒内果然是一万两千三百余块中品灵晶。
<br/>　　“没错！”苏莫点了点头，继续道：“龚长老，拍卖会既然已经结束，那我们也告辞了！”
<br/>　　“请！”龚忠微微颌首，也没有挽留。
<br/>　　“龚长老，不知你们拍卖场，有没有后门？”吕罡沉声问道。
<br/>　　“后门？”龚忠闻言一怔，随即便摇了摇头，道：“并没有后门，只有一个正门！”
<br/>　　吕罡闻言一叹，没有后门那就只能从正门走了，总不能直接毁掉人家拍卖场，破墙而出吧！
<br/>　　“吕罡，我们走！”苏莫招呼一声，便大步走出了包间。
<br/>　　苏莫丝毫不担心，扬天成等人若是敢堵截他，他会让他们后悔。
<br/>　　吕罡见此，便也没有多言，急忙跟着苏莫走了出去。
<br/>　　龚忠见此，心中有些疑惑，不清楚发生了何事。
<br/>　　苏莫虽然没有带银缤面具，但也是改变了容貌的状态，所以龚忠并不认识。
<br/>　　而且，龚忠没有见过苏莫，也不熟悉苏莫的气息。
<br/>　　苏莫和吕罡大步而行，还没接近大门之处，远远的，他们便能看到，杨天威和扬天成两人，正伫立在大门之外。
<br/>　　两人的身上，个个气势腾腾，杀机森然，直接堵在了大门前。
<br/>　　而那一名绿衣女子，却是距离两人稍远，站立在不远处。
<br/>　　不仅如此，大门之外，还围拢了大量的武者，均是从拍卖场中出来的武者。
<br/>　　这些人，见到杨天威和扬天成两人，如此阵势的堵在拍卖会场的门前，哪里还不清楚两人要干嘛？
<br/>　　“杨天威和扬天成，是要找谁的麻烦？”
<br/>　　“不清楚，好像也没有人得罪他们啊！”
<br/>　　“看两人的模样，好像杀气腾腾，怕是要出大事了！”
<br/>　　众人议论纷纷，个个心中好奇不已，不知道杨天威和扬天成两人，是要找谁的麻烦！
<br/>　　人群之中，乙薇带着几名道宗弟子，伫立在远处，同样没有离开。
<br/>　　“不知道谁要倒霉了！”一名道宗弟子幸灾乐祸的说道。
<br/>　　“看看吧！”乙薇淡淡的说道，杨天威号称是天虚宗真传弟子第二人，这次说不定能看出对方的实力，从而判断出厉恨天的实力。
<br/>　　毕竟，要不了多久，古魔墓地开启，杨天威、厉恨天都是她的对手。
<br/>　　天狼亦是没有离开，一头银发、高大魁梧的天狼，站在人群之中如同鹤立鸡群，极为醒目。
<br/>　　而天狼身边的武者，均是不由自主的微微与其拉开了距离。
<br/>　　“搞什么鬼！”天狼望着杨天威两人，眉头微皱，他本来还想等杨天威离开古桐城之后，他直接从后者的身上，将源石抢过去，没想到后者居然在这里挑事。
<br/>　　“苏莫！”吕罡遥望着门外的情况，顿时有些紧张了起来，杨天威已经摆开了阵势，要对付苏莫了。
<br/>　　不过，此刻的苏莫，却是并没有在意外面的杨天威等人，他的目光落在了前方一名白衣女子的身上。
<br/>　　在他的前方不远处，一名白衣女子，面上戴着轻纱，款款而行，正在向门外走去。
<br/>　　这名女子身姿高挑，轻柔妙曼，看着其柔美的身姿，都是一种莫名的享受。
<br/>　　“此女……！”苏莫望着白衣女子的背影，神情怔怔，他感觉这背影——好熟悉！
<br/>　　这不是——青璇吗？
<br/>　　苏莫脑中一震，因为这个白衣女子的背影，和青璇的背影几乎一模一样。
