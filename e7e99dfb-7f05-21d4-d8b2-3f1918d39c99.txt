　　“平——乱——诀！”
<br/>　　如同死神之音，淡漠无比，带着浓浓的死亡之意，以一种誓要荡平天下****的信念，巨剑横空，于虚无之中走出，荡平一切****。
<br/>　　这平乱诀是公孙轩辕一生征战所创，结合葬剑诀、乱剑诀等其他剑诀，蕴含强烈的杀戮法则，更带着一种人族皇者的威压霸道的剑势。如今虽然只是草创，但其威力也绝对不容小觑。
<br/>　　看到这一剑横空而来，蚩尤脸色顿时大变，这一剑的威力已经能够威胁到他了，他必须小心应对，否则必定会阴沟里翻船。
<br/>　　“魔神斩之弑神灭！”
<br/>　　蚩尤双翅振天，手中虎魄魔刀连舞，强大的毁灭刀气带着浓浓的血腥味，不断冲击着人的神经，同时一种无形的力量，如同一只滔天巨兽，发出噬人的魔光，斩向朝他斩来的惊天巨剑。
<br/>　　“轰隆隆！”
<br/>　　巨剑与魔刀相触，霸道的剑气，噬魂的魔气，在这一刻完全爆炸开来，如同一朵火花在虚空绽放，充满了美丽，但同时也带着浓浓的毁灭气息。
<br/>　　蚩尤轻轻松了一口气，至少这一剑自己挡住了，但还来不及喘气，就在此时，他又感觉到一股强烈的死亡危机，比之前还要强烈。
<br/>　　“不好！”
<br/>　　蚩尤心中暗道一声，连忙聚刀向着身后劈去，这一刀他使出了全力，不敢有丝毫的保留，巫族强大的肉身带着强悍无匹的力量，逆斩而上。
<br/>　　“尔敢！”
<br/>　　“大胆！”
<br/>　　九黎一方传来愤怒的喝斥声，与此同时，天地一片昏暗，一道巨大如山岳般的大印从天而降，带着强悍的力道与镇天的威势，向着战场中的蚩尤狠狠地砸落下来。
<br/>　　翻天印！
<br/>　　一击大印，可翻苍天！
<br/>　　轰！
<br/>　　蚩尤仓促之间抵挡，就算用了全力，还是没能挡住这翻天一印，最终被砸落进地下，甚至就连战场上一大部分地面，都被砸落的深陷下去数千丈，周边满是裂缝，场面凄惨无比。
<br/>　　翻天印，第一次亮相便给人一种无可匹敌的印象，不愧翻天之名，不周山所炼，没有一样是简单的。
<br/>　　显然，刚才是广成子借机暗中偷袭了一下蚩尤。战场本就是如此，不是你死就是我活，哪里还有机会给你一对一的对决。
<br/>　　轩辕一方被广成子这一击弄得目瞪口呆，先前还是好好的，看着蚩尤大战轩辕，怎么眨眼间就成了这个样子？
<br/>　　与此同时，蚩尤一方却是愤怒无比，看着广成子，眼中满是杀机与怒火。
<br/>　　“混蛋，竟然偷袭！”
<br/>　　“真是该死！”
<br/>　　“上，杀了他们！杀啊！”
<br/>　　“吼！给我杀，为首领报仇！”
<br/>　　“……”
<br/>　　顿时，九黎部落的大军举起战旗，敲响战鼓，在雨师、风伯、刑天、重山九婴等大巫的带领下，举兵冲杀过来。
<br/>　　面对如此大的阵势，轩辕一方自然是不能退缩，公孙轩辕在第一时间做出决定，高高举起手中轩辕圣剑，冲着前方挥下，同时口中大声喝道：
<br/>　　“杀！”
<br/>　　顿时，双方部落勇士，冲杀在一起，厮杀起来。天上飞舞的，地上奔跑的，地下遁地的，御使妖兽的，飞剑横空的……
<br/>　　无数修士拼杀起来，符篆漫天飞舞，神通玄法随处爆炸，法宝满天轰杀，冰雨，风雷，毒雾，沼泽，火海，陨石……
<br/>　　战阵变化，演化出各种阵势，变化出滔天巨兽，相互撕咬，战斗。
<br/>　　各种各样的玄术神通施展出来，爆炸开来，血雾弥漫，如同一朵朵血色而妖艳的花朵在战场上盛开，美丽而诱人，但却充满了凄凉。
<br/>　　生命是如此的脆弱，一花开放，一花凋零，生命又是如此的美丽，可惜只能是昙花一现，战场，本就是如此残酷，战士的宿命，便是战死。
<br/>　　入眼所见，满目凄迷，那漫天的喊杀声、惨叫声、爆炸声，伴随着野兽的怒吼，大地的颤动，天空的闷响，显示出大战的残酷与凶猛，激烈。
<br/>　　天地作鼓，风雷击罄，为战死的勇士送行。
<br/>　　“风来！”
<br/>　　一声怒吼，在战场上响起，如怒风狂啸。
<br/>　　紧接着，在战场上便吹起了强劲的风暴，这风暴刚开始还小，但随着时间的推移，只是转瞬间，便成了狂暴的龙卷风暴。
<br/>　　数十上百的龙卷风暴从战场四周席卷而来，卷起无边狂沙，冲击进战场。不知多少修士被卷进那庞大的风暴之中，被无尽风力撕裂成粉碎。又有无数风刃吹过战场，无尽战士为此送命。
<br/>　　眼看众多修士为此送命，还有更多的修士要被这狂风所杀，公孙轩辕不由得有些着急了，他看向周边的众仙，希望有人能够抵挡这狂暴的风力。只是大多数仙人对此只有苦笑，他们都是一些散修，哪有法宝来抵挡啊。轩辕无奈，只得看向阐教一方，希望他们能够有办法制止这一切。
<br/>　　“师侄莫急，看我神通！”
<br/>　　这时，一道桀骜淡漠的声音不急不慢的响起。紧接着，人群中走出一个满头金发，眼神锐利，鼻尖微勾的道人，正是阐教的大罗金仙迦楼罗，也是天地间第一只金翅大鹏，是孔宣的兄弟。
<br/>　　“有劳师叔了。”
<br/>　　轩辕一听，急忙感谢道，他还是知道这位师叔的本领的。
<br/>　　金翅大鹏微微点头，只见他轻轻一踏，飞上高空，紧接着他变幻出本体金翅大鹏。大鹏横空，全身金灿灿，满身金色翎羽如同一柄柄锋利的剑刃，体型庞大无比，如同山岳，两翼广六万里，双翅一振便是云程九万里。
<br/>　　却见那金翅大鹏双翅朝着那数十龙卷飓风轻轻一扇，便有无数狂风呼啸而出，挡住那战场中前进的死亡飓风，紧接着他又轻轻一扇，更加强大的风力呼啸而出，龙卷风开始倒卷而去。之后，他再次一扇，却见那数十龙卷风向着敌方阵营席卷而去。
<br/>　　三扇，仅仅三扇，便将那强大的龙卷风扇回去，真是强大。
<br/>　　“哼！”
<br/>　　九黎一方中，大巫风伯见此，冷冷一哼，大喝一声：“风散！”
<br/>　　只是眨眼的时间，之前那狂暴无比的死亡之风，便缓缓消散于无形之中。
<br/>　　高空中金翅大鹏却又再次变幻出人性，他却没有回来，而是将目光盯住了大巫风伯，那锐利的眼神，牢牢锁定风伯，就像是猎鹰锁定猎物一般，仿佛能够将人深深刺穿。
<br/>　　“雨来！”
<br/>　　又是一声大喝，却是那九黎一方的大巫雨师，在施展强大巫术神通，召唤大雨前来降敌。
<br/>　　“哗啦啦啦——”
<br/>　　不过眨眼的时间，战场上方的天空便下起了淅淅沥沥的小雨，紧接着，雨点渐渐变大，开始变得如石子般大小，一个雨点落下，便能砸死一个普通修士。一个个雨点落下，便能葬送一大片军队。
<br/>　　雨点越来越大，雨速也越来越快，狂暴的倾盆大雨，如瀑布一般，冲刷下来，战士们都被淋湿了，而且这雨也给战士们造成很大的阻力，哪还有精神战斗？
<br/>　　“轰隆隆！”
<br/>　　片刻后，山石崩塌，无边山洪狂涌而来，冲进战场，滔滔洪水淹没无尽修士，带走无数生命。
<br/>　　轩辕大军一方，再次面对巨大危机！
