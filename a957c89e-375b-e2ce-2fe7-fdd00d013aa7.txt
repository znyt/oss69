　　慕清霄神识在山洞周围扫视一圈，并没有任何隐藏的未知危险，调动灵气，双手快速结印，口中轻吐而出：“封！”
<br/>　　事先在洞口设下结界，自然是为了避免炼丹过程中，受到魔兽的干扰。
<br/>　　封印禁制，不仅是防御禁制，攻击禁制，更包括幻术禁制，哪怕没有掌握禁制施法方式，设定幻术禁制对他来说也是轻而易举。
<br/>　　可别忘了，他的永恒万花筒写轮眼，其中可是寄存着极其恐怖的幻术。
<br/>　　布置完禁制后，只要慕清霄不解除，哪怕外界有修士路过，也看不到瀑布旁明显的山洞。
<br/>　　山洞不是很大，但用来炼制丹药却是有足够的空间，况且慕清霄也不是居住在这里，炼制完丹药后，他就会离开此处。
<br/>　　随意盘坐在地上，慕清霄取出在乌坦城买来的普通炼丹炉，随后将三种灵药取了出来，开始炼制三品丹药‘清灵丹’。
<br/>　　清灵丹的效果，能够使服用者放空心灵，抛弃杂念，快速进入修炼状态，而且能够避免走火入魔的风险，对许多修士杂念繁多的修士来说，是修炼途中必备良药。
<br/>　　虽然丹药品级不高，但价值却相当于四品丹药，拿去拍卖行，也能拍出不低的价格。
<br/>　　取出炼丹炉后，一股浓郁的药香味就弥漫在山洞中，慕清霄手心窜起一缕九霄琉璃焱，毫不犹豫的就丢尽炼丹炉中。
<br/>　　九霄琉璃焱在炼丹炉中闪烁，却没有将丹炉烧毁，自然是因为他的控制力强大到了不可思议的程度，炼丹的同时，又能不损坏炼丹炉。
<br/>　　好的炼丹炉，往往炼制出的丹药药效会强上不少，慕清霄手中的只是一鼎较为普通的炼丹炉，炼制一般的丹药足以。
<br/>　　若是想要炼制更高级的丹药，这样的普通丹炉往往在炼制过程中，便顶不住丹药力量的撞击，瞬间炸裂开来。
<br/>　　所以，炼制三品丹药已经是极限，慕清霄也想着，等有机会就去找一鼎好的炼丹炉，毕竟掌控炼丹术，对他未来修炼有许多的帮助。
<br/>　　燃起火焰后，慕清霄用灵魂力量控制着灵药，先将辅助药材，龙阳果投入丹炉中，只是一瞬间，龙阳果就化作一滩红色的液体，悬浮在炼丹炉中。
<br/>　　随后，又控制着其他两种主药材，相继投入炼丹炉中，三种灵药互相搅拌在一起，隐隐有融合的趋势。
<br/>　　慕清霄结印，微微加强九霄琉璃焱的火力，然后将炼丹炉盖上，闭上双眼，如同老僧入定，盘坐在山洞中，至于丹药的炼制过程，就需要静静的等待了。
<br/>　　三品丹药，一般只需要两三天的功夫就能炼制完毕，炼丹需要消耗极强的精神力量，例如时刻要保持火焰的强弱，偏差一丝可能就会影响丹药的炼制。
<br/>　　炼丹过程中，需要保持百分百的清醒，对于药尘那种大陆第一的炼药师来说，做到这一点自然不难，对于别人来说，却是苛刻到了极点。
<br/>　　当然，慕清霄是个列外，他虽然不知道自己的灵魂力量有多强大，但只要看自己的神识就清楚，区区一些精神力量，对于他来说，无伤大雅。
<br/>　　而且，比起其他炼药师，慕清霄最大的优势就是他血脉与火焰，不仅能够增强药效，而且也不会有一丝的灼热感。
<br/>　　像许多高级炼药师，炼制高级丹药的时候，炼化药材，炼制过程中都需要将火焰调到极高的温度，而且要在极高的温度下，忍耐几天几夜。
<br/>　　甚至比较长的，几个月也是不少，而慕清霄本身就是高贵的凤凰血脉，身处灼热火焰中本就像普通人沐浴阳光一样舒服，虽然九霄琉璃焱没有一丝的温度。
<br/>　　它存在，却又仿佛不存在，说它不存在，但它又拥有焚烧一切的力量...
<br/>　　炼丹过程中，慕清霄盘坐在山洞中，身如槁木，浑身没有一丝的气息波动，仿佛已经坐化，但炼丹炉中的火焰却一只保持适当。
<br/>　　而且，不时就有一朵鲜花，莫入他的口中，这些自然是能够增强灵魂力量的灵药，将极品灵药当成糖果一样食用的，恐怕也就他这一家了。
<br/>　　时间在不知不觉中流逝，到了第二天，炼丹炉中的火焰变得越来越弱，直到第三天，炼丹炉中的火焰熄灭，彻底消失不见。
<br/>　　随即，一股浓郁的清香就从炼丹炉中溢了出来，闻到这股清香，慕清霄剑眉一颤，缓缓睁开双眼，嘴角勾起一丝笑容，道：“三品清灵丹，成了。”
<br/>　　掀开丹炉，任由顶盖丢在一旁，慕清霄目光落在炼丹炉内，其中正躺着数十颗丹药，散发着乳白色的光晕，一阵清香席卷而来。
<br/>　　满意的点点头，慕清霄将丹药装进容器中。
<br/>　　拍了拍白衫上的几丝灰尘，站起身来，刚想要收拾东西，就有一道剧烈的能量波动，以及狂暴的狮吟声，忽然在天空中犹如闷雷炸响。
<br/>　　听着这蕴含着狂暴气息的狮吟声，慕清霄脸色微变，双手快速结印，解开禁制后，也顾不得破旧的炼丹炉，直接冲出山洞，目光紧紧盯着远处的天空。
<br/>　　那里，正是狂暴能量的传出地点，虽然距离有些远，但慕清霄却能够清晰的看见，是一个身姿妙曼，雍荣华贵的女子身影。
<br/>　　“六阶魔兽，紫晶翼狮王，相当于斗皇强者，不会这么巧吧，难道这女子就是云韵？”
<br/>　　慕清霄嘴角抽搐了几下，低声呢喃，真是强大的女人。
<br/>　　云韵秀雅绝俗，自一股轻灵之气，美目流盼，含辞未吐，气若幽兰，就如冰山雪莲般纯洁清丽、素雅高洁，这是一位堪称完美卓绝的女子。
<br/>　　饶是见惯了倾城美女的慕清霄，眼中都透露出难以掩饰的惊艳之色。
<br/>　　蔚蓝的天空上，剧烈的能量波动不断的制造出一声声宛如闷雷般的声响，就是慕清霄隐匿气息站在几百米外，都能感受到其中蕴含的能量。
<br/>　　哪怕是斗灵级别的修士，若毫无准备的近距离接触战场，恐怕都会被这股力量震的七窍流血而亡。
