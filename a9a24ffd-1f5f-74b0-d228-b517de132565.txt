　　萧家府邸内，每一个角落，都充斥在激烈的战斗中。
<br/>　　兵器的碰撞声，武技的对轰声，不绝于耳。
<br/>　　一方是慕容家的人，如狼似虎；一方是萧家的人，顽强抵抗。
<br/>　　但萧家，显然处于下风，形势很不妙。
<br/>　　“该死，慕容家哪里那么多的武者。就算陨星山脉那边的人手回来了，也不至于有那么多武者。”
<br/>　　一个中年女子一手拖着一个少女，一手杀敌，愤愤地说道。
<br/>　　“如意执事，有很多敌人都是生面孔，明显不是慕容家的人...”一个萧家武者说道。
<br/>　　他的话还未说完，已经被数个敌人围攻而死。
<br/>　　“混账。”中年女子数道剑气劈出，将那几个敌人劈死。
<br/>　　嗖、嗖、嗖...
<br/>　　数个敌人立刻赶来，将中年女子团团围住。
<br/>　　“萧家第一执事，萧如意，果然厉害，哼，便先杀了你吧。”数人立刻发起攻击。
<br/>　　中年女子，也就是萧如意浑身真气澎湃而出，一人挡住数人，却丝毫不落下风。
<br/>　　“哼，慕容伤，你这个慕容家第一执事越活越回去了，竟需要联手才敢与我战斗吗。”萧如意冷哼一声，一剑将这数人震退。
<br/>　　这数人，皆是慕容家执事。其中一个还是慕容家第一执事，后天九重武者，慕容伤。另外三个也是精英执事，后天八重武者。
<br/>　　“废话少说。”慕容伤冷笑一声，“成者王败者寇，你们萧家今天必败无疑。紫云城，将再没有你们萧家。呵呵，你现在还带着个小女娃，我看你如何能在我们手中活命。”
<br/>　　萧如意一身实力不俗，远超其它后天九重武者。
<br/>　　可惜，她还带着一个少女，被拖了后腿，战斗渐渐落入了下方。
<br/>　　嗤嗤两声，一个不慎，她的手臂被刺伤了两剑，鲜血染红了衣袖。
<br/>　　“如意执事，您不要管我了，全力杀敌吧。”少女着急地说道。
<br/>　　少女长得清纯可人，美丽出尘，正是依依。
<br/>　　“不行，我答应过那小子，要照顾好你的，你跟紧我。”萧如意快速说了一声，再次挥剑应敌。
<br/>　　萧家府邸内，处处都是战斗。
<br/>　　但大多是萧家武者被数个敌人围攻，渐渐地已经出现了伤亡，一个又一个萧家武者倒下，被乱剑杀死。
<br/>　　萧如意这边，手臂上又中了一剑，但仍旧苦苦支撑，着急无比，“该死，大长老被牵制住了，二长老又要游走于战斗中，其余长老也有对手，无法支援。再这样下去，萧家危矣。”
<br/>　　另一边，又是数个萧家族人战死。
<br/>　　萧如意身上也多了数道伤痕，伤势越来越重。
<br/>　　“如意执事，不要管我了。”依依一把挣脱开萧如意的手，打算与敌人拼了。
<br/>　　“依依，别做傻事。”萧如意大急。
<br/>　　“萧如意，先管好你自己吧。”慕容伤牵制住她，而后狞笑一声，对身后的一个执事道，“去将那小丫头宰了。”
<br/>　　“是。”一个执事狰狞一笑，一个跳跃来到依依面前，“死吧。”
<br/>　　依依吓得闭上了眼睛，她不想拖累萧如意，宁愿死。
<br/>　　但在这个危急关头，她心中竟还傻傻地呢喃着，“少爷，依依死了，你会伤心吗？也不知以后能不能照顾好自己...”
<br/>　　恰在此时，一只温暖而有力的手掌搭在了她可爱的脑袋上，宠溺地摸了摸。
<br/>　　“你说呢？”
<br/>　　熟悉的声音传入依依耳中，依依立刻睁开了眼睛，她知道，那是萧逸少爷的声音。也只有萧逸的声音，才会让她这么激动。
<br/>　　“少爷，你回来了？”依依激动地说道。
<br/>　　“嗯。”萧逸点点头。
<br/>　　这时，依依看了眼自己前方，刚才那个欲要杀她的慕容家精英执事，竟已经倒在了地上，脖子上有一条醒目的血痕。
<br/>　　萧逸一把抱住依依，而后一个跳跃，来到萧如意身边。
<br/>　　“如意执事，替我保护好依依，其他的交给我。”萧逸将依依推向萧如意，而后独自迎上了慕容伤等人。
<br/>　　萧如意见一道人影忽然袭来，先是一惊，当看到是依依时，连忙接住。
<br/>　　而后看到萧逸的出现，顿时大急，“萧逸，你怎么回来了？”
<br/>　　“诶呀，该死，不过是个后天一重的小子，怎么这么冲动。萧逸你快回来，如今萧家危矣，你绝对不容有失....”
<br/>　　然而，她的话还未说完。
<br/>　　只见萧逸手中火剑划过，一道火红之芒划过，慕容伤等人应声倒地，脖子上血流如注。
<br/>　　“嘶。”萧如意倒吸一口凉气，“慕容伤，外加两个后天八重武者，就这样死了？这小子怎么变得这么强了。”
<br/>　　待得击杀这三人，萧逸看了眼周遭不容乐观的情况，顿时皱起了眉头，“怎么回事，慕容家怎么有这么多武者。”
<br/>　　萧如意跑过来，道，“慕容家不知从哪请了帮手，人数远多于我们萧家。”
<br/>　　萧逸追问道，“大长老和二长老呢，有他们二人在，慕容家武者再多也无济于事。”
<br/>　　萧如意面露急色，道，“现在情况危急，不知该如何向你解释，总之，大长老被牵制了，情况很不妙。”
<br/>　　“二长老则在游走着，若不是他在帮忙，以我们萧家武者远少于慕容家的情况来看，我们恐怕早就全死了。”
<br/>　　萧逸看了眼，果然，场上有一道人影在不断飞跃，快速地击杀着敌人，给萧家武者减轻了不少压力。
<br/>　　人影正是二长老。
<br/>　　萧逸眉头一皱，快速道，“保护好依依，我去杀敌。”
<br/>　　萧如意道，“还是我去吧，你保护依依，毕竟我的实力更强。”
<br/>　　她的话还未说完，发现萧逸已经消失在原地。
<br/>　　“好快的速度。”萧如意吓了一跳，自己这个萧家第一执事的眼睛，竟完全跟不上萧逸的速度。
<br/>　　偌大的萧家，成为了一个血腥的战场。
<br/>　　而这战场上，忽然出现了一道火红色人影。
<br/>　　火红色人影所过之处，必有一个敌人应声倒地，血流如注。
<br/>　　火红色人影，如同一道燃烧的疾风，所过之处，将敌人烧成了灰烬。
<br/>　　一个萧家执事本在苦苦支撑，几乎撑不住了。然而，一道火红色人影掠过，他却惊讶地发现，自己的对手已经死了。
<br/>　　“好强...是萧逸，不，是少家主。”这个执事立刻大喜。
<br/>　　“去帮其他人。”萧逸喝了一声，再次杀敌。
<br/>　　“是。”该执事应答一声。
<br/>　　随着萧逸的加入，萧家族人的压力明显减轻了不少。敌人一个个倒下，萧家执事得以腾出手来帮助其他族人。
<br/>　　这是一个此消彼长的关系，萧逸杀敌越多，便有越多的族人能去帮助其他人。
<br/>　　不多时，数十个敌人死于萧逸剑下。
<br/>　　二长老也发现了萧逸的出现，顿时大喜，一掌将数个敌人击毙，而后来到萧逸身边，道，“小子，你可能抵挡这些敌人？”
<br/>　　“放心。”萧逸自信道，“如今的我，先天之下无敌，杀后天九重如杀鸡。”
<br/>　　“杀后天九重如杀鸡？好，很好。”二长老大笑一声，道，“如今各大长老都在和慕容家的长老在战斗。”
<br/>　　“大长老那边更是被牵制住了，我得立刻去助他。”
<br/>　　萧逸点点头，道，“好，这里交给我。”
<br/>　　二长老一个瞬身，消失在原地，往家族深处而去，那里是先天武者的战场。
<br/>　　萧逸这边，再次疯狂杀敌。
<br/>　　在他面前，根本无一合之敌。无论是凡境武者，抑或强大的后天武者，遇到他都是被一剑击毙。
<br/>　　慕容家执事已经注意到萧逸，并完全重视起来，七八人立刻联手。
<br/>　　“碧波叠浪掌。”
<br/>　　八人一同释放武技，威势冲天。
<br/>　　“不好，少家主小心。”萧家执事大惊。
<br/>　　萧逸怡然不惧，冷笑道，“来得好，崩山斩。”
<br/>　　火剑之上，一股崩山裂地般的威势出现，一剑将这八道威势冲天的波涛劈碎。而后，火剑余威不减，顷刻间将这八人斩于剑下。
<br/>　　八个后天武者，在萧逸面前瞬间被秒杀。
<br/>　　萧逸如今已是后天八重修为，配合升龙的增幅，已是先天实力，加上玄阶中级武技崩山斩，杀他们的确如杀鸡般轻松。
<br/>　　“少家主威武。”族人们被激起了强烈的战意。
<br/>　　半个时辰后，战斗形势完全扭转，反倒是萧家这边占了上风。
<br/>　　偌大个萧家，遍地尸体，有的是萧家武者的，有的是敌人的。
<br/>　　刺鼻的血腥味弥漫着整个空气。
<br/>　　“犯我萧家者，一个不留，杀光他们。”萧逸大喝一声，声音传遍了整个萧家。
<br/>　　“谨遵少家主之命，杀。”萧家族人们视死如归，强烈地反扑起来。
<br/>　　一个时辰后，萧逸气喘吁吁地停下了不断杀敌。他几乎全场游走了一遍，慕容家执事，全死于他手上。
<br/>　　鲜血，染红了他的衣衫。但他仍旧如一把锋芒利剑一般，锐气无可挡，杀意滔天。
<br/>　　在慕容家族人眼中，萧逸俨然成了一个杀神，一个让他们丧命的恶魔，一个他们无法抵挡的噩梦。
<br/>　　如今，场上的后天武者，只剩下萧家的执事。慕容家那边，最强的只剩下一个凡境九重的武者。
<br/>　　虽然慕容家的武者更多，但他们的败亡只是时间上的问题了。没有后天武者，萧家的执事几乎将他们一个接一个地秒杀着。
<br/>　　萧逸来到一个执事身旁，道，“这里就交给你们了，尽量减少族人的伤亡，快速杀敌，我去帮长老们。”
<br/>　　“是。”执事恭敬地应答一声，他虽然贵为执事，但此刻看向萧逸的目光，却是无比地信服与尊敬。
<br/>　　萧逸脚步一动，往萧家府邸深处而去，“萧重叔叔，千万不要有事阿。”
<br/>(本章完)
