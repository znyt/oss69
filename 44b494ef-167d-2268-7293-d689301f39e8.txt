　　渡过天人大劫，羽化登仙，是所有半步天人境强者梦寐以求的希望。
<br/>　　哪怕是在沧海星域之中，真正能渡过天人大劫，羽化登仙的人物，千年来也就寥寥几位。
<br/>　　许多半步天人境的强者，终其一生，因为积累不够，都没有去冲击过天人大劫。
<br/>　　可若是有所机会的话，哪怕是这沉沦海王，文供奉，也有着想要去冲击天人大劫的想法。
<br/>　　“九域虽然没落，不值一提，不过如今三大圣域，也已经是盯上了九域，我们要前去，也要做好万全的准备才行。”
<br/>　　文供奉比起沉沦海王，要冷静一些，想了想开口道。
<br/>　　“这件事，等祭海帝寿宴之后，再去好好计划一番。”
<br/>　　“恩，海十四得到大机缘的事，现在其他商会应该还不知道，我们现在要趁着这个机会，好好拉拢他……”
<br/>　　沉沦海王跟文供奉交谈了一阵，也纷纷离开了。
<br/>　　“看来沧海星域，也都得知了关于九域的消息了，这消息传递的倒是够快的。”
<br/>　　楚风眠在镇海海城之中闲逛着，脑海之中却是想这刚刚文供奉跟沉沦海王交谈的话。
<br/>　　月海商会之中，布下了层层的禁制阵法，就是为了不让人窥探，闯入其中。
<br/>　　不过楚风眠的灵识强横无边，这禁制阵法，根本拦不住楚风眠的灵识。
<br/>　　刚刚二人的交谈，也都被楚风眠听在了耳中。
<br/>　　这二人的交谈之中，也提到了九域，九域出世的消息，已经是传播的越来越广了。
<br/>　　沧海星域，可是距离九域足足数十个星域的距离，楚风眠到来沧海星域都花费了不少的时间。
<br/>　　现在这九域出世的消息，已经是从沧海星域之中流传起来了。
<br/>　　未来涌入九域的强者，定然是会越来越多，曾经圣域，这四个字，对任何的武者妖神，都有着极大的吸引力。
<br/>　　倒时候九域，只会比起现在还要混乱的多。
<br/>　　“先去海剑宗，将那剑道图腾拿到手再说。”
<br/>　　楚风眠身形加快，向着镇南海城的中心走去。
<br/>　　镇南海城，有着通往海剑宗的门户，不过在这一路上，楚风眠还在翻阅这海十四的记忆，在看着关于古海的事。
<br/>　　沉沦海王，文供奉，都是将楚风眠伪装成的海十四，实力增强，算在了古海的头上。
<br/>　　查了一番海十四的记忆，楚风眠也清楚了，这古海，其实是相当于归墟天一样的存在，埋葬这无数古老时代海族的宗门遗迹。
<br/>　　同样埋葬的，还有无数宝藏。
<br/>　　不过比起归墟天，古海还要更加危险，哪怕是实力再强的人物，踏入其中，都是九死一生。
<br/>　　海十四的记忆之中，都对古海无比的恐惧，他还从未踏入过古海。
<br/>　　对于古海的记忆，也都是在他人的描述之中听闻的。
<br/>　　古海危险重重，可同样是有着无数的宝藏，吸引着沧海星域的武者妖神，纷纷前去探索，一旦是可以从古海之中活着走出来的武者妖神，都会得到巨大的好处，一步登天都不为过。
<br/>　　“古海遗迹，正好，就用这个作为理由，反正古海的神秘，没有几个人知道。”
<br/>　　楚风眠化为了海十四，比起真正的海十四要强大的多，本来还想要找找理由的，现在正好可以用这古海遗迹，来作为理由。
<br/>　　而且海剑宗也是以力为尊，实力越强，在其中办事，也就会越发的容易一些。
<br/>　　楚风眠脑海思考的时候，他已经是来到了镇南海城的中心。
<br/>　　一道虚幻的门户，出现在了楚风眠的面前，这门户，半虚半实，让人都一时间无法看出其中的玄妙。
<br/>　　这就是海剑宗的门户，海剑宗弟子，都可以通过这门户，回到海剑宗之中。
<br/>　　一眼望去，在这门户前，已经是聚集了不少的海剑宗弟子。
<br/>　　海剑宗弟子，足有百万之多，每日往来的弟子都不在少数。
<br/>　　一些海剑宗弟子，也在维持着秩序，指引那些海剑宗弟子，进入门户之中。
<br/>　　海剑宗的宗门所在，其实并不算是在沧海星域之中，而是独立于外的一处小世界之中。
<br/>　　穿越这镇南海城的门户，需要横渡一段空间，才可以回到海剑宗之中。
<br/>　　这其中的空间乱流，凶猛无比，也算是一道天然的屏障，哪怕是古帝，都不敢冒险涉足。
<br/>　　这些海剑宗弟子，排队等待的，正是要集合在一起，一起御使方舟，好度过空间乱流，回到海剑宗之中。
<br/>　　楚风眠到来，看了一眼这些排队的海剑宗弟子，便是身形一闪，直接是从人群上面飞了过去。
<br/>　　那些维持秩序的海剑宗弟子，看到楚风眠的遁光飞来，没有人阻止，而都是纷纷恭敬的行李了起来。
<br/>　　“这是谁？不需要排队，这不合规矩啊。”
<br/>　　一些海剑宗新人弟子，看到这一幕，不由的出声道。
<br/>　　“住嘴，这是十四公子，你的话要是让他听见，小心你死都不知道怎么死的。”
<br/>　　在他旁边的另一位海剑宗弟子，急忙的说道。
<br/>　　“十四公子，这就是我们海剑宗的核心弟子？”
<br/>　　那一名新人弟子，看着楚风眠一个人踏入到了这虚幻门户之中，吃惊的说道。
<br/>　　“他难道是要自己一个人横渡这空间乱流？”
<br/>　　“哼，海剑宗的诸位公子，一个个都是神仙一般的人物，无所不能，这小小的空间乱流，我们需要借助方舟，竭力抵挡，对他们来说，却是挠痒痒一样。”
<br/>　　那一名海剑宗弟子，冷哼一声说道。
<br/>　　“据说三公子，曾经一个人，可是独战三名半步天人境强者联手，都是斩杀了其中两人，让赫赫有名的琳琅三魔，就此落荒而逃，诸位公子的神通，可不是你可以了解的。”
<br/>　　“独战三名半步天人境的魔头，斩杀其中两人？”
<br/>　　那海剑宗新人弟子都是咽了一口口水，有些不可置信。
<br/>　　“努力修行吧，我们现在只是外门弟子，只要是晋升到内门，未来身份都是无比尊贵，个个宗门都会礼待。”
<br/>　　看着楚风眠离去的身影，这名海剑宗弟子憧憬的开口道。
