　　闻听苏莫之言，嘈杂的武斗场，稍稍安静了下来。
<br/>　　不少人环目四顾，却是根本无人上场。
<br/>　　众人都见识了苏莫的实力，别说同阶之人，就算是和褚彦一样的一品高级修为之人，也没有敢上场。
<br/>　　“要不……我去试试？”
<br/>　　沁儿面带沉吟之色，向骆妤和姬月松问道。
<br/>　　“沁儿，你就算了吧！”
<br/>　　姬月松摇了摇头，道：“没有二品的修为，你是无法击败此人的！”
<br/>　　姬月松心中清楚，沁儿的战力，也就和褚彦差不多，就算会稍强一些，但也强不到哪里去。
<br/>　　此种实力，上去不过是白白送源石而已。
<br/>　　“无人敢战吗？”
<br/>　　战台上，苏莫环顾四周，再次朗声问道。
<br/>　　这些人，难道都是庸才吗？就没有一个妖孽人物？
<br/>　　“呵呵，我们一战如何？”
<br/>　　就在此刻，看台上传出一道轻笑之声。
<br/>　　霎时之间，所有人都循声望了过去，苏莫同样看了过去。
<br/>　　只见在看台最上方的座位上，端坐着一位极为普通的灰衣青年，其貌不扬，毫无亮点。
<br/>　　不过，就在众人的目光，落在灰衣青年身上之时，此人面貌与体型瞬间大变。
<br/>　　眨眼之间，普通的灰衣青年，变成了一名身穿红衣的男子。
<br/>　　这位男子五官精致，肤色如羊脂白玉，可谓俊美无比。
<br/>　　男子的头发很长，垂落到腿下，十指指甲极长，如同一柄柄秀气的弯刀。
<br/>　　这是一位，比女人还要美丽的男子。
<br/>　　“是兽王宫貂族的红烨！”
<br/>　　“他居然在这里！”
<br/>　　“哈哈，有意思了！”
<br/>　　看到俊美的红衣男子，不少人惊呼出声，因为这名红衣男子，是一位妖孽天才。
<br/>　　兽王宫，有很多妖族种群，貂族不算强大，但是，红烨却是貂族这数万年来，天赋最强的一人。
<br/>　　红烨在轮回神朝名气不是很大，这主要是修为还低，但是在兽王宫，却是声名响亮。
<br/>　　“原来是红烨！”
<br/>　　姬月松面带笑意，他对于此人，倒是有所耳闻，据说在兽王宫同阶无敌。
<br/>　　“真美！”沁儿不屑的撇了撇嘴，一个男人居然长得这么美。
<br/>　　不过，她的眼眸之中，明显有这一丝小嫉妒。
<br/>　　这个男人，长得比她都毫不逊色。
<br/>　　“这个红烨战力很恐怖，说不定真能战胜那人！”骆妤沉声道，她对于各大王宫的妖孽，都有些了解。
<br/>　　这个红烨的战力，绝对是同阶之中，最顶级的存在。
<br/>　　“上台吧！”苏莫目视着红烨，沉声说道，他不惧任何人，也不管对方是什么级别的天才？
<br/>　　就算对方再天才，也不可能堪比古神吧！
<br/>　　神域之中，苏莫目前还不了解古神的消息，不知道那些远古乃至太古的古神都在哪里。
<br/>　　看台之上，一身红衣的红烨，微微一笑，身形一闪，便轻飘飘的落在了战台之上。
<br/>　　“你很强，但是，我不和无名之人交手！”
<br/>　　红烨凝视着苏莫，继续道：“你是妖孽，强大的妖孽，既然如此，那就报上你的名号吧！”
<br/>　　听闻红烨之言，所有人都望着苏莫，想要看他会不会报上名号。
<br/>　　“我名——冷无敌！”苏莫淡淡的说道，既然别人三番两次的让他报名号，他便随便取一个便是。
<br/>　　冷，这是因为，他如今的面容，是冷邪的面容。
<br/>　　无敌，便代表着他同阶无敌的自信。
<br/>　　别说同阶，就算是高他一两个小境界，也没有什么人能击败他。
<br/>　　“冷无敌？”红烨闻言，顿时眼眸微眯了下来。
<br/>　　这个名字，未免太嚣张了吧？
<br/>　　他不傻，他知道这个名字很可能是假的，但是，就算是假的，取这种名讳，也极度的自大。
<br/>　　“冷无敌？”
<br/>　　“没听过啊！”
<br/>　　“假的吧？谁会取这么嚣张的名讳？”
<br/>　　“真假不清楚，但是名字确实嚣张！”
<br/>　　武斗场之中，众人听闻苏莫之言，顿时议论纷纷。
<br/>　　实在是冷无敌这个名字，太过嚣张霸道了。
<br/>　　“好自大的人！”姬月松嗤笑一声，敢以无敌之名自称的人，可是够嚣张的。
<br/>　　在轮回神朝，有天王宫和地王宫那两位禁忌一般的存在，居然还有人敢自称无敌？
<br/>　　“这一定是假名字！”沁儿秀美一撇，笃定的说道。
<br/>　　“此人的确有这个资本嚣张！”骆妤面色凝重，她心中亦是无比的好奇，这个自称冷无敌之人，到底是何方神圣？
<br/>　　“冷无敌，你要赌多少？十万下品源石？”红烨沉声问道。
<br/>　　他没有纠结苏莫是否报的真名，因为既然不愿意暴露身份，他再追问也是无用。
<br/>　　“十万吧！”苏莫点了点头，他只有十万源石，想赌多点都不行。
<br/>　　红烨点了点头，随即取出一个储物戒，抛向管理台。
<br/>　　苏莫同样取出了他赢取的那枚褚彦的储物戒，其内也是十万下品源石。
<br/>　　交取了源石，苏莫和红烨两人，互相凝视，沉默无语。
<br/>　　红烨的面色很凝重，他知道苏莫强大，心中并没有百分百的把握。
<br/>　　毕竟，他的实力虽然比褚彦强不少，但是，他也能看出，苏莫有所保留。
<br/>　　苏莫手握太云神剑，面色倒是很淡然，他有十足的自信。
<br/>　　而且，这一战，他不会再向和褚彦交手那般浪费时间。
<br/>　　而是，要以雷霆之势解决对手。
<br/>　　唰！
<br/>　　红烨动了，整个人如同一道红色的幻影，瞬间出现在了苏莫的右侧，一抓狠狠的抓向苏莫的脑门。
<br/>　　这一击，快到极致，快到巅峰，红烨的身形速度和攻击速度，都要远远超过了褚彦。
<br/>　　一爪抓出，五道犀利的爪芒，瞬间形成了一道符印，直接就临近了苏莫的脑袋。
<br/>　　这一击，威势滔天，如瀚海咆哮，更是蕴含一种诡异大势。
<br/>　　苏莫面临攻击，最能感受的到这种大势，他不惊心中一惊。
<br/>　　快！
<br/>　　这是一种规则之力，快之规则，让得红烨的攻击速度，快到了极致，快到让人反应不及。
<br/>　　面对这一击，苏莫已经没有了出手抵挡的时间，因为对方太快了。
<br/>　　他被打了个措手不及，没想到居然还有如此玄妙的规则之力。
<br/>　　当然，苏莫可不会坐以待毙，电光火石之间，他心念一动，立刻爆发出了自己巅峰的防御力。
