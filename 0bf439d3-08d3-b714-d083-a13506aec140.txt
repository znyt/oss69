
<br/>　　<br/>　　    男人为女人，用得最多的一个词叫冲冠一怒为红颜，懵懂的王惊蛰也不知道小草姑娘算不算是他的红颜，毕竟对方对于他的态度向来都是很冷淡，还带着那么一点小欺负，但这些都不重要，重要的是王惊蛰看见茅小草被围攻在中间有点急眼了。<br/>　　<br/>　　    王惊蛰来的时候，秋成子已经被逼的捉襟见肘了，被群起而攻之的后果就是他双拳难敌四手，一把桃木剑都被打飞了，只能靠双拳牢牢的护着后面的茅小草。<br/>　　<br/>　　    茅小草却没怎么动手，她给人的表现就是我乃一介女流之辈不适合打打杀杀的，所以也就没人知道茅小草要是动手的话，会是什么效果，而她似乎也没机会出手了，因为王惊蛰来了。<br/>　　<br/>　　    茅小草和秋成子被逼的节节败退的时候，没料到后面来了人，所以茅小草踉跄着往后退的时候，身子忽然就撞到了人，然后一股挺熟悉的味道传到了她的鼻孔里，撞到的人软软的胸膛热乎乎的，还伴随着急促的呼吸声，那是王惊蛰跑动的时候因为着急而气喘吁吁了，那股熟悉的味道和上一次的间隔并不是很久，才不过几天的时间而已，很容易就让茅小草回忆起了对方是谁。<br/>　　<br/>　　    然后，小草姑娘就理所当然的问了一句：“你怎么来了？”<br/>　　<br/>　　    王惊蛰心直口快的说道：“赶上了，我也没想到会在这里碰见你”<br/>　　<br/>　　    茅小草翻了翻白眼，说道：“碰上了还能怎么的，看见这么多人欺负我，你还敢帮什么忙嘛？”<br/>　　<br/>　　    王惊蛰挺了挺胸膛，刚想说从昨天开始我已经差不多要把全天下的道门都给得罪了，在这得罪几个也不是算回事，我现在就是块唐僧肉谁都想咬我一口，但话到嘴边他又给咽了回去，王惊蛰觉得要是这么说的话不够煽情，这种英雄救美的时刻，不得需要一番话来渲染一下气氛才对的么。<br/>　　<br/>　　    “为了你我得罪全世界都不怕······如果，再给我一次选择的机会，我大抵还是会这么做的，我愿意为了你站在所有人的对立面”王惊蛰低着脑袋，脸色通红的说道。<br/>　　<br/>　　    茅小草膛目结舌的瞪着眼睛，憋了半天后才说道：“你是猪么？”<br/>　　<br/>　　    王惊蛰的脸更红了，他觉得自己刚才那句话说的太没头没尾了，这绝对是钢铁直男癌晚期患者的表现。<br/>　　<br/>　　    王惊蛰是不是猪这件事有待商榷，但秋成子这时候人脑袋差点被打成了猪脑袋，身上的道袍坏了好几处，被木簪扎在脑后的头发也散开了。<br/>　　<br/>　　    “你们两个，能，能不能帮个忙，别光我自己打啊，凭什么啊”秋成子委屈的喊道。<br/>　　<br/>　　    王惊蛰拉了下茅小草将她拽到了身后，小草姑娘感觉手腕被人握住之后有点不太习惯，就想要挣扎开，但到了王惊蛰的后面就看见他挺着胸膛宽阔的后背忽然想起了一个词叫做安全感，别管是多么强势多么聪明的女子，在安全感这个词上都是没有抵抗力，十分敏感的。<br/>　　<br/>　　    王惊蛰松开右手，伸进斜挎着的包中，再拿出来的时候，手上多了一把黄色的符纸，他平淡的眼神扫了面前的一圈人。<br/>　　<br/>　　    道门中人交手无外乎就是两方面，一是拳脚比谁的功夫深，就像武当山上的道士，一身太极炉火纯青。<br/>　　<br/>　　    二就是斗法，拼的就是谁掌握的道术精纯。<br/>　　<br/>　　    对于妖魔鬼怪，孤魂野鬼和与人为敌，其实交手时都差不多。<br/>　　<br/>　　    上一次和南昆仑的崔玄策比拼风水法阵，已经让王惊蛰对自己的身手起了不小的信心，被称作这一代那颗最闪亮星星的崔玄策他尚且都能轻易应付得来，此时再面对这些人，他心里也有底了，就想试试看赊刀一脉的道术在交手方面能有几斤几两。<br/>　　<br/>　　    奇门，遁甲中的奇门之术，最善迎敌！<br/>　　<br/>　　    看见王惊蛰摊开手掌露出一叠黄纸，围着的人群神色都稍微有些凝重起来，先前秋成子拿着桃木剑无非就是拼上了拳脚而已，但看王惊蛰此时所做，明显用的就是道法了，众人都知道敢用道法来迎敌的，那都是对自己一身道术有着毋庸置疑的信心的。<br/>　　<br/>　　    “唰”王惊蛰突然用两根手指夹起一张符纸，屈指一弹符纸飘然飞去，随即他手指快速的朝着那张符纸接连点了几下。<br/>　　<br/>　　    “幡悬宝号，普利无边，诸神卫护天罪消愆······急急如律令，燃！”<br/>　　<br/>　　    王惊蛰口中快速的念了一串咒，那张飞去的符纸上顿时闪现出一道符箓来。<br/>　　<br/>　　    对面几人当即就有人脸色大变，脱口而出道：“隔空画咒！”<br/>　　<br/>　　    以往在符纸上写道家符箓基本都是用毛笔蘸着朱砂或者是狗血等驱邪避凶之物来书写，但据说也有人把符咒练到极致的时候，可以凭空画符，这需要的就是一身道法极其精纯了才可以。<br/>　　<br/>　　    谁也没有料到，王惊蛰上来就撒出了一记隔空画咒。<br/>　　<br/>　　    那道飞出去的符纸，即将碰触到人群的时候，突然之间“噗”的一声就冒出了一道火光，剧烈的燃烧起来，好几道火星纷纷扬扬的朝着四周溅了出去。<br/>　　<br/>　　    离得最近的几个人顿时四处躲闪，仓惶躲避着。<br/>　　<br/>　　    “经完幡落，云旆回天，各遵法旨不得稽延·····雷”<br/>　　<br/>　　    第二张符纸被王惊蛰再次弹出去后，纸上闪现出一道复杂的符箓。<br/>　　<br/>　　    惊雷符，此符一出可引雷光乍现。<br/>　　<br/>　　    “轰”<br/>　　<br/>　　    一道白光突然从符纸中冒出，雷声先行响彻在半空中，紧接着一道拇指粗细的雷蛇就蜿蜒而出了。
