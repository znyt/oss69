　　门外的守卫把门打开，布克尔特走进了屋子，看着坐在桌子边的两个女人。
<br/>　　原本正在“交谈”的佟千雪和娜塔莉娅也停下了谈话，转头看向站在她们面前布克尔特，三个人就这么对视在一起。
<br/>　　敏锐的直觉让布克尔特觉得自己眼前这两个女人似乎是在隐瞒着自己什么，他盯着两个女人，问道：“你们在聊什么？”
<br/>　　“布克尔特什么时候对女人间的话题也有兴趣了？”佟千雪反唇相讥道：“难道女人间私密的事情也要向你报告吗？是不是还要我告诉你，我每个月月事的时间，还有使用什么牌子的卫生巾？哦，我忘记了，这里是南美洲没有我平时用的那个牌子卫生巾，如果我需要你是不是要帮我空运过来？”
<br/>　　如果是别的男人，听到这些跟女人私密有关的话题，肯定会面红耳赤很不自在，但是布克尔特完全不会，别说是女人私密的话题，就是一个女人脱光了站在他面前，他也会明目张胆的看着，不会有丝毫的心虚。
<br/>　　“我觉得你们聊得似乎不是这个话题。”
<br/>　　布克尔特盯着佟千雪的眼睛，像是要从她的眼睛中读出她隐藏的秘密来。
<br/>　　佟千雪心跳的很快，她喉咙有些发干，不过佟千雪明白自己这个时候不能避开眼神，不能表现出丝毫的心虚，如果身上藏着钥匙的秘密曝光，自己可能不会死，但是她身边的娜塔莉娅一定会死。
<br/>　　佟千雪紧张，娜塔莉娅也同样紧张，她生怕布克尔特看出些什么来，那样的话，自己谋划了这么久全都成了无用功，她肯定会被布克尔特杀死，指望他会对女人怜香惜玉，还不如指望母猪会上树。
<br/>　　清楚这样下去早晚会被布克尔特看出点什么，佟千雪从凳子上站了起来，对着布克尔特张开双手，说道：“既然你有所怀疑，那你就动手来搜吧。”
<br/>　　佟千雪在赌，赌布克尔特的原则，她身上一共就一件单衣、一条裤子和一条内裤，如果布克尔特真的动手搜，肯定会被他找到钥匙，到时候娜塔莉娅必死无疑。
<br/>　　还好，佟千雪赌赢了。
<br/>　　布克尔特并没有动手在佟千雪身上搜，当然，也不排除布克尔特对自己的自信，他自信在这里，就算佟千雪有什么小动作也不会起到什么作用。
<br/>　　“不要想在我的面前耍花招，那是没有用的。”
<br/>　　布克尔特转身朝着屋外走去，走到门口的时候，停了下来，背对着屋子里的佟千雪，说道：“告诉你也没有关系，你所期待的营救你的特种部队很有可能已经到了，并且正策划着如何营救你；可是我可以很明确的告诉你，不管他们如何策划都是没有用的，特种部队那套东西我也懂，他们来这里营救你，只能是自寻死路，我会等着他们来，然后当着你的面宰掉他们！不要在这里停留很长时间。”
<br/>　　对着娜塔莉娅叮嘱了一句后，布克尔特走出了屋子，门外的守卫重新把门锁了起来。
<br/>　　等到布克尔特离开后，佟千雪一屁股坐回到凳子，饱满的胸口起伏着，显然是对刚才自己的豪赌而心有余悸。
<br/>　　“你疯了！”
<br/>　　娜塔莉娅显然是对佟千雪刚才的举动感到很不满，她完全是拿自己的命开玩笑，压低声音说道：“刚才如果他真的动手搜你的身，我就死定了！”
<br/>　　“可是如果不这样做，一直被他这么盯着，我不知道自己能隐瞒多久。”佟千雪说道：“我的丈夫也是特种部队的，他们接受过专业的审讯和反审讯训练，他们甚至不需要做太多的审问，只是盯着你的眼睛看，就能知道你有没有隐瞒什么东西！”
<br/>　　“还好，你赌赢了！”娜塔莉娅抚着自己的胸口，说道：“我刚才真怕布克尔特突然拔出手枪来，一枪把我给杀了。”
<br/>　　“也许是赌赢了，也许是输了。”
<br/>　　佟千雪回想着布克尔特离开屋子前对自己所说的那些话，说道：“有可能他真的是没有发现什么，有可能是他发现了，但是出于他对自己的自信，觉得我们这些小动作对他根本没有任何作用，无法改变什么。”
<br/>　　本来听到娜塔莉娅提及龙隐部队可能已经抵达这个国家，准备实施营救的时候，佟千雪心中真的很激动；可是看到刚才布克尔特的反应时，佟千雪却又有些希望慕容飞他们不要来救自己。布克尔特抓自己是为了什么？不就是为了要把慕容飞他们这些人引来吗？既然这才是布克尔特本身的目的，那么他又怎么会没有准备？
<br/>　　等待慕容飞他们的或许真的是一个死亡陷阱！
<br/>　　……
<br/>　　巴坦伦秘密别墅外山路边的丛林中，迷彩服外披着伪装网，脸上涂着伪装迷彩的莱昂、布谷、天蛇三个人潜伏着。
<br/>　　玄龟正在山路上埋设着炸药，天蛇正在用无人机观察着四周的情况，布谷则是架起了XM2010增强型狙击步枪盯着玄龟的方向。
<br/>　　玄龟的动作很快，花了不到五分钟的时间就在这条唯一可以让车辆通过的山路上埋设好了地雷，因为地面上下过雨泥泞的关系，本身地面上的脚印和轮胎印就很杂乱，所以玄龟没有刻意再去清理地上的痕迹，抬头往别墅大门的方向看了一眼，转身就跑回了路边的丛林中。
<br/>　　“但愿不会有车子通过，把地雷引爆了。”
<br/>　　天蛇把无人机收了回来。
<br/>　　“就算有汽车从上面压过去也没关系。”玄龟从口袋中拿出一个遥控器，说道：“地雷是经过改装的，需要我启动，引信才会被激活。引信没有激活之前，别说是汽车往上面压，就是开一辆坦克来都不会爆炸。”
<br/>　　“可以啊，倒是没发现你还有这么一手。”
<br/>　　布谷对着玄龟竖起一个大拇指。
<br/>　　“必须的！”
<br/>　　玄龟不客气的揉了揉鼻子，一副拽的恨不得想要揍他两拳的表情。
<br/>　　“现在就看狂龙他们那边的进展了。”
<br/>　　莱昂眼睛死死地盯着巴坦伦秘密别墅的方向。
