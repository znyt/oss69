　　陆岑从房间里离开的时候，两只脚都是软的。
<br/>　　做一行也不是一天两天了，接过的客人数也数不清，但是能让陆岑软着脚走出的房间，屈指可数。
<br/>　　监控室内，三儿让人把钱鑫房间里那段视频拷贝下来后，离开了监控室。
<br/>　　停车场，当三儿来到他那辆奥迪车旁边的时候，陆岑已经是在那里等着了，陆岑一只手扶着奥迪车，稳住自己的身体。
<br/>　　三儿按下遥控器，示意陆岑坐到后排座位上去；三儿拉开左侧后排座位的车门钻进奥迪车后，把手放在陆岑大腿上，说道：“刚才看你很爽啊？”
<br/>　　“唉哟，三爷~”
<br/>　　陆岑立刻腻倒在三儿身上，用手指在三儿胸口画着圈，说道：“那还不是三爷的东西好，那傻比就是个废物，你又不是没看到，我给他戴个套就射了，我就没见过这么快的。”
<br/>　　陆岑刚才叫喊的有些用力，现在说话的时候有些嘶哑，眉角还带着高潮后留下的媚态。
<br/>　　三儿放在陆岑大腿上的手开始不老实起来，向上游动；陆岑如何能不知道三儿的想法，顺势就贴了上去。
<br/>　　“要不要我也戴个套啊？”三儿在陆岑脸上摸了一把，说道。
<br/>　　“做我们这行的，哪有这么容易怀孕。”陆岑跨坐在三儿大腿上，动手去解他的腰带，说道：“我们这种人如果不带节育环，怀了孩子都不知道是谁的种。”
<br/>　　“那你还让钱鑫戴套？”
<br/>　　三儿喉间发出一声满足的闷哼声，把手伸进陆岑的衣服中。
<br/>　　“那傻比怎么能跟三爷比，让那傻比射在我身体里，我他妈想想都觉得恶心。”
<br/>　　陆岑咬着嘴唇发出充满魅惑之意的哼声，身体耸动起来。黑暗中奥迪车不断的晃动着，车窗上也是结了一层雾气……
<br/>　　约莫半个小时后，奥迪车中的两个人已经重新穿好了衣服，陆岑全身发软的靠在三儿身上。
<br/>　　“是我厉害，还是钱鑫厉害？”三儿一脸满足的对陆岑问道。
<br/>　　陆岑脸上带着媚态，伸手攀上三儿的脖子，说道：“钱鑫那傻比怎么能跟三爷比呢？对了，你给我的到底是什么东西，居然能让钱鑫变这么厉害？”
<br/>　　“这是力哥从国外弄回来的新货，叫‘水晶珠’。这东西，对男人那方面的提升特别大。”三儿右手还在陆岑衣服里不老实的动着，说道：“我也只是听力哥说起过它的厉害，但是真没想到会这么厉害，一粒就让一个快枪手变得金枪不倒。”
<br/>　　“可有了这东西，您让我们以后还怎么做生意？”陆岑对三儿撒着娇说道：“这一粒一个小时，我们这些姐妹不是要被折腾惨了。”
<br/>　　“这你可以放心，‘水晶珠’现在我们一共就没多少，而且后面的价格会很高，一般人可消费不起。”
<br/>　　三儿从口袋里拿出烟盒，咬住一根烟抽出烟盒，陆岑很识趣从自己包里拿出打火机帮三儿点上，对三儿问道：“您说力哥是怎么回事，不卖粉怎么做起别的了？”
<br/>　　“谁说力哥做别的了？”三儿抽了口烟，说道：“这‘水晶珠’是力哥从境外弄回来的全新合成型毒品，一次成瘾，极难戒除。碰上了，就甩不掉了。你想想来这种地方玩的，谁不想玩个尽兴，要是都像钱鑫那样戴个套就射，玩得不尽兴不说，还要被看不起。男人嘛，脸面最重要。这时候如果把‘水晶珠’拿出来，就算我卖五千块一粒，他们也会心甘情愿的掏钱，而且尝试过一次后就会欲罢不能，会想着继续享受这种征服的快感，这东西来钱比别的可要快多了。”
<br/>　　“力哥还说了，‘水晶珠’的成瘾方式跟以往不同，成瘾后，会特别想要女人，可是没有‘水晶珠’的帮助他们就会发现自己不行了，心痒难搔却又没办法正常发泄，他们会对‘水晶珠’格外有需求，届时我坐地起价，每天限量供应，而且是以拍卖的方式进行销售，你觉得我能赚多少钱？”
<br/>　　“三爷，你真是太厉害太聪明了。”陆岑赶紧一记马屁拍了上去。
<br/>　　“那是。”三儿降下车窗，在陆岑胸前狠狠捏了一把，说道：“以前我是没机会，现在机会来了，我当然要好好把握！”
<br/>　　三儿的眼眸中闪烁着一种叫做“野心”的东西！但是三儿也知道，自己的野心不能在李霄力面前暴露，不仅是李霄力比他更狠，最重要的是李霄力把控着他的命脉，李霄力不给货，他三儿就算是本事再大也翻不起什么浪花来。
<br/>　　……
<br/>　　黑色奥迪车缓缓停在一个小区门口，在陆岑准备推门下车的时候，三儿叫住了她，说道：“你手里那两粒‘水晶珠’藏好了，回头我会让钱鑫再来找你。”
<br/>　　“我还要陪他啊？”陆岑一脸的不乐意。
<br/>　　“力哥要从他身上得到一些东西。”三儿对陆岑说道：“最多也就再陪两次，等到那小子利用完了，就会被当成‘皮箱’送到T国去，再也不会出现在你面前，放心吧。”
<br/>　　“那可说好了，最多只有两次。”陆岑说道。
<br/>　　“最多就只有两次。”三儿很肯定的说道：“你手里一共就只有两粒‘水晶珠’，没有了‘水晶珠’，那小子就是个废人，你就是脱光了站在他面前，他都硬不起来，跟个太监没区别。”
<br/>　　陆岑离开后，三儿拿出手机拨通了李霄力的电话，说道：“力哥，钱鑫那小子入套了。”
<br/>　　“很好。”李霄力语气冷冽的说道：“吊着他，如果他想要接着爽，就拿情报来换，拿不出我想要的东西就憋着！”
<br/>　　“明白了力哥，我知道该怎么做，我已经叮嘱过陆岑那娘们了。”三儿回答道。
<br/>　　“三儿，我提拔你，是因为你聪明。”李霄力在电话中，用警告的语气对三儿说道：“你聪明归聪明，但是别把你的聪明用在不该用的地方，尤其是用在我身上，我的话是什么意思，你应该明白。记住了，做好你应该做的事情！”
