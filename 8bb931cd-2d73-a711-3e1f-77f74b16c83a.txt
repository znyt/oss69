　　龙都城的南部区域，坐落着一个巨大的广场，占地方圆两万余丈，非常辽阔。
<br/>　　今日，广场上人满为患，因为，龙族的祖龙大典，便是在广场上举行。
<br/>　　广场中央搭建起了高达百丈的礼台，而广场的东、西、北三个方向，均是搭建起了高达数十丈的观礼台。
<br/>　　北部观礼台上，就坐的是一众龙族高层人物，有近五十位龙族长老，和三百多名龙族执事，吞虚龙帝则是端坐在最中央。
<br/>　　东部观礼台之上，端坐的是龙族的比较核心的一些族人，数量极多，不下数千人，如龙城、龙双双等人，尽在其中。
<br/>　　而西部观礼台之上，就坐的是各大势力前来观礼之人，如紫霄、天尘宇、地藏神王等人。
<br/>　　西部的观礼台，每一个席位，都相对独立，配上了案几。
<br/>　　各大势力之中，前来之人也不少，足有六七百人之多。
<br/>　　南方没有观礼台，地面上则是聚集了无数的普通龙族子弟，甚至还有很多龙族奴仆。
<br/>　　此刻，广场中央的礼台上，空无一人，龙腾还未现身。
<br/>　　整个广场之上，嘈杂的声音此起彼伏，有各大势力之人在低声交谈，也有龙族之人在不停的议论。
<br/>　　这时，吞虚龙帝见各大势力之人，已经悉数到齐，便长身而起，站了起来。
<br/>　　随着吞虚龙帝起身，全场陡然安静了下来，便的落针可闻。
<br/>　　“诸位，首先感谢大家，百忙之中抽空来我龙都城观礼，在此，龙某有礼了！”吞虚龙帝极为客气，双手抱拳，向西方观礼台各大势力之人拱手。
<br/>　　“龙帝客气了！”
<br/>　　“龙族出现祖龙，乃是大喜之事，我等自然不能不来！”
<br/>　　“龙帝的威严，更甚当年啊！”
<br/>　　不少各大势力的强者，纷纷起身，向龙帝回礼。
<br/>　　众多势力之中，前来此地的神王级别强者，足有一百多位，但是，高级神王只有寥寥几位。
<br/>　　而与龙帝同地位的人物，基本上没有。
<br/>　　比如，轮回神朝的掌权者混沌元王、亘古魔国的掌权者亘古魔帝等人，均是没有来此。
<br/>　　不过，这些来到此地的神王，均是代表着自身所在的势力，龙帝也不能怠慢了。
<br/>　　“呵呵，诸位客气了！”
<br/>　　吞虚龙帝淡然一笑，随即吩咐道：“来人，给诸位贵客奉上神酒和神果。”
<br/>　　龙帝话音一落，便有不少龙族之人，从上空飞来，各种神果和神酒，如同坠落的星光，落在了各大势力之人身边的案几之上。
<br/>　　“诸位，这是我们天龙神国特有的龙血神果，以及我龙族特制的神酒，大家细细品尝！”吞虚龙帝朗声说道。
<br/>　　“多谢龙帝！”
<br/>　　“多谢龙帝！”
<br/>　　“多谢吞虚龙帝款待！”
<br/>　　众多强者纷纷道谢，紫霄倒了一杯神酒，轻抿一口，朗声道：“龙帝前辈，时辰已经不早了，我们便饮美酒，便观祖龙大典如何？”
<br/>　　“是啊，时辰已经不早了，还是尽快开始吧！”天尘宇附和道。
<br/>　　紫霄和天尘宇两人，已经有些迫不及待了，他们仔细观察过全场，并没有苏莫的身影。
<br/>　　心中也在思索，苏莫肯定不会现身，但是，他们有办法让苏莫不得不现身。
<br/>　　“如此正和本帝之意！”吞虚龙帝微微点头，随即朗声吩咐道：“祖龙大典，现在开始。”
<br/>　　随着龙帝的话音一落，远处的祖龙府之中，有数道身影冲天而起。
<br/>　　这数道身影，为首之人，正是龙腾。
<br/>　　龙腾的身后，跟着十数人，有龙族大长老、二长老、还有几位一些执事。
<br/>　　一众人虚空踏步，很快便来到了广场礼台的上方。
<br/>　　今日的龙腾，格外的耀眼，头戴金冠，穿着一身五彩之色的高领长袍，衣袖极为宽大，长袍之上绣着一条条栩栩如生的小龙。
<br/>　　这些小龙，仿佛活物一般，给人一种拥有灵性之感。
<br/>　　如此的穿着，配上龙腾高大的身躯、刚毅的面容，气场无与伦比。
<br/>　　这一刻，所有的人，目光都紧紧的盯在龙腾的身上，想要看看这所谓的新的祖龙，有何不凡之处？
<br/>　　对于龙腾，众人都很陌生，以前从未见过。
<br/>　　姜枫然、战乱和闻人天都，静静的望着龙腾，心中颇为感慨。
<br/>　　没想到，现如今，龙腾在他们几人之中，居然又走到了前面。
<br/>　　他们三人，目前的修为，都只是二品，而龙腾修为已达四品。
<br/>　　当然，他们也不会嫉妒龙腾，因为他们个个都是万古天骄，甚至是比神域一般的万古天骄的天赋要好的多，千年之内达到神王之境，成为无上神王都是指日可待。
<br/>　　各大势力之中，一些最顶尖的妖孽，个个眸光犀利，甚至眼眸之中战意澎湃。
<br/>　　祖龙？
<br/>　　这可是不可多得的对手。
<br/>　　唰唰！！
<br/>　　龙腾和大长老、二长老等人，纷纷落在了中央的礼台上。
<br/>　　“鄙人龙腾，欢迎诸位朋友前来观礼！”龙腾大袖挥动，双手作揖，向众人介绍自己。
<br/>　　“龙腾小友果然气度不凡！”
<br/>　　“来到龙都城数日，龙腾小友的威名如雷贯耳，今日一见，果然名副其实！”
<br/>　　“恭喜龙腾小友！”
<br/>　　很多大势力的神王级别强者，纷纷微笑着开口，仿佛是同辈一般。
<br/>　　虽然龙腾目前修为不高，但是，作为新的祖龙，将来成为至强者，都不会有太大的难度。
<br/>　　“诸位前辈过奖了！”龙腾微微一笑，道。
<br/>　　这时，大长老上前一步，朗声道：“组龙大典，正式开始！”
<br/>　　现场安静了下来，没有人再开口，只见大长老大袖一挥，观礼台前方便出现了一个高达百丈左右的雕像。
<br/>　　这雕像，乃是用玉石雕刻而成，是一位中年人的模样，此人浓眉大眼，面容略显粗犷，这正是洪荒祖龙的雕像。
<br/>　　此雕像一出，所有人都面带崇敬之色，因为，这是龙族的始祖，一位纵横太古无敌的强者之一。
<br/>　　“行三拜九叩大礼！”大长老朗声高喝。
<br/>　　随后，龙腾大袖一挥，便在雕像前跪了下来，行三拜九叩之礼。
<br/>　　祖龙大典，并不复杂，在龙腾行过三拜九扣大礼之后，大长老便施展了一种龙族的秘法，笼罩全场所有的龙族之人。
<br/>　　无数微不可查的荧光，从无数龙族之人身上升起，飘上半空，最后汇聚在一起，融入了龙腾的体内。
<br/>　　这是气运之光，代表着龙腾将引领龙族的气运，这虽然无法增强龙腾的实力，但是，却能让龙腾拥有龙族最强的气运。
<br/>　　整个过程，持续了半个时辰，才最终结束。
<br/>　　“本人龙昌，龙族大长老，在此郑重宣布，龙腾成为龙族第二任祖龙，尊号苍穹祖龙，日后引领龙族！”大长老朗声高喝，浑厚的声音，震动整个龙都城，浩荡诸天环宇。
<br/>　　即便在数百万里之外，都能真切的听到大长老的声音。
<br/>　　“拜见苍穹祖龙！”
<br/>　　“拜见苍穹祖龙！”
<br/>　　“拜见苍穹祖龙！”
<br/>　　大长老话音一落，无数的龙族之人，纷纷抱拳行礼，朗声高喝，声震九霄云外。
<br/>　　祖龙，不仅是崇高的身份，还是无上的地位。
<br/>　　吞虚龙帝面色淡漠，这一切仿佛都与他无关，他只是静静的看着。
<br/>　　“众位族人，免礼！”龙腾环视四方，朗声说道，此时此刻，他的心中亦是万分激荡。
<br/>　　甚至，他有一种错觉，仿佛自己成了洪荒祖龙，俯瞰百万族人。
<br/>　　“谢苍穹祖龙！”众人齐声高喝，洪亮的声音涌荡四野，滚滚而去。
<br/>　　其它各大势力之人，望着意气风发的龙腾，面色各异，这位新的祖龙，能否像当年的洪荒祖龙一样，铸就传奇？
<br/>　　真一切都是未知之数，毕竟，再妖孽的人物，没有成长起来之前，都有夭折的可能。
<br/>　　“呵呵，苍穹祖龙果然威震八荒！”
<br/>　　这时，紫霄面带微笑的开口了，继续道：“现在祖龙大典，也算是结束了，但在下心中还有好奇之心。”
<br/>　　“有何好奇之心？”龙腾目光凝视着紫霄，沉声问道，他知晓对方的身份，也知晓此人安奈不住了。
<br/>　　“听闻龙腾小友之所以能进化血统，是因为得到了祖龙塔第十层中洪荒祖龙的精血，而据我所知，祖龙精血乃是一个叫屠霄的天才闯进了祖龙塔第十层，才得以获得此精血？”紫霄微笑着说道。
<br/>　　各大势力之人闻言，个个眸中精芒闪烁，这个消息他们都知晓，并不是什么秘密。
<br/>　　相比于龙腾，他们对于这个屠霄，同样的好奇，此人到底又何不凡之处。
<br/>　　当然，也有很多来到龙都城较早之人，听到了一些屠霄的情况，再结合外界的传闻，心中都隐隐有了猜测。
<br/>　　“不错！”龙腾微微点了点头，这众所周知，他也不必隐瞒。
<br/>　　“呵呵，这个屠霄能闯进祖龙塔第十层，当真是天赋冠绝神域，不知是龙族那位天才？”紫霄轻笑一声，目光环视四方，朗声问道。
<br/>　　“是啊，不知哪位是屠霄？能否出来一见？”一位体型壮硕，面容粗犷的中年男子附和着说道。
<br/>　　此人，正是雷焱神王殿的雷火神王，当初为了追查苏莫，亲身前往终极神王殿。
<br/>　　“天某人也极为好奇，这屠霄有何不凡之处？”天尘宇同样开口了，声音低沉而浑厚。
