　　危机关头，涂小安不得不出现，否则这位很傻很天真的少女，估计也会死在鸡冠王的手中。
<br/>　　他昂首而立，将李清照挡在了身后，跟鸡冠王对峙了起来，此时，空气像煮沸了一样沸腾起来。
<br/>　　涂小安觉得面对鸡冠王，比面对吸血蛇还要有压力，这条二阶中等生物，不知道有什么强大的手段。
<br/>　　他本以为自己的等级暴涨，任何蛇类都要臣服在自己的脚下，如此看来，天下之大，无奇不有。
<br/>　　一高更比一山高，他的眼界还完全不够。
<br/>　　“还不快走，真想死在这里不成”
<br/>　　一道诡异的声音出现在李清照的脑海中，旋即她看到这条金银色的蛇转头望了她一眼。
<br/>　　仅仅一眼，李清照就确定，这声音是这条蛇发出的。
<br/>　　很诡谲，很惊世骇俗，但她知道这是真的。
<br/>　　她呆呆滞滞，像个泥塑木雕的人，忘却了身体的本能，坐在地上发傻。
<br/>　　“真是耽误事”
<br/>　　涂小安一双蛇眸像是烙印在李清照身染，然后浑然一眯，一股无形的意识力骤然投放在她的身上。
<br/>　　忽然，李清照腾空而起，轻飘飘的飞出了赵家宅院之外。
<br/>　　做完这一切，只见鸡冠王通灵摄厉的发出咆哮：“你若阻我，我连你也杀”
<br/>　　冰冷的声音，透着嗜血杀意。
<br/>　　这是正儿八白的发出人的语言，这种二阶生物，果然具备口吐人言的本领。
<br/>　　涂小安已经见怪不怪，拖油瓶已然送走，他现在可以毫无顾虑的跟对方一战。
<br/>　　在蛇的世界里，败者就要沦为对方的食物。
<br/>　　在则，如今对峙的两条蛇，都已经脱离了凡蛇的范围内，不说超凡脱俗，也已然成精成妖。
<br/>　　“谁死还不一定呢”
<br/>　　涂小安不甘示弱，身上散发出来的气息愈演愈烈，普通人要是在他身边，根本就连呼吸都做不到。
<br/>　　他今天来干嘛的，就是来拿蛇，拿住这条杀人蛇，然而吞噬，从而进化。
<br/>　　就在两蛇即将在展开生死决战的时候，赵家的人反应过来了，陆续的来到了灵堂前，大约有七八个人，男女老少都有。
<br/>　　其中更有一位被人推着轮椅的老者到来，此人便是赵宅村的村长，德高望重的赵老爷子。
<br/>　　赵老爷子先是被人推到了灵堂前，看到赵大跪在地上嘴巴张的大大的，脖子上一条蛇纹显而易见，顿时老泪纵横。
<br/>　　因为他最后的一个大儿子也死了，他生了四个儿子呀，现在全部死绝了。
<br/>　　“老爷，你看宅院”
<br/>　　推着轮椅的管家满脸惊恐的开口。
<br/>　　赵老爷子旋即看到院中对峙的两条与众不同的毒蛇，顿时浑身颤抖，他瞬间认出了其中的一条蛇。
<br/>　　那便是他赵家的家蛇啊，那条蛇的样子他永远都忘不了。
<br/>　　二前年，家蛇出现在赵家人的面前，从而离开，现在却回来杀人。
<br/>　　一天一人，五天五人，现在第六天，第六个死者也出现了。
<br/>　　如此想来，这条蛇早就有计划，二年前那次显身，更像是跟赵家人宣战。
<br/>　　看看院中的那条家蛇，看到跪在地上的赵大，赵老爷子的面色白的就如他一头斑白的头发，痛苦的高呼：“你要杀，就杀我这个老头子，为什么要杀我的儿子，他们都是无辜的”
<br/>　　“我才是最该死的那个人”
<br/>　　赵老爷子在轮椅上痛苦不堪，看着院中的鸡冠蛇充满了复杂情绪。
<br/>　　鸡冠蛇看到赵老爷子，身上的杀意顿时汹涌澎湃而出，发出如怨如泣的冰冷声音：“你自然该死，但我要你先看着自己的亲人一个个的死去，最后我在送你去死”
<br/>　　蛇说话了，须根之间，整个空间变得寂静无声，天空似乎要垮塌下来了，赵家人，不管男女老少，都如同雷轰电掣一般，呆住了。
<br/>　　但如今蛇说话并不算什么了，关键的是，这条蛇要让他们全部都死，这才是最恐怖如斯的地方。
<br/>　　同时众人心头浮现一个谜团，当年到底发生了什么事情，这条守护赵家的家蛇，为什么要杀光赵家的所有人。
<br/>　　老爷子为什么说该死的人是他。
<br/>　　此时鸡冠蛇的蛇眸横扫出现在现场的每一个人，老爷子唯一的女儿，赵家的老管家，他的孙子，孙女，每一个人接触到那双蛇眸的人，都浑身颤栗了起来，害怕，恐惧。
<br/>　　那条蛇的眼神太可怕了，让人如临地狱深渊。
<br/>　　死亡的恐惧感侵袭了每一个人的心，但此刻没有一人敢动弹一分。
<br/>　　“是我对不起她，你杀了我，你就收手吧”
<br/>　　赵老爷子从轮椅上挣脱出去，爬在地上。
<br/>　　“老爷，老爷”
<br/>　　“爷爷，爷爷”
<br/>　　一帮人围了过来。
<br/>　　“别管我，你们退下”
<br/>　　赵老爷子一声叱喝，朝着鸡冠蛇的方向爬起。
<br/>　　其实当年赵老爷子的身子很健朗，只是二年前，家蛇出手，他痛苦的高呼，‘家蛇已走，赵家要败了。
<br/>　　从此双腿就中风瘫痪了。
<br/>　　赵老爷子这是求死，他想让鸡冠王杀了他，从而结束这场杀戮。
<br/>　　鸡冠王泛着冷冰的蛇眸望着赵老爷子非常吃力的爬过来，它的毒牙像是在摩擦，发出滋滋的声音。
<br/>　　仇恨，所有人都能看出这条蛇对赵老爷子充满了仇恨。
<br/>　　如今的局面变的很透明，涂小安本来就要跟鸡冠蛇展开生死搏斗，因为众人的出现，变的很尴尬了起来。
<br/>　　他被人完全的忽略了。
<br/>　　拜托，他也是主角好不好。
<br/>　　但其实涂小安并不愿意在人前现身，因为他所做的事情，跟能力都是惊世骇俗的，但此时的局面已然失控。
<br/>　　同时他跟赵家人一样，都非常的好奇，好奇赵老爷子当年到底做了什么事情，能让守护的家蛇如此仇恨。
<br/>　　这条变异的鸡冠蛇，要将赵家灭门。
<br/>　　一条蛇，让一户家大业大开枝散叶的家族灭门，完全是一件很滑稽很无厘头的事情。
<br/>　　但...这条蛇做得到，它有这个能力。
<br/>　　五口棺材，一位跪地的死者赵大，都是赫赫的例子。
<br/>　　所有人呼吸都停止了，只能眼睁睁看着赵老爷子朝着鸡冠蛇一点点的爬，一点点的接近。
<br/>　　赵老爷子苍白的脸上，随着跟鸡冠蛇距离的拉近，反而变的轻松了起来，像是赎罪，像是解脱。
