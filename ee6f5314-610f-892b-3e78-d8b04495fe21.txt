　　“将军？”
<br/>　　麦克米兰通过无线电试图跟詹姆斯取得联系，但是耳麦中传来的却是Damien中校的声音。
<br/>　　“将军在十分钟前已经离开了‘鹰巢’，前往现场。我已经让人接通将军，你们很快就能跟将军进行对话，稍等。”Damien中校说道。
<br/>　　片刻之后，詹姆斯的声音从耳麦中传来过来，“Alpha1，Bravo1，我正在赶往现场。”
<br/>　　刚坐上一架湾流商务机的詹姆斯看了眼手表上的时间，说道：“我大概会在两个小时后跟你们汇合，在此之前，你们必须稳住西伦科特。”
<br/>　　罗昊听到耳麦中詹姆斯的话，心中同样对詹姆斯会亲自赶到现场有些疑惑，但是也没有提出反对的意见。
<br/>　　“两个小时后，联合行动司令部的詹姆斯?沃尔特将军会来这里亲自跟你交谈。”麦克米兰看着西伦科特说道：“我们并没有最终的决定权。”
<br/>　　“詹姆斯?沃尔特？”
<br/>　　听到麦克米兰的话，西伦科特先是愣了愣，随后脸上突然露出意味深长的笑容，用只有他一个人能听到的声音低声说道：“没想到这家伙亲自来了，果然是忍不住了么？”
<br/>　　“也好，我正好跟这个老朋友谈谈，我想他是不会拒绝我所提出的要求的。”
<br/>　　西伦科特点头答应了下来，他显得很笃定，似乎可以肯定詹姆斯一定会答应跟他的交易一样。
<br/>　　三十分钟后，湾流商务机降落在一处军用机场，詹姆斯走下商务机后，立刻上了一架早已在机场待命的直升机。
<br/>　　詹姆斯爬上直升机，在直升机向上爬升起飞的时候，对机舱中的机组成员问道：“我要的东西带了吗？”
<br/>　　机组成员将一个长方形的迷彩手提箱推到詹姆斯面前，詹姆斯接过箱子也没有回避，直接打开，箱子里面是一支狙击步枪，一支订制的M200狙击步枪！
<br/>　　詹姆斯将嵌在箱子泡沫塑料中的狙击步枪拿了出来，拉开伸缩枪托加以固定，右手握着狙击步枪，枪托顶在右肩上，把枪架在左手臂上，左手环绕过狙击步枪对狙击镜进行着校准。
<br/>　　当詹姆斯对狙击镜完成初步校准后，直升机已经飞出了机场，詹姆斯从箱子从拿起一个压满了子弹的弹匣拍进弹匣槽，拉动枪栓，将子弹推上枪膛，要求飞行员将直升机悬停在空中，进行实弹射击对狙击步枪弹道进一步的校准，打空了一个弹匣后，詹姆斯满意的收起狙击步枪，拍了拍自己身边的机组成员，对着他竖起一个大拇指。
<br/>　　詹姆斯年轻的时候，在Force-Recon（美国海军陆战队远征军直属侦察连）服役，参加过对越战争，曾经是SOG（MACV-SOG）的一员，配合过绿色贝雷帽特种部队、中央情报局、海豹突击队完成过多次政府不承认的“黑色行动”。在force-recon服役的时候，詹姆斯就是狙击手，并且他所创造的最远狙杀纪录保持了很长时间，直到“9.11”事件后，全球反恐战争拉开序幕，他的狙杀记录才被打破。
<br/>　　悬停在空中的直升机加速朝着飘荡在大西洋上的远洋货轮飞去。
<br/>　　……
<br/>　　大西洋，远洋货轮。
<br/>　　西伦科特和联合行动特种部队之间进入了一种僵持的状态，西伦科特左手提着那个装有P病毒瓶子的背包，右手握着沙漠之鹰，枪口始终指着背包中的P病毒玻璃瓶，完全没有丝毫放松。西伦科特很清楚，此时站在自己面前的，可以说是代表了当今世界特种作战最高水准的存在，这些人没有一个是简单的存在，全都是百发百中的神枪手和快枪手，表面上他们似乎是跟自己达成了暂时的和平，但是西伦科特清楚他们一直在寻找着机会，只要自己有丝毫的放松，他们就会闪电般的出手，开枪击毙自己，所以西伦科特不敢放松。
<br/>　　这时，一艘游艇朝着远洋货轮的方向快速靠近，在距离远洋货轮还有五六百米的时候，游艇关闭了引擎停止了前进，显然是游艇上的人发现了远洋货轮上的异常，所以才停止了前进。
<br/>　　游艇上人影绰绰，数名雇佣兵手持武器出现在游艇上，狙击手也爬上了游艇最顶层，趴在甲板上，支起狙击步枪瞄准了远洋货轮的方向。
<br/>　　西伦科特对着游艇甲板上那个戴着墨镜，嘴里咬着一根粗雪茄，上身穿着一件紧身背心，下身穿着一条丛林迷彩长裤，黑鹰丛林作战靴的佣兵队长点头示意了一下，随后抬起手，看了眼手表上的时间，对罗昊和麦克米兰说道：“来接我离开的船到了，而我的老朋友还没有出现，你们决定了没有，是要跟我达成一笔对你们有利的交易，还是我打爆背包中的玻璃瓶我们同归于尽？”
<br/>　　“呼呼呼呼……”
<br/>　　一架直升机快速从海面上掠过，气流将海水向直升机四周推开，在靠近远洋货轮的时候，直升机机头向上扬起，快速爬升，然后悬停在甲板上方。
<br/>　　站在甲板上的龙隐部队队员和海豹突击队队员纷纷回头朝着自己的直升机看去，戴着一副墨镜的詹姆斯就坐在直升机舱门口，嘴里咬着一根雪茄烟，墨镜背后，詹姆斯眼神冷漠的看着站在船舱门口的西伦科特。
<br/>　　西伦科特看到詹姆斯对着大声喊道：“老朋友，难道你不准备下来跟我谈谈吗？我手里可是有着你非常想要得到的东西，背包中的东西如果遭到了损伤，恐怕你也不好交代吧。”
<br/>　　不过詹姆斯已经没有丝毫要让直升机下降的打算，只是坐在直升机上，用力的吸着嘴里那根足有拇指粗细的雪茄烟。
<br/>　　“嗞啦——”
<br/>　　詹姆斯缩着腮帮子又一次吸了一大口雪茄烟后，突然把咬在嘴里的雪茄烟吐掉，还剩下大半截的雪茄烟在空中打着转，朝着甲板坠落下去，在吐出雪茄的瞬间，詹姆斯一把抓起放在机舱地板上，已经将子弹上膛的M200狙击步枪，瞄准了站在船舱门口的西伦科特，然后果断的扣动了扳机……
