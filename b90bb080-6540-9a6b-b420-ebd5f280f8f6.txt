<br/>				    ***    等火烧的差不多了，陈默又走进去看了看，除了被烧成黑渣滓的尸体外，还弥漫着一股别样的味道。<br/>　　<br/>　　    此时，洁净术就有了用武之地，直接来一个洁净术后，顿时山凹中除了清新的空气外，没有了任何其他的味道。<br/>　　<br/>　　    陈默又掏出一个爆裂符箓，看看感觉不够，又掏出一个。然后退到较远的位置站定，朝手中的爆裂符箓真元一激发，直接将两个爆裂符箓扔进洞中。<br/>　　<br/>　　    他制作好爆裂符箓以后，就从来没有试验过，所以也不知掉爆裂符箓的能量有多大，因此就多加了一个，生怕能量不足。<br/>　　<br/>　　    没有想到的是，爆裂符箓被他扔进山凹以后，直接爆发出了强大的力量，而且还是两个同时爆发，将这个山凹给震的坍塌不，还将山凹上面的山石也震落下来。<br/>　　<br/>　　    山坡上很多石头和泥土，本来就被雨水弄得有些松软，再这么一震，就形成了面积的泥石流。<br/>　　<br/>　　    陈默感觉不对的时候，就迅速逃窜，好在他本来就站的比较远，另外就是他的速度不错，跑的快！要是他还在山凹下面的位置，绝对会被埋住。<br/>　　<br/>　　    爆裂符箓的威力不错呦！这要是几个同时引爆，那岂不是非常的爽歪歪？如果以后有谁招惹到自己，直接来个集束爆裂符箓，一次十二个在一起，然后扔出去，那场面绝对震撼！<br/>　　<br/>　　    不过一想到绘制符箓的制作过程，陈默就头痛，不仅制作成品率还比较低不，而且非常耗费真元和精神力，每次制作完都是浑身无力的感觉。要不是有灵液能够使用的话，那种感觉绝对的欲仙欲死！<br/>　　<br/>　　    不过今天晚上战斗过后，陈默决定还是要多多制作符箓，这玩意可以多多益善，等遇到高阶后天武者的话，拿着符箓砸死他，哈哈！想到这个地方，陈默也是开心不已。反正自己恢复真元很快，只要准备好灵液稀释水就成。<br/>　　<br/>　　    其实陈默的这个想法，要是让修真界的人知道后，绝对会暴打陈默，然后拿出雷暴符箓，将陈默轰击一百遍啊一百遍！<br/>　　<br/>　　    麻麦皮的，灵液这种东西，就是在修真界也是稀世珍宝级别的东西，却让他像喝水一样使用，无论是谁知道以后，都会有揍死陈默的冲动。<br/>　　<br/>　　    回到皮卡车的位置，将车箱稍稍整理了一下，一些地方能掰直的就掰直。双手有着几百公斤的力量，虽然对战的时候不够给力，但是稍稍修复一下后车厢，还是可以的。<br/>　　<br/>　　    一些钢板被陈默掰直，还有一些梁柱什么的，他就没有办法了，只能等回去后将车送去维修。<br/>　　<br/>　　    陈默很郁闷，自己好像不是开车的料，你看吧！上一辆车还没有行驶几公里，就返回了维修间，大修了一回。<br/>　　<br/>　　    这个皮卡车虽然开了一段时间，但是首保都还没有到呢，这车厢什么的就成这个样子，直接都被砸穿了！唉！郁闷的要死！难道老天看到自己成为修士后，就感觉自己以后是坐飞剑的主，所以想让自己和汽车绝缘么？<br/>　　<br/>　　    虽然天上下着雨，但是陈默却并没有受到雨淋。金刚符箓的作用还在，能量也没有被消耗完，还一直依附着陈默，在他身体一圈形成保护层，雨水什么的，也就淋不到他。<br/>　　<br/>　　    倒是让他想到了金刚符箓的另外用处，可以作为雨伞用！哈哈！<br/>　　<br/>　　    开心一下而已，这制作符箓的难度，还有成功率，实在是太低了，等自己以后能随手就制作成符箓的话，当雨伞用还有可能，要不才舍不得。<br/>　　<br/>　　    自己学会绘制符箓以后，在这么长时间的制作中，金刚符箓自己也就制作成功了三张！今天晚上还用掉了一张，所以回去后，还要抽时间再制作一些。无论如何，保命的东西要多多益善，有备无患。<br/>　　<br/>　　    等回到车里后，他开始查看起自己刚才搜索过来的东西。先是看木盒，一打开后，却发现里面是根须皆的人参，看其年限，可能也就五十年以上药龄。感觉还不错，等拿回去后入药，给父母练出点补药来，让他们也吃吃。<br/>　　<br/>　　    对于人参，陈默并没有多么的珍惜，自己乾坤珠内种植的人参很多，而且只要等过个几年，自己再解开一层乾坤珠的禁制后，可能百年的人参就多的不要不要的了。<br/>　　<br/>　　    将木盒关上收到乾坤珠内，然后就拿出那个像书本的包装。上手摸去，果然和猜测的一样，里面是本书。<br/>　　<br/>　　    书本被外边的塑料包装的很好，看来其主人很看重这本书，保护的如此之好。打开包装后，发现这本书是线装书，应该是手工制作的，有种古风古气。但是这个不是重点，重点是书名，王家长拳！<br/>　　<br/>　　    陈默看到书名，并随手翻了几页纸以后，就明白这个是拳谱。王家，呵呵！<br/>　　<br/>　　    看来是个姓王的世家，来杀自己的人可能就是王姓世家的。虽然不一定，但是却有很大几率。<br/>　　<br/>　　    不过这个书名是什么鬼？王家长拳！这么简单的名字？难道就不能起个什么伏虎、金刚之类的，就是冲拳炮拳也比这个名字好听啊！要是能有个逍遥，毁天灭地的名字也是可以的啊！<br/>　　<br/>　　    怪不的袭击自己的人，就是一拳打来，然后又是一拳砸来，这不是长拳是什么？陈默忍不住吐槽，但是身体却很诚实的将王家长拳的书收入乾坤珠内。呵呵，自己终于有本武林秘籍了！<br/>　　<br/>　　    其实他身体现在还有发抖，不过已经是极其细微罢了。今天晚上有多么凶险，他的内心是非常清楚的。要不是他还算勤快，学习符箓之术，并且制作了几种符箓，尤其是金刚符箓，不仅可以抵御来敌，还利用符箓将来敌消灭！<br/>　　<br/>　　    要不然的话，不是死亡就是重伤，虽然凭借修士的手段，最终可能逃离，但是这些就会将他的身份暴露出来，那么后果，细思极恐啊！<br/>　　<br/>　　    陈默的战斗经验还是比较低的，如果没有符箓，而是用法决的话，就不应该让人靠近，直接使用法决快速解决来敌才是。<br/>　　<br/>　　    所以逢人莫装比，装比死的快！陈默今天晚上还是有些托大了，要是金刚符箓禁不起敌人的几下，那么他绝对就是受伤的命。<br/>　　<br/>　　    死是死不了的，至少洗筋伐髓之后，他的皮肉骨髓杂质很少，坚韧性和恢复性都是不错的，但是重伤是不可避免。<br/>　　<br/>　　    将拳谱收入乾坤珠内，自己正好缺一些招式，拿来自己学着正好，也能参考一下，看看这些武林世家的格斗技艺是不是很好，也能有个了解。<br/>　　<br/>　　    钱包就是那种普通的男士钱包，里面除了有大概几千元的现金外，就是一些卡片，各种信用卡和银行卡，还有一个驾驶证。<br/>　　<br/>　　    驾驶证上的名字是王琦，和照片一对比，就是刚刚想杀死自己的人。看了看其地址，是另外一个城市的。<br/>　　<br/>　　    陈默想了想，这个王琦究竟是谁呢，最近自己能够得罪的人，除了沈博文以外，还真没有其他人了。当然一些暗地里做的事情，陈默相信被人绝对察觉不到是自己，那么这个王琦来杀自己是为了什么呢？<br/>　　<br/>　　    手机，是现在比较普遍的国产智能机。不过好像设置的有密码，打不开。虽然他知道怎么消除密码，但是一旦消除密码后，手机内所有的东西就部完蛋了，都会被删除然后恢复出厂前的设置。<br/>　　<br/>　　    陈默手中拿着手机，有些费心思量，究竟是谁想要来杀我？为什么会知道自己回家的路线，还有在公路的旁边堵着自己<br/>　　<br/>　　    自己现在该如何做呢？是直接回家还是原路返回？想想，还是决定先不回去的好。到底是谁想要杀死自己，这个事情要是搞不清楚的话，那么后面肯定还有一连串的事情等着自己。<br/>　　<br/>　　    所以，今天晚上还是不回去了。反正这个时候也就不到夜里十点钟，路上再往前走，就到了镇子上，有修车的地方也有租车的地方。自己的皮卡车需要修理，手机也需要破解密码。<br/>　　<br/>　　    但是这么做，会不会被今天晚上想杀自己的同伙给撞见？要是同伙逃跑了怎么办？<br/>　　<br/>　　    希望大家多多支持，给个推荐和收藏，学生就会有动力写下去，谢谢了！<br/>　　<br/>　　    7***<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
