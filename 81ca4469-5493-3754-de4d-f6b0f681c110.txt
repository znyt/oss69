        &lt;h3&gt;第三卷第十九章  傅海生哭了&lt;/h3&gt;
<br/>
<br/>        洛城陈子乔为迎接弟弟妹妹举办的晚餐结束了，陈子乔问陈子昂：
<br/>
<br/>        “子昂，你累不累，如果不累了我们去唱唱歌？消遣消遣？”
<br/>
<br/>        “子昂？”傅海生也看着陈子昂，她唱歌就像是百灵鸟一样好听。
<br/>
<br/>        “俞轩，要不我们去唱唱歌？”
<br/>
<br/>        “好啊，我可是你的粉丝。”江俞轩笑着说。
<br/>
<br/>        “我们子昂可是百灵呢。”陈子霖也很开心，他也有很久没有听见他家百灵唱歌了。
<br/>
<br/>        一行几人去了附近的酒吧，陈子昂想起来五姐和她当初的生活，心里百感交集，当时要不是五姐出事，她有可能会回去上大学了，她和傅海生会不会在一起？
<br/>
<br/>        傅海生过得好吗？听四姐的意思，他们夫妻好像不在一起？而且夏颜和四姐也认识的？
<br/>
<br/>        等一会好好和他聊一聊。
<br/>
<br/>        傅海生没有想到的是，他被陈子昂的聊一聊，聊得他，抱着她放声大哭。
<br/>
<br/>        大家都想听陈子昂唱歌，陈子昂想到傅海生给他写的那些信被张函压在书低下，深深的叹了口气，年少的心啊，总是那么纯真和美好，年少的情感啊，只是来不及发现就消失得无影无踪。
<br/>
<br/>        雁南飞，雁南飞，雁叫声声心欲碎；不等今日去，已盼春来归。今日去原为春来归，盼归莫把心揉碎，莫把心揉碎，且等春来归。
<br/>
<br/>        陈子昂的一首雁南飞唱得如泣如诉，原本她的歌声清脆空灵，再加上想到她和张函、傅海生的纠葛，见到傅海生，心里顿生感慨，只把人唱得心碎不已。
<br/>
<br/>        每个人都在陈子昂的带动下各有心思。
<br/>
<br/>        江俞轩唱了一首刘天王的“一起走过的日子。”
<br/>
<br/>        江俞轩也是人才，一口粤语歌唱的声情并茂，他边唱便看着陈子昂，是啊，他们一起走过的日子，有你有我有情有义有海有地，以前一起走过，以后也会一起走过，子昂，此生我们就一直相伴着你，陪着你一起奋斗也是一种幸福。
<br/>
<br/>        陈子霖唱了一首“爱拼才会赢”，他要和妹妹一起奋斗，没有伞的孩子是要学会奔跑的，他和他妹妹都是没有伞的孩子，只有自己长成参天大树才能让父老乡亲纳凉。
<br/>
<br/>        陈子乔唱的是“你究竟有几个好妹妹”她的歌声没有陈子昂的好听，但是也不逊色。
<br/>
<br/>        临到马总了，马总唱的是“心会跟爱一起走。”他伸出手准备想邀请陈子昂跳舞，陈子乔站起来拉着他的手俩个人一人一个话筒，对唱了起来。
<br/>
<br/>        “海生，这几年好吗？”
<br/>
<br/>        “还好，你呢？”没有你的日子都是一样的无趣，哪里还有什么好不好的？
<br/>
<br/>        “我，也还行吧，去了北京之后就一直在努力的工作。”
<br/>
<br/>        “这一次回来，没有带孩子和家人吗？”
<br/>
<br/>        “孩子要上学呢。”陈子昂提起俩个小宝心里充满了骄傲，俩个小家伙真正是聪明，比拼着拿他们学校的第一。
<br/>
<br/>        傅海生看着陈子昂那温柔的、满足的眼神，心里空了一块却又觉得充实了很多，只要陈子昂幸福就好，他给她写的信，大概她都没有看到吧。也好，安静的看着她幸福就好。
<br/>
<br/>        “海生，夏颜和孩子呢？”陈子昂问傅海生，他们肯定也很恩爱。
<br/>
<br/>        “孩子在泉城，夏颜调到西城了。”
<br/>
<br/>        “你们一家三口三地分居啊？”难怪四姐说傅海生一个人。
<br/>
<br/>        傅海生微微一笑，他和夏颜的关系如果可能的话就差在离婚书上签一个字了。
<br/>
<br/>        “工作的需要嘛，都是没办法的事情。”
<br/>
<br/>        “也是，你们都是吃公家饭的。”
<br/>
<br/>        “子昂，我还记得你写的诗，我念给你听：
<br/>
<br/>        我站在楼上，看着满树盛开的樱花
<br/>
<br/>        你从时光中走来
<br/>
<br/>        阳光照耀下，樱花与你
<br/>
<br/>        谱出了动情的旋律
<br/>
<br/>        我伸出手，抓了一把阳光
<br/>
<br/>        摊在掌心，都是你的影子
<br/>
<br/>        怕将时光丢失，我将影子藏在了怀里
<br/>
<br/>        连同自己的心
<br/>
<br/>        悄悄的交给你
<br/>
<br/>        盛开的樱花羞涩了一地的妩媚
<br/>
<br/>        化成春泥，欲与你生生世世。
<br/>
<br/>        傅海生念诗的时候声音是颤抖着的，而且已经泪流满面。
<br/>
<br/>        “海生，你还记得那么清楚，我都忘记了。”陈子昂年少的时候怀揣着一个做诗人的梦，写下了很多幼稚的诗，没有想到傅海生还记得那么清楚。
<br/>
<br/>        “子昂，和你在一起的日子我都记得。”
<br/>
<br/>        “是啊，在一起的日子真是让人怀念呢，来，好哥们，我们走一个。”
<br/>
<br/>        陈子昂拿起酒瓶和傅海生碰了一下，五彩的灯光转过来，打在傅海生的脸上时，陈子昂看到了傅海生满脸的泪水。
<br/>
<br/>        “海生，还能喝吗？”
<br/>
<br/>        “怎么不能喝呢，来，子昂，好哥们，走一个。”她永远都当他是好哥们，那就做一个好哥们吧。
<br/>
<br/>        人生不就是分分合合，人生不就是聚散无常吗，人生不就是由无数的遗憾组成的么。只要你安好，对我来说就是每一天的小确幸。
<br/>
<br/>        他那没有说出口的爱就让他深深的藏在心底，就让他做她的好哥们吧。
<br/>
<br/>        他作茧自缚了很多来年，如今看着她如此光彩照人，看着她很幸福的样子，他的心突然满足了。
<br/>
<br/>        傅海生拿起话筒唱了一首“最远的你是我最近的爱”，无论何时，无论你离得有多远，我想，你都是我最近的爱。
<br/>
<br/>        “海生，过去的都已经过去了，我们要向前看，我们要学会放下，学会爱自己。只有有能力爱自己，才能有能力爱别人。”陈子昂等着傅海生唱完歌，递给傅海生一杯水。
<br/>
<br/>        她觉得傅海生今天喝得有些多，不过傅海生酒量有多大，她并不清楚。
<br/>
<br/>        “子昂，你说的话我都会记得，我也会去实行的，你放心，我一定会好好爱自己。”
<br/>
<br/>        大家都喝了一些酒，马总准备要离开了。
<br/>
<br/>        “我得回家了，要不然你们大嫂会打电话给我的。你们几个再玩一会？”
<br/>
<br/>        陈子昂坐了半天的车，其实也有些累了，于是就说：“那大家一起走吧？”
<br/>
<br/>        陈子霖要开车，喝得很少。《那几年是没有酒驾这一说的》
<br/>
<br/>        陈子乔送马总回家。
<br/>
<br/>        傅海生离单位不远，准备步行回家，但是陈子昂看着喝多了的傅海生，要傅海生和他们一起走。
<br/>
<br/>        傅海生一直努力的克制着自己，等到下车时，傅海生心里翻江倒海。
<br/>
<br/>        他刚见子昂，就要分别，他是多么的想要时光能够停留啊。
<br/>
<br/>        “海生，明天见！”
<br/>
<br/>        “子昂，我可以，抱，抱，你吗？”傅海生看着陈子昂。
<br/>
<br/>        陈子昂沉默了一会，走向傅海生。
<br/>
<br/>        傅海生刚接触到陈子昂，脊背就剧烈的抖动着，然后是抑制不住的放声大哭
<br/>
<br/>        “子昂，我知道我一直在做梦，我知道我每次都会梦见你，我知道梦一醒你就会消失。可是我还是想，抱，抱你。”
<br/>
<br/>        “子昂，我想你啊，我想了你很多年，为什么你都是杳无音信？我给你写了那么多信，我以为你不回我信，可是我的信被她给截走了。”
<br/>
<br/>        “子昂，子昂，你还好吗？你过得好吗？”
<br/>
<br/>        傅海生醉了。
