　　末世三年似乎整个世界都在动荡，神秘家族之间的大战爆发的突如其来，也沉寂的快速异常，在陆玟丞接到猎王暗中送来的情报之后，这场战事之后的种种似乎就被可以埋藏了起来，两位老人回到上京继续管理着众多基地，猎王也带着猎者联盟人就此消失如同事发之前一样的低调，高家和罗家更像是什么都没有发生过一般，正常往来，一切照旧。
<br/>　　好似这场战事根本没有发生过，当然知道这件事的人除却当事人，整个华夏也不超过十个。
<br/>　　除此之外，远在西南地带的狼牙战团，也好似就此消失在了众人视野，除却留守在那处山脉之下的本部少许人，整个狼牙战团多达八万人的军队，就这么没入埋骨之地的方向再也没有任何消息传出来。
<br/>　　南方边境的战火依旧，整条防守线上丧尸的碎尸和血液灌满了大地，它们前仆后继的攻来，一次又一次的被联盟军挡下，双方就这么长久的对峙，不死不休的好像就看谁能坚持到最后。
<br/>　　在这样混乱的年代似乎进入了某种规律型发展模式时，狼牙基地内被划为绝对禁地的那座小屋，终于来了两位不速之客。
<br/>　　白允儿一身白衣，与身旁身披大红色长裙的步纱站在一起，两女颜色分明的样子宛如一道刺目的风景线，完全相悖的气场更让这间小屋的内外都平添一份诡异感。
<br/>　　“就是这里，你进去吧。”白允儿轻声开口，毫不在意她们两人大摇大摆的进入这里，会不会引发整个狼牙的最高级警报。
<br/>　　步纱冰冷的看了白允儿一眼，略带苍白的面孔可以看出她这一个月内付出了极大代价，而一路赶来这里却只是为了救人，这违背她的原则更完全不符合她的行·事作风。
<br/>　　步纱很清楚，白允儿还有很多东西没有告诉她，比方说这次救楚涵不可能简单，而她会因此变成什么样更是未知，或许还会有生命危险。
<br/>　　但她心甘情愿，义无反顾。
<br/>　　没有太多的言语，步纱上前轻推开门，她的双手在颤抖，眼神更附上了一层雾色。
<br/>　　吱呀——
<br/>　　门开。
<br/>　　嘭——
<br/>　　门闭。
<br/>　　看到步纱进入屋内，白允儿这才移开目光，望向了不远处疾步赶来的尚九娣。
<br/>　　这一次不像上回与白羽秘密潜入，白允儿带着步纱而来，自然逃不过这狼牙基地处处防备的暗哨，哪怕步纱已经十阶，哪怕白允儿的可怕战力无法估计，但这处禁地十步一哨的恐怖依然让尚九娣第一时间得到了消息。
<br/>　　尚九娣看到白允儿站在屋外，没有多想便朝着木门而去，她当然知道有另外一个人与白允儿一同前来，自然担心屋内的楚涵会有意外。
<br/>　　但是白允儿的速度让尚九娣望尘莫及，在尚九娣以最快速度冲到门边之际，一个白影闪过，白允儿就挡在了她的面前。
<br/>　　“救人，勿扰。”白允儿的声音依然清丽，冰冷的眼神更带着让尚九娣无比陌生的强硬感。
<br/>　　尚九娣嘴唇蠕动了一下，道：“进入的是何人？”
<br/>　　穿着红裙，样貌美艳至极的年轻女人，让尚九娣下意识的警惕，太诡异了，实在诡异的让她不敢多想。
<br/>　　白允儿红唇轻勾：“你应该听说过她，蛇蝎妖姬美阎罗。”
<br/>　　尚九娣的瞳孔剧烈一缩，不禁失声大喊：“你疯了！那是个杀人不眨眼的女人，你让她独身进入这间屋子？！你到底在想什么！？失忆让你也忘了你与楚涵曾经的一切了吗？！你想让楚涵死吗！”
<br/>　　尚九娣的咆哮带着剧烈的恐惧，以及情绪的即将崩溃，她根本无法想象美阎罗会对楚涵做什么，那可是蛇蝎妖姬啊，猎者联盟内最难伺候的高级猎者，只要是她想要杀的人，就没有逃过一劫的，更在江南一带有着一步一杀的凶名！
<br/>　　白允儿的笑容很诡异，淡然的样子与尚九娣的激动形成强烈对比，她道：“步纱可以杀光全世界的男人，但唯独不可能做出伤害楚涵的事。”
<br/>　　尚九娣呆愣原地，眼神空洞无措的看着白允儿：“我不是很明白你的意思……”
<br/>　　白允儿没有解释的意思，只是挡在了门外，视线转移到四周环视，对尚九娣察觉不到的那些轻微又密集的脚步声，报之淡然一笑。
<br/>　　在尚九娣撕心裂肺吼出那些话的时候，很快消息便传到了其他军官的耳朵里，美阎罗混进了狼牙并且进入了禁地这件事，一瞬间便传到了所有驻守在基地的人之中。
<br/>　　于是没有一颗耽误，只要不是手无缚鸡之力的人，统统都放下了手头的事情赶来，他们之中只有极个别人知道那间木屋里的人是谁，大部分人都只是有所怀疑和猜测，但无论怎样那都是谁也不能靠近的禁地。
<br/>　　而美阎罗，这个女人太可怕了，早就是狼牙内部名册上排名极高的重点关注对象。
<br/>　　同一时间在陆玟丞的办公室内，顾凉晨在得知这一消息后近乎是一跳而起，惊恐万分的样子好似末世再次来临，这也是他长期被称为冰山以来第一次如此惊慌失措。
<br/>　　陆玟丞一把将其拉住，让顾凉晨不至于失控。
<br/>　　“你干什么？美阎罗！她要杀楚涵！”顾凉晨的双目腥红，充满了即将爆发的杀意：“白允儿带来的，白允儿失忆了，白家，还是什么？放手！让我过去！”
<br/>　　陆玟丞不过一名文职，虽说有着不弱的战力但在顾凉晨这种高阶新人类面前，还是被完全压制。
<br/>　　感受到眼前这个冰山即将黑化，那恐怖的生命波动步步膨·胀着即将爆出体外，陆玟丞强忍着要吐血的感觉开口：“那是你大嫂，大嫂，你淡定……”
<br/>　　顾凉晨显然一时半会儿没能接受这等设定，一脸的莫名之间更因为爆发的能量忽然受到心情影响而回收，差点被反噬，只见他面色苍白下意识反问：“什么大嫂？”
<br/>　　“你大嫂有很多个，你先别冲动。”陆玟丞也同样有种失控的感觉。
<br/>　　他完全没想到白允儿会带着步纱过来，而步纱的真实身份以及与楚涵的关系，除了猎王和他这位情报部的部·长，还的确没其他人知道。
