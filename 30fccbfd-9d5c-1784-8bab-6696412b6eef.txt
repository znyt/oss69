　　因为下午睡了一觉的关系，补充过睡眠的关系，到了晚上，罗昊并没有什么睡意。
<br/>　　这个季节，北方城市虽然还没有开始供暖，不过因为房子本身的隔热设计，云熙在卧室即便只是穿睡裙也不会觉得冷。
<br/>　　当罗昊从背后抱住自己的时候，云熙的眼中闪过一丝狡黠之色，打算罗昊在准备对自己做点什么的时候，捉弄他一下；可是这次出乎云熙的意料，罗昊这次却是出奇的规矩，只是抱着自己什么过分的动作都没有。
<br/>　　又不想就这么放弃捉弄罗昊的机会，云熙决定要主动出击，身体往罗昊怀中靠了靠，柔声说道：“我们都有一段时间没这么安静的在一起了……”
<br/>　　要让云熙主动说出跟那方面事有关的话，云熙还是说不出口，可是她这一句话却也已经把自己要表达的意思，隐晦的表达了出来。
<br/>　　罗昊深吸一口气，对靠在自己身上的云熙说道：“云小熙，别以为我不知道你打什么主意，你可不要逼我浴血奋战，浴室垃圾桶里我可是都看到了，你就收起你这点小心思，别想着捉弄我，把我逼急了，我可是什么事情都做得出来。”
<br/>　　云熙翻了个身，可怜兮兮的看着罗昊，说道：“我对你这么好，你舍得那么对我吗？难道你真的忍心？”
<br/>　　“云小熙，没想到你还有这天赋，你要是去演艺圈发展，说不定就帮我们国家把第一座最佳女演员的小金人给拿回来了。”罗昊保持着冷静在云熙鼻子上点了一下，说道。
<br/>　　“哼，你一点都不配合，不好玩，不玩了！”云熙皱了皱鼻子，翻身背对着罗昊，耍着小脾气，还故意踢了踢身上的毯子。
<br/>　　“这么大的人，都两个孩子的妈了，怎么还跟个孩子一样。”罗昊把毯子帮云熙盖上，说道：“早点休息，明天还要回龙穴一趟，试试能不能从龙猫那里找到解决办法。”
<br/>　　……
<br/>　　龙隐部队特种作战基地，赵天宇办公室。
<br/>　　此时时间已经是快十点了，可是赵天宇办公室的灯却还亮着，窗帘拉着，不过透过窗帘，还是能看到办公室中还有人。
<br/>　　赵天宇的办公室中除了赵天宇之外，还有云岳和朱致阳两个人，茶几上的烟灰缸中塞满了烟头，办公室中还有未散去的烟雾。
<br/>　　赵天宇手中拿着一份行动计划，从机场回来，三个人就关在这间办公室中一直在讨论着什么，赵天宇手里这份行动计划，他已经翻了不下十遍，不说把里面的内容都记了下来，就连标点符号都快要被下来了。
<br/>　　这份行动计划书就是“追魂”计划，是由云岳一手制定的，交由赵天宇进行评估。
<br/>　　“暗影，你确定要执行‘追魂’行动？”赵天宇放下行动计划书，对云岳问道。
<br/>　　“如果想要找到小丑，这是最好的机会。”云岳说道：“错过这次机会，以后可能还会有机会，但是我们需要更长时间的等待。我从‘炼狱’那边得到消息，玛丽科娃还不打算放弃‘crystal-ball’，她已经对关押在‘黑蛇’监狱里的Derrick采取了几次行动，目前还没有得手，按照这个趋势，玛丽科娃还会继续动手，如果真的让她从Derrick手中拿到了‘crystal-ball’的配方，这个行动计划就失去意义了。”
<br/>　　赵天宇看向朱致阳，问道：“老狐狸头，你怎么看？”
<br/>　　“‘追魂’行动的风险的确有些高，所谓富贵险中求，这的确是次很好的机会。”朱致阳回答道：“从原则上，我是赞成这个行动计划的。”
<br/>　　“既然你们两个都没有意见，那就按照计划执行。”
<br/>　　赵天宇也不是那种磨磨唧唧，婆婆妈妈的人，当即决定执行这个代号为“追魂”的行动计划。
<br/>　　“无常那边谁去说？”云岳看向赵天宇问道。
<br/>　　“这个……”赵天宇考虑了一下，回答道：“有个人去比我们仨去都要合适。”
<br/>　　云岳当即反应过来，点头道：“的确，他算是无常的伯乐，的确由他去是最合适的。”
<br/>　　……
<br/>　　第二天一早，罗昊和云熙就驱车赶回了龙穴。
<br/>　　罗昊找到了司徒空，把佟千雪的情况跟司徒空尽可能详细地说了一遍，司徒空闻言，沉默了好一会儿，才是开口说道：“通过中医调理的确是有可能恢复的，可是就你目前所说的，我也不好确定佟千雪的确切情况，而且我从小更多接触的是武学，在中医药理这一块不如我的那几个叔伯堂兄弟，这样，我跟无常联系一下，如果他们方便的话，最好是去东海市找我家里人，这样也能对症下药。”
<br/>　　罗昊点头，这样是最好的，毕竟他也只是把从赵天宇那边了解到的情况进行转述，情况是不可能说清楚的，让佟千雪本人去东海市找司徒空家里人确诊是最理想的。
<br/>　　司徒空先是跟家里取得了联系，然后又是把电话打到了慕容飞的手机上。
<br/>　　杭山市，佟千雪公寓。
<br/>　　慕容飞看到手机上的来电显示，立刻变得紧张起来，以为又是有什么行动，需要自己紧急归队，可是仔细看了看电话号码，才发现这是警通连值班室的电话。
<br/>　　“无常，我是龙猫。”司徒空开门见山的对慕容飞问道：“佟医生在你身边吗？我有些话要跟她说。”
<br/>　　慕容飞疑惑地皱了皱眉，但还是拿着电话走进卧室，把躺在床上休息的佟千雪叫醒，把电话递给她，说道：“是龙猫的电话，说有话要亲自跟你说。”
<br/>　　佟千雪也是跟慕容飞一样疑惑，不明白慕容飞的战友为什么会指名道姓的要找自己接听电话，按道理，自己应该跟他们并没有太多的交集才对。
<br/>　　心中有疑惑，可佟千雪还是接过了慕容飞递过来的手机，“你好，我是佟千雪。”
<br/>　　“佟医生，我是龙猫，我们见过面。”司徒空简单的介绍了自己后，便是直奔主题，说道：“你的情况，我有所了解。我们司徒家除了是武学世家也是中医药世家，我跟家里联系过，你的情况或许能有转机。”
