　　第350章
<br/>　　宋游渠不能不恐惧，他们来自长安，而只要你去过长安，见识过那个干燥血腥的夏天，就不可能没听过那个名字。
<br/>　　一柄大斧，将整个帝国的体面劈个洗把脸。那一日武科场荡起的烟尘，还经常出现在诸人的噩梦中。
<br/>　　钟武才在对方面前搬出宋伯康、薛相，以期吓住对方，现在看上去像个笑话。就算宋伯康站在对方面前，怕也有心惊胆战，拿出十二分精力应对。
<br/>　　“宋哥哥，他会杀了我们么？”薛问婷怕了。如今大家在山贼窝子里，山贼是怎样的存在已经不需多言。多的是山贼屠人满门，占人妻女的传说。更何况自己又生的娇美，薛问婷不能不怕。
<br/>　　“他们……不会杀我们的。”宋游渠终于道。
<br/>　　大家一听这话，都看向宋游渠。
<br/>　　“我们已经抬出身份，他终究要顾忌我们背后的势力。敢杀我们，他不想活了么。”
<br/>　　如今众人都心慌意乱，宋游渠说出这话，无论如何都给众人带来安慰。
<br/>　　而事实上，宋游渠心里也没谱。对正常人而言，总要顾忌相府、凉州牧的存在。可程大雷是个正常人么……根据以往发生的事情判断，没有任何证据证明他正常。
<br/>　　“宋哥哥，全靠你了。”薛问婷低低的声音道。
<br/>　　两家在极力促成这门亲事，携伴同游便有这方面的考虑。薛问婷情长如野草，早已倾心。宋游渠却虽然经常板着冷脸，却也已对出身模样都不错的薛问婷动心。
<br/>　　此刻薛问婷一句话，激起他男儿气魄，昂起头道：“婷妹放心，自有我照应。”
<br/>　　听到这句话，薛问婷瞬间羞红了脸。就觉得这暗室之内，铁牢之中，也有了些温香。
<br/>　　咚咚咚。
<br/>　　沉重的脚步声响，程大雷在徐神机、银眸的陪伴下来到牢房。他一步步走进，在昏暗的光线下一张丑脸显得更加恐怖，宋游渠的心立刻紧张起来。
<br/>　　程大雷目光扫了一圈，嘴角笑容显得有些意味深长。当然，在宋游渠等人眼里看来，这笑容就是魔鬼的微笑。
<br/>　　“诸位感觉怎么样，没受什么委屈吧？”程大雷让银眸搬了张椅子过来，坐在诸人对面。
<br/>　　“哼，你若敢动我一根头发，凉州大军杀来，要你千万条尸体偿命。”宋游渠。
<br/>　　“凉州牧么？”
<br/>　　程大雷抿嘴笑笑，宋伯康来到凉州的消息，程大雷很早就知道了。之前的凉州牧被戎族杀了，这职位一直空着，当然，上上下下的主要人物都是相府的人，程大雷也不是一无所知。
<br/>　　关键是，这宋伯康来此地是做什么，会不会是冲着自己来的。这是程大雷心里琢磨的事情。
<br/>　　见他面露犹豫，宋游渠松了口气，他换了个口气，道：“不过，你也不用怕，现在凉州正是用人之际，只要真心向我父投诚，家父必定会原谅你的所作所为，还会对你委以重任。”
<br/>　　程大雷不明白宋伯康的来意，宋游渠不可能不知道。
<br/>　　琴川关产铁，当初莫鸣米劫掳劳力日夜挖铁，类似不计其数，就是为了供应给相府。说白了，原先的莫鸣米就是相府一个工头而已。
<br/>　　可莫鸣米被程大雷取而代之，相府就缺少了兵器来源。想到程大雷是怎样的人，相府不能不重视。于是派了宋伯康来理凉州事。
<br/>　　对于程大雷，上下的意思是一样的：能用就用，不能用就杀。
<br/>　　宋游渠心中想的明白，若自己可以拉拢到程大雷，一场危机便会变成自己平步青云的机会。
<br/>　　看他目光闪烁，当是已经有几分动心。宋游渠决定再加一把力。
<br/>　　“你不必担心，有我作保，必能给你一个官位，保你光宗耀祖。”
<br/>　　“这个……那个……”程大雷闪烁其词：“该不会骗我吧？”
<br/>　　“我宋游渠出口，一字千金，今日之事，不过是一场误会而已，我怎么会放在心上。”
<br/>　　“多大官？”
<br/>　　“七品侍卫如何？”
<br/>　　“呃……有点小。”
<br/>　　“六品游骑将军？”
<br/>　　“还是有点小。”
<br/>　　“呵，你这人不要太得寸进尺，只要你好好做事，日后何愁不能升官加爵。”宋游渠道：“你要多大的官？”
<br/>　　“你觉得……”程大雷起身：“帝国天子这个位置配得上我么？”
<br/>　　宋游渠一怔：“你想造反么！”
<br/>　　程大雷摊开双臂：“我本来就是反贼呢。”
<br/>　　紧跟着，一片笑声。隔壁一间牢房内，有人探出头道：“大当家，这货该不会是个傻子吧。”
<br/>　　程大雷扭过头，诧异道：“高飞豹，你又喝酒闹事了。”
<br/>　　“啧，也没喝多少么，大当家，你既然来了，就放我出去吧，在这里待着，嘴里淡出个鸟来。”
<br/>　　“就特么该一直关着你。”程大雷恨铁不成钢道。
<br/>　　周围笑成一片，宋游渠一张脸由白转红，恍然意识到自己成了一个笑话。
<br/>　　程大雷无语的看着对方，道：“你爹妈生你的时候，难道把你丢了，把胎盘养大了。”
<br/>　　“什么意思？”宋游渠下意识道。
<br/>　　“不然的话，你为何没有脑子。”程大雷若有所思。
<br/>　　宋游渠满脸涨得通红，紧紧咬着牙齿。
<br/>　　“好啦。”程大雷道：“我也没时间和你们逗闷子，宋伯康来凉州的目的是什么？”
<br/>　　“狗贼，你休想从我嘴里知道一言一句。”宋游渠。
<br/>　　“好啦，好啦，你们细皮嫩肉的，估计也撑不过三五遍刑，现在老实交代，落个体面。非让我把你们折腾得死去活来，也没意思不是。”
<br/>　　众人心头砰砰乱撞，目光都看向宋游渠。
<br/>　　“说与不说！”程大雷忽然喝了一声：“来人，与我动刑！”
<br/>　　“说，说，我告诉你们。”钟武才大喝。
<br/>　　“好，现在有一个了，其他人呢？”程大雷目光转了一圈。
<br/>　　“大当家，我看也别费这个劲了。一人敲断五根手指，这些人什么都会招了。那个小娘皮交给我，我保管折腾得她死去活来。”徐神机。
<br/>　　“呀。”程大雷看向徐神机：“军师，你堕落了呀。”
<br/>　　“招，我们招。”
<br/>　　十几人哭得喊娘，争着大喊，生怕落在别人后面。
