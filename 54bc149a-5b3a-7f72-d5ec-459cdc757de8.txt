        童筱颖伸出手来，浑身都在颤抖，却不知道自己的手该触碰他的哪儿是好。
<br/>
<br/>        因为，面前这个躺在地上，微微抽搐的男生，被人打得鼻青脸肿，嘴里还不停地有鲜血溢出来，旁边的黑框眼镜已经镜架分离，支离破碎。
<br/>
<br/>        “马智杰……马智杰……你……要挺住！我……我……”童筱颖鼻子一酸，泪水瞬间模糊了整个视野，她开始语无伦次起来。
<br/>
<br/>        她不知道自己该怎么去急救一个被打成重伤，至口吐鲜血的人。
<br/>
<br/>        外婆没有教过她，她现在……
<br/>
<br/>        童筱颖忽然间想起了自己的行李箱，她的行李箱里有她随身携带的药箱，药箱里有特制的止血药。
<br/>
<br/>        对！
<br/>
<br/>        童筱颖想到这里，不禁喜上眉梢，刚准备起身离开的时候，手腕突然被人给抓住。
<br/>
<br/>        “不、不要走……小、以、筱颖……能、能再见到……你……真、真好！”马智杰吃力地说道。
<br/>
<br/>        童筱颖蹲下身来，按住他的手背，安慰道：“我不会走，我只是去给你拿药。你一定要挺住！我去给你拿止血药！是我按我外婆的方子配置的，很管用！我去给你拿，你一定要挺住！”
<br/>
<br/>        “不！不……我、我好冷……”马智杰抬起另一只颤抖着的手来，一把抓住童筱颖的手，用尽最后一丝力气，紧紧地握住，“小、小筱颖……我……”
<br/>
<br/>        “呕”地一口鲜血，从马智杰的嘴里喷了出来，堵住了他欲要说下去的话。
<br/>
<br/>        童筱颖顿时瞪大了眼睛，双瞳微缩，马智杰两眼一翻，眼皮子缓缓地合上，整个人就躺在地上一动也不动了，而他的双手，却紧紧地握着她的手不曾放开。
<br/>
<br/>        刚刚这个人还拉着她的手，在跟她说话，而现在却……
<br/>
<br/>        “马智杰……你醒醒啊！现在不是睡觉的时候！你起来，好不好？不要吓我！我、我经不起吓！马智杰……你起来啊！”童筱颖支支吾吾地说，抬起另一只手去按了按马智杰的手腕。
<br/>
<br/>        她一个人说了许久，在他的手腕上也按了许久，完全把不到他的脉搏之后，看着这个躺在地上，永远也听不到她所说话的人，忍不住失声痛哭了起来。
<br/>
<br/>        他最后那句没说完的话，成了她心里永远也无法释怀的伤。
<br/>
<br/>        他一定是有遗愿要跟她交代的，然而，他连他的愿望都未来得及跟她说，他就这么去了……
<br/>
<br/>        此时此刻，不远处的救护车的鸣笛声姗姗来迟。
<br/>
<br/>        童筱颖一直跪坐在地上，陪在马智杰的身边，寸步不离。
<br/>
<br/>        直到有医护人员上来，给马智杰做了初步检查，确定他已经停止心跳，没有了生命体征后，打电话报了警。
<br/>
<br/>        当警察和法医赶到现场时，童筱颖的手仍旧被马智杰的双手紧紧地握在了手心里。
<br/>
<br/>        或许，这样的姿势，其实并不称得上是“握”，而是“捧”。
<br/>
<br/>        他将她的手，紧紧地捧在了自己的两只手心里。
<br/>
<br/>        童筱颖能感觉到，马智杰的手，从余热中渐渐地变凉。
<br/>
<br/>        因为法医要做尸检，马智杰最后的姿势不能动，童筱颖就一直毫无怨言地，心甘情愿地跪坐在地上，守护在马智杰的身边。
<br/>
<br/>        她的两只眼睛里，泪水止不住地往外流，太多的泪痕，慢慢地刺痛了她的脸颊，她却无法再哭出声来。
<br/>
<br/>        法医费了很大的劲，才将童筱颖的手从马智杰的手中挣脱出来。
<br/>
<br/>        “你可以起来了！”此时，耳畔不知道是谁对她发出了命令声。
<br/>
<br/>        童筱颖整个人，神情有些恍惚。
<br/>
<br/>        当她试图从地上站起来，然而，大腿刚使力，小腿却突然发麻。
<br/>
<br/>        她起身的动作进行到一半，双腿因此而突然发软，眼瞧着即将栽倒在地时，双脚下腾然间落了空。
<br/>
<br/>        童筱颖只觉自己的身子撞入了一个温暖的怀抱中，当她抬头时，看到的却是傅夜沉的那张俊脸。
<br/>
<br/>        “大哥？！”
<br/>
<br/>        童筱颖嘴唇微微颤抖着唤了声，她的脸色一片惨白，眸子里黯然失色，眼角还噙着晶莹剔透的泪珠。
<br/>
<br/>        傅夜沉剑眉微蹙，双手横抱着她的身子，带着她只身离去。
<br/>
<br/>        一旁的警察，一个个看着，却一句话也不敢多说。
<br/>
<br/>        “不！我不走！我知道他是被谁打死的！我……”
<br/>
<br/>        “这件事情，警察会去处理，他们查清楚，将凶手绳之以法，自会让死者安息。无需你卷入其中！”傅夜沉一边抱着童筱颖往楼下走去，一边冷冷地打断了童筱颖的话。
<br/>
<br/>        本来，他是派了司机来接她，因为他知道，她一定不想看到他。
