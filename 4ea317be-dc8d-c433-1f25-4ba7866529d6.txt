　　猎妖令，美其名日接此令者，为猎妖殿效命，其家人自当受到猎妖殿庇护。
<br/>　　而事实上，很多武者为了躲避仇家，又放心不下家人，便会兑换这猎妖令。
<br/>　　萧逸没有回答，而是问道，“这猎妖令，真的有用？”
<br/>　　“当然。”总执事回答道，“受庇护的个人或者家族，任何势力胆敢攻击，等同挑衅猎妖殿。”
<br/>　　“若是暗影楼这等大势力呢？”萧逸追问道，“以他们的力量，等闲家族，挥挥手就被灭掉了吧。”
<br/>　　总执事一笑，想起来易霄与钟无忧的一战，道，“哦，原来你是担心暗影楼报复你的家人。”
<br/>　　“放心吧，他们不敢这样做。若是受庇护的家族被灭，北山郡九大总执事，十殿执法队，数十万猎妖师，会在一天之内将这个势力荡为平地，暗影楼也不例外。”
<br/>　　萧逸顿时眼睛一亮。
<br/>　　但这时，总执事却道，“不过，猎妖令有两个弊端。”
<br/>　　“首先，它的功效只有一年。”
<br/>　　“其次，若兑换此令的猎妖师死亡，它的功效会自动消失。”
<br/>　　“也就是说，若你兑换了此令，你被杀了，猎妖令就会自动失效。”
<br/>　　“提醒你一点，猎妖殿只庇护你的家族，而无义务保护你。”
<br/>　　总执事继续道，“所以说，这猎妖令，看起来厉害，实则是个鸡肋。”
<br/>　　“毕竟，如果真有本事保护自己的家族，根本无需兑换此令。”
<br/>　　“若没有本事，就算兑换了，被仇家找上来杀了，猎妖令会自动失效，仇家还是可以对付你的家族。”
<br/>　　萧逸点点头，明白其意思。
<br/>　　猎妖令虽然有着很强的作用，但它的弊端也很大。
<br/>　　任何势力都不敢坏猎妖殿的规矩，但却可以来猎妖殿查询兑换猎妖令的人是谁，而后将此人击杀，直接让猎妖令失效。
<br/>　　就算此人躲起来了。
<br/>　　这个势力也可以等满一年时间，等候此人自己回来家族。
<br/>　　一年过后，兑换者需要回猎妖殿再付出2000任务点，兑换新的猎妖令，而后再亲自将新的猎妖令交到家族手中。
<br/>　　就是因为这两个弊端，所以才让猎妖令效果这么大的东西，仅仅需要2000任务点就能兑换。
<br/>　　“诶。”这时，总执事看了萧逸一眼，道，“你现在还有任务点吗？你问个屁阿。”
<br/>　　“我...”萧逸一愣，他发现，自己真想暴揍这个贱人一顿。
<br/>　　“能否用银两直接买？”萧逸恳求道。
<br/>　　“不行。”总执事坚决地摆摆手，道，“猎妖殿有猎妖殿的规矩，只有任务点可以兑换殿内的东西。”
<br/>　　“哼，猎妖师可是一份光荣的职业，任务点等同你鲜血与荣耀背后的奖励，岂能用金钱去衡量。”
<br/>　　总执事忽然变得一副义正言辞的样子，让萧逸愣了愣。
<br/>　　下一秒，总执事忽然说道，“你真的有必须保护的人或者家族？”
<br/>　　萧逸认真地点了点头。
<br/>　　“好吧，那我便教你一个法子。”总执事道。
<br/>　　“什么法子？”萧逸追问道。
<br/>　　总执事笑道，“用钱是不可能兑换殿内任何东西的，只有任务点可以。你可以找一个拥有足够任务点的猎妖师为你兑换，只要他愿意。”
<br/>　　“对阿。”萧逸双手一拍。
<br/>　　事实上，很多猎妖师都不会在猎妖殿内兑换任何东西。
<br/>　　因为，他们需要的东西都能在外面的商行买到，除非是特别稀有的物品。
<br/>　　但特别稀有的物品，一般的猎妖师也用不上。
<br/>　　如果某个家族或者个人，迫切需要某些稀有的东西，而外面商行又暂时没有卖，便会以数倍的价格，找猎妖师帮忙兑换。
<br/>　　像培源朱果，一般很难在外面的商行找到。但实则，他的价格顶多1000两，而任务点却是100点。
<br/>　　1000两易赚，100任务点难攒。
<br/>　　若是某个炼药师迫切需要，便会找猎妖师帮忙兑换，但价格起码要三千两。
<br/>　　对于炼药师来说，三千两是小事。
<br/>　　而对于猎妖师来说，100任务点兑换不了什么好东西，三千两，却足够自己买更好的武器或者买些救命的丹药了。
<br/>　　这是两方都占便宜的事，很多猎妖师都愿意干。
<br/>　　这时，萧逸皱了皱眉头，2000任务点，一般的三级猎妖师肯定没有，可是四级猎妖师又不知去哪找。
<br/>　　一旁的总执事笑道，“是不是在想到哪去找拥有这么多任务点的猎妖师帮你阿？”
<br/>　　“不错。”萧逸点了点头。
<br/>　　“简单。”总执事贱笑道，“你面前不是有一个。”
<br/>　　“你？”萧逸一愣，疑惑道，“你不是说....”
<br/>　　“咳咳。”总执事一副仁慈的样子，说道，“易霄，虽然老夫很不喜欢那些黄白之物。只是，老夫见你似乎很在意担心你的家人，于心不忍阿，便破例一次，帮你一把吧。”
<br/>　　“哦？”萧逸可不觉得这个老奸巨猾的老头会这么好心。
<br/>　　果然。
<br/>　　总执事笑道，“易霄，2000任务点太多了，老夫很是心疼的。这价格嘛，自然也贵些。这样吧，十万两，老夫替你兑换下来。”
<br/>　　“十万两？”萧逸吓了一跳，随和便是满脸黑线，这老家伙，真是个贱人。
<br/>　　刚才坑了自己1500任务点，现在又狮子开大口。
<br/>　　“得，我换了。”萧逸咬了咬牙，就当敬老了。
<br/>　　“易霄，别一副很为难的样子，你不想换就算了，老夫可没勉强你。”总执事满脸贱笑。
<br/>　　“不不不。”萧逸强忍心头怒火，道，“总执事您大仁大义，易某岂会感到为难。”
<br/>　　“诶，这就对了嘛。”总执事拍了拍萧逸的肩膀。
<br/>　　这时，女子已经回来，将令牌还给萧逸，笑道，“易霄先生，这是您的令牌，已经替你办好手续了，从今日起，你便是猎妖殿的执事。”
<br/>　　萧逸咬了咬牙，接回令牌，而后拿出十万两交给总执事。
<br/>　　总执事大手一挥，笑意盈盈地将银票收下，道，“小子，等一会儿，我现在去拿猎妖令给你。”
<br/>　　走时，大摇大摆，还自顾地说道，“啧啧，今日白赚500任务点，又得十万两，赚翻了。这个世界，笨蛋还真多。”
<br/>　　萧逸气得全身发抖，要不是实力不如他，肯定上去暴揍这贱人一顿。
<br/>　　一旁的女子轻笑道，“易霄先生不要介意，总执事经常这样捉弄别人，一把年纪了，没个正经。”
<br/>　　半晌，总执事回来，将两个金色令牌交给萧逸。
<br/>　　“猎妖令，分子母两块。母令放在你自己身上，子令交于你家族重要之人佩戴。”
<br/>　　“佩戴子令者，若遭受攻击，会自动显露，警示敌人，同时感应母令。”
<br/>　　“哪怕你远在万里，也可以通过母令，感知子令的情况。”
<br/>　　“若佩戴子令者死亡，会自动断裂，距离最近的猎妖殿会立刻派遣执法队前往救援。”
<br/>　　“都明白了吗？”
<br/>　　总执事问道。
<br/>　　萧逸点点头。
<br/>　　“还有，记清楚了。若有人对付你的家族，子令显露后，他人可凭着子令的气息，到猎妖殿查询是谁的子令。”
<br/>　　“那时，你就等着你的敌人满世界追杀你吧。一旦你死了，猎妖令便自动失效了。”
<br/>　　“有劳总执事提醒。”萧逸拱拱手，道，“易某告辞。”
<br/>　　总执事笑道，“慢走，不送。”
<br/>　　萧逸转身离去，背后还传来一句，‘小子，下次需要兑换东西，再来找老夫，老夫一定帮你。”
<br/>　　萧逸一个趔趄，差点没站稳，而后加快了离开的脚步。
<br/>　　他怕自己走迟几秒，会被这个贱人气死。
<br/>　　......
<br/>　　待得萧逸离去，房间内，老者收敛了所有贱笑，意味深长地捋了捋自己的花白胡子。
<br/>　　一旁的女子轻笑道，“殿主，易霄先生被你气坏了。”
<br/>　　殿主？若是萧逸在此，一定会惊讶至极。
<br/>　　因为，整个北山郡，十大猎妖殿，从来都只有一位殿主。
<br/>　　女子继续道，“如果我刚才没有看错的话，刚才你给易霄先生的，是殿主级别的猎妖令。”
<br/>　　“需要二十万任务点才能兑换，有效时间为5年。兑换者的身份和信息列为殿内机密，除了您以外，任何人都无法查询。”
<br/>　　“呵呵。”总执事，不，老者笑笑，道，“那小子是个痛快人，却在说到他的家族和需要庇护的人时，一副愁眉苦脸、忧心忡忡的样子。”
<br/>　　“老夫不想他不痛快，便帮他这一把。”
<br/>　　女子摇摇头，回到一旁继续自己的工作。
<br/>　　这时，老者目光一凛，自语道，“易霄，看在那老伙计的份上，今日老夫便帮你一把，可不要让我失望。”
<br/>　　说罢，老者转过身，对一旁的女子道，“对了妙妙，我要回郡都了。这二十万任务点，记你家总执事头上。”
<br/>　　“阿？”女子一愣，连忙道，“殿主，这不合规矩阿。总执事要是回来了，非骂死我不可....”
<br/>　　她的话还未说完，老者已经化作一阵轻烟，消失在原地。
