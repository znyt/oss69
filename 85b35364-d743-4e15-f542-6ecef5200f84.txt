　　第691章
<br/>　　在小鹿三女眼里，程大雷眯缝着眼睛，露出几分谄媚模样。相貌自然谈不上好看，可若是定神细看，也谈不上有多丑陋。
<br/>　　面对程大雷的询问，三人答应不是，不答应也不是。这一位，虽然被视为山贼，天下多少人口诛笔伐，然而，提起他的名字谁不是又恨又怕。
<br/>　　能令天下人敬仰也是一种本事。
<br/>　　看着三人沉默的脸，程大雷并没有丝毫不满。他向来是个好脾气的人，尤其是对模样不错的女孩。此刻仔细端详着三人，小鹿是个瓜子脸的美女，两条茁壮的长腿洋溢着青春；小刀则有些婴儿肥，沉默如冰，但却稚气未退。至于小犬，则有些丰腴，是三人中发育得最好的。
<br/>　　“据说夫人为你改了名字，唤做香珠是不？”程大雷满意的点点头：“夫人还是有眼光的，珠圆玉润，也是个好名字。什么猫啊狗啊，让你父母知道估计也不会开心。”
<br/>　　小犬，或者说香珠沉默以对，确实也是不知该如何应对。她们三人都是孤儿，早已忘了生身父母，将自己当成相府的狗。
<br/>　　但在程大雷这里，她们确实得到了难得的尊重。是当成一个人的平等对待。
<br/>　　提起香珠这两个字，程大雷抽了抽鼻子，目光落在崔白玉身上。
<br/>　　据说这白玉美人，身怀异香，能将男人迷得神魂颠倒。程大雷今日方才知道，这传言不虚。
<br/>　　崔白玉依旧翻阅着手中的资料，心乱如麻，但不像表露出任何，免得被程大雷看穿任何。但程大雷色眯眯的眼神，还是被其发现了。于是内心更加慌乱，不知这恶贼会如何处置自己。
<br/>　　“喂，有没有看出什么？”程大雷打断了崔白玉的思绪。
<br/>　　崔白玉合上书卷，道：“他们在等？”
<br/>　　程大雷皱起眉头：“等什么？”
<br/>　　“等相府的意思。”
<br/>　　程大雷忽地恍然大悟，困扰他许久的难题被崔白玉一句话点破。钱通玉、白空集、孙别山这三人长久被相府控制，崔相令其往东，他们便往东；令其往西，他们便往西。
<br/>　　这次程大雷拿下凉州城，他们之所以一直没有反应，就是在等崔相大人的态度。如果崔相大人要打，那么就里应外合，攻打程大雷。若崔相大人要和，他们也不介意与程大雷保持表面上的和平。
<br/>　　“那么，以你估计，相府会有什么反应？”
<br/>　　崔白玉沉默的摇摇头。
<br/>　　程大雷板起面孔，道：“崔姑娘，我一直是很好说话的，但你不要因此把我当傻瓜。如果你希望日子好过些，最好能证明你的价值不止美貌。”
<br/>　　崔白玉开口：“不是我不肯相告，实在是我揣测不明父亲的心意。他会不会放弃凉州，我也拿不准。”
<br/>　　程大雷看其态度不像是撒谎，沉下神心里琢磨着。根据崔白玉的话判断，目前要紧的便不是钱通玉三人的态度，关键是相府的态度。至于相府会不会放弃凉州，程大雷也推算不出来。
<br/>　　崔白玉三人在等，恰恰程大雷也在等，而等就需要消耗时间。但只要给程大雷足够的时间，要不要放弃凉州，便不是相府说了算。
<br/>　　“蛮好，蛮好。”程大雷还是很满意崔白玉的表现，他目光落在小鹿三人身上，将一盘糕点推到她们三个面前：“来，尝一尝，你们当也是饿坏了吧？”
<br/>　　三人面面相觑，不知该如何是好。过了半晌，小鹿颤巍巍伸出手，摸向那盘点心。
<br/>　　程大雷脸上露出一丝笑容。
<br/>　　恰在这个时候，骤变突起，小鹿谨小慎微的眼神变得犀利。小刀不知何时绕到程大雷身后，猛地扑了过来，身子缠绕在程大雷身上。而小鹿与香珠一左一右，猛扑过去，若发怒的小猫，擒住程大雷两条胳膊，顿时将他缠得结结实实。
<br/>　　不仅崔白玉在思考脱身之策，小鹿三人也在想。这次行动，她们没有与崔白玉打招呼，完全是自己做主。按照三人的想法，其实在床第之间，等程大雷魂游天外时是最好的机会。但这次机会也不错，三人不想错过。
<br/>　　门口的小蝶发出一声惊呼，用手捂住嘴巴。
<br/>　　程大雷皱起眉头，感觉有些头疼。小鹿三人在发力，企图折断程大雷的关节，废掉程大雷一身功夫，然后以他为人质离开凉州城。这当然需要冒险，但相比噩梦般的未来，还是值得冒险的。
<br/>　　但程大雷的身体铁铸的相仿，浑身肌肉紧绷，硬得像山。三人已经使出吃奶的力气，额头渗出细密的汗珠，可还是没能撼动程大雷分毫。
<br/>　　程大雷深吸一口气，猛地发力，右手直接将缠绕在上面的小鹿甩脱。抓起背后的小刀，将她丢在墙脚。最后一把扯起香珠，手卡住她的脖子。
<br/>　　香珠呼吸越来越艰难，脸上肌肉绷紧，模样有些难看。
<br/>　　程大雷平静的看着她，道：“本当家给你做人的机会，看来你是宁愿当狗了？”
<br/>　　“放，放开她。”
<br/>　　崔白玉颤抖的声音响起，不知何时，她抽出怀里一把剪刀，用剪刀制住了小蝶。
<br/>　　这剪刀她一直随身携带，便是在等自己被逼到走投无路时，给自己再留一条路。却是没有想到，这把到会用在今日。
<br/>　　这姑娘是个用脑子的，不习惯用刀，此时手腕不停颤抖。但确实又制住了小蝶，小蝶一张脸吓得脸色苍白。
<br/>　　程大雷笑看着她，二人距离不过五步之摇。忽地，程大雷松开手，抄起桌上一杆笔丢了出去。
<br/>　　啪地一声，剪刀落地，崔白玉的身子也无力委顿在地上。小蝶惊魂未定，忙跑到程大雷身后，胆战心惊如一只小猫。
<br/>　　程大雷目光扫过四人，倒是不如何生气。而四人眼里却充满惊恐，刚才小鹿三人严密配合，已经使出了看家本领，可以说天时地利俱全。但即便是如此，也没能伤到程大雷分毫，被他轻而易举拿下。
<br/>　　程大雷摸了摸鼻子，看来这首领光环有用，但也作用有限。并不能靠此洗刷四人对自己的仇恨。
<br/>　　他迈步走过去，扳起崔白玉的下巴，居高临下的望着对方。
<br/>(本章完)
