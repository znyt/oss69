　　　　苏莫闻言，眼眸扫视四方，望着这无数的武者。
<br/>　　这些表示要加入苍穹神宫的人，人数简直不下千万。
<br/>　　这么多人，若是全部加入的话，的确是能让苍穹神宫的总体实力大增。
<br/>　　并且，能直接弥补苍穹神宫的断层。
<br/>　　目前的苍穹神宫，几乎是没有武皇强者，除了苏莫一人之外，其他人都是太弱太弱了。
<br/>　　不过，苍穹神宫或许能容纳这么多人，但苏莫可没有这么多资源供养。
<br/>　　“各位，若是今日大家能平安脱险，愿意加入我苍穹神宫的人，只要天赋尚可，能通过考核，便能成为苍穹神宫的弟子或长老！”稍一沉吟，苏莫朗声高喝。
<br/>　　“苏宫主，若是我能活命，必定去苍穹神宫！”
<br/>　　“我也是，定然去参加苍穹神宫的考核！”
<br/>　　“我们都会去的！”
<br/>　　一道道呼声响起，声浪震天。
<br/>　　这些面色均是颇为坚定，并不是敷衍的神情。
<br/>　　苏莫见此暗暗点头，苍穹神宫的确是要吸收新鲜血液了。
<br/>　　就在此刻，苏莫双眉一挑，因为他察觉到，虚界神图之外的火炼八荒大阵，已经消散了。
<br/>　　苏莫立刻神情一凝，他没有急着出去，因为他发现，虚界神图之外，是一片如混沌般的世界，充斥着滂湃到极致的水属性力量。
<br/>　　火炼八荒大阵，之所以消散，便是受到了这些水属性力量的侵蚀。
<br/>　　水火相克，这些水属性力量，天生便克制火炼八荒大阵。
<br/>　　而且，火炼八荒大阵，失去了阳鼎天的超控，威力已经大减。
<br/>　　苏莫神情高度集中，随时准备冲出去。
<br/>　　就在此刻，他感觉虚界神图，受到了一股无形的大力，被推送了出去。
<br/>　　……
<br/>　　外界。
<br/>　　半空之上，二魔老、阳鼎天等五大强者，正在与帝释和东海龙王对峙。
<br/>　　东海龙王，依旧是蛟龙之躯，横亘在虚空之中，庞大无比。
<br/>　　几人身上的气息极为恐怖，化为无形的飓风，扫荡九天。
<br/>　　二魔老五人的气势，连成一片，形成惊涛骇浪，不断的向帝释和东海龙王冲击。
<br/>　　而帝释和东海龙王，气势并不弱多少，和二魔老五人分庭抗礼。
<br/>　　几人均没有率先动手，均是想在气势上先占据上风，好一股作气，击败对方。
<br/>　　对于二魔老人等人而言，帝释不足为惧。
<br/>　　当他们忌惮的是东海龙王，这位名义上的苍穹世界第一强者。
<br/>　　轰隆隆！！
<br/>　　几人的气势不断碰撞，让得方圆数千里之内，空间不断崩碎塌陷，下方的大地不断的沉.沦崩裂。
<br/>　　上官昊和其他神宫之人，以及黄泉魔宗等势力的强者，早已退到了万里之外，根本不敢靠近。
<br/>　　“东海龙王，难道你也要与我黄泉圣宗为敌？”二魔老眸中杀机浓郁至极，直视东海龙王巨大的眼瞳。
<br/>　　昂！
<br/>　　巨大蛟龙之躯的东海龙王，闻言咆哮一声，随即大口对着二魔老猛然一张。
<br/>　　“不好！”
<br/>　　二魔老见此，心中一惊，正欲后退，便见到东海龙王并不是攻击他，而是口中吐出了一副画卷。
<br/>　　这幅画卷正是之前火炼八荒大阵中，苏莫遗留下来的空间宝器。
<br/>　　唰！
<br/>　　画卷乍一出现，一道人影便凭空出现，人影正是苏莫。
<br/>　　当虚界神图从那处水属性的空间中来到外界，苏莫便第一时间从神图空间中出来了。
<br/>　　“苏莫！”
<br/>　　二魔老等人见此，顿时心中一惊，随即几人脸色均是黑了下来。
<br/>　　他们没想到火炼八荒阵，居然这么快就被东海龙王给破除了！
<br/>　　虽然他们知道，大阵被东海龙王吞了之后，最终肯定会被破开，但也没有想到会这么快！
<br/>　　这才几个呼吸而已。
<br/>　　昂！
<br/>　　东海龙王吐出虚界神图之后，身上立刻青光闪烁，化为了人形。
<br/>　　东海龙王是一位老者，外表看起来正是花甲之年，身穿青袍，面色略显古板。
<br/>　　“苏莫，辛会！”东海龙王向苏莫微微点头，声音浑厚异常。
<br/>　　“这是……？”苏莫看了看东海龙王，又瞥了一眼旁边的帝释，一时间还不知道具体情况。
<br/>　　“苏莫，这位便是东海蛟龙一族族长，东海龙王！”
<br/>　　帝释指了指东海龙王，继续道：“龙王刚才救了你，为你破除了那火焰大阵！”
<br/>　　“哦？”苏莫闻言惊讶，原来面前这位，就是苍穹世界威名赫赫的东海龙王！
<br/>　　“多谢龙王前辈！”苏莫立刻向东海龙王抱拳道谢，今日若不是对方，那就真的有些危险了！
<br/>　　想到刚才虚界神图是从对方口中出现，苏莫心中亦是暗暗惊讶，难道之前那水属性之力磅礴无比的空间，是东海龙王的腹中！
<br/>　　东海龙王微微颌首，目光扫视二魔老等人，道：“苏莫，闲话以后再聊，我等先对付敌人吧！”
<br/>　　苏莫闻言点头，同样目光扫视二魔老人等人，随即轻笑一声，道：“诸位，很不幸，你们的算计要落空了！”
<br/>　　二魔老、阳鼎天等人，闻听苏莫之言，本已黑下来的脸色，彻底铁青了下来。
<br/>　　这一次这么好的机会，居然被东海龙王破坏了，让得苏莫丝毫无伤的脱了身。
<br/>　　不过，既然计谋没有得逞，那目前就只能正面交战了。
<br/>　　“苏莫，你得意的未免太早了，今日谁都救不了你！”二魔老声音低沉，心中杀机沸腾。
<br/>　　苏莫闻言冷笑一声，随即意念一动，海明江出现在了他的身边。
<br/>　　“四对五，谁胜谁负还不一定呢？”苏莫轻笑着说道。
<br/>　　海明江乍一出现，也是有些搞不清楚状况，但看到东海龙王和帝释，他瞬间便明白了，原来是这二人救了他们。
<br/>　　而后，海明江望向阳鼎天，大声怒吼道：“阳鼎天，我天涯海阁和你阴阳神宫不共戴天，今日我们不死不休！”
<br/>　　怒吼声一起，他便立刻出手了，手中光芒一闪，一矛向阳鼎天刺去。
<br/>　　海明江心中的杀意，早已抑制不住了，现在得以脱身，立刻便彻底炸开了。
<br/>　　当然，海明江也不是鲁莽之辈，他之所以敢率先出手，是因为己方四人的实力，完全不比对方五人差。
<br/>　　海明江可是和东海龙王相识已久，他非常清楚东海龙王的实力。
