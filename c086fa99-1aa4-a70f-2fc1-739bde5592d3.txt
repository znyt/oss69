<br/>　　这才是他的真正实力，之前他所展现出来的实力，只是其中的一小部分。
<br/>　　展现出一小部分实力的冷面魔君，都是绝对的巨头，更何况他如今实力暴涨数十倍。

<br/>　　看台之上，那通天阁老者，看到冷面魔君突然的变化，都是脸色惊变。
<br/>　　就算是他，似乎都没有想到这一幕。

<br/>　　“这下可要糟了，他们二人都是神力境的境界，这楚风眠还有着一战的资本，但是面对生死境，就算是他的剑术再强，想要逆转这数十倍的力量，也是一件难事啊。”
<br/>　　通天阁老祖，喃喃自语道。
<br/>　　而在他旁边的毒手魔君，却是冷笑连连，看向楚风眠，像是一个死人。
<br/>　　冷面魔君已经是展现了真正的实力，那楚风眠，便是只有死路一条。
<br/>　　“可惜啊，楚风眠，你嚣张一时，却没有想到吧，我的境界，早已经是度过了生死大劫，达到了生死境的地步。”

<br/>　　冷面魔君讥笑一声道。
