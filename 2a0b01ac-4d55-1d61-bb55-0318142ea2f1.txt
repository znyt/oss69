        “筱颍，快开门，我在你公寓门外，救我……我受伤了……”
<br/>
<br/>        来不及先声询问，听筒里便传来了傅昼景那气若游丝的呼救声。
<br/>
<br/>        童筱颖惊愕地瞪大了眼睛，连忙放下手中的勺子，一边举着手机，一边往家门口走去。
<br/>
<br/>        透过门上的猫眼，她确实看到了傅昼景。
<br/>
<br/>        他的脸色惨白，额头上大汗淋漓，表情痛苦地凝望着门上的猫眼。
<br/>
<br/>        童筱颖犹豫了一下，才将门给打开，谁知，傅昼景看起来体力不支，直接往她身上扑了下来。
<br/>
<br/>        “昼景？昼景？”童筱颖单手揽住傅昼景，将他搀扶着进屋后，一脚往后踢关了大门。
<br/>
<br/>        当她扶着他往沙发上趴着的时候，她才无意间发现自己手上和手臂上全是鲜血。
<br/>
<br/>        “我、我、我给你打……”
<br/>
<br/>        “不要……不要筱颍……我知道你……你这儿有创伤药。”傅昼景吃力道。
<br/>
<br/>        童筱颖慌忙跑回卧室里，拿出了药箱。
<br/>
<br/>        她顺便搬来板凳，坐在了傅昼景的身旁，拿着剪刀，剪开了傅昼景后背上伤口处的衣服。
<br/>
<br/>        “这是？这是刀伤？还是砍刀？你、你……”童筱颖有些难以置信地支吾起来。
<br/>
<br/>        傅昼景闭了闭眼。
<br/>
<br/>        童筱颖顿时揪心地接着说道：“伤口有点深，需要缝线。我这儿没有麻醉药，你忍一忍。”
<br/>
<br/>        傅昼景点了点头。
<br/>
<br/>        待童筱颖替他把伤口处理好后，傅昼景整个人都痛得虚脱了。
<br/>
<br/>        她知道他没力气说话了，所以也没有对他叽叽喳喳地问个不停。
<br/>
<br/>        或许是因为体力不支，傅昼景最后昏睡了过去。
<br/>
<br/>        再待他醒来的时候，这已经不是客厅了，也不沙发上，而是卧室的大床上。
<br/>
<br/>        他被换了一身干净的衣服，尺寸大小刚好。
<br/>
<br/>        睁开眼睛的时候，他整个人仍旧是趴着，所以脖子有些落枕。
<br/>
<br/>        傅昼景支撑起身子，背后传来一阵皮肉撕扯的痛感，让他忍不住呲牙皱眉。
<br/>
<br/>        离开卧室后，看到客厅上的壁钟，他才知道自己竟然昏睡了一天一夜。
<br/>
<br/>        此刻，童筱颖正躺在沙发上休憩，厨房里传来中药的苦味。
<br/>
<br/>        就在傅昼景准备靠近她的时候，她猛然惊醒，随手拿起沙发上的枕头指向傅昼景。
<br/>
<br/>        “筱颍，是我。”傅昼景苦笑道。
<br/>
<br/>        童筱颖定了定神，松了口气，便将手里的枕头给放下了。
<br/>
<br/>        她从沙发上下来，一边穿上拖鞋，一边说道：“我去给你把药端过来吧！我去附近的中医诊所，让医生给你开的内服和外敷的药。”
<br/>
<br/>        “筱颍，我饿了。”傅昼景的脸色仍旧有些惨白。
<br/>
<br/>        童筱颖接着说道：“那先吃饭吧！晚饭，我已经做好了。”
<br/>
<br/>        “好！”傅昼景微微一笑。
<br/>
<br/>        因为他受伤失血过多，所以，童筱颖做的菜大多是补血补气的。
<br/>
<br/>        两个人安安静静的吃饭，童筱颖并未过问傅昼景的事情。
<br/>
<br/>        “你不问我，这伤是怎么来的吗？”因为气氛太过安静，傅昼景按捺不住地主动挑起话题。
<br/>
<br/>        童筱颖抬眸看了傅昼景一眼，什么话也不说，继续往嘴里塞饭。
<br/>
<br/>        “筱颍，这段时间，我得在你这儿住到伤口愈合，可以吗？”傅昼景抿了抿唇，低沉地说道。
<br/>
<br/>        童筱颖顿了一下，二话不说地便拒绝了：“吃了这餐饭，你就赶紧走吧！”
<br/>
<br/>        “筱颍，你真的要对我这么绝情吗？除了思妍的那件事情，我并未做过任何对不起你的其他事。”傅昼景左手握成了拳。
<br/>
<br/>        童筱颖拧了下眉头，接着说道：“我现在不是对你绝情，而是在避嫌，懂吗？”
<br/>
<br/>        “避嫌？！避什么嫌？”傅昼景诧异。
<br/>
<br/>        童筱颖淡定道：“我现在是你的大嫂。”
<br/>
<br/>        “筱颍，别开玩笑了。大哥早就已经抛弃你了。”傅昼景苦笑，“我知道，你心里一定很难受。一定就像当时你等待我一样，在等待着大哥回到你身边来，对不对？”
<br/>
<br/>        “昼景，请你不要再自以为是了。我和大哥已经领证结婚了，而且还在爷爷和婆婆的见证下拜的堂。没有通知你去参加我们的婚礼，是我的主意。还有，我是为了工作方便，才搬来这里住的。等我忙完体育馆的项目，我便会出国去陪大哥。”童筱颖慢条斯理地解释。
<br/>
<br/>        以前她藏着掖着，是顾及他傅昼景的感受。
<br/>
<br/>        但是经过这段时间后，她发现他变了，变得偏激与执着。
<br/>
<br/>        而且，私生活好像也很混乱。
<br/>
<br/>        现在的傅昼景，已经完完全全不是当初她所认识的那个傅昼景了。
<br/>
<br/>        “你为什么会爱上大哥？”傅昼景眸光黯然，深沉地问道。
