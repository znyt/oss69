　　　　转眼一个星期后。
<br/>　　佘山之上，两位腰揣蛇篓，脚裹绑腿的捕蛇人一起进山捕蛇。
<br/>　　捕蛇人这个工作虽然危险，但依然是白镇人民的第一职业，纵然是很多年轻人，也会选择这份工作。
<br/>　　毕竟要是捕到一条罕见毒蛇，就是几万到十几万的RMB，绝对是高收入，只是高收入伴随的高风险。
<br/>　　如果运气逆天，捕抓到一条珍稀蛇种，那么最少都是一百万到手。
<br/>　　佘山百蛇出没，一般进入密林，就能看到一些毒蛇的身影，可今天奇怪的是，这两位捕蛇人在大山中游逛了数个小时。
<br/>　　居然都没看到一条毒蛇的影子。
<br/>　　“老王，怎么回事啊，这蛇都去那里了呀”陈师傅郁闷的开口，满头是汗。
<br/>　　王师傅看了他一眼，反问：“老陈，我还想问你呢，是不是你出门没拜蛇王大人”
<br/>　　“瞎说什么，蛇王不拜，我怎么敢进山”
<br/>　　捕蛇人都有规矩，进山捕蛇要先拜一拜蛇王，让蛇王保佑自己，能捕到一条好蛇。
<br/>　　这个规矩从古代就传下来了。
<br/>　　黑社会拜关公，捕蛇人拜蛇王，求子拜观音，各路神仙都有属于自己圈子的粉丝。
<br/>　　在佘山，几个小时都没看到一条蛇，这是非常非常罕见的状况。
<br/>　　要知道百蛇出没可不是吹牛的，尤其是当地的捕蛇人，那里有蛇，心里都跟明镜似的。
<br/>　　蛇类喜居荫蔽，潮湿，人迹罕至，杂草丛生，树木繁茂，有枮木树洞或乱石成堆的地方。
<br/>　　你在这些地方找蛇，基本上不会扑空，一找一个准。
<br/>　　忽然，陈师傅鼻子嗅了嗅，眼睛明亮起来：“我闻到了蛇的气息，很浓郁，前方一定有很多的毒蛇汇聚在一起”
<br/>　　王师傅笑了起来：“这附近都没蛇，前方蛇类气息浓郁，该不会是被我们撞见群蛇交配繁衍的事情吧”
<br/>　　现在是夏季，天气虽炎热，但也是蛇交配的好季节。
<br/>　　曾经在佘山，就出现了一次骇人听闻的群蛇交配事件。
<br/>　　方圆几十里的毒蛇都汇聚在一处阴凉的地带，群蛇熙熙攘攘，不断蠕动，首尾相连，形成了一条宽约百米，长达数公里的“群蛇阵”。
<br/>　　那个画面，简直叫人头皮发麻，看的让你怀疑人生，足够让你三天吃不下饭。
<br/>　　如果真的碰到群蛇在交配，那么两位捕蛇师傅只能打道回府了。
<br/>　　还是那句话，捕蛇人有捕蛇人的规矩，尤其是正统的捕蛇人，信仰蛇王，更是心有忌讳。
<br/>　　他们有所谓五不捕规矩，第一种是正在进食的蛇不捕，第二种是还未成年的小蛇不捕，第三种便是身上有伤的蛇不捕，第四种是非常漂亮，颜色很鲜艳的蛇不捕，第五种是正在交配的蛇不捕。
<br/>　　尤其是交配的蛇，捕抓更是大忌，它们在交配繁衍后代，你要上去打扰它们，将它们捕了，那么会非常的吸引蛇的仇恨。
<br/>　　基本上，属于找死的行为。
<br/>　　曾经就有一位捕蛇人，偶遇了几十条毒蛇在一起交配，他看到大喜过望，因为蛇交配等于任人宰割，你可以满地的抓。
<br/>　　它们也不会逃。
<br/>　　那捕蛇人将这几十条一起交配的蛇都给捕抓了。
<br/>　　等一个月之后，他在上山捕蛇，顿时周围涌现几百条的毒蛇，同时攻击，直接将他硬生生的咬死，极度残忍。
<br/>　　这叫蛇仇恨，你身上有了蛇仇恨，所到之处，只要有毒蛇，就会立刻出来攻击你。
<br/>　　上佘山就更是找死。
<br/>　　这蛇仇恨，看似危言耸听，但有真实的案件摆在你面前，你不得不信。
<br/>　　“先去看看在说”
<br/>　　两位捕蛇师傅一商量，决定先上前看看，真的附近碰到群蛇交配，那么他们就打道回府，或许是绕路，换一块区域捕蛇。
<br/>　　步行了五百米，蛇类的气息越发的浓郁，要知道捕蛇人是非常敏感的，常年跟蛇打交道，附近有蛇，用鼻子就能闻出来。
<br/>　　这也才是正经八百的捕蛇人，像一些半路出家的，就完全没这方面的专业技能了。
<br/>　　步行的陈师傅跟王师傅，现在基本确认前方就有大数量的毒蛇汇聚，否则不可能蛇类气息如此的浓郁。
<br/>　　可谓冲天而起，扑面而来。
<br/>　　是群蛇交配不会有假了。
<br/>　　除了大数量的群蛇汇聚在一起，否则万万不可能有那么强烈的蛇类气息。
<br/>　　因为蛇不是群居动物，没有社会性，只有它们在交配的时候或者冬眠的时候，才有可能汇聚在一起。
<br/>　　现在是夏季，冬眠不可能，也只能是交配了。
<br/>　　确认是确认了，但两位捕蛇师傅还是想过去看上一眼在走。
<br/>　　走着走着，前方一片灌木草丛中传出了密集的沙沙之声，一排排大树上，顿时群鸟腾飞，像是受到了什么惊吓，纷纷的逃走。
<br/>　　“有野兽？”
<br/>　　群鸟腾飞，是预兆。
<br/>　　两位都是非常有经验的捕蛇人，佘山可不单单只有蛇这一种动物，这是蛇数量品种太多，而闻名罢了。
<br/>　　忽然，两位师傅的身子猛然的僵硬起来，促而痉挛地呼了一口气，像是看到了什么不可思议的事情。
<br/>　　只见一条庞大的蛇躯在灌木草丛中露出了狰狞的一角，蛇类气息铺天盖地的散发开来。
<br/>　　他们并没有看到整条蛇，只看到水桶腰似的蛇躯，布满似火般的鳞片，可光这一幕，就让两位捕蛇师傅吓的冷汗涔涔。
<br/>　　很显然，这不是蛇，他们是遇到巨蟒了，水桶腰似的蛇躯，粗的过分，一眼就让人惊慌失措。
<br/>　　就在这时，灌木丛中却走出了一位黑袍男子，他微微抬起下巴，露出比厉鬼还要可怕的脸，面目可憎的挂着笑。
<br/>　　这种笑既阴险又恶毒，好像被厉鬼缠上了。
<br/>　　“你们是不是来捕蛇的，不好意思，我的宝贝将这附近的毒蛇都给吃光了，你们恐怕要空手而回了”
<br/>　　黑袍男子看到这位捕蛇师傅非常的高兴，又开口了：“既然都空手了，那么也就别回了，我的宝贝肚子正好也没饱，就拿你们充饥”
<br/>　　“吼...”
<br/>　　一道凶煞的咆哮声冲天而起，紧接着便是一张血盆大口狰狞的吞噬而来。
<br/>　　“啊...巨蟒，救命...”
<br/>　　这片林中传出了凄惨的叫声，可叫声就叫了那么几声，就没有下文了。
<br/>　　“哈哈...我江行泽又回到佘山了，那里跌倒，我就要在那里爬起来。”
<br/>　　黑袍男子欣赏着蛇吃人的残忍画面，随后眼睛微微一眯的大声吼出来。
<br/>　　他这眼睛一眯，却仅仅是一只右眼，因为他的左眼已瞎。
<br/>　　原本就面目可憎的江行泽，因为被蛇博士陈默的银针插入左眼，直接瞎掉，现在的样子更为的吓人。
<br/>　　让人用言语都难以形容，天底下居然有如此之丑的人。
<br/>　　被厉鬼撞见了，都要嫌弃他的丑。
