        &lt;h3&gt;第三卷第一百九十章  壁咚&lt;/h3&gt;
<br/>
<br/>        凌辰买醉了好几天，也被家里催着相亲了好多次，突然接到狸猫的信息，简直是悲从中来捂着被子放声大哭，哭着哭着又开始大笑。
<br/>
<br/>        他的雅儿没有男朋友，哈哈哈，真是太好了。
<br/>
<br/>        原来她是和胡婷他们一起出去了，凌辰急需将这个喜悦要一起分享给刘诚之，似乎他也只能分享给刘诚之。
<br/>
<br/>        “诚之，我有个好消息告诉你。”凌辰抱起电话，打了好几次刘诚之的电话，电话才接通。
<br/>
<br/>        “凌总，什么好消息？”刘诚之被他老爸提溜到安教授家陪客人，几个人正在说话，凌辰的夺命连环扣打过来了。
<br/>
<br/>        “宁雅没有男朋友，哈哈哈，她还没有男朋友，她现在在韩国，和狸猫在一起。”凌辰在电话那头开心的笑着。
<br/>
<br/>        “不是，凌总，合着你年前没有见到小雅姐姐，不但没有见到，人家还和别人一起去了韩国。”刘诚之还以为凌辰见到宁雅。
<br/>
<br/>        “你说的什么话？我去北京的时候宁雅已经去了韩国。”
<br/>
<br/>        “要是我，我就追到韩国去了，莫不是这几天你天天喝醉了哭鼻子？”刘诚之一语中的。
<br/>
<br/>        “你好像是长着千里眼。”
<br/>
<br/>        凌辰没有去想刘诚之是不是千里眼，虽然刘诚之打击了他那颗火热的心，可是他不在意：
<br/>
<br/>        “等宁雅回来，就去她家提亲，你说好不好？”
<br/>
<br/>        “好啊，去吧，准备好礼金，我等着喝你喜酒。”刘诚之看着大家看着他，想挂了电话，但是凌辰还在电话里说话。
<br/>
<br/>        “诚之，你说，我提亲请谁去？没有个合适的人怕是不行吧？”凌辰因为兴奋抱着电话就不停。
<br/>
<br/>        “我现在和董总和安教授在一起，等会回你电话好吗？”
<br/>
<br/>        “啊，亚子和你在一起？嘿嘿，你逃不出她的手掌心。”凌辰心里心花怒放，他的小雅没有男朋友，亚子也爱上了刘诚之，两全其美啊。
<br/>
<br/>        刘诚之看了一眼董亚子，她正在和安教授讨论问题，看起来眉飞色舞，小娃娃脸红扑扑的，灵动而可爱，董亚子发现刘诚之看向她，冲着刘诚之甜甜一笑，刘诚之立马收回眼神。
<br/>
<br/>        “我等会回你信息，他们在说话。”刘诚之看着大家都看着他，如果不停的聊下去不妥。
<br/>
<br/>        “好，那你帮我想一想怎么样去提亲吧，我这个都不知道该怎么办了。”
<br/>
<br/>        “好吧，我想好了给你打电话。”刘诚之感觉有些好笑，凌辰这种猴急的样子是他不曾看到过的。
<br/>
<br/>        凌辰在家里转着圈，曾经的煎熬变成了喜悦，当父亲又来安排去相亲的时候。
<br/>
<br/>        “爸，不瞒你说，我已经有女朋友了，你就别操那份心了，过几天我就提亲去。”
<br/>
<br/>        “当真？今天介绍的可是隔壁老王家的女儿，老王家女儿温柔贤淑，是持家的一把好手，你这经常不着家的，该找个这样的媳妇儿回来。”
<br/>
<br/>        “哈哈哈哈，你是找儿媳妇呢，还是找保姆？”凌辰开怀大笑，今天的心情特别好有没有？连老爸都这么可爱了。
<br/>
<br/>        这样想着，骚操作也出来了，拿起手机：“冬天很冷，但是我感觉很温暖，因为心里住着你。”
<br/>
<br/>        “没有你的日子何止是一日三秋，简直是度日如年，过了几十年，你还长在我心田，可见已经根深蒂固，与我血脉相连。”
<br/>
<br/>        “我喝着酒，不，那不是酒，那是我想你的泪，辛辣而甘甜，苦涩而温暖。”
<br/>
<br/>        “隔着山、隔着海、隔着桑田、但是隔不了我的爱意在蔓延。”
<br/>
<br/>        “我想牵着你的手，看日出看潮汐、听鸟语花香、伴夕阳西下、柴房骏马、海角天涯！”
<br/>
<br/>        “我的情话只对你说，没有你，我便才思枯竭，如同浮萍摇摆不定。”
<br/>
<br/>        “每天，我都在等，等你的早安、午安、晚安、每天我都会对你说早安、午安、晚安，只是你听不见，因为我已经在心里说了几千遍。”
<br/>
<br/>        “不敢告诉你，心里藏着你，怕一不小心暴露了自己的心迹，怕你会离去，怕嘲笑让我自卑。”
<br/>
<br/>        “努力的活着我自己，却一不小心活成了你想要的样子！”
<br/>
<br/>        他的思维就像是飞奔的骏马，弛聘在广袤的草原上，曾经压抑的情感像是爆发的火山，燃烧着他的心，既痛苦又热烈，他一条又一条的信息发给宁雅。
<br/>
<br/>        日子好漫长啊，他迫不及待的想要去韩国，想要去见他的宁雅。
<br/>
<br/>        “要不，现在定机票去韩国？”
<br/>
<br/>        “不，现在去韩国了，错过了怎么办？”
<br/>
<br/>        “不，还是在家里准备一下，等她回来的时候就去提亲，给她一个惊喜。”
<br/>
<br/>        “不行，如果她不同意怎么办？还有她心里要是没有我怎么办？”
<br/>
<br/>        “小雅，那一片彩色的云，铺开了天际的爱，在尘世之中的花仙子，可愿意牵着我的手漫步在田间街头，品人间烟火、尝世间欢乐？”凌辰一条一条的信息轰炸着，就是不去说“宁雅，我想娶你，你愿意吗？”
<br/>
<br/>        刘诚之的手机一会一个qq消息，响得他都有些不好意思了，看着安教授和董亚子，苦笑了一下：“我也不知道今天怎么回事，这么多信息，估计都是拜年的。”
<br/>
<br/>        “没事，年轻人嘛，有消息就是好事情，广交朋友，海纳百川。”安教授笑着说。
<br/>
<br/>        刘诚之实在是有些尴尬，他将手机静音了。
<br/>
<br/>        “诚之，带着亚子去我院子里走走，我和你爸爸说会话。”安教授吩咐刘诚之。
<br/>
<br/>        “诚之，亚子很少来杭州，你多带她转转，在她回去之前你就陪着她逛逛杭州。”刘爸爸也笑着说。
<br/>
<br/>        “这个董亚子活泼中透着稳重，言谈举止让人感觉风趣幽默，有女孩子的典雅，也有男孩子的思维，不错不错！”刘爸爸已经是很满意眼前的这个未来的儿媳妇了。
<br/>
<br/>        “她？她很少来杭州？”刘诚之又看着董亚子：“你怎么这么会装？让我爸认为你很少来杭州的？你是怎么认识安教授的？”
<br/>
<br/>        “我......”董亚子被刘诚之的连环问给气得满脸通红，谁说我骗你爸了？
<br/>
<br/>        “诚之，你这孩子！”刘爸爸着急的想阻拦刘诚之。
<br/>
<br/>        “走，不是没来过杭州吗？我带你逛去，不是说能来也得来，不能来也得来吗？你厉害，走吧！”刘诚之拉起董亚子的手。
<br/>
<br/>        “安教授，爸，我们逛去了啊，你们不用管我们。”董亚子被刘诚之拉着跌跌撞撞的朝外走去。
<br/>
<br/>        刘爸爸看着安教授：“这孩子.....”
<br/>
<br/>        安教授：“哈哈哈，年轻人别管他们，来，我们喝茶！”
