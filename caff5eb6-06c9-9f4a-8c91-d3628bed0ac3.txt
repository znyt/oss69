　　面对紫霄神王的胁迫，绫裳整个人怔在了原地，心中对于对方的好感，一瞬间便降到了冰点。
<br/>　　“我们走吧！”苏莫看向绫裳，声音低沉的说道，他并未发怒，因为面对神王境界的人，他连发怒的资格都没有。
<br/>　　所以，他只能让绫裳离开，既然对方不救，他就再想办法。
<br/>　　他不可能为了救芊寻月，而让绫裳受辱，或者说让绫裳牺牲清白，这绝对不可能。
<br/>　　“你若走了，此女将再无救治的希望！”紫霄大马金刀的坐在主座之上，淡淡的说道。
<br/>　　“什么意思？”苏莫阴沉着脸问道。
<br/>　　“此女时日无多，最多百年，将无法救治，那时候就算本宫主也不可能救活她！”
<br/>　　紫霄神王虽然是在告诉苏莫，目光却是望着绫裳，继续道：“而整个神域，除了本宫主之外，再也找不到第二个有能力救治她的人。”
<br/>　　苏莫闻言，心瞬间沉到了谷底，只有百年的时间了吗？
<br/>　　还是说，紫霄神王危言耸听，故意蒙骗于他？
<br/>　　“紫霄，我们相识数十万年，没想到，你却是如此卑鄙！”
<br/>　　这时，绫裳开口了，面色清冷的望着紫霄，目光之中蕴含着怒火。
<br/>　　“哎！”
<br/>　　面对绫裳的斥责，紫霄叹了口气，无奈的说道：“绫裳，你也知晓，荒界纪元终结已经不远了，为兄也没有办法；为兄已经在中级神王之境，停留了五万载，必须早日突破，在纪元终结之前，成为至强者！”
<br/>　　“成为至强者，也难逃终结的命运！”绫裳冷冷的说道，纪元终结一事，她自然知晓，只不过，纪元的终结的具体时间，没有人知晓，可能还有数十万年，也可能还有数百万年。
<br/>　　“成为至强者，终归是有希望！”
<br/>　　紫霄神王声音低沉，叹了口气，继续道：“绫裳，我知晓你心中，一直对为兄抱有爱慕之心，这也不算是强人所难吧？”
<br/>　　他并不觉得，自己是在胁迫绫裳，因为对方对他也有感情，两厢情愿之下，何来的胁迫之说？
<br/>　　“不，我以前对你的确有好感，但现在，已经荡然无存了！”
<br/>　　绫裳摇了摇头，随即，心有愤怒的大声道：“紫霄，我绫裳从未求过你什么，难道就这一件事，你都不愿意帮助吗？”
<br/>　　“爱莫能助！”
<br/>　　紫霄淡淡的摇了摇头，根本无动于衷，修为到了他这个地步，不会被情感所左右。
<br/>　　“秦陨，我们走！”
<br/>　　绫裳不再多言，招呼苏莫一声，便大步的向外走去。
<br/>　　苏莫见此，便收起棺椁，阴沉着脸，跟着绫裳离开。
<br/>　　走出大殿，苏莫便再次催动了天幻神丹，他的真实相貌，还是不能展露在外的。
<br/>　　“绫裳，为兄等待你的答复！”望着绫裳离去的背影，紫霄神王淡淡的说道。
<br/>　　他不担心对方不答应，因为他能看出，那棺椁之中的绝色女子，对于绫裳和苏莫都很重要。
<br/>　　而整个神域，目前只有他，有这个能力救活那个女人。
<br/>　　他也不着急，毕竟，绫裳现在的修为，远远没有恢复。
<br/>　　只要绫裳答应他，待到绫裳恢复到神王之境，时机就到了。
<br/>　　……
<br/>　　绫裳一路无言，俏脸寒霜，大步的返回了绫裳剑宫，回到了闭关的塔楼顶层。
<br/>　　而苏莫，也跟着绫裳，来到了此地。
<br/>　　两人均是沉默不语，心中即愤怒，又无奈。
<br/>　　“我自己再想办法吧？”
<br/>　　良久之后，苏莫看着绫裳，沉声说道。
<br/>　　事到如今，事不可为，只能再想其它办法了。
<br/>　　“苏莫，你需要多久，能超越紫霄？”绫裳突然问道。
<br/>　　“超越？”苏莫闻言，顿时眼眸一凝，略一沉吟，道：“不确定！”
<br/>　　他的确是不确定，那个紫霄神王，乃是中级神王，又是无上神王，战力比如恐怖无比。
<br/>　　而他，距离神王之境，还有非常遥远的距离，具体需要多久，他的实力才能超越紫霄，这是未知之数。
<br/>　　“我可以答应紫霄，让他救活芊寻月师尊！”绫裳声音悠悠的说道。
<br/>　　“你的意思是……先拖着？”苏莫眸光一闪，他明白了对方的意思。
<br/>　　对方是想要先答应紫霄，让紫霄救活芊寻月，然后却不与紫霄双修。
<br/>　　一直拖到他的实力，能超越紫霄为止，到时候，事情如何，就不是紫霄说的算了。
<br/>　　“不错，就算我答应他的要求，他现在也不可能立刻要求双修！”
<br/>　　绫裳眸中，闪过犀利的精光，继续道：“他必然是要等我修为恢复到神王之境。”
<br/>　　“你多久能恢复到神王之境？”苏莫急忙问道。
<br/>　　“这还不是我自己说了算？”绫裳看着苏莫反问道。
<br/>　　苏莫了然，的确是如此，绫裳完全可以延缓修为恢复的速度。
<br/>　　只不过，对方如此，就会浪费大量的时间。
<br/>　　“这是唯一的办法了！”绫裳微微叹了口气道，本来她想过，前去请求终极神王殿主，也就是他的师尊帮忙。
<br/>　　但是，她又放弃了这个念头，因为，殿主也不会干涉紫霄的想法。
<br/>　　最主要的是，转世之后的她，在殿主心中的分量，已经一落千丈，远远比不上紫霄。
<br/>　　“貌似只能这样了！”苏莫叹了口气，想要救芊寻月，只能这么做了。
<br/>　　不过，他必须要在绫裳恢复到神王境之前，先一步达到神王境，不然对方只能从了紫霄。
<br/>　　随后，苏莫和绫裳，又交流了片刻，便再次前去找紫霄。
<br/>　　不多时，两人在紫霄剑宫的大殿之中，再次见到了紫霄神王。
<br/>　　“绫裳，我就知道你会回来的！”紫霄依旧端坐在主座之上，面带和煦微笑，望着走进殿中的绫裳。
<br/>　　他知道绫裳会改变主意，只不过，他没想到会这么快。
<br/>　　“我答应你的要求，但你立刻就要施手救治。”绫裳面色清冷的说道。
<br/>　　“这是自然，只要你答应了，我必然全力相救那位女子！”紫霄微笑着说道。
<br/>　　苏莫没有开口，静静的望着紫霄，他在紫霄的笑容之中，看到了一股耐人寻味之色。
<br/>　　这让他心中微沉，担心对方会不会又耍什么手段？毕竟对方不傻，不难猜到绫裳会假意答应。
<br/>　　不过，事到如今，苏莫只希望紫霄神王能一言九鼎，救活芊寻月了。
<br/>　　“那就好！”绫裳点了点头，随即看向苏莫，示意他将棺椁取出来。
<br/>　　苏莫见此，意念一动，再次将芊寻月的棺椁取了出来。
<br/>　　而后，紫霄缓缓起身，从主座之上走了下来，来到了棺椁之前。

<br/>　　随即，他手掌平升，一股无形的力量，将棺椁之内的芊寻月牵引，缓缓的从棺椁内漂浮了起来。
