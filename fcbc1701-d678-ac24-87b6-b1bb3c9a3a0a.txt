
<br/>　　<br/>　　    一艘快艇从岸边驶来，然后快速抵达，医护人员经过简单的检查过后，发现秋雨暂时没有性命之忧，不过得马上送往医院救治，于是赶紧送上快艇离去，等到医院之后进行诊断，初步估计他是头部受到重创导致人昏迷不醒，就是不知道受伤后人是不是自主呼吸或者有没内脏受伤，这就得等到医院检查过才清楚了。<br/>　　<br/>　　    随即，没过多久剩余几名考古队员和蛙人也返回到了船上，几人都比较慌，出师不利有点太不吉利了，都是没有深潜经验的人，一个人出事其他的也毛了，再一个这次对古城外围的考察，差不多进行到了三分之二就中断了，得算是半途而废了。<br/>　　<br/>　　    人送走了后，不少人的眼光挺复杂的就落到了王惊蛰的身上，黄詹主动走过来，谨慎的问道：“王先生，你让秋雨别下水，就是知道他会在水下出事？”<br/>　　<br/>　　    “猜到了一点”王惊蛰含糊的说了一句，然后摆手说道：“至于为什么猜到，我还是那句话，我解释了你也未必能信，你就当是我感觉太灵敏了吧”<br/>　　<br/>　　    黄詹笑了笑，说道：“惊蛰，我可比你年长了差不多四五十岁，这辈子从事历史和考古方面的研究至少也有五十几年了，这么多年中我参与过的挖掘古墓工作还有历史事件的论证，我自己都数不过来有多少了，所以你别和我说什么你的话我接受不了，我见过的匪夷所思的事，可能你都未必知道的有我多”<br/>　　<br/>　　    王惊蛰沉默着点了点头，淡淡的说道：“关键是你不一定信啊”<br/>　　<br/>　　    黄詹叹了口气，很认真的说道：“这就是体制内和体制外的区别了，有些事我信不信是一回事，我认不认就是另外一回事了”<br/>　　<br/>　　    黄詹说的无非就是学术上的严谨态度，国内不提倡封建迷信和怪力乱神这种事，没有证据的东西在理论上不能承认，所以呢有些事拿到表面上，是说不了的，他的意思是，在私人角度上，他还是有着自己的看法的。<br/>　　<br/>　　    “本着实事求是的态度，不耻下问虚心求教，一直是我的人生信条，惊蛰接下来关于古城的探索，我希望你还能和我们一起，我不介意坦诚的和你道个歉，为我之前的态度和你说一声，抱歉了！”<br/>　　<br/>　　    王惊蛰其实挺佩服这位老教授的，从两人第一次见面开始到现在，对方作为国内考古和历史领域最顶尖的专家，始终都没拿有色眼镜来看他，这其实挺难得的，黄詹身上没有那种迂腐和冥顽不灵的态度，这是真真的一位科学工作者。<br/>　　<br/>　　    重伤了一个，没死人，就不算是个大事故，考察古城的工作还得继续，但第一段勘察外围的计划就得暂时搁置了，以后再补，直接就进行到下一步，准备深入古城内部了。<br/>　　<br/>　　    王惊蛰和陈三岁换上潜水服，耿明在旁边叮嘱道：“待会下水你俩跟着我，最远不能超过十米，这个距离我还有足够的机会和时间在你们出事的时候赶过去救援，还有水下环境复杂，训练是训练，实战是实战，再加上我们从来都没有进入过古城中，所以不可抗拒因素可能会出现很多，到时咱们多留意四周状况吧”<br/>　　<br/>　　    陈三岁说道：“水下沟通全靠手，哎呀，出事不出事全看命吧，真要是有问题了喊一嗓子也听不见，就说之前那家伙吧，王哥要是不事先提醒一句话的话，人直接都他么喂鱼了”<br/>　　<br/>　　    “我替秋雨谢谢你”陈三岁身后，李菲儿曲线尽显的走了过来，诚挚的朝着王惊蛰点头说道。<br/>　　<br/>　　    陈三岁尴尬一笑。<br/>　　<br/>　　    王惊蛰淡淡的说道：“尽人事听天命，你也不用谢我”<br/>　　<br/>　　    李菲儿叹了口气，说道：“是我们见识浅薄了，忠言逆耳没有听，那这次下水你还有什么意见嘛？”<br/>　　<br/>　　    “我也不是摆摊算卦看吉凶的，你问我没用，下了水我自己都不知道自己是啥状况呢”王惊蛰伸出手，比划道：“但我对你们的意见就是两字，听话！”<br/>　　<br/>　　    李菲儿被他噎的顿时没话了，挺知性的脸蛋上表情有点发愣，似乎从来没有碰到这种被人冷嘲热讽的时候，她愣了几下后说了声一会见吧，转身就回去了。<br/>　　<br/>　　    这次下水的人还是六个，除了王惊蛰，陈三岁和耿明外，考古队那边是李菲儿和文峰还有另外一人，不过配备的蛙人则是多了两个，到时在水下就单对单的负责照应了。<br/>　　<br/>　　    几个人潜水服都穿戴完毕，站在栏杆旁准备下水，黄詹皱着眉头叮嘱道：“从秋雨受伤的状况来看，古城四周的水域情况比较复杂，如果到了内部，可能就更不好说了，所以你们倒是一定加以小心，之前定制的计划可能得要稍微有点变动了，就是你们彼此之间距离不要拉的太远，一旦有事就以灯光为信号……还有，以王惊蛰为准吧”<br/>　　<br/>　　    那位研究所的张所长这回老实了，挺尴尬的杵在在一旁一个字都没吭，这就是个典型的体制官员作风，啥本事没有全都是嘴皮子上的工夫，你真要是让他下水考察，他肯定死活都不会下去的。<br/>　　<br/>　　    临近中午十二点，第二次考察继续，王惊蛰等人和几个蛙人依次进入水中，随即就快速的朝着一百六十米深的古城潜入，他们所携带的氧气，只能够支撑两个多小时，所以得尽量在有效的时间内，完成有效率的工作计划。<br/>　　<br/>　　    王惊蛰和耿明还有陈三岁，下水之后直接就头冲下，迅速下潜，后面二十米左右跟着李菲儿和文峰等人，水压逐渐增大，能见度逐渐降低，在下水十几分钟后，古滇国古城的影子渐渐的显露了出来。
