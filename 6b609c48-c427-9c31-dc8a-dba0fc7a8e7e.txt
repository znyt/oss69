<br/>    顺着悠扬的歌声，陈明可以听得到在前方有着河流被翻动的声音，定睛一看，从迷雾之中行来一艘艘船，船头挂着一盏红灯笼，而在船上，有着一群目中无神的人。
<br/>    
<br/>        他们从陈明的身旁飘摇而过，让陈明觉得说不出的诡异。
<br/>    
<br/>        这里莫非是黄泉？
<br/>    
<br/>        陈明站在木排上，让下面的野猪们继续朝着前方游去，这些野猪毕竟是凶兽，体力倍儿棒。
<br/>    
<br/>        这些幽魂似乎已经是没有了自己的意识，只是迷茫地唱着歌。
<br/>    
<br/>        走了一阵，陈明忽然看到在一艘游船之上有着一个浑身珠光宝气的富家翁，大腹便便，腰缠三串铜钱，富家翁看向陈明，目光炯炯有神，“你是来买东西的吗？”
<br/>    
<br/>        富家翁看向陈明问道。
<br/>    
<br/>        陈明疑惑不解，怎么回事？
<br/>    
<br/>        买东西，这里到底是什么地方？
<br/>    
<br/>        这个富家翁又是谁？
<br/>    
<br/>        陈明想要看清这个富家翁的实力，却是发现，对方只是普通的凡人，血肉之躯。
<br/>    
<br/>        富家翁看向陈明的眼神逐渐变得深邃，然后笑道：“原来是你啊，我差点没有认出来，你已经付过账了，你要的货，就在前方，今日起，钱货两清，你我互不相欠。”
<br/>    
<br/>        富家翁道完，已经消失在迷雾之中，让陈明摸不着头脑。
<br/>    
<br/>        我什么时候跟他做过交易，怕不是疯了，在自然自语。
<br/>    
<br/>        陈明也没有怎么在意，顺着大河向上，陈明可以看得到，在前方，有着点点晶莹的绿光，如同一颗颗绿色的星辰点缀其中。
<br/>    
<br/>        木排在小岛上停下，野猪首领第一个登上小岛，朝着远处的一棵参天大树指了指，“大仙，就是那棵树的果子！”
<br/>    
<br/>        陈明朝着那棵树走去，这棵树有着十人合围粗细，在树上，挂着一颗颗晶莹剔透，璀璨如同玉石般的清心普提。
<br/>    
<br/>        抬头望去，如同繁星点点。
<br/>    
<br/>        这个数量，应该差不多有上万颗，足够门下弟子一人一颗了。
<br/>    
<br/>        这清心普提，连猪都能上天了，给门
<br/>    
<br/>        下弟子，绝对可以将门下的弟子悟性提升到天才的地步！
<br/>    
<br/>        陈明飞身站在菩提树的枝丫上，只是站在菩提树下，就感受到整个身心一片清净，如同被洗涤过一般澄净透亮，不染凡尘。
<br/>    
<br/>        陈明清楚的感觉到，自己的思绪，从来没有像今日这般清晰，脑海中只觉得一片轻松，没有任何压力。
<br/>    
<br/>        陈明看向这菩提树，“这种宝贝，到底是怎么孕育出来的？”
<br/>    
<br/>        秉承燕山的传统，陈明连树都要了，为了避免带到燕山养不活，陈明回头站在木排上，用仙人的储物戒，直接连整个小岛都收下了！
<br/>    
<br/>        陈明看了周围一眼，百鬼夜游，“速速离开此地，回去！”
<br/>    
<br/>        在野猪的奋力之下，陈明终于是再次上岸，陈明回头看向弱水，“这一行还好有惊无险，走了。”
<br/>    
<br/>        野猪首领看向陈明，“大仙，大仙，带我一起走啊！”
<br/>    
<br/>        陈明扫了野猪一眼，然后道：“幸好你遇到的是我，即日起停止进攻玉虚宫秘境出口，好好生活在里面，我保你们无忧！”
<br/>    
<br/>        野猪首领道：“大仙，我想要出去。”
<br/>    
<br/>        陈明思考片刻，然后道：“将你耳朵里那颗菩提子吞下去，而后忘掉这件事，出去之后，听从我燕山的安排，言听计从，我可以带你们出去。”
<br/>    
<br/>        陈明可不想让这些记名弟子知道他们跟一头猪一样。
<br/>    
<br/>        野猪首领连忙掏出菩提子吞下，点头道：“多谢大仙！”
<br/>    
<br/>        带着野猪首领来到玉虚宫秘境出口处，黑虎见到陈明带着一头野猪出来，连忙上前道：“山主，里面情况怎么样了？”
<br/>    
<br/>        陈明指了指野猪首领，然后道：“这货就是兽潮的首领，从今以后，这些凶兽都是我们燕山的坐骑了，如果有凶兽不听话，你找这头猪就可以了。”
<br/>    
<br/>        黑虎有些疑惑地看向野猪首领，目光带着怀疑，却见这头野猪首领哼哼唧唧来到黑虎面前，“这位妖族老哥，交给我了，谁不听话，我削他！”
<br/>    
<br/>        就是黑虎，此刻也是愣了愣，凶兽都会说话了！
<br/>    
<br/>        陈明安排好了这里的事，直接回燕山去了，回到燕山之后，陈明将燕山山顶开拓出一片可以容纳小岛的大坑来，直接将整座小岛放置在了山顶之上。
<br/>    
<br/>        清心普提子不过刚刚落下，毕方鸟就带着小葵花，小鲤鱼，还有其他的灵魅围了过来，毕方鸟指了指清心普提树，然后看向陈明，“陈明，我觉得这棵树很亲近，我想要在这棵树上休息，它似乎可以让我长得更大！”
<br/>    
<br/>        陈明有些惊讶的看向清心普提树，还有着这个能力？
<br/>    
<br/>        能让天灵脉变得更大，长大，那是什么灵脉？
<br/>    
<br/>        毕方鸟接着道：“在山下的那个小火鸟，有了这棵树，它应该很快就会孕育出来了！”
<br/>    
<br/>        陈明笑着摸了摸毕方鸟的脑袋，不愧是我燕山的灵魅大管家，真是对其他灵魅相当的照顾！
<br/>    
<br/>        毕方鸟忽然眼中闪过一抹精光，“嗯，它已经孕育出来了，这样的小宝宝刚孕育出来都很害怕的，陈明，我们一起去安抚它吧。”
<br/>    
<br/>        陈明自然是点头，这棵清心普提树的效果简直拔群！
<br/>    
<br/>        跟着毕方鸟来到山下，陈明可以看得到毕方鸟一个闪身进入山脉之中，出来的时候，怀里已经抱着一只巴掌大小的小火鸟了，不过现在正挂在毕方鸟手中的糖葫芦上。
<br/>    
<br/>        毕方鸟道：“陈明，你快点给它种下灵契啊！”
<br/>    
<br/>        陈明欣慰地看了一眼毕方鸟，真是贤惠又贴家！手指微动，一个灵契绘制完毕，落在小火鸟的脑袋上，燕山，再多一条灵脉，燕山上，也多了一个活泼的灵魅。
<br/>    
<br/>        种下灵契之后，小火鸟果然对陈明亲近了许多，在毕方鸟的指引下，溜到燕山之上和其他的灵魅玩去了。
<br/>    
<br/>        陈明看着众多灵魅在燕山之上玩耍的样子，心里感到一阵惬意，陈明看向众多的菩提子，是时候给燕山弟子炼制一块腰牌了。

<br/>        此时此刻，燕山的弟子都是感觉到了周围的变化，他们忽然觉得自己的脑海瞬间清晰了很多。修行起来，以前的那些隔阂也在一瞬之间悟透，“原来师父说的入道境瓶颈，是这个意思，只要我将法力运行的路线稍加改变，就可以了！”
