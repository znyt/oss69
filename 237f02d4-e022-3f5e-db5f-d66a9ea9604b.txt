　　“多谢诸位，承蒙龙帝器重，我们才能脱离奴仆的身份！”
<br/>　　苏莫来到祖龙塔之前，向众人拱了拱手，客气的说道。
<br/>　　“哈哈，屠霄，你这是靠自己的天赋！”
<br/>　　一位面色略显黝黑的龙族青年朗笑一声，继续道：“以你们两人的天赋，闯到第七层没有问题，能否闯到第八层，就看运气了；你们若是真的闯到了第八层，那可是真正的震动全族了，地位绝对大涨！”
<br/>　　“呵呵，或许吧！”苏莫笑了笑，第八层就是运气了？对方未免太小看他了。
<br/>　　“祝你们成功！”黝黑的龙族青年笑着说道，当然，他也就是开玩笑，并不认为苏莫和司空炎能闯到第八层。
<br/>　　虽然在他看来，苏莫和司空炎非常不凡，但是，距离闯到第八层的天赋战力，还是有一点差距的。
<br/>　　毕竟，整个龙族上下，也就只有五人闯入过第八层。
<br/>　　“我们一定会成功的！”司空炎笑了笑，第八层而已，他有绝对的信心，至于苏莫，肯定比他厉害，就算无法闯到第十层，第九层还是没有问题的。
<br/>　　“你可真是自信！”
<br/>　　就在此刻，一声冷笑声突然响起，众人转头一看，原来是龙城大步走了过来。
<br/>　　苏莫见此，有些无语，怎么每次都能遇到这个家伙。
<br/>　　“肯定比你自信？”
<br/>　　司空炎轻笑一声，道：“怎么？你特地来观看我们闯祖龙塔？”
<br/>　　现如今，他已经脱离了奴仆的身份，也不会再给龙城好脸色了。
<br/>　　“你想多了，我不过是路过而已！”
<br/>　　龙城冷笑一声，继续道：“别以为你脱离了奴仆的身份，就能在我面前耀武扬威。”
<br/>　　“你这个……！”
<br/>　　司空炎正欲再言，苏莫却是抬手打断了他，道：“少数两句吧，准备进塔！”
<br/>　　苏莫不想让司空炎和龙城耍嘴皮子，一个无关紧要之人，根本不用在意，事实会让对方仅仅的闭嘴。
<br/>　　“嗯！”司空炎无奈的点了点头。
<br/>　　苏莫扫视着眼前的祖龙塔，望着每层塔壁上雕刻的神龙图暗，向身边的众人问道：“诸位，第九成和第十层，那是什么龙？”
<br/>　　苏莫对于龙族的种类，也算是颇为了解，这祖龙塔第一层到第八层雕刻的神龙，他全部都认识，这些神龙第一层的最弱，越往上越强大，但是五爪金龙居然只排在第八层。
<br/>　　而第九层和第十层的神龙，他完全不认识，从未见过，也从未听过。
<br/>　　第九层的神龙，身上居然长满了白色的毛发，体型不大，极为怪异。
<br/>　　而第十层的神龙，身上居然是彩色的，最惊人的是，其拥有九只龙爪，这是前所未有的九爪神龙。
<br/>　　“孤陋寡闻！”
<br/>　　龙城嗤笑一声，朗声道：“这第九层的神龙，是吞虚神龙，也就是龙帝的本体，属于特殊异龙，整个龙族只有龙帝一人，而第十层的神龙，自然是祖龙老祖！”
<br/>　　“原来如此！”苏莫闻言恍然，难怪能排在五爪金龙之上，原来是异种和祖龙。
<br/>　　不过，吞虚龙帝如此的不凡，居然也败北在第八层，那这第十层的难度可想而知了。
<br/>　　随即，苏莫不再多问，大步向祖龙塔中走去。
<br/>　　祖龙塔的入口，是一扇光华如镜的铜门，根本不用打开，苏莫的身形便穿过了同门，进入了祖龙塔之中。
<br/>　　司空炎见此，也并未多留，仅随着苏莫，进入了祖龙塔之内。
<br/>　　“这屠霄和炎空，天赋超强，不知道能闯到第几层？”
<br/>　　“第七层肯定没有问题的！”
<br/>　　“第八层的话，也有一些可能，但是可能性不大！”
<br/>　　“的确是这样！”
<br/>　　在苏莫和司空炎进入祖龙塔之后，广场上的不少龙族之人，顿时议论了起来。
<br/>　　苏莫和司空炎，这段时间名头很甚，天赋众所周知，所以很多人都好奇两人能有什么样的成绩。
<br/>　　虽然闯祖龙塔，不会有什么奖励，但是，取得的成绩越好，越是能在族中受到重视。
<br/>　　“第八层是不可能的！”龙城沉声说道，他可不相信苏莫和司空炎，能闯到第八层，这未免有些太夸张了。
<br/>　　要知道，整个龙族之中，只有五人闯到了第八层，像他这样的万古天骄，都根本闯不到第八层。
<br/>　　虽然苏莫和司空炎的天赋很逆天，超过了一般的龙族万古天骄，但在龙城看来，依旧不足以闯到第八层。
<br/>　　“他们两人，到底有多逆天，不得而知，不过闯塔的结果，要不了多久就能出来了！”有人谈论口气说道。
<br/>　　对于苏莫和司空炎的天赋，他们均是心生羡慕。
<br/>　　此事此刻，祖龙塔上，第一层到第六层的塔壁，其上雕刻的神龙图案，均是留露出淡淡的关茫，这是说明有人在其内。
<br/>　　而第七层至第十层，神龙图案暗淡无关，这是说明无人闯到那几层。
<br/>　　不过，众人相信，第七层应该要不了多久，就能亮起来，炎空和屠霄，肯定能闯到第七层。
<br/>　　祖龙塔，第一层空间之中，苏莫进来之后，立刻打量四周。
<br/>　　他发现，这里一片虚无，仿佛是浩瀚的宇宙星空，但是，没有任何的物体。
<br/>　　昂！
<br/>　　有龙吟声从前方的虚空传来，苏莫定晴一看，是一条庞大黄龙，体型数千丈，龙威浩大，气势如虹。
<br/>　　此黄龙，修为和苏莫相当，同样是三品高级的地步。
<br/>　　这条黄龙乍一出现，便飞速的向苏莫扑了过来，强大的龙威铺天盖地。

<br/>　　此龙同样体型数千丈，修为也是三品高级，气势比之之前的黄龙，更甚一倍。

<br/>　　苏莫摇了摇头，第二层的龙灵只有龙灵本身的两成实力，对于他来说，和一成实力的黄龙没有什么区别。

<br/>　　第六层，出现的是太虚雷龙。
<br/>　　此龙乍一出现，便张开巨大的龙口，吐出了一团硕大的雷球。
