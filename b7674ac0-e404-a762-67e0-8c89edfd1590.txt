<br/>　　“怎么回事？莫非是这天元神丹已经是产生灵识，想要遁逃？”
<br/>　　虚家老祖看到这一幕，脸色大变道。
<br/>　　身形一动，直接也是冲入到了虚空之中去，追了上去。
<br/>　　这突然的变化，是他完全没有想到的。
<br/>　　按理来说，这天元神丹只是刚刚炼制出来的，就算是有着强大的灵力，也是一张白纸，根本不应该懂得会反抗才对。

<br/>　　他的身形一动，遁入到虚空之中，一身灵力全部爆发出来，顿时追了上去。
<br/>　　这天元神丹可是他耗费了无数心血炼制而成的，要是今日这天元神丹自己逃了，只怕他可是会活活气死。

<br/>　　可是这天元神丹，看起来却根本不像是那种刚刚炼成的灵丹一样，不止是拥有这极为强大的力量，更是懂得连连反抗。
<br/>　　这虚家老祖，连连摄取，想要将这天元神丹抓住，最后却都是功亏一篑。
<br/>　　被这天元神丹，连连脱逃。
<br/>　　这天元神丹，仿佛是拥有这自己的意识一样，在虚空中连连穿梭，令那虚家老祖根本无法抓住。
<br/>　　“该死！这天元神丹怎么会有着如此意识？”
<br/>　　虚家老祖心中极为惶恐，距离突破，他只有一步之遥。
<br/>　　可现在这天元神丹，却是直接逃走了。
<br/>　　“老夫就不信，区区一个小小灵丹，还能逃出老夫的手掌心！”
<br/>　　虚家老祖长啸一声，顿时口中都是连连吐出了无数鲜血，一下子燃烧精血，将力量催动到极致，追逐这那天元神丹。
<br/>　　在虚空中，不知道穿行了多久，这虚家老祖的身形，终于是追了上来。
<br/>　　“天地封锁！”
<br/>　　虚家老祖大手一挥，顿时周围虚空都被封锁，这天元神丹，也是被封锁其中。
<br/>　　看到这一幕，虚家老祖的脸色终于才是平静了一些，伸出手，刚刚想要对着这天元神丹抓过去的一刻。
<br/>　　“轰隆！”
<br/>　　一声猛烈的声响，突然传遍整个虚空。
<br/>　　一只巨大无比的手臂，居然是直接突破了这天地封锁，向着那天元神丹，抓了过去。
<br/>　　这一枚天元神丹，看到这一只巨大无比的手臂，非但是没有抵抗，反而是极为欣喜，一下子被这巨大无比的手臂抓走。
<br/>　　令那虚家老祖，都是扑了个空。
<br/>　　“怎么回事！什么人！”
<br/>　　虚家老祖看着眼前的一幕，几乎是无法相信的大吼起来。
<br/>　　“是谁敢来暗算老夫！”
<br/>　　从震惊之下，更是变的无比愤怒，这天元神丹，他辛辛苦苦炼制而出的天元神丹，居然就这样被人抓走了。
<br/>　　同时这天元神丹的变化，也是让他明白过来。
<br/>　　世道如今，他也清楚，这一切，都是因为有人在捣鬼。

<br/>　　“这天元神丹，我都难以炼成，如今看来还要感谢你，让我得到了这枚天元神丹。”
<br/>　　这一道身影，正是楚风眠，刚刚楚风眠便是一路操控着天元神丹，连连躲避这虚家老祖的追击。

<br/>　　那虚家老祖，看到楚风眠的出现，眼神中尽是不可置信。

<br/>　　“楚风眠，怎么可能是你！就凭你这只蝼蚁，也可以算计我？”
<br/>　　虚家家主不可置信的喊道。
<br/>　　他知道自己被人算计了，可是他无论如何也想不到，他居然会被一个还未成就圣者的蝼蚁给算计了。
<br/>　　他可是生死境强者，御风境的武者，在他看来，都是蝼蚁，足以随意拿捏的对象。

<br/>　　“不错，当然是我，你以为你的算计，天衣无缝，难道我还看不出来？”
<br/>　　“今日这天元神丹，归我了，而且你的命，今日也要归我了！”
<br/>　　楚风眠看着虚家老祖，毫不客气的说道。
