        听季雅姗这么一说，童筱颖这才记起，自己最近最后一次见十爷，是那次在婴宁的墓前。
<br/>
<br/>        十爷对着婴宁墓碑上的那张照片，露出了深情与愧疚的目光。
<br/>
<br/>        难道……
<br/>
<br/>        “你有给他打电话吗？”童筱颖问道。
<br/>
<br/>        季雅姗摇了摇头，心情沉闷地回答：“我没有他的任何联系方式，一直都是他来找我，而对于他的行踪，我只能从他身边的手下口中得知。”
<br/>
<br/>        其实，童筱颖也发现了这点，十爷要是找上门来，完全不需要她们的任何联系方式，就能知道她们在哪里。
<br/>
<br/>        童筱颖甚至有种错觉，感觉自己和季雅姗被十爷派人监视了一样。
<br/>
<br/>        希望，这只是一种错觉吧！
<br/>
<br/>        童筱颖看着季雅姗，安慰道：“或许，这段时间，十爷很忙……就像我老公一样，忙着出差！没有时间陪自己的爱人。”
<br/>
<br/>        “爱人？！”季雅姗希冀地看了童筱颖一眼，随后这希冀的眼神又消失了。
<br/>
<br/>        “我真的是他的爱人吗？他的爱人是他的那个亡妻吧！”季雅姗黯然神伤地接着说。
<br/>
<br/>        童筱颖却忍不住笑了起来。
<br/>
<br/>        季雅姗见童筱颖在笑，瘪着嘴，抱怨道：“你居然还笑！”
<br/>
<br/>        童筱颖见季雅姗是真的不开心了，这才收敛笑容，解释道：“你还说你不喜欢十爷，我看你就是已经喜欢上他了！”
<br/>
<br/>        “我……才没有！”季雅姗又开始口是心非起来。
<br/>
<br/>        说白了，她这叫“死傲娇”，喜欢人家又悄悄不承认。
<br/>
<br/>        “十爷本来就是神龙见首不见尾这么号人物，所以，他迟早还是会再出现在你面前的！”童筱颖莞尔一笑。
<br/>
<br/>        季雅姗唉声叹气道：“但愿如此吧！”
<br/>
<br/>        “所以嘛！你开心点！”童筱颖接着安慰。
<br/>
<br/>        季雅姗又傲娇起来：“我可不是因为喜欢他才想见他的，我是因为没钱花了，相见他，问他要零花钱的！”
<br/>
<br/>        她一脸正经的样子，让童筱颖只笑不语。
<br/>
<br/>        季雅姗忽然觉得自己的心情好多了，果然，不开心的事情，向朋友说出来后，心里便舒畅了许多。
<br/>
<br/>        “筱颖，你都没跟我说过你的事情，不如，你跟我说说吧！把你不开心的事情，说给我听，我替你分担出主意！”季雅姗微笑道。
<br/>
<br/>        童筱颖一边点头一边笑着回答道：“我的事情啊！没什么好说的！”
<br/>
<br/>        “那你跟我说说，你和你老公是怎么认识的吧！”季雅姗随便找了个话题。
<br/>
<br/>        她都把她的老底都告诉她了，而她的事，她却一无所知，实在是不像一个称职的朋友。
<br/>
<br/>        一提到老公，童筱颖想起傅昼景来，便有了滔滔不绝的话题。
<br/>
<br/>        季雅姗在听完童筱颖的讲述后，非常佩服地看着童筱颖惊赞道：“你两好幸福噢！一起同甘共苦！”
<br/>
<br/>        “其实，也不算是苦吧！只是，当时住的房子小了点，生活上拮据了点！”童筱颖会意地微微一笑。
<br/>
<br/>        只要能与昼景在一起，似乎每一分每一秒，都能收获幸福。
<br/>
<br/>        “你老公，一定很爱你。”季雅姗憧憬地说。
<br/>
<br/>        童筱颖不禁惊怔地看着季雅姗，讷讷地反问：“你真觉得，他是爱我的吗？”
<br/>
<br/>        “当然，倘若你老公不爱你，那么明明知道你要替父母还债，他还有勇气娶你，还把钱都给了你！这些，都是他爱你的表现啊！”季雅姗条条是道地说。
<br/>
<br/>        但在童筱颖看来，昼景是因为人好，当时把她当朋友，才会出面帮她替她伸出援手。
<br/>
<br/>        面对昼景是否爱自己，童筱颖始终都不太自信。
<br/>
<br/>        两人聊了一会儿，在餐厅里又休息了一会儿，下午一起去了驾校一起练车。
<br/>
<br/>        下午练完车后，一辆加长型的豪车停在驾校门口很是养眼。
<br/>
<br/>        豪车的后座旁，已经有司机候在那儿拉开了车门。
<br/>
<br/>        童筱颖认得那辆车，是十爷的，而季雅姗在众多女学员们的艳羡中，和童筱颖拥抱告别，然后兴高采烈地上了那辆加长型的豪车。
<br/>
<br/>        季雅姗上车前是笑颜如花，上车后，看到车内的人，脸色顿时就变了。
<br/>
<br/>        童筱颖并不知道季雅姗上车后发生了什么，只是在车门关上的那一瞬间，她有看到季雅姗的身子一怔。
<br/>
<br/>        司机回到驾驶座上后，加长型豪车扬长而去。
<br/>
<br/>        周遭的女人们议论纷纷，有人诽谤，有人嫉妒，有人羡慕。
<br/>
<br/>        但是，听到有三个八婆一样的女人因妒生恨，辱骂季雅姗十有八九是小三的时候，童筱颖忍无可忍地走到那三个女人面前，气势汹汹地呵斥道：“那是我朋友的男朋友来接她，请你们注意一下你们自己的素质，不要吃不到葡萄说葡萄酸！”
