　　巨大光幕上，破丹、融脉、凝气泉，三个步骤，一气呵成。
<br/>　　气丹、真脉，融化为气泉空间。
<br/>　　而后，投影再次恢复原状；又再次重复这三个步骤。
<br/>　　一个时辰后，投影上，三个步骤已经重复了数十遍。
<br/>　　萧逸每分每秒都在认真看着，不敢有丝毫懈怠，务求将所有步骤熟记，了然于胸。
<br/>　　这样，轮到自己真正去突破时，才会有百分百的把握。
<br/>　　他的双眼，不敢离开光幕投影半秒，紧紧盯着。
<br/>　　没能注意到，一旁的易老，淡淡地笑着。
<br/>　　“臭小子，老夫说可以指点你，你偏不来；老夫不给你兑换洞玄突破录，你干脆自己出去历练。倒是够狂，也够傲。”
<br/>　　.....
<br/>　　一个时辰后，易老大手一挥，巨大光幕，缩小回原状。
<br/>　　萧逸一愣，显然意犹未尽，但还是马上反应过来，并对着易老鞠了一躬。
<br/>　　“谢易老指点。”萧逸感激地说道。
<br/>　　易老这随手而出的光幕投影，给了他极大的帮助。
<br/>　　详细的讲解，也为他增长了极多的知识。
<br/>　　若是单靠自己出外历练感悟，也不知要耗费多长的时间。
<br/>　　易老摆摆手，道，“说好了要指点你一天，如今不过一个时辰，还没到谢我的时候。”
<br/>　　“定。”易老喝了一声，光幕投影的画面瞬间定格。
<br/>　　定格在气泉空间出现的位置。
<br/>　　“接下来，便要给你讲解气泉的差距，以及洞玄境该如何修炼，若有不懂，可随时问我。”
<br/>　　易老话音刚落下，周围树木瞬间抖动，而后无数绿叶飞舞而来，最终凝聚成两块绿叶蒲团。
<br/>　　“好奇妙的手段。”萧逸微微一惊。
<br/>　　若是植物系武魂的武者，要操控这些树叶很简单。
<br/>　　可易老明显没有用武魂力量，而是靠体内真气。
<br/>　　也就是说，这是一种手段。
<br/>　　“坐。”易老淡淡一笑，道，“这些手段，你突破洞玄境后也一样能使用。禁锢空间，操控万物。当然，待会儿我也会给你讲解。”
<br/>　　两人在蒲团上坐下。
<br/>　　“好，言归正传，先说这气泉。长宽高各十丈，它容纳的真气，显然远远多于以前储存在气丹以及真脉中的真气。”
<br/>　　“多了百倍都不止。”萧逸惊讶道，“我感觉，到了洞玄境，已经超出武者的范畴了。”
<br/>　　“不错。”易老点头道，“你以为洞玄境武者飞天遁地的手段怎么来的，各种奇妙手段又是怎么来的，自然是靠体内庞大的真气。”
<br/>　　“而且，洞玄境之后，可称之为修士。”
<br/>　　“在普通人，乃至低级武者眼中，洞玄境，简直就是如同神仙一样的存在，无所不能。”
<br/>　　“当然了，修士同样是武者。这个世界，一切规则，一切轨迹，皆由武道而成；修炼武道之人，无论什么境界，什么实力，都称之为武者。”
<br/>　　“只不过，洞玄境太强了，乃是远远超越先天境，达到另外一种层次的境界，便被人尊称为修士。”
<br/>　　易老说着，沉声道，“洞玄境，有一种最正确的说法，名为‘入道’，只不过，我不知道你懂不懂，也很难解释，能否理解，只能靠你自己的悟性。”
<br/>　　“入道？”萧逸眼前一亮，自语道，“脱凡入道。”
<br/>　　易老满意地点点头，“不错，你这样的理解也可以，看来你的悟性远超我所料。”
<br/>　　易老忽然说道，“好了，先不说这个。洞玄境到底有多强，还是要等你真正突破，才能自身去感受，现在多说无益。”
<br/>　　“说回区别。气泉，长宽高各十丈，乃是最低的标准。”
<br/>　　“最低？”萧逸吓了一跳。
<br/>　　“嗯。”易老说道，“普遍的洞玄境武者，气泉都有数十丈，一般的天才，能达到五十丈；再厉害点的天才，可达八十丈。”
<br/>　　“若是我裂天剑派的外门弟子，最差都能达到一百丈。如今，外门中最强的弟子，恰恰达到一百五十丈。”
<br/>　　“而内门弟子，则最差都能达到二百丈，如今最强的，达到三百丈。”
<br/>　　“气泉大小，全凭自身而定。气丹的强度、真脉的强度、武魂的品阶高低。”
<br/>　　“我刚才说了，气丹真脉越强，要融化它们就需要越多的力量；自然的，凝聚的气泉会越大。”
<br/>　　易老说罢，萧逸的脸色已经彻底惊骇。
<br/>　　三百丈，这是何等惊人的概念。体内储存如此多的真气，若倾泻而出，战力该达到何等恐怖的地步。
<br/>　　“这就开始惊讶了？”易老笑笑，道，“这就是天才和庸才的区别，也是实力差距。”
<br/>　　“不过，你觉得很恐怖吗？”易老问道。
<br/>　　“这还叫不恐怖？”萧逸惊声道。
<br/>　　“笨蛋。”易老笑骂道，“别被数字吓到了。拿我裂天剑派最出色的弟子来说，三百丈，与一般天才相比，五十丈，不过多了六倍。”
<br/>　　“你觉得我们最出色的弟子，在同境界来说，实力打不过6个外头的普通天才？”
<br/>　　“额，这...”萧逸皱眉道，“易老你是在歪曲事实吧，这明显不能用单纯的算法去计算。”
<br/>　　“武者体内真气，多个几倍，实力可是提升得极多。”
<br/>　　易老笑笑，道，“我且问你，武者那小小的身体，能承受三百丈的真气同时发出？”
<br/>　　萧逸顿时恍然，“我明白了。气泉大小，虽然影响战力强弱，但程度并不大。真正的意义在于，战斗的可持续时间。”
<br/>　　三百丈真气，足够一个洞玄境武者，战斗许久，而不会导致真气枯竭，战力全失。
<br/>　　“对。”易老满意地笑了起来。
<br/>　　他没想到，萧逸的悟性竟如此高，一点就透。
<br/>　　对于一个老师来说，聪明的学生，省心不少。
<br/>　　“还是那句话，等你真正突破洞玄境，你会感受到其中的变化。”
<br/>　　易老说道。
<br/>　　“除了我刚才说的几种区别，传闻，还有一种绝世天才，气泉达到五百丈的夸张程度。当然，我没见过。”
<br/>　　“好了。”易老继续道，“气泉的区别跟你说了，接下来，便是手段的问题。”
<br/>　　“接下来要讲的，才是重中之重，小世界。你要知道，气泉，也不过是小世界中的一部分而已。”
<br/>　　“小世界，由你凝聚，你便是这个小世界的主人。”
<br/>　　“你能在小世界中无所不能，这是一份能力，换到正常的世界，这能力也一样可以使用。”
<br/>　　说到这里，易老顿了顿，停了下来，因为萧逸在思考。
<br/>　　半晌，萧逸回过神来，脸上尽是恍然之色。
<br/>　　“我在小世界中，能随心所欲，甚至调动天地灵气，禁锢一切；在外头的世界，同样可以调动天地灵气。”
<br/>　　“这就是洞玄境武者操控天地灵气，禁锢敌人的手段。”
<br/>　　“对。”易老说道，“视你小世界的强弱，你的能力就越强，在外头的正常世界，也就越加无所不能。”
<br/>　　“不过，我提醒你。小世界的强弱，只与武者本身修为境界有关。修为越高，小世界越强。”
<br/>　　“到了破玄境，你甚至能在你的小世界中凭空创造东西。当然，那是没有灵智的死物。”
<br/>　　“譬如你的三火傀儡球，那三只半步洞玄的火属性傀儡。就是破玄境的炼药师，凭空创造出来，而后禁锢到傀儡球中去的。”
<br/>　　“我明白了。”萧逸点点头。
<br/>　　“好，该说的都说了。现在只剩下最后一点，那就是洞玄境的修炼问题。”
<br/>　　“洞玄境之后的修炼，就是吸收灵气，转化为真气，直至将你的气泉填满；气泉满一成，你便是洞玄一重；满九成，就是洞玄九重；全满，即可准备突破破玄境。”
<br/>　　“同时，你的小世界，也会在你的修炼过程，不断扩展；修为境界越高，小世界会越变越大，能力也将更强。”
<br/>　　易老说罢。
<br/>　　萧逸疑惑道，“小世界会随着修为变大，那气泉呢？”
<br/>　　易老摇摇头，道，“不可。气泉，是你的根基，你突破时，是多大就是多大，那是定数。只有小世界，拥有无限的可能。”
<br/>　　“气泉，永远是那样的大小，永远处于小世界的正中央。”
<br/>　　”小世界，根据修为而逐渐扩展。“
<br/>　　.....
<br/>　　不知不觉，易老教导萧逸的时间，竟过去了半天。
<br/>　　萧逸该懂的，也都懂了。
<br/>　　这时，易老从蒲团上站起来，道，“好了，与你说了那么多，估计你头脑在发懵吧。”
<br/>　　萧逸点点头，道，“是有些发懵，不过问题不大，稍微捋一捋，很快就能通畅，完全接受。”
<br/>　　“嗯。”易老点点头，继续道，“我最后与你说一句，你便懂了。”
<br/>　　“传闻，武道的终点，便是武神之境。”
<br/>　　“武神之所以凌驾于天地规则之上，掌控世界，就是因为武神体内的小世界，与真正的世界一样大。”
<br/>　　“武神，因此而无所不能，甚至创造真正的生灵；总之，没有武神做不到的事。”
<br/>　　“我这样说，你现在可懂了？”
<br/>　　易老循循善诱着。
<br/>　　“阔然开朗了。”萧逸笑笑。
<br/>　　说着，萧逸站起身来，鞠了一躬，道，“谢易老教导，小子感激不尽。”
<br/>　　易老摆摆手，道，“不急，现在才半天，还有半天时间。”
<br/>　　“哦？”萧逸面露疑惑之色。
<br/>　　“笨蛋。”易老铁着脸骂道，“你那控火兽武魂，乃是最低级，最无用的东西；你何时才能吸收够足够的天地灵气去破丹融脉。”
<br/>　　“还有，以你的情况，日后就算凝聚气泉和小世界，虽然也不会大到哪里去。”
<br/>　　“但，以你的控火兽修炼速度，一辈子都别想突破到洞玄一重，更别说填满气泉了。”
<br/>　　“你只有一条路可走，炼药。剩下的半天，便教你炼药了。”
<br/>　　说着，易老白了萧逸一眼道，“我可说明白了，小子，你现在才知道我厉害，后悔也没用。”
<br/>　　“我说了，只指点你一天。能学到我多少本事，就看你自己的了。”
<br/>　　“要是你连一品炼药师都达不到，也只能怪你自己之前不把握机会找我指点。”
<br/>(本章完)
