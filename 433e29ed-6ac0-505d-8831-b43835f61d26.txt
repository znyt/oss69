　　以萧逸的眼力，自然一眼看出这五个武者的修为。
<br/>　　两个先天六重，三个先天四重。
<br/>　　他懒得浪费时间，手中凝聚的火焰乃是全力一击。
<br/>　　然而，另一边毒龙的眼力也不差，一眼就看出了萧逸手中凝聚的火焰威力，顿时出手，并大喝了一声，“快退，那小子起码是先天八重的实力。”
<br/>　　他想让手下退，但终究慢了一些。
<br/>　　萧逸已经全力一掌打出。
<br/>　　那带着炽热高温的火焰瞬间席卷这五人。
<br/>　　轰的一声爆炸，三个先天四重被秒杀，两个先天六重也顷刻间重伤，吐血倒地。
<br/>　　“混蛋，我要你死。”毒龙怒不可遏。
<br/>　　那五个先天武者，可是他最得力的手下。一下子就被秒杀了三个，剩下的两个也是半死不活，这对于毒龙山贼团来说，无异于元气大伤。
<br/>　　但说到底，是他自己小看了萧逸。
<br/>　　在他看来，一个少年罢了，就算天赋再高，顶天了也就先天三、四重左右的实力，他的手下足以制服。
<br/>　　可他万万没想到萧逸竟然强到这个地步，而且在他反应过来前已经将他的手下重创。
<br/>　　这时，他已经化作一条巨大的黑蟒，全力朝萧逸攻去。
<br/>　　萧逸脚步一动，轻松躲开，淡淡道，“原来是七纹黑蟒，呵，小小毒蛇也敢妄称毒龙。”
<br/>　　七纹黑蟒武魂，黄阶兽武魂，体型巨大，力量不凡；蛇身上有七道黑纹，蕴含剧毒。
<br/>　　“死到临头，还敢大言不惭。”毒龙冷声说道。
<br/>　　话音刚落，四周同时冒起了黑烟。黑烟扩散的速度极快，不出片刻便将方圆数百米笼罩起来。
<br/>　　“黑纹毒雾。”萧逸眉头一皱，立刻警惕起来。
<br/>　　黑雾遮盖了视线，让人如同置身黑夜之中。
<br/>　　毒龙的身影已经消失，萧逸甚至看不清周围数米之外的情况。
<br/>　　“怎么回事，区区先天九重，不可能在这么短的时间内将毒雾扩散到这个程度。”萧逸微微一惊。
<br/>　　黑纹毒雾带有毒性和腐蚀性，将他的真气外衣腐蚀得劈啪作响。
<br/>　　“哈哈哈哈。”
<br/>　　这时，一片漆黑中，传来毒龙的大笑声。
<br/>　　“小子，这里可是我的地盘，我怎么可能没有后手。”
<br/>　　“我早已将我的毒雾事先用玉瓶装好，藏在四周。只要我真气发动，便可瞬间引爆这些玉瓶。不出数秒，就能将这里变成一块剧毒之地。”
<br/>　　“哈哈哈哈，如今你且看我到底是小小毒蛇还是毒龙。”
<br/>　　“我的黑纹毒雾，先天五重之下，不出半个时辰便会抵挡不住，剧毒攻心而死；先天八重之下，头晕目眩，实力大降，最终也只是成为我刀下亡魂罢了。”
<br/>　　萧逸暗骂一声狡猾，但却没有害怕。
<br/>　　如今他的实力，单靠真气已有先天六重的战力，使用升龙后，更是不亚于先天八重。
<br/>　　若加上武技，要战胜毒龙并非难事。
<br/>　　毕竟，若武者修为相聚不大，有武技的一方必然会占优势，若是强大的武技，甚至可以做到同修为秒杀。
<br/>　　至于烈焰手套，对付一个普通的先天九重，还无需用到。
<br/>　　以这毒龙的实力，倒是可以让他练练手。
<br/>　　这时，毒龙愤怒无比的声音再次传来。
<br/>　　“小子，本来我要杀你，轻而易举。但，你竟将我的地盘毁成这番不堪模样，更杀我数名得力手下。若不让你凄惨死去，难消我心头之恨。”
<br/>　　“待得你真气外衣破碎，毒雾加身之时，你会后悔当初我没有一刀杀了你，哈哈哈哈。”
<br/>　　毒雾之中，毒龙藏身于萧逸数十米之外，猖狂大笑着。
<br/>　　他要让萧逸绝望而死，唯有如此，方能稍稍让他压抑怒火。
<br/>　　然而，下一秒，一道恐怖的火刃忽然从他背后劈出。
<br/>　　同一时间，一声冰冷的呢喃在他耳边响过，如鬼魅之音，又如深渊寒冰，让人不寒而栗。
<br/>　　“噬火百刃。”
<br/>　　“嘭”的一声，火刃直接将他的后背炸得皮开肉绽，而后整个人被轰飞十数米。
<br/>　　“噗。”当他站稳身子后，猛地吐出一口鲜血，忍着后背的疼痛，不可置信地看着后方原本自己的位置。
<br/>　　“怎...怎么可能...”
<br/>　　不知何时起，萧逸竟已经站到了他原本的位置，刚才更是一招将他重创。
<br/>　　“我的黑纹毒雾应该让你眼不能视才对，加之这片黑雾笼罩的范围如此之广，你怎么可能找到我？”
<br/>　　毒龙满脸不可置信。
<br/>　　“你不觉得你的话太多了吗？”萧逸冷冷地说道。
<br/>　　“你寻着我的声音找到我？”毒龙一惊，道，“不可能，这片黑雾在我掌控之中，我能察觉你的一举一动，你不可能无声无息地出现在我身后，更将我重创。”
<br/>　　这时，毒龙似是想到了什么，顿时脸色一变，惊声道，“难道你不单止是猎妖师，还是一个擅长隐匿的杀手？”
<br/>　　“不...不可能。”毒龙又推翻了自己的想法，“杀手我见多了，不可能如你这般有如此高超的隐匿本事。”
<br/>　　破玄城的那些家族和商贾，除了在猎妖殿中发布悬赏任务外，也另外请过不少杀手，打算暗杀毒龙。
<br/>　　可是，这些杀手无一例外都死在了毒龙手上。
<br/>　　所以毒龙对杀手很熟悉，也丝毫不惧。
<br/>　　但，如今站在他面前的，乃是曾经纵横地球的杀手之王，萧逸。
<br/>　　在萧逸眼中，这个世界的杀手，毫无技术可言。像之前的杨伏，除了一身武者修为外，暗杀技巧、藏匿功夫等等，简直只能用拙劣去形容。
<br/>　　两者的差距，天差地别。
<br/>　　区区黑雾，不单止没能阻碍他，反倒是给了他一层更好的掩饰‘外衣’，更简单地接近毒龙。
<br/>　　“没什么不可能的。”萧逸冷冷说道，“毒龙，若你只有这点本事，那么，你可以死了。”
<br/>　　“小瞧不起人了。”毒龙勃然大怒。
<br/>　　“小子，是你逼我的，待我将你一块块皮肉削下，看你还能否这般嚣张。”
<br/>　　话音刚落，毒龙的身影再次消失。
<br/>　　嗖，忽然一道剑吟声响过。
<br/>　　嘶的一声，萧逸还未反应过来，手臂上已经多了一条血痕，鲜血汩汩之流。
<br/>　　“是风？”萧逸眉头一皱，刚才好像是一阵风将自己刮伤了。
<br/>　　嗖嗖嗖....
<br/>　　转瞬间，萧逸身上再次多出了几道血痕。
<br/>　　“不对，是剑，好诡异的攻击，轻盈如风，又锐利无比。”萧逸猛地一惊。
<br/>　　萧逸暗暗想道，没想到毒龙还有这种底牌，看来不能藏拙了。
<br/>　　这时，又是一声剑吟。
<br/>　　但，却没能伤到萧逸。
<br/>　　因为，此刻的萧逸，身上竟出现了一团紫炎。
<br/>　　紫炎仿佛能焚烧万物，竟连飘荡着的黑雾也瞬间被点燃。
<br/>　　在紫炎面前，浓郁的黑纹毒雾，竟毫无反抗之力。
<br/>　　空气中响起一阵噼噼啪啪之声，不出片刻，方圆数百米的黑雾被焚烧一空。
<br/>　　这时，毒龙的身影显露了出来，身上拿着一把剑，满脸惊骇之色。
<br/>　　“什么东西，竟然能焚尽我的黑纹毒雾？”
<br/>　　萧逸没有回答，而是打出了一道噬火百刃。
<br/>　　只不过，这道噬火百刃由紫炎而成。
<br/>　　毒龙一惊，连忙提剑抵挡。
<br/>　　不愧是先天九重的修为，一剑就将噬火百刃击碎。
<br/>　　只是，噬火百刃被击碎后，紫炎却没有消失，溅射到他的剑上以及身上。
<br/>　　嘶....
<br/>　　紫晶灵炎，在焚尽物体前都不会消失。
<br/>　　不出数秒，整把剑连带剑身剑柄被烧成了灰烬，而后紫炎消失。
<br/>　　而毒龙身上的紫炎，也是快速焚烧着毒龙，吹不熄，掐不灭，撇不掉，诡异至极，且又强大至极。
<br/>　　毒龙看了眼已成灰烬的剑，再看了眼身上将自己灼烧得疼痛无比的紫炎，面露恐慌之色。
<br/>　　“喝。”毒龙浑身真气澎湃而出，这才堪堪将紫炎消灭。
<br/>　　紫炎是萧逸用真气凝聚出来的，自然也被毒龙用真气抵消了。
<br/>　　“小子，你...你到底是什么人...”毒龙惊骇地看着萧逸，此刻的他，根本不觉得自己有能力赢萧逸。
<br/>　　“记好了，我是猎妖师，易霄。别到了黄泉之下，死于谁人手中都不知道。”
<br/>　　萧逸说罢，十数道噬火百刃同时打出，全是以紫炎凝聚。
<br/>　　毒龙拼死一搏，挥手抵挡，将火刃全部挡下。
<br/>　　只是，破碎后的紫炎还是溅射到他身上。
<br/>　　十数道火刃破碎后的紫炎，顷刻间便吞没了他的身影。
