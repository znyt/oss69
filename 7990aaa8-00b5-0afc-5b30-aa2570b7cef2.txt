　　佟千雪现在的感觉很糟糕，非常糟糕。
<br/>　　她只觉得自己腹部一阵阵的刺痛，好像有什么东西正在从自己身体中离开一样。
<br/>　　佟千雪双手捂着自己的腹部，想要阻止它从自己身体中离开，阻止自己失去它；佟千雪用自己微弱的嘶哑，呼救着，她奢望着外面的人能进来帮助她。
<br/>　　“送我去医院，送我去医院……”
<br/>　　微弱的求救声，从屋子的木门背后传出来，声音尽管十分微弱，可是布克尔特还是听到了。
<br/>　　布克尔特跨上台阶，走到门口，看到倒在地上的佟千雪，还有她身下那滩刺眼的血迹，脸色立刻变得非常难看。抬腿，直接一脚踹在门上，那扇木门受不了冲击，直接是被布克尔特一脚踹开。
<br/>　　“olivia！”
<br/>　　布克尔特对着跟在自己身后的一个武装毒贩招呼了一句，立刻有一个脸上涂着伪装迷彩，穿着老式四色丛林迷彩服的白人走了出来。
<br/>　　olivia退役之前是个军医，他能在战场上给人完成截肢手术，动脉缝合。在退役后，olivia在一家私人诊所里给一个医术平庸的医生当助手，直到西伦科特找到他，并且把他送到布克尔特身边。
<br/>　　西伦科特不希望布克尔特身边的人只会战斗，除了学会使用轻武器外，西伦科特还要他们掌握更多的东西，比如战场救治。
<br/>　　olivia虽然是个外科医生，可他还是一眼就看出了佟千雪的情况。olivia抬起头，看着布克尔特说道：“她怀孕了，但是现在情况很糟糕，如果不及时救治的话，她很有可能会没命。”
<br/>　　“救活她！”
<br/>　　布克尔特伸手按在olivia肩膀上，说道：“一定要救活她，她如果死了，我们所做的一切都没有了任何意义，而且我们还会遭到肆无忌惮的报复，他们将不会有任何的忌惮，会玩了命的追杀我们！”
<br/>　　olivia点了点头，将佟千雪从地上抱了起来，对普提查说道：“我需要一个干净的房间，并且把你村子里懂医术的人都找来，最好是女的。”
<br/>　　普提查对着阿卡盼使了个眼色，阿卡盼立刻带着olivia朝着村子中的诊所走去。
<br/>　　普提查的人有时候也会跟T国政府，或者是其他势力的人发生冲突，所以在渔村里设立了一个诊所，里面储备了药品和医疗器械，渔村里也有一个会医术的女人。
<br/>　　这个女人是乌克兰人，是被普提查的人抓来这里的，因为她会医术，所以她活了下来，而不是被普提查和他的那些手下玩弄致死。
<br/>　　布克尔特转身看着普提查，冷声问道：“普提查，你不觉得你应该给我一个合理的解释吗？”
<br/>　　如果不是跟着布克尔特过来，普提查也不知道发生在这里的事情，他冷眼看着阿扎普和另外一个负责看守的毒贩，让他们两个人给自己一个解释，也给布克尔特一个解释。
<br/>　　“这个女人想要逃跑，所以我进去教训了她一下，可是我并不知道她怀孕了，并不知道会发生这样的事情。”阿扎普低着头，说道。
<br/>　　阿扎普很清楚，事情到了这个时候，想要抵赖是不可能的，因为负责在这里看守的就他们两个人，现在佟千雪发生这样的事情，要说是佟千雪自己在屋子摔倒造成的，这样弱到爆的借口就算是一个傻子都不可能会相信。
<br/>　　只不过，阿扎普的解释实在太苍白，太无力了，刚才那样的情况，就算是阿扎普把嘴皮子都磨破，也很难让布克尔特相信他的鬼话。
<br/>　　“你不觉得你的这个解释太低级了吗？”
<br/>　　布克尔特盯着阿扎普看了一会儿，说道：“你这样的谎言别说是我，就算是一个刚上小学的小学生也很难相信你刚才的话。”
<br/>　　布克尔特迈步走到两条腿已经在不停打颤的阿扎普面前，说道：“你知道我最恨什么吗？强奸！我不介意我的手下去玩女人，但是我绝对不会纵容他们去强奸，任何违反这个规矩的人全都已经死了，所以你……”
<br/>　　阿扎普这个时候如何还能不知道布克尔特是要对自己动手了，他想要跑，立刻就跑，跑得越远越好，可是偏偏在这个时候他的双脚就像是被灌了铅一样，根本就挪动不了，别说是跑了，就连脚都抬不起来。
<br/>　　布克尔特的手指在他挂在腰间的那把爪刀刀柄上轻轻敲了两下，食指突然勾住了爪刀刀柄末端的指环，勾住爪刀的瞬间，将刀拉出刀鞘，只是一个抬手的动手，锋利的刀刃就在阿扎普脖子上留下了一道血痕，布克尔特手中爪刀刀尖上，有着血滴滴落。
<br/>　　滴答！
<br/>　　第一滴血滴滴落的时候，阿扎普脖子上那道血痕被想要从他体内喷涌而出的鲜血给撑开。
<br/>　　滴答！
<br/>　　第二滴血滴滴落的时候，大量的血液已经是从阿扎普脖子上喷出，“沙沙沙”，血液喷出的声音，就像是秋风吹过麦田，麦穗轻轻摇晃时所发出的声响。
<br/>　　阿扎普捂着脖子倒在地上，他眼睛睁得滚圆，他身体因为失血严重而发生痉挛、抽搐，很快，倒在地上的阿扎普就没有了任何的动静，血液依旧从他的脖子中往外涌出，他的身体下那一大滩血迹，将佟千雪之前留在地上的血迹给淹没了……
<br/>　　不到一秒钟的拔刀动作，终结了阿扎普的性命。
<br/>　　站在门口的另外一个毒贩看到发生在屋子里的一幕，吓得直接尿了裤子，但是很快他就从恐惧中反应过来，一瞬间爆发出了自己全身的力量，飞快的朝着海滩的方向跑去。
<br/>　　虽然佟千雪的事情中，他没有参与进去，但是不可否认，他跟这件事情有不可推卸的关系，阿扎普死了，自己会是下一个，他不认为布克尔特会大发慈悲的放过自己。
<br/>　　“Boss，他跑了，需要我留下他吗？”
<br/>　　渔村西北方向的丛林中，潜伏在丛林中的狙击手盯着TAC-50狙击步枪上的16被瞄准镜，通过耳麦向布克尔特做出请示，只要得到布克尔特的指令，狙击手就会毫不犹豫的扣动扳机，然后看着狙击镜中正在逃跑的毒贩被撕裂成两截。
