　　国王将王冠递给楚涵的一幕吓得一干人差点跪下，但楚涵只是取走了其上的蓝色宝石就将王冠抵还，又让一干人连忙闭上嘴。
<br/>　　只是没想到国王又将王冠递了过去：“王冠给你，留作纪念吧！也是我所能拿出最珍贵的东西！”
<br/>　　一干人急的跳脚，几个大臣已经快速的想好陈词准备上前说话。
<br/>　　楚涵想了想，没有推脱的收下：“也对，天运国的国运不在什么王冠，而在与坐在王位上的人。”
<br/>　　一席话说的极其有水平，也终于让国王扬起笑脸，很自信的笑容。
<br/>　　刚想说话的几个大臣连忙退了回来，这话堵的他们无话可说。
<br/>　　真是一波三折！
<br/>　　最终楚涵还是走了，在王宫一处幽闭的地方走的，只不过临行前国王给他送行，又哭的稀里哗啦，就差抓·住他的裤脚不让他走了。
<br/>　　对此楚涵也很无奈，他对国王的感情挺特别，亦徒亦友。
<br/>　　“我会回来看你。”楚涵留下这句话，便跟随着考核石壁的提示音消失。
<br/>　　反正天运国已经出现在了回炉系统上，只要积分够，楚涵随时能开启时空之门过来。
<br/>　　国王呆愣在原地良久，将这句话记得尤其牢。
<br/>　　……
<br/>　　这一考核到异度空间点的一行，让楚涵身心俱疲，等到回归考核石壁的黑暗时，才感受到真正的真实。
<br/>　　只是很奇怪，楚涵被弹出考核石壁的地方，并不是阴阳谷，而是在一片草原，四周也不见考核石壁。
<br/>　　回炉碎片已经被他用最快的时间用在回炉系统的修复上，当下的回炉系统，也终于修复到了100%！
<br/>　　苏醒的旺财记忆也恢复到了70%，似乎是长大了一些，话变少了。
<br/>　　楚涵看了眼天空，略一思索后便朝着一个方向疾行而去。
<br/>　　他差不多是在中原地带，鬼知道为什么会出现在这里！
<br/>　　也不知道白允儿什么时候能到……
<br/>　　……
<br/>　　阴阳谷内，阳谷里的冰湖已经不是黑夜，实际上在楚涵的考核结束后，考核石壁上就不再放送其考核画面，也就是之后的异度空间点情况，观看的人都无从得知。
<br/>　　只是他们还围聚在这考核石壁处没有离开，直到楚涵离开了异度空间点，十阶考核石壁上出现了楚涵的排行与名字，不少人才彻底放心。
<br/>　　同样在华夏的各个地方，很多人都第一时间注意到了考核石壁上的名词变动，楚涵这个名字就像是神明一样，每一次都在最耀眼的地方。
<br/>　　各个城市的丧尸清扫已经步入尾声，大城市都清理的干净，只剩下一些小城市还未有着落。
<br/>　　这事也不是一朝一夕就能完成，总要慢慢来。
<br/>　　文明也在渐渐恢复，不少艺高胆大的人，就这么进入往日的城市中，尝试居住了起来。
<br/>　　各大基地的发展欣欣向荣，民生已经走上正轨，流民几乎不存在了。
<br/>　　狼牙又一次壮大，狼牙基地也成为了基地界的标杆，只要是狼牙推行的政策，其他基地都纷纷效仿。
<br/>　　步纱作为孕妇，在狼牙地界生活的很好，只是很奇怪的是，都好几个月了她依然只是小腹微微·隆·起，根本看不出来已经怀·孕。
<br/>　　对此研究部在结束了一轮基因研究后给出了一番理论，大致意思就是当下的新人类怀·孕，周期会根据孩子父母的基因变动。
<br/>　　非新人类的正常人周期很短，可能不到五个月就会出生。
<br/>　　而新人类，等阶越高，周期越长，长到不知道会怀·孕多久。
<br/>　　步纱此时已经十阶，楚涵当初让步纱怀·孕时等阶也不低，这孩子还真一时半会儿生不下来！
<br/>　　于是步纱也就安了心，闲来无事还会帮尚九娣管理一下城市。
<br/>　　……
<br/>　　楚涵在疾走了几天几夜后，没着急去阴阳谷，而是回到了狼牙地界。
<br/>　　这是他最熟悉的地方，也是狼牙起步的地方，承载着太多回忆，两大城市更是全部依照了他当初的蓝图在发展，发展到空前盛世的地步。
<br/>　　闻着这里的空气，楚涵都心情好！
<br/>　　只是好巧不巧，回到狼牙的时候又是夜晚，尚九娣又在加班。
<br/>　　楚涵不习惯大张旗鼓，没通知任何人摸索到了尚九娣的办公室窗外，窗外的那棵树也不知道什么原因，竟然没被砍。
<br/>　　于是楚涵和旺财就这么坐在树杈上，又干起了偷·窥的事。
<br/>　　“我说，第几次？你幼不幼稚！”旺财不忍直视。
<br/>　　楚涵瞥了它一眼，不屑道：“你不是人，你懂个屁！”
<br/>　　偷偷摸·摸的才刺激啊！
<br/>　　尚九娣加班似乎累了，捂住嘴打了个哈欠。
<br/>　　楚涵一看时机成熟，便轻轻一跃，悄然无息的就从窗户里钻了进去。
<br/>　　太快也太轻，没让任何人发现，只有对面坐在办公桌后面的尚九娣瞪大了眼睛。
<br/>　　似乎也是三番两次的经历这种事，让她没再吃惊叫出声，而是很快适应了过来，‘噗嗤’一笑。
<br/>　　紧接着，尚九娣起身将窗户关上：“你怎么总是这么进来？”
<br/>　　楚涵满脸笑容，轻轻将尚九娣环抱住：“我说，咱俩的婚事是不是推迟太久了？”
<br/>　　尚九娣一挑眉：“对，还有答应了上官雨馨的婚事，你聘礼都没送呢！还有步纱，一个名分都没有吗？”
<br/>　　至于白允儿，尚九娣没说，这个名字已经在狼牙成了禁忌。
<br/>　　楚涵笑了笑，道：“等白允儿回来，就大婚！”
<br/>　　四个老婆，想想都爽歪歪。
<br/>　　尚九娣瞳孔一缩，她认真的看了眼楚涵，确认对方不是在说胡话后，心跳顿时漏跳了半拍。
<br/>　　白允儿的名字早就不在考核石壁上，这种情况只有可能是死亡，楚涵怎么会……
<br/>　　尚九娣担心被楚涵看在眼里，他没多说，也解释不清。
<br/>　　太空啊飞船啊这些东西他自己刚看到的时候都差点没震惊到怀疑人生，更何况压根没接触过这些的尚九娣？
<br/>　　尚九娣眼神闪烁了一下，连忙转移话题：“对了，你爸的情况有所好转，应该很快就会苏醒！”
<br/>　　听到这样的好消息，楚涵也眼神亮了起来，他在末世纪元拼命的最初目的就是一家人团聚，想必他老妈袁熙叶肯定高兴坏了！
