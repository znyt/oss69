　　“你也应该知道这是不可能的。”
<br/>　　莱昂摊手，说道：“我们只有四个人，而随后巴坦伦派来的人就不会再只是8个，可能是80个，甚至是更多，你觉得我们4个人能对抗那么多人吗？”
<br/>　　“可是你们不留下来，这里这些孩子，还有修女和教父都有可能会死！”塔尼特冲着莱昂喊道。
<br/>　　“我们留下来，他们也一样有可能会死！不仅他们会死，我们也有可能会死！”莱昂冲着塔尼特说道，语气也比之前要严厉了一些。
<br/>　　“你们难道就没有一点同情心吗？”塔尼特质问道。
<br/>　　“同情心？”莱昂冷笑一声，说道：“我是雇佣兵，不是上帝派来拯救你们的天使，虽然我不愿意对女人使用粗暴的手段，但是如果你执意要留在这里，或者是要带走这里的其他人，我不介意对你使用一些暴力手段。”
<br/>　　“你！”
<br/>　　塔尼特狠狠地瞪着莱昂，恨不得用眼神在他身上戳出两个洞来。
<br/>　　“你如果被巴坦伦带走，所有的一切就全都结束了。”莱昂看着塔尼特说道：“这个国家将没有人可以对抗巴坦伦，你会死，你的父亲也会死，而这个国家的cocaine种植、加工和销售就会变得合法化，你应该很清楚这意味着什么，这里将除了cocaine外，再也没有其他农作物，这个教堂里的修女和教父要么为布克尔特种植cocaine，要么被杀死！至于这些孩子，会变成娃娃兵，成为巴坦伦手中的杀人工具。”
<br/>　　塔尼特沉默地看着莱昂，她知道，莱昂说得是实情，巴坦伦担任这个国家的总统，莱昂刚才所说得一切都会真实的发生。
<br/>　　“好吧，我跟你们走，我也会帮你们联系到我的父亲。”
<br/>　　塔尼特终于还是向莱昂做出了妥协。
<br/>　　“塔尼特，你的决定是正确的。”
<br/>　　这所教堂的教父走向塔尼特，满是皱纹的脸上带着慈祥看着塔尼特，说道：“我们这些人死了没有关系，但是这个国家不能落入到巴坦伦手中，不然将会有更多的人死去。”
<br/>　　“杰克教父……”
<br/>　　塔尼特眼中带着泪花，她知道，杰克教父出来说出这番话，这是他和其他修女已经做出了决定。
<br/>　　“年轻人。”杰克教父看向莱昂，说道：“我有一个要求，请你们把这些孩子带走。”
<br/>　　莱昂脸上现出为难之色，但最后还是答应了杰克教父的请求。
<br/>　　“谢谢。”
<br/>　　杰克教父摘下挂在自己脖子上的纯银十字架，挂在莱昂的脖子上，在他额头上轻吻了一下，说道：“上帝会保佑你。”
<br/>　　“杰克教父！”
<br/>　　“杰克教父！”
<br/>　　周围那些孩子围住了杰克教父，他们已经知道，这次跟杰克教父再见，将会是永别，他们将再也不会见到慈祥的杰克教父。
<br/>　　“去吧。”杰克教父对着塔尼特挥了挥手，说道：“照顾好这些孩子。”
<br/>　　时间有限，所以塔尼特和那些孩子并没有携带什么行李，只身跟随莱昂离开了教堂。还好这次过来的时候开的是一辆皮卡车，不然要把这些孩子全部带走真不是一件容易的事情。
<br/>　　罗昊、齐凡、莱昂、布谷带着塔尼特和那些孩子离开教堂不久，巴坦伦的警卫队长就带着人来到了教堂。
<br/>　　正如莱昂所预料的那样，这次来的人远远超过了之前，足有一百多人！除了AK自动步枪外，他们还携带了火箭筒这样的重火力武器，冲进教堂的吉普车上，架设着.50口径的勃朗宁重机枪。
<br/>　　在塔尼特和那些孩子离开后，杰克教父就把教堂中所有的修女都集中到了教堂中，他们没想着逃跑，这里就是他们的家，所以他们不会离开这里。
<br/>　　嘭！
<br/>　　教堂门被警卫队长从外面踹开，一队警卫队员端着AK自动步枪冲进了教堂，将教堂中的杰克教父和修女包围起来。
<br/>　　警卫队长手中握着沙漠之鹰走到杰克教父面前，眼神凶戾的盯着杰克教父，问道：“教父告诉我，塔尼特在哪里？”
<br/>　　“她走了。”杰克教父回答道。
<br/>　　“她去了哪里？”警卫队长问道。
<br/>　　“我不知道。”杰克教父摇头道。
<br/>　　“她！去！了！哪！里！”警卫队长一字一顿又对着杰克教父问了一遍。
<br/>　　“我不知道她去了什么地方。”杰克教父还是摇头。
<br/>　　警卫队长抬头看了眼杰克教父还有他身边那些修女，看到他们脸上的表情，他知道自己应该是没办法从他们嘴里问出什么东西了，对着自己身边的警卫队员做了个手势，转身朝着教堂外走去。
<br/>　　冲进教堂的警卫队员也跟随着警卫队长离开了教堂，留下三个警卫队员取出三颗手雷，拉开保险销，甩手朝着教堂内的杰克教父和修女扔去。
<br/>　　当啷！当啷！当啷！
<br/>　　三颗手雷被丢进教堂中，在地上跳动了一下，朝着杰克教父和那些修女滚去。
<br/>　　看着朝自己滚来的手雷，杰克教父和教堂中的修女一动不动的站在原地，看着手雷在自己面前爆炸，看着火光将自己吞没。

<br/>　　听到从教堂方向传来的爆炸声，坐在车斗中的那些孩子忍不住嚎啕大哭起来，塔尼特也是红着眼睛，眼泪直掉。
<br/>　　谁都知道，这三声从教堂方向传来的爆炸意味着什么。罗昊没有丝毫要停车的意思，深踩油门，加速前进，他知道，如果自己这个时候停车，车上的这些孩子一定会不管不顾的朝着教堂的方向跑去，那样只能是自投罗网，杰克教父和那些修女的牺牲也就白费了。
<br/>　　“有地图吗？”
<br/>　　皮卡车上，塔尼特向莱昂要来了地图，用笔在上面画出一个地方，把地图交给坐在副驾驶上的齐凡，说道：“去这个地方，那里很安全，我会通知我的父亲到那里跟我们见面，这些孩子我也会拜托我的父亲安置妥当。”
<br/>　　莱昂看了眼坐在车斗中，不断抹着眼泪的孩子们，对塔尼特说道：“他们将会是优秀的战士，仇恨的力量会让人变得无比强大。”
