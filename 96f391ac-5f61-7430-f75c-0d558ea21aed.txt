　　“拖住，要赢了！”看着自家的炮车还在那里坚挺的活着，看着自家距离胜利只有咫尺之遥，香蕉已经是激动的在那里大声的喊了起来。

<br/>　　此刻只要自家的队友能够拖住对方几人，香蕉是完全有自信利用这几个小兵将对方的最后一座基地塔给推掉的，只要那个基地塔没了，自己就完全没有任何顾忌了，到时候可以直接推掉这个水晶。
<br/>　　听着香蕉的大喊，这个时候夜雪狂歌几人也是铁了心的跟在对方几人的身后，一副要去送命的架势。
<br/>　　如果这一场比赛放到赛场上的话，可以毫不夸张的讲，这绝对是香蕉的封神之战，一个人夸张的单吃掉了楚白衣的中单，然后以一己之力推掉对方的水晶，当然了这是需要队友的帮忙拖延的，但是毕竟他才是这里的主角。
<br/>　　等到香蕉将对方的水晶的血量推到半血的时候，皇族这边在教练的密切的关注下，终于是有一人回到了家里，正是他们的上单大树。
<br/>　　白队这边毕竟仅仅只有三人，对方四人分散逃跑，他们肯定是有一人会被放掉的，而在这个时候在叶帝的喊声中，夜雪狂歌几人直接将大树这个英雄给放掉了。
<br/>　　因为在大家的心中，这是一个对武器大师的威胁比较小，而且当时那种情况大树距离他们比较远，所以几人只能就近拖延其他几名对手。
<br/>　　不过到了这个时候一切都已经无关紧要了，现在无论是放对方的谁回家，肯定都是阻挡不了武器的脚步的，作为一个单挑王，他是不惧怕对方任何一个英雄的。
<br/>　　无论是谁回来都只有死路一条，不过看到对方其他几人此刻已经被追击到了中路的二塔的时候，香蕉却是没有多加理会这个大树。
<br/>　　这个时候如果再在这个大树的身上耽搁一些时间的话，也许对方几人就已经徒步走回来了，到了那个时候自己面对这么多人的情况下，恐怕就真的要死了。
<br/>　　随着武器的一声声敲击的动作，听着那砰砰砰的敲击声，看着那血量越来越少的基地水晶，这一刻皇族几名队员的心都沉了下去。
<br/>　　而大树除了控制一下武器之外，根本没有做出其他的作用，香蕉的武器在那里推塔，大树在一旁攻击着武器，只不过他的伤害打在一个开大的武器身上，却是像挠痒痒一般，不痛不痒的。
<br/>　　随着武器的最后一下攻击击打在水晶，当水晶破碎的一刹那，皇族这边众人都是有些颓然的瘫坐在了椅子上。
<br/>　　在大喜大悲面前，任谁都不能将自己的心态保持在一个很好的水平。
<br/>　　而反观此刻白队的训练室，随着香蕉最后一下将对方的基地击碎的一瞬间，叶帝等人已经是大声的欢呼了起来。
<br/>　　“nice，这一把如果是比赛的话，香蕉就要封神了，牛逼，真的牛逼。”
<br/>　　“我都以为要输了呢，大嘴刚刚那输出真的是太恐怖了，刚刚看到香蕉单杀白衣的机械先驱的时候，我都没有回过神来。”
<br/>　　“这一把夜雪哥也很稳啊，如果没有这条大龙buff的加成，恐怕香蕉还真不一定能够将这个防御塔给推掉，香蕉现在是真的强啊，无敌！”
<br/>　　“666666666666，这一把香蕉mvp，咱们这个体系虽然被对方给针对住了，不过香蕉他们却是针对不住香蕉这个人，今晚给香蕉加餐。”
<br/>　　训练室内众人开心的喊着。

<br/>　　他们都是用自己的大号和对方打排位赛的，没有退出游戏会在自己的面板上留下战绩的。
