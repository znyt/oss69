　　家中，萧逸继续观看着武技噬火百刃。
<br/>　　萧家的底蕴还是很强的，武技不下十数本。玄阶低级武技一本，黄阶的则有十多本。
<br/>　　但，凡境修为的族人，还有一般的执事，只能修炼黄阶武技。
<br/>　　只有那些受家族充分信任的执事，才有机会修习噬火百刃。
<br/>　　当然的，越是高级的武技，越难领悟，也越难炼至大成。
<br/>　　像黄阶巅峰武技，凡境修为的族人中，暂时还没有炼至大成，也就少数几个后天执事和长老们完全大成了。
<br/>　　至于噬火百刃，则是没有任何一人大成。
<br/>　　包括武痴二长老，修炼了这么多年，也仅仅凝聚出九十道火刃罢了，距离一百道火刃大成还有一段距离。
<br/>　　大多长老都只在六十道左右。
<br/>　　萧逸知道噬火百刃的强大。
<br/>　　对于武技，他同样渴求。毕竟，形意五绝充其量只是近身攻击，或者说是身法和招式。
<br/>　　但像噬火百刃这种武技，是一种攻击手段，可远距离攻击，并产生爆发性威力，对他的战力提升非常大。
<br/>　　不过，他也知道噬火百刃的困难，想短时间炼至百道火刃大成不现实。
<br/>　　而且，就算练成了，以自己体内的真气，也释放不出来，倒不如先将修为练上去。
<br/>　　将噬火百刃的信息记入脑海后，萧逸合上了这本武技，看向下一本武技。
<br/>　　那个挂名父亲留下来的武技，到底如何呢？
<br/>　　萧逸翻开首页，顿时眼前一亮。
<br/>　　“崩山斩，玄阶中级武技。泰山之大，吾剑立于山峰之巅，千丈高山，吾一剑破之。”
<br/>　　萧逸不禁赞叹一声，“好强大的剑技，霸道，威势，威力骇人。”
<br/>　　他迫不及待地将整本书籍看完，然而，当他看到最后一页时，却陷入了疑惑。
<br/>　　因为，在书的最后，竟有着‘上卷’二字。
<br/>　　“怎么回事，这本武技不完整。”萧逸暗暗皱眉。
<br/>　　以他将形意拳练出真意的本事，可以感觉得到，若是这本武技完整，其描述的武技，确实极为强大。
<br/>　　他甚至能想象到，那一剑之下，千丈高山被瞬间毁灭的恐怖之意。
<br/>　　但，这本武技不完整，单单书籍上记载的修炼方法，威力却凭空下降了百倍都不止。
<br/>　　若是完整的崩山斩，其等阶，绝对不止玄阶中级，怕是地阶巅峰武技都有可能。
<br/>　　而且，这上卷中，似乎并未存在崩山斩的真正精粹，这才导致它只是玄阶中级。
<br/>　　半晌，萧逸恍然，并淡淡一笑。
<br/>　　也是，创造出崩山斩的武者，必然是一位叱诧风云的强大武者。
<br/>　　这样的武技，又怎么会是一个普通家族中的家主所能获得的呢。
<br/>　　按萧逸的猜测，自己那个挂名父亲，应该只是偶然得到这本崩山斩上卷，并留在了家族中。
<br/>　　“崩山斩与噬火百刃不同，噬火百刃只要是拥有火属性武魂的武者都能修习并使用。但崩山斩，明显是一本剑技，若手中无剑，根本施展不出来。”萧逸喃喃思索着。
<br/>　　至夜深，萧逸才停止了对崩山斩的修炼，并拿出了大长老给的书籍。
<br/>　　书籍上的炼药知识，萧逸的兴趣少了许多。但现在自己已是后天一重，需要更高级的丹药了。
<br/>　　他的双眼陡然间变为了一红一黑，太阴太阳之眼出现。
<br/>　　而后，凝聚出冰鸾剑剑身。
<br/>　　剑身上那复杂玄奥的符文，在太阳太阳之眼下，动了起来，转化为一道道文字信息。
<br/>　　同一时间，记在脑海中的炼药知识变为明悟。
<br/>　　之后，他看了眼‘后天丹’的丹方。
<br/>　　霎那间，脑海中出现了一个神秘的丹方，‘金脉丹’。
<br/>　　“嗯？冰鸾剑这次创造的丹药怎么这么奇怪？”萧逸暗暗皱眉。
<br/>　　按照金脉丹的介绍，这不是一种类似‘后天丹’提升修为的丹药，反而是一种升华主脉的灵药。
<br/>　　而且，金脉丹无需服用多粒，一粒就能起效，九条主脉，服用九粒即可。
<br/>　　不过，萧逸看了眼那些材料，顿时傻眼。
<br/>　　“培源朱果，三级妖兽内丹，这....”
<br/>　　炼制金脉丹的其他材料都好说，很容易得到。唯独这两样东西，培源朱果，三品药材，一颗的价格起码在千两以上，而且是有价无市。
<br/>　　而三级妖兽，那可是等同于先天武者的存在阿，怎么可能得到它的内丹。就算有些商行有卖，其价格也高得吓人。
<br/>　　萧逸无奈地摇摇头，“看来金脉丹是暂时无法炼制了。”
<br/>　　萧逸发现无事可做，干脆蒙头睡觉去。
<br/>　　....
<br/>　　翌日，当萧逸醒来，他还是决定今日就出发，前往陨星山脉。
<br/>　　依依得知此事，连忙哀求道，“少爷，带上依依好不好，那种深山野地的环境很差呢，依依能随时照顾服侍您。”
<br/>　　萧逸摇摇头，语气决绝，道，“不行。”
<br/>　　“少爷...”依依哀求着。
<br/>　　萧逸脸色一板，呵斥道，“说了不行就是不行。”
<br/>　　“少...少爷...”依依被萧逸吓到了，满脸委屈。
<br/>　　萧逸放松了脸色，柔声道，“依依乖，我又不是去游山玩水，而是去找慕容千军算账的。”
<br/>　　“陨星山脉内妖兽纵横，又有许多险地，危机重重。你一个普通人，跟着我会有危险的，而且很可能会成为累赘，拖累我，反倒让我陷入危险。”
<br/>　　“如果你想下次能跟着我一起出去历练，那便好好修炼。只有你强大起来了，我才放心你跟着我。”
<br/>　　萧逸循循善诱着，希望诱导依依努力修炼。
<br/>　　这个世界，很危险。萧逸更希望，自己不在的日子内，依依不用照顾自己，可以将心思都花在修炼上面。
<br/>　　而且，直觉告诉萧逸，慕容家的人绝对不会单单散布个谣言就罢休。那群阴险的家伙，绝对会有一连窜的后手。此次离开萧家，少不得和他们交锋。
<br/>　　这也是萧逸不许依依跟随的原因。
<br/>　　......
<br/>　　依依最终还是选择听萧逸的话，为萧逸收拾好行李。
<br/>　　直到萧逸来到萧家大门，依依还是不舍地默默跟着。
<br/>　　“逸儿。”不知何时起，三长老竟早就在萧家大门等待萧逸。
<br/>　　“萧重叔叔。”萧逸走向三长老，将两本武技交给他，道，“两本武技的内容我已经记下，噬火百刃交还家族，而崩山斩，则交由您处置吧。”
<br/>　　“嗯。”三长老点点头，将一个乾坤袋交给萧逸，道，“这是我为你这次历练准备的东西。”
<br/>　　“谢萧重叔叔。”萧逸拿过乾坤袋，而后道，“萧重叔叔，我拜托您一件事行不行？”
<br/>　　三长老忽然玩味一笑，道，“我知道你想说什么，让我帮忙照顾依依是吗？”
<br/>　　“额。”萧逸一愣，“是的。”
<br/>　　“臭小子，我就知道你舍不得依依，放心吧，我早有准备。”三长老笑着说道。
<br/>　　这时，一个中年女子忽然走了过来。
<br/>　　萧逸认得她，萧家的第一执事，后天九重武者。而且，实力远超同境界的后天九重武者，堪称先天之下第一武者。
<br/>　　虽然她不是长老，但在萧家中的威信，却丝毫不亚于长老。甚至，对不少执事来说，她的命令比长老们的命令更有用。
<br/>　　但，她是个高傲之辈，至今未嫁。有两个传言，其一是她痴于武道，无心****之事；其二，她不嫁，似乎和三长老有关系。据传，他们两人以前是青梅足马，年轻时甚至有一段外人不知的故事。
<br/>　　“可以阿萧重叔叔，我听说，我们萧家的这位第一执事，连大长老都叫不动，你一句话就把她叫来了....”萧逸满脸打趣地问着。
<br/>　　三长老连忙打断道，“臭小子，胡说些什么。哼...”
<br/>　　三长老扯开了话题，道，“我已经说服她当依依的武道指导老师，你不用担心依依。”
<br/>　　“嗯，谢萧重叔叔。如此，我去陨星山脉了。快则半月，慢则一月，萧家会收到慕容千军败于我手中的消息。”
<br/>　　“好，有志气，我信你。”三长老点点头。
<br/>　　“依依，照顾好自己，专心修炼，我很快就会回来。”萧逸对依依交待了一句，快步走出了萧家。
<br/>　　萧家门外，早有准备好的一匹千里骏马。萧逸一跃而上，扬鞭绝尘而去。
<br/>　　后面，三长老和依依远远地看着，目光中尽是不舍与担忧。
<br/>　　说到底，这是他们心中所关爱着的最重要的人，第一次远行。
