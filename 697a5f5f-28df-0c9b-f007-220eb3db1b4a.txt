　　“不！”
<br/>　　小魔主被轰飞，刚刚反应过来，便是看到这一道剑芒，向着他直接斩杀过来，脸色骤然大变，嚎叫一声。
<br/>　　他已经感觉到，在这一剑之下，就算是有着天魔宝车的保护，他也是要必死无疑。
<br/>　　“天魔相助！万魔降临！”
<br/>　　小魔主声嘶力竭的狂吼一声，在他的胸口之中，突然一道玉符，似乎感受到了小魔主的危险，突然飞了出来。
<br/>　　这一道玉符上，蕴含有着滔天魔意，这魔意的可怕，令楚风眠都是脸色震惊无比。
<br/>　　这玉符，便是小魔主最后的底牌，也是他最后的手段，他的师尊，九魔城魔主给他留下的玉符。
<br/>　　“轰！”
<br/>　　突然之间，玉符炸裂开来，无穷无尽的魔意，突然降临，万魔降临，其中一道魔头，顿时抓起小魔主的身躯，硬生生的将这宫殿都给撕裂开来，猛然化为一道遁光，窜了出去。
<br/>　　“此人手中，居然有着如此强者留下的玉符！”
<br/>　　楚风眠都没有料想到，小魔主的手中，居然还有这样一道保命玉符。
<br/>　　这玉符，绝对是一尊无上强者凝聚而出的，将这宫殿硬生生都给撕裂开来，足以证明这玉符的力量到底有多强大。
<br/>　　不过幸好的是，这玉符仅仅只是一道逃命玉符，就像是曾经玄北圣者给楚风眠的那一道一样，无法用来对敌。
<br/>　　不然这玉符上的力量，若是全部轰击楚风眠，楚风眠只怕都要当场陨落。
<br/>　　“小魔主逃不了多远，在这天羽神域之中，在强大的力量都要受到压制，小魔主逃不了多远！”
<br/>　　楚风眠瞬息之间，他的身躯也是一动，催动起魅影身法来，直接向着小魔主所逃的方向追了过去。
<br/>　　现在是斩杀小魔主最好的机会，要是让小魔主这次逃走，恢复了天魔宝车的全力。
<br/>　　只要他不在跟楚风眠死战的话，楚风眠都对他无可奈何，所以今日楚风眠必须要斩杀了小魔主才行。
<br/>　　不然在这天羽神域中，小魔主终究是一个威胁。
<br/>　　他的身形，在这天羽神域之中每一次催动身法，都需要消耗身躯中大半的灵力。
<br/>　　一般武者，都不敢这样催动身法来赶路，但是楚风眠却是拥有神脉，消耗灵力，瞬间就能够恢复。
<br/>　　楚风眠的灵识一动，横扫周围，突然就察觉到了这小魔主的方向。
<br/>　　“魅影！”
<br/>　　楚风眠身形再次催动，猛然冲了过去，在这天羽神域中，武者消耗的灵力是数倍，数十倍。
<br/>　　但是对楚风眠却是丝毫不受影响，可以说在这天羽神域中，他的实力甚至是要比起在外面，还要强大几分，因为他可以肆无忌惮的动用灵力。
<br/>　　楚风眠的身形，只留下一道道虚影，向着小魔主的方向追了过去。
<br/>　　“幸好师尊给我留下了这道玉符，不然今日还真的要死在此人的手中！”
<br/>　　小魔主在丛林之中，连连穿梭，他的脸上都是露出了几分冷汗。
<br/>　　就在刚刚，绝对是让他尝到了临近死亡的滋味，若不是这玉符，在刚刚楚风眠那一剑下，他就必然陨落。
<br/>　　“这小子的实力，实在是太可怕了，整个天羽神域中，都没有人单打独斗可能是此人的对手了，我必须要去联合天剑宗，圣王宗的强者，一起去围攻他才行！”
<br/>　　小魔主死里逃生，心中都已经算计起来，如何对付楚风眠了。
<br/>　　他小魔主一直以来，都是算计别人，还从来没有被人逼迫到如此狼狈的地步，在这天羽神域中所有九魔城弟子都已经陨落，只剩下他一人。
<br/>　　这对他们九魔城来说都是一个极大的损失，不杀了楚风眠，小魔主也难以平息。
<br/>　　“想要联络其他武者来杀我？可惜，小魔主，你不可能还有这个机会了！”
<br/>　　就在小魔主刚刚放松之际，一道身影，却是从远方直接追了上来。
<br/>　　几乎是一瞬间，楚风眠的身躯便是超越了小魔主，直接站在了他的面前，将小魔主直接逼停了下来。
<br/>　　“什么？你到底是谁？你怎么可能追的上来的？”
<br/>　　小魔主看到楚风眠追了上来，眼珠子都要掉了下来，脸色已经没有了之前的狂妄，尽是恐惧。
<br/>　　那玉符，足足是带他离开了十里距离。
<br/>　　十里距离，在外面，对武者来说，便是一瞬之间的事，但是在这天羽神域中，无法催动遁光，催动身法都需要消耗大半灵力。
<br/>　　十里距离，让一些武者动用身法去追，只怕是灵力耗尽都追不上。
<br/>　　所以小魔主才会觉得，自己已经逃走了，万万想不到，楚风眠居然追了上来。
<br/>　　“该死！跟此人拼了！”
<br/>　　小魔主一瞬间脸色变得狰狞无比，将身上仅剩的灵力，全部注入到天魔宝车之中。
<br/>　　顿时这天魔宝车，再度唤醒，冲着楚风眠撞了过去。
<br/>　　“天魔宝车，这种灵器留在九魔城也太过可惜了，拿来吧！”
<br/>　　楚风眠看到小魔主催动天魔宝车撞了过来，冷哼一声，顿时运转起身上的战龙血脉来，顿时一股龙威，直接冲着天魔宝车，笼罩而去。
<br/>　　同时一道战龙之爪也是凝聚出来，一把抵挡住天魔宝车的撞击，将这天魔宝车，抓在了手中。

<br/>　　楚风眠怒喝一声。
<br/>　　随着这一个字喊出，天魔宝车，被楚风眠紧紧抓在手中，顿时上面的无穷无尽的魔意，便是冲着楚风眠反击而来。
<br/>　　但是楚风眠心神一动，更是运转了更多的灵力，一股股龙威，也是源源不断的笼罩下去，逐渐将这天魔宝车上的魔意净化，镇压。
<br/>　　小魔主看到楚风眠将天魔宝车收取，脸色已经是惨白了。
<br/>　　失去天魔宝车，他简直便是失去了任何可以跟楚风眠对抗的机会。
<br/>　　“小魔主，没想到居然是你？”
<br/>　　几乎就在这时，一股庞大无比的灵力，突然从天而降，化为了一道巨大无比的手掌，猛然向着小魔主抓了过去。
<br/>　　同时也是一股灵力，向着楚风眠镇压过来，将楚风眠死死的给镇压在了原地。
<br/>　　“有强者出手了！”
<br/>　　楚风眠目光一转，自然明白过来，这是有强者突然出手了。
