夏夕绾回了幽兰苑，将买回来的芭比娃娃送给了陆老夫人，老夫人开心的像个孩子一样，迅速连拍了几张照，要发微博。
<br/>
<br/>        “绾绾，你有微博么？我关注你啊，以后奶奶就是你的头号粉丝！”
<br/>
<br/>        夏夕绾没有想到老夫人竟然玩起了微博这种东西，她确实刚开了微博，这段时间不断上热搜，她已经有80万的粉丝了。
<br/>
<br/>        “奶奶，我们互关啊。”
<br/>
<br/>        夏夕绾将老夫人的账号找了出来，老夫人的微博名是奶奶今年18岁~
<br/>
<br/>        面对只有18岁的奶奶，夏夕绾精致的眉梢里都染上了笑意。
<br/>
<br/>        老夫人将自己和芭比娃娃的合照放了上去，还配文孙媳妇送的，谢谢我的小绾绾，么么哒~
<br/>
<br/>        夏夕绾就觉得老夫人好厉害啊，微博玩的溜，流行语也会说了，她迅速给老夫人点了一个赞。
<br/>
<br/>        “绾绾，奶奶也有一样东西要给你啊。”老夫人神秘兮兮的将一个荷包袋塞给了夏夕绾。
<br/>
<br/>        夏夕绾看着这个荷包，荷包相当的精致，应该是苏杭那里的丝绸锦缎做的，上面还用金丝线绣了一个“陆”字。
<br/>
<br/>        “奶奶，这是什么呀？”
<br/>
<br/>        “这个荷包里放着我从寺庙里求回来的得子福，绾绾，你可要收好了，不能丢了。”
<br/>
<br/>        得子福…
<br/>
<br/>        果然还是奶奶啊，三句话就能绕回抱重孙的主题。
<br/>
<br/>        夏夕绾扯了一下红唇，收下了，“好的奶奶，我一定会妥善保管，不会丢掉的。”
<br/>
<br/>        这时草坪上有两道明亮的车灯透过蹭亮的玻璃窗镀了进来，老夫人笑道，“绾绾，寒霆回来了。”
<br/>
<br/>        陆寒霆回来了。
<br/>
<br/>        这一次夏夕绾出门了七天，两个人许久没有见了，她快速的起身来到门边，抬头就看到了停在草坪上的那辆劳斯莱斯幻影。
<br/>
<br/>        驾驶车门打开，一道高大英挺的身躯闯入视线。
<br/>
<br/>        今天的陆寒霆穿了一身手工版的黑色西装，脖间打着领带，禁欲系男神，举手投足弥漫着优雅倨贵。
<br/>
<br/>        夏夕绾看见他迈着稳健的步伐走了过来，随着他每一步，那剪裁如刀锋的黑色西裤都能走出商界精英冷硬迷人的气场，还有天生王者的淡漠疏离。
<br/>
<br/>        夏夕绾本来想跑出去迎接他的，但是想一想她还是躲在门框后面，准备吓一下他，给他一个。
<br/>
<br/>        小圆圆蹲在夏夕绾的脚边，不知道主人在干什么喵喵喵~
<br/>
<br/>        夏夕绾迅速将纤白的手指放在唇边做了一个噤声的手势。
<br/>
<br/>        小圆圆不叫了。
<br/>
<br/>        夏夕绾小手趴着门框，探出一双黑漉漉的翦瞳偷偷向外看去。
<br/>
<br/>        这一看，她发现陆寒霆走近了，也看了过来。
<br/>
<br/>        夏夕绾吓了一跳，迅速躲着不敢看了。
<br/>
<br/>        陆寒霆早已经发现了她，门边露出了一小截的裙摆，他知道她躲在那里。
<br/>
<br/>        刚才他都看到她那黑漉漉的眸子里，被发现时像小鹿乱撞一样，又缩了回去。
<br/>
<br/>        陆寒霆勾了一下薄唇。
<br/>
<br/>        夏夕绾等啊等，没有等到他过来。
<br/>
<br/>        怎么回事，这么一小段路，他怎么走这么久？
<br/>
<br/>        夏夕绾探出小脑袋一看，陆寒霆已经不见了。
<br/>
<br/>        咦，他去哪里了？
