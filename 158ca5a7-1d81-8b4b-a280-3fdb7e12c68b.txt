　　10月16日，世界赛正式开赛，一大早国内的媒体便是争相报道起来，其他各大城市的玩家一个个也都是非常的激动。
<br/>　　当然了最激动的还是要属上海这座城市，一大早这座城市的年轻人们便是向着现场赶去。
<br/>　　这一次举办世界赛的场地是上海最为著名的体育馆，能够容纳20万人，而在门票刚刚预售的时候，这些门票便是已经被一抢而光。
<br/>　　而火神的各路主播，这个时候也都是来到这里开启了户外直播，来蹭世界赛的热度，很多主播都在体育馆门外等着，等着这些职业战队的到来。
<br/>　　到了比赛这一天，叶帝他们也都是非常的激动，努力了一年就是为了这一刻，如今终于到来，这一次他们注定依旧要辉煌下去。
<br/>　　“小叶子加油啊，千万不要丢人，现在台下可是有着20万观众在那里看着呢。”还没离开基地，吃早饭的时候，楚南在那里开口说着。
<br/>　　“是啊，这20万观众，国内最少也要有19万，第一场比赛如果要输了的话，那国人肯定会很失望的，虽然压力很大，但是你们一定要顶住。”秦霄这个时候也是跟着开口说道。
<br/>　　“放心吧，今天这场比赛我们肯定会尽全力的。”叶帝郑重的点着头，他自然是知道今天这场比赛的重要。
<br/>　　吃过早饭后，众人在基地又休息了一会后，便是乘车向着现场赶去，因为今天是开赛日，各大战队都要到场参加开幕仪式，所以就算是没有比赛的秦霄他们也是跟着一起去了。
<br/>　　当叶帝他们到达体育馆还没有进入到现场的时候，在外面他们便是已经感觉到到了气氛的热烈。
<br/>　　体育馆外人山人海，人声鼎沸，所有人的脸上都洋溢着一副热情的表情，在他们的手里都拿着红色的国旗，很多人还将国旗印在了脸上，以此来表达他们对于国内战队的支持。
<br/>　　当看到叶帝他们走下保姆车的时候，场外这些没有买到门票的观众，顿时爆发出了一阵热烈的欢呼声，紧接着场内已经入位的众多观众也是受到了场外气氛的影响，跟着欢呼了起来。
<br/>　　将近二十五六万的观众一起爆发出来的欢呼声，仿佛要掀破天际一般，让得半个上海城都能够听到他们的声音。
<br/>　　场内二十万观众已经入场，而场外如今也有着五六万的观众，这些人没有买到门票，但是他们想要在现场来跟战队一起分享胜利的喜悦，所以一个个也都是顶着火热的太阳赶到了这里。
<br/>　　好在体育馆对此早就有了准备，支起了大大的太阳伞，而且场外有着众多巨大的LED显示屏，这些屏幕会同步直播场内的情景，所以到达现场的这些观众也都可以看到比赛。
<br/>　　当叶帝等人进入到现场的一刹那，那欢呼声更加的剧烈了起来，甚至就是连叶帝几人的耳膜这一刻也都是轰轰作响。
<br/>　　提前一步到达现场的其他各国的战队，看到这样一副画面，一个个也都是哗然变色。
<br/>　　他们虽然预想过华夏的观众对于他们的本土战队会很热情，但是这样的一副画面，他们却是从来都没有想过。
<br/>　　“这个下马威有点太大了，现在比赛还没开始，我就已经感受到压力了。”在后台的金相赫听着现场那巨大的欢呼声，一脸苦笑的开口说道。
<br/>　　“确实，华夏的这群观众真的是太热情了。”金在贤这个时候也是一脸的苦笑，到了现在他也没什么话可以用来安慰队员的了。
<br/>　　不过金在贤对于自家的队员还是很相信的，经历过这么多场大赛，他相信金相赫他们的心理素质肯定能够应付现在的场景。
<br/>　　时间到达1点钟的时候，开幕仪式正式开始，率先出场的自然是特意来到这里为叶帝他们加油的叶芷澜。
<br/>　　看到叶芷澜这位国际巨星出现，台下的众多观众更加的热情了起来，一个个在那里疯狂的嘶吼着。
<br/>　　紧接着国内还有国外的众多一线明星也都是陆续演出，可以毫不夸张的讲，单单这一场开幕式的规模，已经远超以前几届世界赛的规模了。
<br/>　　无论是阵容还是明星，亦或者是现场的专业设备。
<br/>　　“接下来让我们有请60支参加世界赛的队伍登场。”表演结束过后，伴随着米粒的一声大喊，各个战队开始陆续登场。
<br/>　　.......
<br/>　　“最后让我们有请荣耀白队登场，他们在春季赛以连胜的战绩闯入到决赛，以2：3的战绩惜败荣耀蓝队，在夏季赛中他们成功复仇，夺下了华夏赛区的种子名额，今天他们将以东道主的身份来迎战世界综合实力排名第一的ST战队，期待他们今天的表现。”
<br/>　　当最后叶帝他们登场的时候，现场的气氛顿时达到了顶峰，这一刻现场的20万观众，场外的五六万观众都不要命的嘶吼了起来。
<br/>　　这欢呼声让得其他的那些战队脸上也都是哗然变色，一脸艳羡的看着一身黑色队服登场的叶帝等人。
<br/>　　因为ST战队这边有着小花生的突然加盟，他们的综合实力已经重新回到了第一位，荣耀蓝队排在第二，白队则是排在第三。
<br/>　　当然了官方给出的综合数据，他们几支战队之间的差距是非常小的，如果不是那么严格的来讲的话，他们三支战队完全可以并列第一。
<br/>　　随着叶帝一行人的登场，开幕仪式已经是接近了尾声，在介绍完叶帝他们后，众多战队便是陆续退场。
<br/>　　当然了他们并没有离开，而是来到了台下专门为他们预留的座位上等待着观看着今天这场荣耀白和ST带来的揭幕战。
<br/>　　对于这两支战队的比拼，他们也都是非常关注的，毕竟这一场比赛能够很大概率看出两支战队的差距。
<br/>　　当时间到达下午4点钟的那一刻，荣耀白队和ST战队的比赛终于是正式开始。
<br/>　　因为是小组赛阶段，所以比赛都是采取bo1双循环的模式来打的，也就是说今天他们只和ST打一场，等到循环完一组比赛后，他们才会再次迎战ST。
