　　苏莫跟着伏卓，绕过了连绵的阁楼和院落，来到了千渊府的后方。
<br/>　　这里更加的僻静，只有九座宫殿，一字排开，分别相隔数百丈而立。
<br/>　　“上官师弟，这里便是天九殿，修炼宝地。”
<br/>　　伏卓带着苏莫，来到了最边上的一座宫殿之前，面带羡慕的说道。
<br/>　　“修炼宝地？”苏莫闻言惊讶，难道宫殿之中，有上好的修炼室不成。
<br/>　　他打量了一番眼前的宫殿，普普通通，并不精美，也不大气恢弘，殿门之上刻有‘天九’而二字。
<br/>　　“你进去就知道了，用你的身份令牌，应该可以打开！”伏卓笑着说道。
<br/>　　苏莫点了点头，随即向宫殿内大门走去，刚刚走到大门之前，宫殿上传出一个强大的排斥之力，将他阻挡住了。
<br/>　　苏莫拿出身份令牌，并没有什么动作，排斥之力便消散一空。
<br/>　　推开宫殿的大门，殿内亦是普普通通，入门是一处大殿，除了一些座椅，别无它物。
<br/>　　“伏师兄，在此小坐片刻如何？”苏莫没有急着查看这座宫殿，转身对门外的伏卓道。
<br/>　　“呵呵，恭敬不如从命。”伏卓立刻应允，他知道，苏莫刚到神域，有诸多事情需要了解，他不介意做个领路人。
<br/>　　再说了，苏莫受殿主看重，将来前途无量，他自然要极力较好。
<br/>　　随后，苏莫和伏卓，在宫殿中坐了下来，两人畅聊了起来。
<br/>　　当然，主要是苏莫再问，伏卓回答。
<br/>　　约莫一个时辰之后，伏卓在苏莫的恭送之下，离开了天九殿。
<br/>　　“伏师兄，慢走！”宫殿外，苏莫向伏卓抱拳道。
<br/>　　“上官师弟，以后我会多来叨扰你，你有什么不懂之事，随时可以去接引府找我！”伏卓拱手道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，目送伏卓离去之后，他便返回了天九殿，闭上了殿门。
<br/>　　天九殿不大，但是，其内的房间倒是不少，有炼丹室、修炼室、炼器室等等。
<br/>　　走进修炼室，苏莫静静的打量了起来。
<br/>　　这间修炼室不大，长宽高均不足十丈，地面和墙壁，均是漆黑之色，更加惊人的是，无论是地面之上，还是墙壁之上，均是布满了繁杂无比的符文。
<br/>　　这些符文，足有百万个，像是阵法，又不似阵法。
<br/>　　“这修炼室，又何玄妙之处？”苏莫面带惊疑，随即在修炼室的中央坐了下来。
<br/>　　沉吟了片刻，他试着向身下的符文阵法之中，注入了一丝玄力。
<br/>　　因为，他体内的混沌之力，分解了很多，分化成了无形玄力，所以，他注入的是火属性玄力。
<br/>　　刹那之间，修炼室猛然大变，浓郁的火属性源气，从无数符文之中狂涌而出。
<br/>　　整个修炼室之中，温度几乎是在刹那之间，便提高了千倍不止。
<br/>　　“好浓郁的源气！”苏莫心中惊讶，这源气极为浓郁，非常的精纯。
<br/>　　所谓的源气，其实就是蕴含本源之力的灵气，需要武者吸收炼化。
<br/>　　随即，苏莫收回了火属性玄力，改为注入水属性玄力。
<br/>　　猛然之间，修炼室再变，无数的符文之中，涌出大量的水属性源气。
<br/>　　浓郁的水属性源气，瞬间便充满了整座修炼室，让的修炼室，如同被一个汪洋大海包裹。
<br/>　　“太玄妙了，难怪伏卓说是修炼宝地！”
<br/>　　苏莫大喜过望，随即，在他的期待之中，他立刻注入了混沌之力。
<br/>　　不过，苏莫想象的混沌源气狂涌的场面并没有出现，而是五行源气。
<br/>　　无数符文之中，五行源气狂涌，金木水火土应有尽有。
<br/>　　“不错！天助我也！”苏莫并未失望，反而相当惊喜。
<br/>　　有五行源气就够了，他自己再融合就是，不是一定要混沌源气。
<br/>　　“想必这个修炼室之中，远不止五行源气这么简单！”苏莫心中暗道，估计，要是修炼其它属性之力，比如雷源气、阳元气等等，此修炼室也会有。
<br/>　　苏莫深深的吸了口气，有了这个修炼室，他的修炼速度，能暴增一大截。
<br/>　　此地的源气之浓郁、之精纯，远非下界可比的。
<br/>　　随后，苏莫暗自思索了起来，之前，他从伏卓的口中，了解到了很多消息。
<br/>　　比如，在神域，真神共有九品，被称为九品真神。
<br/>　　每个品阶，还有低级、中级和高级之分。
<br/>　　伏卓，只是一品中等的修为，而殿主的修为，高达九品低等。
<br/>　　在九品真神之上，便是神王，神灵中的王者，一个界域的巅峰强者。
<br/>　　据说大夏神王殿，有四位神王，非常强大。
<br/>　　神王也有低级、中级和高级之分，而且据说每个级别之间差距极大。
<br/>　　至于神王之上，便是那些神朝的创造者，来自太古洪荒时代的至尊。
<br/>　　在神域，有八大至尊，被公然为神域最强者。
<br/>　　这八大至尊，便是混沌神帝、亘古魔祖、幽冥至尊、天星尊主、苍天霸主、不死神尊、洪荒祖龙、轮回天尊。
<br/>　　当然，在神域还有一些至尊人物，和这八大至尊同一境界。
<br/>　　但是，这些至尊人物总体实力，比之八大至尊，却是略有不如。
<br/>　　而这至尊人物，大多都汇聚在天池神朝。
<br/>　　天池神朝，不仅地域最广，也是总体实力最强大的神朝，强者如云。
<br/>　　因为，天池神朝，囊括了人族和其它很多强大的种族，是很多至尊人物共建而成。
<br/>　　只不过，正因为天池神朝是很多种族共建，血统并不纯粹，各个势力之间明争暗斗，实力较为分散。
<br/>　　天池神朝八十一个界域，几乎是各自为政，互不相干。
<br/>　　可以这么说，天池神朝，除了有神朝之名外，并无神朝之实，亦或者说，天池就是一个各个种族、各个势力汇聚的复杂地域。
<br/>　　“还是先修炼吧，小目标，十年之内成神！”
<br/>　　苏莫深深的吸了口气，因为这修炼室的缘故，他距离既定目标更近了。
<br/>　　本来，他感觉自己二十年之内，能否成神都未可知。
<br/>　　但是，有了这绝佳的修炼室，他十年之内，成神有望。
<br/>　　随即，苏莫再次向地面的阵法符文之中，注入一股混沌之力，便开始修炼了起来。
<br/>　　虚神第四变，被称为是化神变，化神变算是虚神和真神之间的过度。
<br/>　　所谓化神变，便是要将元神与神丹融合在一起，做到这一步，元神将彻底超脱，超然物外。
<br/>　　这个时候，武者不仅更加强大，元神更是彻底的摆脱了肉身的束缚。
<br/>　　苏莫被浓郁的五行源气包裹，快速的吞噬了起来，他要将圣丹凝练的更加强大，才能尝试冲击虚神境第四变。
<br/>　　时间缓缓流逝，苏莫忘却了时间，整个人完全沉浸在了修炼之中。
<br/>　　而在他修炼之时，千渊殿中却是不平静了起来。
<br/>　　“一个下界的飞升者，加入了千渊殿，并且入住了天九殿，成为了天九殿的主人！”
<br/>　　这个消息，没出两个时辰，便传遍了整个千渊殿，数千弟子一片哗然。
<br/>　　天九殿，可是很多弟子梦寐以求的宝地，毕竟，普通的阁楼和院落，可没有那种绝佳的修炼室。
<br/>　　很多天才弟子，一直无法将天九殿弄到手，可是现在，天九殿居然被一个下界来的人抢去了。
<br/>　　很多人郁闷的吐血，而苏莫，也因此成为了众矢之的。
<br/>　　一时之间，很多人都在打探苏莫的情况，伏卓的接引府，几乎被人踏破了门槛。
<br/>(本章完)
