　　长剑刺出，剑尖之中突然爆射出无数的剑气。
<br/>　　这些剑气很微小，如同银针一般，但是数量奇多，密密麻麻数之不尽，无数微小的剑气，呈扇形辐射而开，向灵晔所在的方向笼罩而去。
<br/>　　剑气的范围非常的广，一下子便封锁了一方空间。
<br/>　　灵晔见此，一边身形急退，一边手中长枪挥舞，将袭来的剑雨击碎。
<br/>　　现在剑雨浩大，封住了一方空间，他也无法在自由的瞬移了。
<br/>　　轰轰轰！！
<br/>　　爆响声不断的响起，这剑雨虽然分散，每一道剑气都很微小，但是犀利无匹，威力依旧很强大。
<br/>　　猛烈的爆炸，让得战台上一片汹涌，气浪滔天。
<br/>　　灵晔虽然不简单，但完全不是卓剑豪的对手，一招剑雨就被逼到了看台的边缘处。
<br/>　　“败吧！”
<br/>　　一声轻喝突然响起，在剑雨将要消散直击，一股庞大的剑威升腾而起，卓剑豪手握长剑，一剑战天。
<br/>　　咻！
<br/>　　剑光夺目，璀璨耀眼，一道恐怖的剑气，仿佛横贯天地的长河，已无与伦比的速度，直击灵晔。
<br/>　　“什么？”灵晔见此，顿时面色大变，如今他已经退到了战台边缘，周围剑雨还未完全消散，他根本无法瞬移躲避。
<br/>　　感受到这一剑的威势，他甚至都有一种整个人被撕裂的感觉，全身冰冷，如坠冰窖。
<br/>　　退！
<br/>　　毫不犹豫，灵晔身形急退，直接退出了战台。
<br/>　　强大的剑气暴冲三万余里，都快要接近看台了，才缓缓消散。
<br/>　　“卓剑豪胜！”
<br/>　　褐袍长老的声音响起，随后，卓剑豪抬脚返回，没有丝毫的得意。
<br/>　　对于他来说，击败区区的灵晔，完全是挥手之间的事情。
<br/>　　众人惊讶，这个卓剑豪貌似比炎澜天、扬天成等人都要强大啊！
<br/>　　当然，炎澜天等人还没有展现真实的战力，具体孰强孰弱，还不一定。
<br/>　　但可以肯定的是，第一名的人选，就在卓剑豪、扬天成等五位妖孽之中产生了。
<br/>　　“的确是挺厉害！”苏莫亦是暗暗点头，心中暗赞，不过，这也是那个灵晔太弱了，除了瞬移之外，战力并没有多强。
<br/>　　随后，会武继续进行。
<br/>　　幽狼登场了，不过他并未挑战苏莫，因为苏莫已经战过一场了，这一轮无法再战。
<br/>　　幽狼挑战了一名雷云禽族的天才，一招便强势的击败了对方。
<br/>　　战斗一场接着一场，由于人数较多，非常的耗时间。
<br/>　　当然，这是因为没有什么人被淘汰，只要等到三轮挑战一过，连败三场的人，就会被全部淘汰，那时候进度就快了。
<br/>　　闪灵族的灵枫上场，同样是一招秒败对手，黑炎族的炎澜天、雷云禽族的雷云皓亦是如此，全部都是只用一招。
<br/>　　或许是卓剑豪的强势，让他们也不想再慢慢击败对手，个个都是展现出了恐怖的实力。
<br/>　　良久之后，又轮到扬天成上场了。
<br/>　　唰！
<br/>　　英俊不凡的扬天成，面上挂着淡淡的笑容，缓缓的飞落在了战台上。
<br/>　　“神女，这是我的第二战！”扬天成向云悠悠抱拳，微笑着说道。
<br/>　　“嗯！”云悠悠轻轻点了点头，小嘴却是微微一撇，你第几战也不必和我说吧！
<br/>　　这时，扬天成依旧没有选择对手，而是深深的吸了口气，微笑道：“神女，适才空暇之时，我做了几句诗词，特意赠于神女，以表雅兴，希望你能喜欢！”
<br/>　　望着云悠悠绝美无暇的容颜，扬天成表面平静，心中却是异常的激动，只要云悠悠给他机会，他就一定能将对方追到手。
<br/>　　对于女人的心，他再了解不过了，以往就是那些冷淡的冰山美女，都逃不出他的攻势。
<br/>　　扬天成此言一出，所有人都是惊讶不已，素闻扬天成不仅天赋妖孽，才情也是不凡，原来果真如此，现在居然要展现文采了。
<br/>　　“卧槽！”苏莫闻言眼眸一瞪，玛德，这个家伙居然卖弄起文采来了，这是在想方设法的讨好青璇吗！
<br/>　　只不过，苏莫心中暗暗摇头，这个世界武道为尊，都是粗.人，这扬天成能有多高的文采。
<br/>　　云悠悠闻言神情一怔，面现迟疑之色，她并不傻，也知道对方在讨好她，但是她并不想和对方有什么交集。
<br/>　　“什么诗句，说来听听！”
<br/>　　云悠悠还没开口，其身边的灵萱，倒是率先开口了，满脸感兴趣之色。
<br/>　　“嗯！”
<br/>　　扬天成郑重的点了点头，微微呼出一口气，随即目带深情的望着云悠悠，轻吟了起来：
<br/>　　我痴蓝怡心迷离。
<br/>　　绝爱相思润心底。
<br/>　　香月神灵牵宿缘。
<br/>　　九天玄女梦相依。
<br/>　　一字一句，扬天成将诗句轻吟了出来，幽幽的字音在整个武斗场飘荡。
<br/>　　“好厉害！”
<br/>　　“不错，果然有文采！”
<br/>　　“扬天成不仅天赋超绝，没想到文采也如此的不凡！”
<br/>　　“当真是才情无双啊！”
<br/>　　听完扬天成的诗句，不少人开口赞扬，虽然很多人根本就听不懂，但却是装作很懂的样子。
<br/>　　“什么破诗！”苏莫心中腹诽，差点笑出声来，这也能叫诗吗？
<br/>　　不过，少倾之后，他的眼眸不禁眯了下来，因为他发现，这诗中居然暗藏玄机。
<br/>　　“我爱神女！”苏莫呢喃自语，这四句诗句之中，第一句的第一个字，第二句的第二个字，第三句的第三个字，和第四句的第四个字，组合起来，正是——我爱神女！
<br/>　　这让苏莫的脸色阴沉了下来，他就是脾气再好，也不禁有些怒了，这个扬天成居然如此的不要脸，在这大庭广众之下，这么明目张胆的表白。
<br/>　　这让他心中不由得生出一丝危机感，青璇该不会真的被这种人吸引吧！
<br/>　　若是如此的话，就有麻烦了！
<br/>　　“简直是找虐！”苏莫心中沉吟，随即轻笑一声，对方敢在他面前向青璇表白，等会定让对方后悔！
<br/>　　北方看台上，云悠悠听了扬天成的诗句，不知道是否发现了其中的奥妙，微笑道：“扬公子才情不凡，悠悠佩服！”
<br/>　　“呵呵，只要神女喜欢便好！”扬天成闻听云悠悠之言，顿时心中大喜，暗暗激动了起来，不过，他的面上依旧是一副儒雅的笑容，显得不卑不亢，气度非凡。
