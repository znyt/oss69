<br/><br/>　　    现在问题就来了，莫克做市委书记也才没多长时间，在省委的时后他就是一个搞政策研究的公务人员，没什么油水的一个职务，他个人合法的收入水平显然还达不到这个水准，甚至孙守义估计莫克的收入还不能够买到朱欣在城邑集团的那套房子。莫克在做市委书记之前，只是一个比较边缘性的省委副秘书长，而这个职务听上去好像是很大的，但实际上却是没什么重要性的。因此理论上他应该不会有什么不合法的灰色收入的。<br/>　　<br/>　　    既然正当的收入和灰色的收入都没有多少，那莫克是怎么搞到城邑集团这套房子的房款的，就很令人怀疑了。按照孙守义的推测，莫克朱欣拿到的这套房子很可能是束涛送给莫克的，莫克一定是跟束涛私下达成了某种交易，束涛这才帮莫克兑现了给朱欣的条件。<br/>　　<br/>　　    既然有交易，就一定有蛛丝马迹可循的，孙守义就开始关注城邑集团有没有新的行动，果然，城邑集团看中了海平区的一个地块，参与了这个地块的竞标。孙守义就侧面打了电话给海平区的陈鹏，问了一下陈鹏这个地块的情况。陈鹏就以为孙守义对这个地块也有意染指，就暗示说这个地块有领导已经打了招呼了，他不好再答应孙守义什么了。<br/>　　<br/>　　    孙守义一下子就明白这个有领导打了招呼的领导是指谁了，在海川排名在他前面的领导并不多，金达和于捷在这一类的事情是很爱惜羽毛的，剩下来最大的可能就是莫克了。联系到朱欣的那套房子，孙守义基本上就可以确定打招呼的就是莫克。而这个招呼一定是为城邑集团打的。<br/>　　<br/>　　    明白了这一点，孙守义就明白了莫克和束涛的操作思路了。他们放弃了比较显眼的海川市区，转而把目光放到了海川周边的县市。莫克一定是知道他这个市委书记目前在周边县市的影响力肯定是会大于在海川城区的影响力的。没有一个下属县市的主官敢忽略他这个市委书记的指示的。而这么做更大的好处是比较隐秘，下属县市的主官们绝对不会喧嚷莫克跟他们打过招呼的，人们就不会知道莫克出面帮城邑集团争取地块了。<br/>　　<br/>　　    莫克还真是有够狡猾的，也是够胆大的，竟然就在人们对朱欣和束涛私下往来的关注度还很高的时候，波澜不惊的就帮束涛把事情给办了。这家伙玩起小伎俩来还真是有一手的。<br/>　　<br/>　　    虽然来海川之后，莫克办了不少拙劣的事，但这一次手法却是玩的很高明的。这个时候朱欣拿束涛那套房子，表面上看似乎会引人怀疑，但是仔细分析一下就会知道，莫克是进退有据的。<br/>　　<br/>　　    首先朱欣在这一次离婚事件中算是一个利益的受损者，家庭破裂，市委书记的丈夫跟她离了婚，人们这个时候大多会忘记造成朱欣离婚的主要原因是朱欣的贪婪，而只会看到朱欣是一个失婚的妇人，是一个应该被同情的弱者。谁会在这个时候去追究一个失婚妇人房子是怎么的来的吗？追究下去干什么，要把她赶尽杀绝吗？那样子似乎也太没有人情味了。<br/>　　<br/>　　    其次，就算是追究下去，房子是束涛送的，人们也只会认为束涛这是对朱欣的一种补偿，别忘了，朱欣是因为跟束涛会面才被逼离婚的。束涛是朱欣的离婚的主因，送出一套房子作为补偿，也是应该的。再说，这个时候朱欣接受这套房子已经无法在帮束涛做什么了，你无法把这件事情定义为行贿受贿，而只能定义为私人的赠送。<br/>　　<br/>　　    最后，也是最主要的，整件事情当中你看不到一点莫克参与的证据，表面成立的证据都在说事情是在朱欣的主导下办的，就算是真要追究责任，也只能追究朱欣的责任了，于莫克无关。<br/>　　<br/>　　    整件事情最高明的就是这一点，莫克一定是看透了这一点，才敢不避嫌疑的让朱欣去城邑集团拿那一套房子。<br/>　　<br/>　　    孙守义相信此刻在莫克内心中一定会认为他自己是最聪明的，别人都被他蒙骗了，都是傻瓜。但实际上在政坛上谁也不比谁傻多少，很多人不是看不出莫克在玩什么把戏，而是不愿意去拆穿他，这些人是在装傻，而不是真傻。<br/>　　<br/>　　    而目前孙守义也没有任何打算要去拆穿莫克的把戏，他的选择也很明确，他要跟别人一样装傻，因为他明白了别人会装傻的真正原因。这个别人当中也包括他现在的顶头上司金达。<br/>　　<br/>　　    金达在莫克到海川任职市委书记之后，整个做人做事的态度是有了很大的改变的。以前的金达虽然对一些事情也会忍让，但是还没到被人家骑到头上来还忍让的程度。他的个性不允许他这么做。他跟前任市长徐正，前任市委书记张林之间的冲突就充分说明了这一点。<br/>　　<br/>　　    但是莫克来了之后，金达的态度却来了一个一百八十度的转变，首先一点，就是朱欣工作安排跟莫克产生的冲突，金达是应莫克的要求才提出来给朱欣一个很好的安排的，就算是你莫克不满意，你也是不好公开驳金达的面子，更别说事后还拿这件事情在报纸上大搞宣传了，你这等于让金达的面子彻底扫地啊。如果换在以前的金达，他肯定无法接受莫克这么做的，就算是不去找书记吕纪，他也会直接跟莫克闹翻的。<br/>　　<br/>　　    但是金达没有这么做，不但没有这么做，还把这一切都隐忍下去了，他跟孙守义的的解释是为了维护班子的团结，还让孙守义要尊重莫克。<br/>　　<br/>　　    接下来就是云山县县委书记孙涛被抓到工作时间离岗的事情，这件事情于捷找莫克求情，莫克不但没给于捷面子，还把于捷说了一通，并趁机提出要整顿工作作风。对此于捷找到了金达，金达却只是做了和事老，并没有跟莫克对着干，相反还支持了莫克开展整顿工作作风的做法。<br/>　　<br/>　　    再下面就是旧城改造项目了，莫克接受了朱欣居中的请托，想要帮束涛拿下这个项目，就站出来主动把旧城改造项目的领导权给揽了过去。金达对此的态度也是全力支持莫克，事事为莫克的指示是从。这一次如果不是丁益联合中天集团玩了一下束涛，把束涛跟朱欣的见面照片给发出来，莫克就把项目给了束涛了。<br/>　　<br/>　　    在后面莫克去北京争取云泰公路项目资金失败，谁都知道责任不是在傅华身上，但是莫克为了推卸责任，抓住傅华没有安排他见郑老这个理由，对傅华大批特批，对此金达甚至连为傅华说句公道话都不肯。<br/>　　<br/>　　    上述种种，如果都归结在金达想要跟莫克搞好关系，想要维护班子的团结，所以才会这么忍气吞声，显然是不太可能的。金达没这么熊蛋包，他的性格也不会让他这么忍耐的。<br/>　　<br/>　　    但是金达就这么忍耐下去了，这其中一定是有别的原因的。一度孙守义对此感到有些困惑，难道金达真的因为他跟张林的争执而受到了教训，变得胆小了起来吗？<br/>　　<br/>　　    后来孙守义从别的渠道听到了一些情况，才恍然明白为什么金达会这么做的。东海省委的一位朋友私下跟孙守义谈到了吕纪把金达和莫克叫去批评的情况，那位朋友是当做政坛的一件趣谈轶事说给孙守义听的，他说了吕纪最后要求金达看好莫克，别让他在惹出麻烦了。<br/>　　<br/>　　    孙守义听到这里心中才恍然明白金达为什么会有这些变化的，金达并不是在忍让莫克，而是在忍让莫克背后的背景。金达为什么不去忍让徐正，不去忍让张林，也都是因为徐正和张林背后没有莫克的背景。莫克背后的背景不是别的，正是现在的书记吕纪。<br/>　　<br/>　　    不管怎么说，莫克能做上海川市的市委书记，是吕纪举荐的，莫克的身上就打上了吕纪的烙印。金达也许可以对抗莫克，但是他绝对不会去对抗吕纪的。<br/>　　<br/>　　    严格的讲，金达的背景其实也是吕纪。吕纪和金达一样都是前任书记郭逵的嫡系人马。郭逵去了北京，东海的政治势力就遗留给了吕纪继承，金达也是郭逵遗留下来的政治势力的一部分，他也必然被吕纪接收了。<br/>　　<br/>　　    于是金达和莫克在政治上的归类就都应该是吕纪的人马，他们应该是同一派系，也因为如此，即使莫克做的再离谱，金达也不好出来反对莫克，如果金达出来反对莫克，就等于是吕纪派系的人马在闹内讧。那样子的话，不但是莫克和金达会受到很大的影响，恐怕吕纪的权威也会被很大程度上的削弱。因为他的嫡系都在自己反自己，他这个书记还怎么去服众啊？<br/>　　<br/>　　    因此金达的忍让就不是做面子给莫克了，而是做面子给吕纪。金达维护莫克，实际上也是在维护吕纪，他不想让他跟莫克的冲突成为别人攻击吕纪的。<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
