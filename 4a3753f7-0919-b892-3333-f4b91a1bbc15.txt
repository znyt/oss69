　　李承乾被老娘到两仪殿的时候完全就是一头雾水。
<br/>　　要知道长孙皇后平时没事儿的时候根本就不会找他，而这次竟然破天慌的叫他去两仪殿，难道是出了什么事情？
<br/>　　带着心中的一份怀疑，李承乾踏进了长孙皇后的居所，却不想刚一进去，就发现了坐在一旁的长孙无忌，心中便已经隐隐猜到了些什么。
<br/>　　不过现在的确不是说开口说这个的时候，于是便在给老娘问了安之后，试按着问道：“母后您找儿臣？”
<br/>　　说实话，李承乾说这句话的时候，真怕长孙皇后接一句：“你舅舅找你，想让你给他两座矿山。”
<br/>　　不过还好，长孙皇后并没有那么直接，而是笑着问了他一些最近的琐事，然后才不紧不慢的说道：“乾儿，我听你舅舅说，这段时间钢材的生意不好做，你看看能不能给想点办法？”
<br/>　　嗯，如果这样的说的话，表示事情还有商量的余地，李承乾暗暗点头，心中也在佩服老娘会说话。
<br/>　　于是看向一旁的老长孙问道：“舅舅，钢材的生意不好么？我记得船舶和铁路上的生意，有百分之五十都是采购自咱们家中吧？”
<br/>　　“高明，你不知道，现在和以前的行前不一样了，家里现在是卖的越多，亏的越多，如果再这样下去，只怕舅舅就只能卖孙子了！”长孙无忌的演技还真不是盖的，那眼泪是说来就来，还没怎么样的，就先掬了一捧老泪，看的李承乾牙疼不已。
<br/>　　可是没办法啊，到底还是自己的舅舅，而且老娘还在一边看着呢，他总不能拍拍屁股就走，于是只能耐着性子问道：“舅舅何出此言？”
<br/>　　“高明，运费高的离谱了啊，你知道，自从八年前，各大世家全都在钢铁行业投资之后，钢铁的价格便是一跌再跌，现在眼看着就只有以前的七成了。”
<br/>　　“可是运费呢？运费没变啊，还是以前的那些，有些时候甚至还会更高一些，现在的老百姓精贵，你钱给的少了，他就去别人家干去了，反正大唐需要用人的地方又不止咱们一家。”
<br/>　　“铁路方面也是，你说这铁路越修越长，成品钢材的运输距离也就越远，随之而来的就是运费上涨，零零总总算下来，现在的运费几乎都要占到成本的一成到一成半了。”
<br/>　　长孙无忌扳着手指一样一样的给李承乾细数着炼钢的成本，从运费到源材料，从人工费到机械设备费，这一说就是大半天，把个李承乾说的目瞪口呆。
<br/>　　当然，这并不是说他在佩服老长孙的口才，而是实实在在被长孙无忌所计算的数字给吓到了。
<br/>　　毕竟他现在以前不像以前那样，经济会从头到尾的关注一件事情，钢铁厂这方面一般来说他只管分配计划，比如每年在国有钢厂购进多少钢铁，像长孙无忌这样的私营钢厂又要购进多少钢铁。
<br/>　　所以这一忙下来，他就忘了成本核算这一关节，硬是等到长孙无忌来找他，这才想起似乎老长孙并没有说错，这其中似乎真的有问题了。
<br/>　　想到这里，李承乾在老娘若有深意的目光注视下，想了想说道：“舅舅，您先别为这件事着急，此事还得容我再想想。”
<br/>　　而就在此时，长孙皇后也说话了：“高明，你有什么想法现在就说说吧，正好母后我也想听听。”
<br/>　　敢情这位母仪天下的皇后娘娘也听出来的，自家哥哥并不是在无理取闹，而是真的过不下去了，所以心中便也没了什么偏见。
<br/>　　李承乾被老娘这一说也没了办法，只能赔着笑说道：“母后，这事情可不是一句两句话就能说清楚的啊，成本核算要经过调查之后才会有机结果，孩儿必须等到结果出来了才能下结论。”
<br/>　　“乾儿，你可不要忘了，那是你舅舅，小时候可还抱过你呢。”长孙皇后盯了李承乾一眼。
<br/>　　这话说的可就有点重了，弄的李承乾好不尴尬，就好像他不准备办事儿一样。
<br/>　　不过转念想想也是，他的这个便宜老娘可是从小就与哥哥相仪为命的寄居在高士廉家里，若论起关系，兄妹间的感情那自然是不会差了。
<br/>　　若是李承乾真的要坑长孙无忌，除非长孙皇后不知道，若是知道了，那是说什么都不会同意的。
<br/>　　李承乾显然也意识到了这个问题，可就眼下来说，他也知道自己说什么老娘都不相信，于是只能叹了口气，对长孙无忌说道：“舅舅，您有什么想法就说一下吧，如果可以承乾无不答应。”
<br/>　　哎，这就对了嘛，一听李承乾如此说，长孙皇后的脸上露出一抹微不可查的笑意。
<br/>　　而长孙无忌倒也真没客气，略一沉默便接着说道：“高明啊，最近这不是要在东北搞开发么？你看能不能把那矿山给舅舅留一座……”。
<br/>　　“不可能！”一听长孙无忌竟然索要矿山，李承乾当时就急了，把头一摇说道：“舅舅，这事儿不是我李高明小气，实在是做不了这个主，您也知道现在大唐所有的矿山都是归国家所有，任何人不得私采，若是被我送出去一座，您觉得我要怎么向满朝文武交待？”
<br/>　　“这……”长孙无忌又迟疑了一下，将目光看向长孙皇后，显然是要让妹妹发话。
<br/>　　但这次李承乾并没有给老娘说话的机会，而是斩钉截铁的说道：“舅舅，您不用看了，如果您的想法是要一座矿山，这个我无论如何也不可能答应。”
<br/>　　“为什么？”长孙皇后强插一句问道。
<br/>　　“母后，眼下大唐的钢铁产量远居周边各国之首，而且我们的钢材质量也比他们要好的多，但您也知道，这东西是禁止外销的，大唐的钢铁每年外流的数量都被控制在一个范围之内。”
<br/>　　“而掌握这一数量的方法便是所有矿山都被国家所控制，所以我们每年都可以统计出大致的钢铁产量，用了多少，外流了多少。”
<br/>　　“可如果矿产私有，那么这就等于失去了控制，每年将会有大量的钢铁流失到境外，或许或可以换回一部分资金回流，但对于大唐的将来是十分不利的，所以孩儿无论如何也不会同意将矿产私有。”
<br/>　　李承乾没给长孙无忌留任何情面，丝毫没有掩饰怀疑他走私的想法，这让老长孙的脸上多少有些挂不住，可目的没达到又不想离开，于是只能尴尬的搓手再次看向妹妹。
<br/>(本章完)
