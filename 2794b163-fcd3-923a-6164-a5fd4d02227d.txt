　　登基当上掌门，改制教务，一切如鱼得水，直到遇到那个传说中的叶少阳……
<br/>　　道风虽然更强，但他是不管的，毕竟道风名义上早就不是茅山的人，在法术界得罪的人也太多，被人视为邪魔外道，不管他有多强大，各大门派都不会买他的账。
<br/>　　他不是人间法师。
<br/>　　但叶少阳不一样……一开始，他是有雄心壮志跟叶少阳好好斗上一番的，之前龙华会上，他当上了法术界的盟主，但也只是个空头盟主，叶少阳虽然不受人待见，树敌太多，但那是他做人原因，就连跟他最有仇的慈心师太等人，也都承认他是当今法术界第一强者。
<br/>　　之前，慕寒一直觉得，那是因为自己没混过法术界，大家不知道自己的实力，才会认为叶少阳是第一，这种想法一直持续到不久之前的“海眼防御战”。
<br/>　　他亲眼看到叶少阳是如何在绝境中创造出奇迹，带着一干手下、在千军万马中闯出去的经过。叶少阳天神一般的表现，深深震撼到了他，他知道自己做不到。
<br/>　　因此，他才想到要用计除掉叶少阳这个唯一的劲敌，半小时前，他制定的借刀杀人、坐收渔利的计划，虽然不光彩，但也非做不可。
<br/>　　但他内心，还是不甘心的。
<br/>　　“既生瑜何生亮，叶少阳，你凭什么比我厉害，凭什么在我之上，我不服啊！”
<br/>　　他握紧拳头，一拳砸在山石上。
<br/>　　山石的棱角硌伤了他的手，但他仿佛感觉不到疼，一口气往石头上砸了六七拳，热腾腾的血，顺着山石流下去。
<br/>　　慕寒又大声吼叫了几声，这一连串行为耗尽了他的力气，向后无力地躺在了石头上，大口喘着粗气。
<br/>　　天色一点点黑下来。
<br/>　　慕寒终于坐起来，刚迈步要下山，突然山石的背面，传出一阵讥诮的笑声。是个女人的声音。
<br/>　　慕寒蓦地一惊，声音是从山石后面传来的，但山后面是悬崖，能藏在后面的绝不可能是人类！
<br/>　　问题是，如果是邪物的话，她是什么时候来的，自己怎么全无察觉？
<br/>　　“你是谁！出来！”
<br/>　　一道人影，顺着山岩缓缓飞出来，穿着一身百花长裙，长发从双肩披散下来，裙子下摆露出两只雪白光洁的小腿和玲珑的小脚。
<br/>　　慕寒心头一荡，“摇光仙子？”
<br/>　　来人正是星宿海老大摇光仙子。之前在泰山顶上祭天仪式中，双方见过。
<br/>　　摇光仙子面带微笑地走来，转到山石这一面，顺势靠在了上面，望着慕寒，道：“张大掌门好有雅兴，一个人到这里来赏风景吗？”
<br/>　　慕寒没回答，拱手行了个礼，道：“仙子怎么会在这里？”
<br/>　　摇光仙子也没回答他的问题，反问道：“我早就来了，怕打扰了你，故而一直未曾相见。慕寒，你身为龙虎山教主，却这般怨天尤人，不是真人所为。”
<br/>　　慕寒心中冷笑一声，心想她这真是站着说话不腰疼，但不好明说，只说道：“多谢教诲。”
<br/>　　摇光仙子正色道：“我不是要教诲你，是想告诉你，只要有心，就没有办不成的事，他叶少阳能做到的，凭什么你做不到？”
<br/>　　提到叶少阳，再想到自己之前所做的事都被她看到，顿时有点恼羞成怒，哼了一声道：“仙子你说的轻松，实力不如人，又有什么办法。便是你，也未必是那叶少阳的对手吧？”
<br/>　　摇光仙子眼中晃过一丝怒意，但转瞬即逝，说道：“我为什么要跟他比，我又不是人间法师，对手也不是他。我只是替你不值，明明你才是天选之子，他只是个假的，如今却在你之上，你咽得下这口气？”
<br/>　　慕寒不由自主地握紧双拳，转过身去。
<br/>　　摇光仙子追过来，站在他对面，往下说道：“他靠的是什么，他不过是个假的天选之子，但你早年不露面，天下人都以为他是真的，个个都对他好，成全了他多少机缘，否则哪有今天，你底子比他好，难道你就这样自暴自弃，任凭他骑到你头上？”
<br/>　　慕寒牙齿咬得咯吱响，低下头去。
<br/>　　摇光仙子嘴角露出得意的微笑，但是很快慕寒便抬起头，脸上的愤怒完全消失，又变成了平时那副冷漠的样子。
<br/>　　“我觉得挺好，打不过就是打不过，何必强求。”说完微微躬身施礼，“我会山门还有事，少陪了。”
<br/>　　说完真的转身就走。
<br/>　　摇光仙子愣了几秒钟，急忙追了上去。“喂，你这就走了！我还没说完呢，我特意来龙虎山，可是为了你来的！”
<br/>　　慕寒站住，望着她。
<br/>　　摇光仙子摇了摇头，“行吧，被你看穿了，其实，本来我是想激将你一下，再跟你说的……不过你不得不承认，你不服，不甘心，很想把叶少阳踩在脚下。”
<br/>　　慕寒深吸一口气，道：“你如果坦诚相待，我也一样，你之前观察了我那么久，什么都看到听到了，还用问么。”
<br/>　　摇光仙子对他这语气有点不快，自己怎么说也是阐教金仙，地位比人间法师要高的多，哪怕对方是龙虎山掌教。不过自己有任务在身，也管不了这许多了，深吸一口气，说道：“那好，我之前说的，也都是实话，你与叶少阳之间，差距只在后天。”
<br/>　　慕寒这才静下心来品味这句话。
<br/>　　“平心而论，你后天的机缘也不少，有张无生教导你，茅山那些不外传的秘术，你也全都学了，九段光的法器也有几件，九天元辰尺比他的七星龙泉剑，也差不了许多。”
<br/>　　话锋一转：“但是你有的这样，他都有，他有的，你却没有。”
<br/>　　慕寒抬头看他：“什么我没有？”
<br/>　　“功法。”
<br/>　　慕寒一笑，“我还以为什么，功法的话，他茅山功法冠绝天下，但我龙虎山的奇门秘术也不差。”
<br/>　　“呵呵，看来你真是活该被他踩着，你连他强在哪里都不知道，我告诉你，他一年前还是天师，也就是修炼了大周天吐纳心法，实力才突飞猛进，一路攀升到现在这样！”
<br/>(本章完)
