　　战台之上，褚彦和谷辕江两人，全身气势磅礴。

<br/>　　两道风刃一左一右，威势极为强大，凌厉不可一世。
<br/>　　“杀！”谷辕江怒吼一声，手中光芒一闪，出现了一柄大枪。
<br/>　　大枪舞动，如同蜻蜓点水，瞬间刺出两枪。
<br/>　　犀利的枪芒，正中风刃。
<br/>　　轰！轰！
<br/>　　两声爆响同时炸开，风刃和枪芒同时溃散，化为一阵风浪席卷八方。
<br/>　　“有点实力！”
<br/>　　褚彦面色凝重，而后他身形冲天而起，双手如同抚琴一般，快速的甩动了起来。
<br/>　　一道道更加强大的风刃，疯狂的向谷辕江袭杀而去。
<br/>　　谷辕江毫不示弱，手中长枪同样疯狂刺出，无数的枪影爆射长空。
<br/>　　轰轰轰！！
<br/>　　爆响声震天，褚彦和谷辕江两人，你攻我往，不断的攻伐。
<br/>　　看台上，苏莫看的索然无味。
<br/>　　这褚彦，战力一般，他就算没突破到真神之境，也能击败对方。
<br/>　　战斗还在持续，褚彦和谷辕江两人，不断的交手。
<br/>　　两人的战力，均是发挥到了巅峰，威势极为强大，连神域天浑虚界的空间，都出现了丝丝肉眼难辨的细微裂痕。
<br/>　　没过多久，两人便斗了数百招。
<br/>　　这时，战台之上的两人，都已经毫无保留了。
<br/>　　褚彦乃是双生元神，而且都是风类元神，一种是风刃元神，一种是飓风元神。
<br/>　　而谷辕江，只是长枪元神，不过，其血脉之力极强，不清楚是何血脉，但是对于战力的加成，不弱于一种元神。
<br/>　　战斗了几百招之后，褚彦开始占据上风，将谷辕江压制了下去，胜利的天平开始倾斜。
<br/>　　“谷辕江，你必败了！”
<br/>　　就在此刻，战台之上，响起一声大吼。
<br/>　　之间褚彦整个人，突然化为一股猛烈的狂风，向谷辕江袭去。
<br/>　　狂风浩大，席卷整个战台，根本无处躲避。
<br/>　　狂风之中，充满了毁天灭地的威势，强大的凤属性法则之力，极为磅礴。
<br/>　　“嗯？”
<br/>　　苏莫见此，顿时双眉一挑，这一招倒是不错，威力不小。
<br/>　　“破！”
<br/>　　面对褚彦的强大一击，谷辕江怒吼一声，长枪如龙，暴刺而出，瞬间形成了一支长达百丈的枪影。
<br/>　　瞬息之间，枪影和狂风狠狠的撞击在了一起。
<br/>　　轰！
<br/>　　一声爆响，枪影崩溃，狂风浩荡天地，直接卷起了谷辕江。
<br/>　　啊！
<br/>　　谷辕江惨叫一声，随即向战台之下抛飞而去，重重的砸在了地上。
<br/>　　“哈哈哈！”
<br/>　　狂风化为了褚彦的身形，他大笑一声，缓缓的降落在了战台之上。
<br/>　　“谷辕江，承让了！”
<br/>　　目光凝视着刚刚起身的谷辕江，褚彦面带得色，朗声说道。
<br/>　　“愿赌服输！”
<br/>　　谷辕江脸色难看的说了一句，便转身飞回了看台之上。
<br/>　　“褚彦的实力居然如此强！”
<br/>　　“打败了谷辕江，也算是一流的天才了！”
<br/>　　“风属性之道很精深啊！”
<br/>　　看台之上，议论纷纷，褚彦的胜利，让很多人肃然起敬。
<br/>　　“褚彦的实力确实不错！”

<br/>　　“这还不是褚彦的真正实力！”骆妤轻声道。
<br/>　　“哦？”姬月松闻言惊讶，褚彦居然还隐藏了实力，这是要继续比斗下去吗？
<br/>　　“据闻所知，这几年，褚彦创出了一门绝招！”骆妤道，真神境的武者，每个人的绝招，大多都是自身所创，很少有学习前人的绝招。

<br/>　　“有意思，那就让褚彦先展示威风，败了之后我再出手！”姬月松笑着说道，他心中清楚，只要褚彦一直比斗下去，肯定会败的。
<br/>　　换成任何人都是一样，毕竟再上场的对手，会越来越强。
<br/>　　没有人敢说能一直不败，就算是他，也不敢说比斗几十场不败。
<br/>　　“沁儿，今日你也可以一睹褚彦的风采，比那个段惊天强多了！”骆妤转头，看向身边的沁儿说道。
<br/>　　“褚彦哥倒是隐藏的挺深！”沁儿微微笑了笑，她和褚彦自幼相似，没想到对方的实力，出乎她的预料。
<br/>　　……
<br/>　　“呵呵！”
<br/>　　战台上，胜利的褚彦春风得意，他轻笑一声，便看向管理台前的六位执事。
<br/>　　其中一名执事，手掌一抛，两枚储物戒便飞到了褚彦的面前。

<br/>　　嗖！
