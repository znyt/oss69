　　楚风眠带着白萱，冲入到了丛林之中去。
<br/>　　顿时一声声兽吼响起，无数妖兽猛然从丛林中冒了出来，向着楚风眠追杀而来。
<br/>　　“金纹神虎！”
<br/>　　“异瞳青鸟！”
<br/>　　在这无数妖兽之中，最为强大的，便是两尊实力足有生死境五重的恐怖妖兽。
<br/>　　金纹神虎，异瞳青鸟，皆是远古妖兽，血脉尊贵，实力强大无比。
<br/>　　“在这丛林中，居然是有着这样两尊恐怖妖兽。”
<br/>　　“这两尊妖兽中的任何一尊，我们都不是对手。”
<br/>　　司空轩，云公子二人看到这两尊妖兽，都是心有余悸的说道。
<br/>　　“幸好是让他们二人，前去引诱了妖兽，等这些妖兽引走，我们也就可以直接穿越丛林，前去颜古龙帝的洞府之中了。”
<br/>　　“这楚风眠也是愚蠢，居然自己去送死，不过正好，也少了不少麻烦。”
<br/>　　看着楚风眠引着无数妖兽，罗辰更是心中大笑起来。
<br/>　　当这些妖兽口中的诱饵，迟早会死在这些妖兽的口中。
<br/>　　罗辰的心中，阴冷的想到。
<br/>　　“只是可惜，不能亲手杀了他，不过死在了妖兽的口中，也可以了。”
<br/>　　丛林之中你，楚风眠带着白萱，便是一路上大摇大摆的飞着。
<br/>　　丛林中的无数妖兽，都被直接惊动，惊醒过来，纷纷追逐过去。
<br/>　　白萱回头，看到这两尊妖兽的一刻，白萱的脸色都是变的惨白。
<br/>　　“你为何要答应他们，今日就算是跟他们撕破脸皮，也不用听从他们的话来，前往引诱妖兽啊。”
<br/>　　白萱跟随这楚风眠飞行，看着背后追来的无数妖兽，有些愤怒的说道。
<br/>　　“可恶，这些人居然是直接过河拆桥，让我们来当做诱饵！”
<br/>　　“想要我来当诱饵，就凭他们那小小算计，怎么可能！”
<br/>　　楚风眠冷笑一声，向着白萱说道。
<br/>　　“走，先躲开他们的视线，找个地方，我自然会去对付他们。”
<br/>　　说这，楚风眠飞行的速度便是又变的快了许多，一下子将那些妖兽全部引走，前去了远方，令司空轩等人，已经是察觉不到了楚风眠的身形。
<br/>　　司空轩看着楚风眠飞遁的速度，似乎也是微微吃惊，不过很快他的脸上便是露出极为阴狠的神情来。
<br/>　　“这小子，倒是还有着几分手段，居然是真的可以引来妖兽，不过就算是他真的引走了妖兽，敢来到颜古龙帝的洞府中，我一样会杀了他！”
<br/>　　司空轩说完，便是向着其他说道。
<br/>　　“我们走，趁着这小子将那些妖兽引走的时候，我们先进入到颜古龙帝的洞府之中去。”
<br/>　　“好！”
<br/>　　其他三人，也是点了点头，几道遁光，便是直接向着丛林中心，飞了过去。
<br/>　　“终于动了。”
<br/>　　楚风眠的嘴角，露出几分笑意。
<br/>　　他的灵识，一直都是观察着那司空轩四人，正是在等待着这四人前去开启洞府的一刻。
<br/>　　楚风眠身形加速，一下子将那些妖兽，全部甩开。
<br/>　　“万物化龙！”
<br/>　　楚风眠低声轻喝一声，他的身躯，猛然开始转变，化为了一条真龙的样子。
<br/>　　这突然之间变化的一刻，让白萱的脸上，都是吃惊无比。
<br/>　　“什么？你是龙族？”
<br/>　　看着楚风眠，白萱不可置信的说道。
<br/>　　楚风眠这突然转变的样子，很像是一些龙族，恢复真身的样子，尤其是楚风眠身上的气息，都是跟真龙一族，没有丝毫的区别。
<br/>　　楚风眠道。
<br/>　　“不是，只是一些特殊的秘术罢了，你收敛气息，我也为你改造身形，躲避那些妖兽的追击。”
<br/>　　白萱听到楚风眠的话，也是急忙的将自己的气息，全部收敛了起来。
<br/>　　现在楚风眠跟她是一根绳上的蚂蚱，她也只能选择相信楚风眠。
<br/>　　楚风眠出声说这，他手心一动，一滴精血便是打了出去，融入到了白萱的身躯之中去。
<br/>　　顿时白萱的身躯，也是开始猛然变化，化为了一条真龙的样子。
<br/>　　“这是，什么秘术，居然是足以改变我的身体？”
<br/>　　白萱查看着自己的身躯，脸色震惊无比，她的身躯，在这一瞬之间，便是变化成了跟一条真龙，没有任何一点的区别。
<br/>　　其中的龙威，都是栩栩如生，简直是跟真龙一族，没有任何的区别。
<br/>　　“万物化龙术，只能保持半个时辰的时间，不过已经足够了。”
<br/>　　楚风眠平静的开口道。
<br/>　　“走，我们现在可以躲开妖兽的追杀，直接回去，等到那司空轩开启洞府的一刻，我们就出手，杀了他们。”
<br/>　　楚风眠言语之中，蕴含无尽杀意，他伪装半天，正是要完成这个计划。
<br/>　　那颜古龙帝的洞府前，还是有着一道禁制，这禁制应该只有司空轩一个人能够解开。
<br/>　　所以楚风眠才故意演了这样一出戏，让司空轩他们，认为楚风眠已经死了，等到司空轩开启的洞府的一刻，楚风眠便是出手，杀了他们，进入洞府之中去。
<br/>　　“原来这一切，都是你的算计？”
<br/>　　白萱听着楚风眠的话，也是微微一愣，没想到楚风眠算计的如此之深，从一开始，便是想好的一切计划。
<br/>　　楚风眠冷哼一声说道。
<br/>　　“不错，不然我为何会跟这些废物委曲求全，听从他们的命令，就这些人，我杀了他们，如同杀死几只猪狗一样简单。”
<br/>　　“看来，比起寒月漓，似乎是你，还要更加可怕。”
<br/>　　白萱缓缓的开口道
<br/>　　“这司空轩，居然是算计到了你的头上来，真是愚蠢，我选择跟你合作，倒是明智的多。”
<br/>　　提到司空轩的时候，白萱的话中，也是蕴含着无比杀意。
<br/>　　“走，我们先过去看看。”
<br/>　　楚风眠开口道，他的身形一动，便是跟白萱二人，如同两条真龙一般，向着丛林中心的颜古龙帝洞府前飞去。
<br/>　　颜古龙帝的洞府前，那司空轩四人，已经是穿越丛林到达了。
<br/>　　这丛林之中大部分的妖兽，都已经是被楚风眠引走，剩下的那些，被他们轻易斩杀。
<br/>　　“这周围，大部分的妖兽已经是被引走了，剩下的那些弱小的，都不到生死境的实力，不足为惧。”
<br/>　　云公子扫视了周围丛林一圈，笑着开口道。
