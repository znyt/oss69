　　苏莫来到旁边的宫殿之中，这座宫殿比宫主居住的宫殿略小，不过其内的风格完全一样。
<br/>　　苏莫找到了一间密室，进入了密室之中，殿门和密室的大门，全部关闭了。
<br/>　　密室之中，苏莫拿出了三支玉瓶郑重的打量了起来。
<br/>　　随即，他打开了青色的玉瓶，倒出了那枚天幻神丹。
<br/>　　这枚丹药，如同小指头般大小，通体略显透明，无色无味。
<br/>　　打量了一番，苏莫便将丹药吞了下去。
<br/>　　丹药入腹，没有任何的感觉，只是沉入了腹中。
<br/>　　“奇怪！”
<br/>　　苏莫心中惊疑，以前他从未接触过这种丹药，倒是不知道如何使用。
<br/>　　随即，他稍稍运站源力，试着炼化丹药。
<br/>　　不过，这天幻神丹，却是根本无法被炼化，如同是不朽的神铁。
<br/>　　但是，在他炼化的同时，天幻丹之中，却是散发出一种奇异的药力，将他的身躯笼罩。
<br/>　　“嗯？”
<br/>　　苏莫眼眸一亮，这种奇异的力量，好似受他的控制，能幻化出不同的形态。
<br/>　　这种感觉极为奇怪，这股奇异的力量，并不能改变他的任何容貌或者气息，但是，附着在他身上却是能让人看不透他。
<br/>　　“我明白了！”
<br/>　　略一沉吟，苏莫心中便明白了，这种天幻神丹，并不能改变他什么。
<br/>　　但是，通过天幻神丹，他却能让别人看到他想让人看到的形态。
<br/>　　换言之，只要苏莫一个意念，他在别人目光中的形态，可以是人类，可以是妖兽，甚至是花草树木。
<br/>　　他的身躯、容貌、气息，都不会改变，但是看在别人的眼里，却又是另一番模样。
<br/>　　“居然还有如此丹药，不愧是丹神！”苏莫心中暗赞，这种丹药真是玄妙。
<br/>　　事实上，苏莫不知道的是，这种丹药，就算是人德宫主炼制出来都极为困难，不仅配料难寻，而且难以炼制。
<br/>　　少倾，苏莫又倒出了紫色玉瓶的丹药，这一刻就是混沌源丹。
<br/>　　此丹比天幻神丹略大，乍一倒出玉瓶，便有一种澎湃的气息，从丹药内散发而出。
<br/>　　这气息之庞大，如同涛涛江河，无法形容。
<br/>　　“好精纯的混沌源气！”苏莫惊讶，这比五行源石融炼出来的混沌源气，还要精纯很多。
<br/>　　苏莫心中大动，他没有继续查看第三枚丹药，沉吟着要不要吞下这枚混沌源丹。
<br/>　　他不敢直接吞噬，毕竟，这丹药之中混沌源气太强，会然他瞬间爆体，强大的力量，估计连混沌不灭体都抗不住。
<br/>　　“如此源丹，只能吞服，药力应该不会全部爆开吧！”苏莫低声自喃，随即手中源力吞吐，试着开始炼化。
<br/>　　随着他炼化，混沌源丹之中，浑厚的混沌源气，立刻滚滚涌出。
<br/>　　“还好，可以循环渐进！”苏莫暗暗点头，这种丹药炼化之时，源力不会一次性全部爆开，而是缓缓散出，正适合修炼。
<br/>　　当然，这所谓的缓缓散出，其实速度非常快，只是相对于混沌源丹之中的源力，散出的算是比较慢而已。
<br/>　　随即，苏莫毫不犹豫，立刻将混沌源丹吞服了下去，开始了修炼。
<br/>　　源丹入腹，混沌源气滚滚，苏莫努力炼化，炼化为精纯的混沌源力，融入自身，强大修为。
<br/>　　……
<br/>　　天王宫，金碧辉煌的宫殿。
<br/>　　依旧是那座刻画着六色漩涡图案的密室，蟒袍人端坐在漩涡图案的中央，周身笼罩这模糊的光晕。
<br/>　　模糊的光晕，遮挡了视线，让人无法看清蟒袍人的面目。
<br/>　　在他的身前，伫立这一名白衣青年，五官英挺，俊朗非凡，正是无上神王天尘宇。
<br/>　　“帝一魂，你怎么看？”天尘宇望着蟒袍人，沉声问道。
<br/>　　他已经将轮回古城之中发生的事，告知了对方，根据他得到的消息，以及从弟弟天尘山那里了解的情况，这段惊天非常可疑。
<br/>　　“段惊天，就是苏莫！”蟒袍人帝一魂淡淡的说道，语气无比的笃定。
<br/>　　两道犀利的精光，如同两柄利剑，洞穿了光晕，爆射在了密室的墙壁之上。
<br/>　　“那我们怎么办？貌似人王宫有意帮助苏莫！”天尘宇问道，现在知道了苏莫就在人王宫，倒是不用在盲目寻找了。
<br/>　　“人王宫，是否在你的监视之中？”帝一魂问道，声音非常平淡，并没有多少激动之情。
<br/>　　“一只在监视，收买了一名内宫弟子！”天尘山道。
<br/>　　“苏莫是否还在人王宫之内？”帝一魂又问道。
<br/>　　“在，自从轮回古城回人王宫之后，一直没有出来！”天尘山道。
<br/>　　当他知道轮回古城的消息之后，便立刻传讯，让人密切留意了。
<br/>　　帝一魂沉默了片刻，开口道：“你先去一趟人王宫吧！面见人德，探探其具体态度！”
<br/>　　“嗯！”天尘山点了点头，他知道帝一魂心中的顾忌。
<br/>　　对付苏莫，他们不能大张旗鼓的杀去人王宫，逼迫人德交人。
<br/>　　不然的话，消息让拓跋青云知晓，那就比较棘手了。
<br/>　　再者，若是人德真的要庇护苏莫，他们杀过去也逼迫不了人德。
<br/>　　“多派些人手，严密监视人王宫，决不能让苏莫暗中溜出去！”帝一魂叮嘱道。
<br/>　　“放心吧！”天尘宇点了点头，便大步的离开了密室。
<br/>　　“苏莫啊苏莫，你蹦跶的再厉害，也改变不了自己的命运！”
<br/>　　望着天尘宇离开的背影，帝一魂低声自语。
<br/>　　“呵呵！”
<br/>　　随即，他轻笑一声，无奈的摇了摇头。
<br/>　　对于苏莫，他并未有太多的担心，对方逃不出他的手掌心。
<br/>　　就算人王宫守护，那又能怎样，在绝对的实力面前，最终人王宫也会屈服。
<br/>　　所以，他要尽快晋升高级神王，到时候，整个神域都没有苏莫容身之地，他想要融合轻而易举。
<br/>　　而距离突破到高级神王，他已经算是近在咫尺了。
<br/>　　外界，所有人都以为，他需要几万年，甚至是十几万年，才有可能晋升高级神王。
<br/>　　但是，别人又怎么可能知晓他的天赋，他只需要百分之一的时间，就能晋升高级神王。
<br/>　　到时候，融合两具分魂，他的实力暴涨，一举问鼎神域，超脱纪元也不是不可能。
<br/>　　少倾，帝一魂不再多想，继续修炼了起来，实力才是一切的基础。
