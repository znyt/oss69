　　“你是谁？”女子轻轻问道。
<br/>　　“我是地师家族五十六代传人，你是何方妖孽，在此作祟。”
<br/>　　“地师传人……”女子很认真地点了点头，“你跟你祖先一样，很英勇，你们的下场也会是一样的。”
<br/>　　女子轻描淡写，说完之后，不再理她，而是转头朝叶少阳看过去，说道：“你认识青牛祖师？”
<br/>　　“这个……认识，关系很好。”叶少阳临机一动，撒了个谎。他已经断定，眼前这个妹子跟李浩然一定有渊源，如果说自己被李浩然追杀，估计当场就会被她弄死，还是套个近乎再说。
<br/>　　女子微微一笑，“你撒谎，你是什么人。”
<br/>　　“茅山弟子叶少阳。”
<br/>　　女子表情没有变化，接着说道：“你刚才叫我苏沫，那是谁？”
<br/>　　“额，青牛祖师的师妹，我一个好朋友，跟你长的一模一样，可能认错人了。”
<br/>　　女子皱起眉头，似乎也很疑惑，随后微微点头，说道：“我叫碧清，记住了，哦，不过你记住也没用。”
<br/>　　“什么意思？”
<br/>　　“因为，你很快就要死了，记住我的名字，也没用了。”说完，碧清笑了。
<br/>　　卧槽！这可真是够装逼的。叶少阳在心里骂了一声，不过，他还是从碧清的话语中感受到了一丝寒意，右手捏紧了七星龙泉剑，朝边上几人看去，大伙都是以震惊为主，不过陈晓宇和卢晓清有些按捺不住的样子。
<br/>　　叶少阳了解这种感觉：
<br/>　　面前这个女子，那么美，看上去人畜无害，无形中给了他们一种不是很强的印象，但是却口出狂言，这就让他们很不爽了，随时准备出手。
<br/>　　“怎么办？”毛小方还算比较理智，知道从炼尸缸里爬出来的，不管长的多么可爱美丽大方，都绝不是善茬，于是轻声问道。
<br/>　　妙心和梧桐互相看了一眼，随后说道：“我的祖先，是你杀的？”
<br/>　　“是。”碧清回答得很干脆，“你祖先，还算不错，布阵下这九十八根索魂血线，将我封印在这炼尸缸内，不过……也只是关了我几十年而已。”
<br/>　　妙心听她这么说，心中的震惊无以复加，喃喃道：“难道你早就可以破解封印？”
<br/>　　“我早就冲开了封印，我不走，只是想继续在炼尸缸里修炼……有这么好的阴巢给我用，我为什么要出去？不过你们前来打扰，想要把我重新封印起来，我就不能不出手了。”
<br/>　　叶少阳心中恍然，之前还在纳闷这女鬼既然能轻松撕开阵法禁制，为什么不早就这么做，原来……只是为了躲在炼尸缸里修炼，当下想到了什么，问道：“看在大家都是朋友的份上，回答我一个问题啊，那些我们在地面上看到的鬼魂，跟你有什么关系？”
<br/>　　“你说他们吗？”妙心抬头看着那些被龙气裹挟、在中间漂浮着的鬼魂，说道：“我用这通天彻地大阵，将他们每晚送出去，替我吸取人间的阳气而已。”
<br/>　　原来……一切是这么回事。什么百鬼夜行，封印炼尸缸，这一切都是人间故意弄出来的，人家只是待在这里自己不想走而已……之前那个蛤蟆精还傻呵呵地以为是他的存在，镇压了古墓中的邪物。等等，她这么做，难道就不怕被法师发现，找过来灭了她？
<br/>　　叶少阳随后苦笑，他想到了这问题的答案：她对自己的实力，太过自负，完全不惧怕被人发现，就像眼前这样，她面对着几个准地仙和天师、禅师，丝毫无惧。
<br/>　　牛逼的家伙不一定都自负，但是自负的，一般都挺牛逼的。
<br/>　　叶少阳深吸一口气，问道：“那天我们在蛤蟆谷里遇到的袭击，也是你的手下？”
<br/>　　“是我一缕残念而已。”
<br/>　　一缕残念……居然就那么难对付，她的真身现在就在这里，叶少阳真的不知道说什么了，目光在四周流转，思索着逃走的路线。
<br/>　　“青牛祖师，现在哪里？”碧清继续追问叶少阳。
<br/>　　叶少阳知道她愿意跟他们聊这么多，并不是为了满足他们的好奇，只是想要打听青牛祖师的下落，当下故意说道：“你让我想一想啊。”
<br/>　　随即做出思考的样子，其实也不是装，不过他思考的倒不是李浩然在哪，而是接下来应该怎么办。打吗？好像是打不过的，那就只有逃命了，可是要怎么样才能逃出这里？
<br/>　　叶少阳冥思苦想，碧清最后等不及了，一步冲到他面前，道：“快说。”
<br/>　　“额……我不知道。”
<br/>　　“不知道？”碧清的表情一点点变冷。
<br/>　　不知道，不知道你想个毛啊？
<br/>　　叶少阳摆着双手，说道：“我之前在一个地方遇到过他，我可以帮你去找他，等找到了再来告诉你？”
<br/>　　碧清嘴角弯起一抹冷笑，“那我可以自己去找他，要你做什么？”
<br/>　　抬起芊芊玉手，朝他指了过来，叶少阳猛地一闪，躲到了人陈晓宇身后，道：“师兄，救我！”
<br/>　　陈晓宇一怔，没等开口，碧清已经奔着他去了。
<br/>　　既然是“师兄”，一定比他本人要厉害，一般来说，不管是牛逼的人还是邪物，都喜欢先对牛逼的出手，叶少阳正好利用陈晓宇试试眼前这个家伙的实力。
<br/>　　“叶少阳你妈——”陈晓宇一句话没骂完，碧清已经冲过来，对他伸出了一只指头，陈晓宇内心也是不爽的，拔出拂尘，冲了上去。
<br/>　　“我倒也想看看你究竟是什么东西！”
<br/>　　“对，师兄，跟她干，我相信你！”叶少阳在边上助威。
<br/>　　陈晓宇虽然人品很渣，但毕竟是星月奴的弟子，准地仙牌位，法力十分不弱，拂尘凌空横扫，几道灵光组成了一个赤金色的符印，朝碧清撞过去。
<br/>　　碧清只用了一根手指顶上去，陈晓宇立刻感到一股无形的力量，从他指尖的地方蔓延开来，将符印牢牢托住，丝毫不能前进，不光如此，最让陈晓宇不敢相信的是，自己想要点碎符印，让灵力爆发出去，但是符印好像被定住了，完全不受自己控制。
