        &lt;h3&gt;第三卷第一百四十四章  探望陈子澜&lt;/h3&gt;
<br/>
<br/>        潘青松随着陈子澜单位的同事一起到了甸城。
<br/>
<br/>        他们一同去的四个人，俩个人安然无恙，陈子澜躺在重症监护室，郑培已经失去了生命，甸城调查的结果是司机打瞌睡，发现车速过快，踩刹车已经来不及了，车冲下了山体。
<br/>
<br/>        司机受伤还能轻一点，但是也是昏迷不醒。
<br/>
<br/>        他们问明了情况，商议都暂时撤回，那边的事情交给廖国华负责，有什么问题了及时和泉城方面沟通。
<br/>
<br/>        潘青松在甸城等着陈子澜醒来之后才回家。
<br/>
<br/>        潘青松期间给张函说了甸城的情况，并且也给陈子昂打电话了。
<br/>
<br/>        陈子昂想要去甸城看陈子澜，被潘青松坚决的制止了，潘青松是这样说的：“子昂，你三姐只是出车祸受伤了，问题不大，我等一个礼拜之后就带着她回家。你千万不要来。”
<br/>
<br/>        “三姐夫，可是我心里始终放不下啊。”陈子昂担心的说。
<br/>
<br/>        “子昂，我不希望你来看你三姐是有原因的，你三姐的工作性质你知道，你来了会给我们添乱的你知道吗？”潘青松加重语气。
<br/>
<br/>        “那好吧，三姐夫，你照顾好三姐，你要不要我找个人帮忙？”陈子昂想到了谷强，也许谷强能帮帮忙？
<br/>
<br/>        “你想找谁帮忙？”潘青松没有想到陈子昂在甸城还有关系，便多问了一句。
<br/>
<br/>        “我能找的也就是谷强了，我只认识他。”
<br/>
<br/>        “谷强？”
<br/>
<br/>        “对，谷强，他是甸城最大企业家的儿子人，应该关系不错的。”
<br/>
<br/>        “别费那么多心了，你姐姐现在只是在医院躺着，又没有什么大的问题，不要打扰到那么多人，欠人家的人情不好。有我在呢，你放心，我会24小时守在你姐姐的身边。你不要给家里人说了啊，省得大家担心。”潘青松看着躺在监护室的妻子说。
<br/>
<br/>        从今天起，他就会守在妻子的身边，24小时陪着，所有的药品他要亲自过目。
<br/>
<br/>        隔着玻璃，他会给妻子说话，说他们相见时他的心情，说他们在一起的点点滴滴。
<br/>
<br/>        “子澜，你一定要醒过来，醒过来我就带你回家。”一天过去了，两天过去了，第三天陈子澜终于醒来了，潘青松捂住自己的脸，蹲在门口流泪。
<br/>
<br/>        五天后，陈子澜转到普通病房，潘青松已经胡子拉碴，整个人瘦了一大圈，他熬了三天三夜，但是他还不能休息。
<br/>
<br/>        “子澜。你终于醒了，你吓死我了。”潘青松轻轻的抚摸着陈子澜的头发，如果没有你，我不知道我的世界会是怎么的，只怕是一片灰色，再无色彩。
<br/>
<br/>        “青松。你怎么成这个样子了？我睡了很久吗？”醒来的陈子澜看着眼窝深陷的潘青松，有些心疼。
<br/>
<br/>        陈子澜想伸手摸一下潘青松的头发，身上却撕裂般的疼。她身上伤痕累累，左腿粉碎性骨折。
<br/>
<br/>        “你昏迷了几天几夜。”潘青松心疼看着陈子澜：
<br/>
<br/>        “子澜，你受苦了。”
<br/>
<br/>        “似乎很严重？”陈子澜苦笑了一下。
<br/>
<br/>        “最起码得在床上躺半年。”潘青松轻轻的用小汤勺将白开水递到陈子澜嘴边，滋润着陈子澜那干涩的嘴唇。
<br/>
<br/>        “我以为再也见不到你了。还好，老天爷还算仁慈！！”陈子澜回忆起出车祸的那一瞬间看着潘青松微微的笑着。
<br/>
<br/>        俩个人相视而笑，真好！我们彼此都还在。
<br/>
<br/>        潘青松联系了西城的骨科医院，带着陈子澜回到了西城。
<br/>
<br/>        只要回到西城，潘青松心里就安定下来了，陈子澜被安排在单人病房，有陈子悦照顾着，他在病房里另外的一张床上睡得昏天黑地。
<br/>
<br/>        陈子慧、陈子乔、陈子建都来看陈子澜了。
<br/>
<br/>        陈子霖和陈子昂也从北京飞了回来。
<br/>
<br/>        大家看着躺在床上的陈子澜心里都是唏嘘不已，他们之前并没有感觉到生命的无常，更没有感觉到陈子澜工作的危险性，只是觉得陈子澜的工作也就是一个普通的警，察，那些危险的事情都是故事里的，可是看着陈子澜伤痕累累的躺在床上，他们才惊觉陈子澜干着一份危险的事业，而他们现在的幸福生活只是有着陈子澜这样的人在背后为他们保驾护航而已。
<br/>
<br/>        看着围在自己身边的亲人，陈子澜眼眶湿润，能够活着看见自己的亲人真好，小小的病房围着她的亲人们，温暖而热烈。
<br/>
<br/>        当张函推开门的时候，潘青松还在昏睡着，陈家的姊妹都在。
<br/>
<br/>        张函看到大家都在，他微微怔了一下，然后看着大家说：“你们都在啊？”
<br/>
<br/>        “你来干什么？”陈子霖并不知道陈子昂已经原谅张函了，语气不善的问。
<br/>
<br/>        “我来看看三姐。”张函随口就答应着。
<br/>
<br/>        “这里没有你的三姐，你走吧。”陈子乔也说话了。
<br/>
<br/>        “我......子昂。”张函求助陈子昂。
<br/>
<br/>        “你是来找三姐夫的？”陈子昂想到孩子，给张函找台阶下。
<br/>
<br/>        “是的，我也是来看看三姐的。”张函看着陈子昂，还是子昂了解他。
<br/>
<br/>        “三姐夫睡着了，诺。三姐在床上躺着，有什么事情等三姐夫醒了你再来吧。”陈子昂怕张函呆下去大家都难堪。
<br/>
<br/>        “也好，子昂那我能不能和你说几句话？”张函厚着脸皮问。
<br/>
<br/>        “你要说的我都知道了，人各有志，你就不要多说了。”陈子昂是知道张函要劝她不要去甸城，而且她听说张函的老婆也去了甸城，张函是怕她要面对陆玉吗？
<br/>
<br/>        事情已经过去多少年了，他们都已经不再年轻，难道张函的老婆还会认为她还能和张函有什么个瓜葛不成？
<br/>
<br/>        往事成云烟，难道张函和他的老婆这之间连这一点信任都没有了？
<br/>
<br/>        “子昂。”张函有些艰难的喊了一声陈子昂。
<br/>
<br/>        “张总，谢谢你能来看我三姐，如果没有别的事情，我们兄弟姐妹想说说话。”陈子昂已经对张函心里生出了一丝丝反感。
<br/>
<br/>        你都已经有老婆了，孩子你也认了，难道你还不满足吗？
<br/>
<br/>        陈子昂现在实在是讨厌那种想家里彩旗不倒，室外还想彩旗飘飘的男人，虽然她身边不乏那样的人，但是现在的她，已经不是当初的那个她了，无论是从情感还是生活，她完全有了自己的人生观和价值观。
<br/>
<br/>        宁缺毋滥！两个相爱的人感情应当是专情，忠贞的！
<br/>
<br/>        她不喜欢纠结在前尘往事里，过去了就过去了，虽然有美好，有怀念，也有留恋，但是那都是过去式，人是应该朝前走的，不应该将自己困死在过去里。
<br/>
<br/>        所以对张函见到她还表现得很在乎她的那种感情她都感觉羞耻。
<br/>
<br/>        张函无奈的看了一眼陈子昂和大家说了一句：“对不起，打扰大家了，三姐，你好好养伤。”
