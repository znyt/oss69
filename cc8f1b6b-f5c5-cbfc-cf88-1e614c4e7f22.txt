　　涂小安并不想猫精海棠在失去理智，因为他还有心中的疑问要询问它。
<br/>　　顿时看着猫精道：“你冷静，有我在，没人能抓走你”
<br/>　　猫精海棠的毛发依旧炸立，并没有选择完全相信涂小安。
<br/>　　但涂小安的话，让白楚艳身边的冷酷男子失去耐心，脸上泛着一抹森然，整个人一下子寒冷阴鸷下来，幽幽的盯着他，道：“白家要的东西，没人能护的住，你算什么玩意？不知死活”
<br/>　　他的任务是将这只猫精带回去，可偏偏有蝼蚁自己找死，那么，他不介意踩死一只。
<br/>　　一句话落下，却见，任海龙一下子朝着涂小安而去，速度很快，可谓是欺身上前，迅猛如雷。
<br/>　　他是白家少主的贴身侍卫，要知道白家少主白君然可是即将要准备突破三阶生灵的天才。
<br/>　　能成为这等天才的侍卫，没点硬件实力，岂能有资格。
<br/>　　此刻，他施展的就是一种为《剃子步》的身法，速度奇快，如若幻影，一步等于十步位移。
<br/>　　快，太快，任海龙的速度比世界飞人的速度还要快上许许多多。
<br/>　　涂小安只觉得眼前一花，没有多想，就极速后退，然后，让他诧异的是，任海龙就如鬼灵一样，站在他身前了，与之他面对面。
<br/>　　“告诉我，你怎么护猫精”任海龙一抹冷笑质问，一拳堂堂正正，正面轰击而去。
<br/>　　这一拳，出的让人看不到，任海龙习惯性的仿佛看到涂小安整个人倒飞出去在途中，口吐鲜血的画面。
<br/>　　一个呼吸后。
<br/>　　涂小安的速度更快，骤然三丈外，对方用的是功法武功，这是人类修武者的优势。
<br/>　　刚退开，对方的一拳自然落空，但任海龙脸上没有失落，反而笑意更浓。
<br/>　　轰..．
<br/>　　涂小安胸口位置猛然一震，连连倒退数步。
<br/>　　这一拳看似落空，但无形的力量还是打中了他？
<br/>　　隔山打牛？
<br/>　　“叮...宿主的人形态损坏超过百分之六十”
<br/>　　损坏到百分之八十，人形态就会溃散，恢复蛇躯。
<br/>　　对方这一拳让涂小安的人形态损坏百分之十，可见力量之大，普通人要是中了，那么全身都会散架。
<br/>　　嗯？
<br/>　　这小子没吐血，只是倒退了几步？任海龙微微吃惊。
<br/>　　明明清瘦的身板，居然硬抗自己一拳而没有吐血，还真是稀奇。
<br/>　　“不错，你还是第一个受我一拳，而安然无恙的人”
<br/>　　他看似在夸奖涂小安，但这种夸奖是带着轻蔑跟鄙夷。
<br/>　　涂小安蹙眉的看着任海龙，一个意识洞察悄然的丢了过去。
<br/>　　“叮...任海龙，二十三级，力量觉醒者”
<br/>　　力量觉醒者？
<br/>　　力气大的可以隔山打牛，是不一般了。
<br/>　　但涂小安旋即微微一笑，故意嘲讽对方：“要怪就怪你的力量太小，拳头打在我身上都没感觉”
<br/>　　“哈哈...”
<br/>　　任海龙闻言，好像是听了什么天大的笑话，眼中丝丝冷意：“我怕我力量在大一点，会把你打死”
<br/>　　“求死”涂小安无所谓的耸了耸肩膀。
<br/>　　任海龙杀意如狂风骤射：“那我便送你去死”
<br/>　　两人中间相隔三丈，任海龙却原地不动，一拳隔空打出，恐怖如斯。
<br/>　　空间顿时剧烈的震荡起来，仿佛要被粉碎，大方位的波动涟漪起来。
<br/>　　涂小安的眼睛微微一眯，发光发亮，他看到空气的无形中，有一只沙包大的拳头直取自己的面门。
<br/>　　又是一招隔山打牛，看似对方打出的拳头跟他挨不到边，但无形中已然形成了一个大拳头轰来。
<br/>　　“给我破”
<br/>　　涂小安不屑的笑了笑，妖治的眼睛浑然一眯，无形的意识力笼罩出去。
<br/>　　轰..．
<br/>　　空中忽然一声爆响，一股气浪席卷出去，沙包大的拳头不复存在。
<br/>　　任海龙脸色顿变，：“小子，我到小瞧你了”
<br/>　　不由得，‘咯吱咯吱’，任海龙捏了捏拳头，开始正色了起来，这少年看来不是一个软柿子。
<br/>　　转瞬。
<br/>　　他再次动了，几乎是眨眼，就来到涂小安的面前。
<br/>　　“小子，给我去死吧....”任海龙的眼神中闪过一抹寒光，一声低喝，直接抬起拳头，朝涂小安轰砸而去。
<br/>　　拳头前进，摧枯拉朽，空气阵阵爆响，直击涂小安面门，不愧是力量觉醒者，当自身的力量足够强大，速度也非常快。
<br/>　　眨眼之间，拳头距离涂小安的脸面只有一尺的距离了。
<br/>　　任海龙的狞笑越发浓郁，他仿佛已经看到涂小安这颗头如西瓜一样的爆裂。
<br/>　　不远处的白楚艳想看看，这轻松打败自己的少年，到底有多强，他浑身都透着一种神秘，让人一时半刻看不透。
<br/>　　任海龙的实力有多强，白楚艳是非常清楚的，能挡得住他的一拳，足以自傲，而对方已经挡了他二拳。
<br/>　　要是这一拳还接住，那么白楚艳海真的想将这个人吸收到白家，为白家效力。
<br/>　　任海龙拳已到，就在这时，涂小安噙着一抹放荡不羁的笑，不准备后退了，抬起了手，同样是拳头，迎着任海龙的拳头，对上。
<br/>　　“不知死活，你真的找死”任海龙倒是没有想到涂小安这次不但不退，还敢出拳。
<br/>　　可笑之极，莫非他知道自己躲无可躲。
<br/>　　他的拳，可实锤，可无形，皆是杀招。
<br/>　　电光火石之间。
<br/>　　“碰....”
<br/>　　碰撞了，两拳结结实实的砸在一起，一股气浪如强风席卷出去。
<br/>　　“咔。”
<br/>　　伴随的还有一道骨裂的声音。
<br/>　　白楚艳摇了摇头，任海龙可是力量觉醒者，全力的一拳打出去，不下万斤，这人也敢用自己的拳头去接。
<br/>　　悄然的，她想起了下午自己用腿，对方也用腿打败了她。
<br/>　　现在遇到一个用拳的，他也出拳，他当自己是什么，无所不能？
<br/>　　那可是力量觉醒者，浑身充满了爆炸性的力量，这种修武者，要是修炼到了极致，哪怕面对的是一座小山，都能一拳轰塌。
<br/>　　白楚艳将这一幕看在眼里，不禁失望的摇了摇头，逞能的莽夫罢了。
<br/>　　然而，接下来的一幕却让白楚艳的失望直接转变为错愕和震惊。
<br/>　　只见任海龙没来由的喷出一口血，再看他的脸，苍白到和死人脸一样，狰狞的都看不出人形了。
<br/>　　而涂小安，纹丝不动，神情自若。
<br/>　　“我都说，你的拳头力量太小了，现在相信了吧”他妖治的眼眸闪过一丝玩味。
<br/>　　跟我玩力量，涂小安的本体可是三十米的惊天巨蟒，力量何其大，要不是人形态限制了他绝大部分的力量，就刚一拳，这个人就可以飞出天际去了。
<br/>　　而不仅仅是骨骼碎裂。
