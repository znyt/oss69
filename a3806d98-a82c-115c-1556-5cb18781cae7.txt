　　不知不觉。
<br/>　　林峰在基地外就度过了三天时间。
<br/>　　这三天。
<br/>　　主要就是找怪物浪费了最起码两天。
<br/>　　毕竟这些任务提供的线索都是最新的出没地点，有的怪物丧尸他是会移动的，所以林峰往往都要自觉去寻找。
<br/>　　第三天。
<br/>　　林峰在一家咖啡店内坐着休息。
<br/>　　吃着八宝粥喝着脉动。
<br/>　　林峰在整理空投箱。
<br/>　　三天下来，不多不少，正好二十一个D级空投。
<br/>　　“把这个多的一个给开了。”
<br/>　　咔擦。
<br/>　　林峰打开这个空投箱。
<br/>　　一盒退烧药。
<br/>　　林峰丢入物品栏里。剩下的二十个林峰全都给他融合为C级的空投。
<br/>　　“开空投咯！”
<br/>　　每一次林峰开C级空投都有别样的兴奋。
<br/>　　咔擦。
<br/>　　打开第一个C级空投。
<br/>　　“好家伙。”
<br/>　　入眼就是十发火箭弹。
<br/>　　“及时雨啊，正确这玩意。”
<br/>　　除了火箭弹之外，还有无人机挂载的空对地导弹也是足足八发。
<br/>　　“不错，弹药得到补充了。”
<br/>　　“看看第二个什么玩意。”
<br/>　　林峰掏出第二个空投箱。
<br/>　　咔擦。
<br/>　　空投箱内放着一本书。
<br/>　　《忍术：肉弹战车》
<br/>　　“我了个擦，这不是火影里面那个小胖子叫什么秋道丁次的忍术吗？”
<br/>　　林峰一看，这玩意不错，正好可以给胖子那个大肉球，两百斤胖子来了肉弹战车那简直就和雪崩一样的威力。
<br/>　　狗子不是异能者，那么很简单，林峰就要先富带动后富，把自己身边的人一个个的都变成‘异能者’，胖子就是第一个。
<br/>　　“到时候狗子肯定要乖乖献出菊花来了！！嘿嘿！”
<br/>　　林峰把这本《忍术：肉弹战车》丢入物品栏。
<br/>　　今天两个C级空投还算不错。
<br/>　　林峰直接准备打道回府了，这一次任务完成，光是这几个猎杀任务给的报仇，少说也是几百颗三阶晶石到账了，更不谈林峰在任务之中猎杀所得。
<br/>　　到时候去房产中心看看有没有便宜的房子先给他买了再说。
<br/>　　听苏云老师说，房子的价格似乎每天都在涨。
<br/>　　“这尼玛和末世前一模一样，一群专家说房价要调控要跌，跌个毛，13年买不起的，15年更买不起，16年还买不起的，一辈子都租房住吧。房东让你今天滚，你岂敢留到明日。”
<br/>　　林峰决定先给他买一套再说，几百颗三阶晶石，买是可以买的，都是些小户型，三十平的这种。
<br/>　　不过林峰有个荣誉勋章，这玩意可以打折，到时候可以搞一套大一点的，最起码也要八九十平吧。
<br/>　　林峰想到此立即就朝基地奔去。
<br/>　　走到半路上，林峰忽然愣住了。
<br/>　　这是个小镇子。
<br/>　　镇子里都是丧尸。
<br/>　　林峰在这个镇子里看到了熟人。
<br/>　　正是之前那个跑掉的古家古月。
<br/>　　对方今天没穿超短裙而是穿了个牛仔热裤，屁股都露在外面，穿热裤居然还穿了个黑色丝袜，简直是骚的无极限。
<br/>　　在她身边跟着几个保镖，这群保镖实力不弱，身手也都很好，足足都是三阶的。
<br/>　　这个古月也在猎杀丧尸，实力几日不见，已经抵达六阶了。
<br/>　　“有意思。”
<br/>　　林峰立即就朝古月那边冲了过去。
<br/>　　古月正在杀丧尸，杀的正起劲，忽然背后一个人出现了。
<br/>　　直接把他的两个保镖都给放到了。
<br/>　　古月回过头一瞧，顿时意外道：“是你？烤肉小子！”
<br/>　　“烤肉小子？这是什么称号。”
<br/>　　林峰看着她，“骚货，没想到这么快又见面了。”
<br/>　　古月听林峰叫她骚货一点都不生气，反而还高兴。
<br/>　　“看来你实力又增强了，我这两个保镖居然无声无息就被你干掉了。今天莫非你是来杀我的不成？”
<br/>　　“怎么，你不怕？”
<br/>　　林峰目光微微眯起，他今天来就是斩草除根的。
<br/>　　“若你是担心我会把千年僵尸墓里的事情告诉张家和古家的话，那你大可不必担心，我要说出去，我早就说出去了，就算我不知道你是谁，但是我见到过你的脸，你别忘记了，基地内可是有监控探头的，像你身手这么好的人，肯定会前往曙光基地，到时候只要调用一下他们在基地内的关系，不出三天，就会把你的信息给查到。”
<br/>　　古月道。
<br/>　　林峰眉头微微皱起，他还真的是忘记了这事情。
<br/>　　基地大门口的确有监控，要查的确可以查到。
<br/>　　“放心吧帅哥，我没有透露你的任何信息，原因很简单，我古家被他张家和王家欺负的都要抬不起头了，能够看到有个人杀了他们的人，虽然我也死了点人，但是死的都是些废物罢了，不值一提，我自然是乐见其成。”
<br/>　　古月说着说着就是扑了上来，丰满的胸部在林峰身上蹭了蹭。
<br/>　　然后一手就是掏在了林峰裤裆里。
<br/>　　“当然了，最主要的是，你这个地方特别大，是我喜欢的尺寸。”
<br/>　　古月极其魅惑道。
<br/>　　“尼玛的，猴子摘桃？”
<br/>　　林峰被对方给弄了一下弄的都硬起来了。
<br/>　　“哇，变大了。”
<br/>　　古月顿时媚眼如丝。
<br/>　　“挑逗老子？”
<br/>　　林峰一巴掌就是拍在她屁股上。
<br/>　　啪的一下。
<br/>　　把对方屁股都给拍红了。
<br/>　　“告诉你，美人计对我没什么用。”
<br/>　　林峰后退了几步。
<br/>　　“你确定没什么用？那你的那个地方为什么越来越大？”
<br/>　　古月笑咯咯道。
<br/>　　“这个骚货。”
<br/>　　林峰拔出了斩灵剑就是要给她砍了。起反应归起反应，那是男性荷尔蒙的正常生理反应，是个男的被这样摸都会起反应。
<br/>　　但是林峰心里对这个古月无比忌惮，太骚了，越骚的女人越不好对付。油嘴滑舌的，所以一杀百了。
<br/>　　“且慢。”
<br/>　　也就在这时。
<br/>　　一个人忽然出现了。
<br/>　　这个人来的极其的意外。
<br/>　　林峰侧头瞧去就看到一个男人走了过来。
<br/>　　目光顿时眯起，“七阶的？”
<br/>　　“风哥哥，你怎么来了。”
<br/>　　古月看着对方道。
<br/>　　“我不来，你就死了。这家伙刚才动了杀心。”
<br/>　　被叫做风哥的白衣男子目光落在林峰身上，若有所思道：
<br/>　　“我没有猜错的话，你杀了战门的人吧，我能够感觉到，你身上有追魂索命蛊的气息。”
<br/>　　“你怎么知道？”
<br/>　　林峰略感意外。这个家伙居然看一眼就知道这些事情？莫非他是战门的？
<br/>　　古月笑道：“我古家背后的势力叫做星陨阁，风哥哥就是星陨阁的成员。”
<br/>
