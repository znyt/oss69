<br/>				    冷霜带着人上了酒菜后便行了一礼后退下，轩辕墨泽端起酒壶倒了两杯酒，递了一杯给她：“这一年多在外面，我无时不想着尽快将事情办完好回来。”<br/>　　<br/>　　    “你不在的这段时间，我也遇到了不少事情，尤其是被传送到了那悬浮之地的那一年，更是让人意想不到。”<br/>　　<br/>　　    她笑了笑，道：“虽遇到了不少麻烦，但实力也因此而提升，而且到最后还收服了那么多的修士，得了那悬浮之地，果然福祸是相依的。”<br/>　　<br/>　　    轩辕墨泽喝了口酒，道：“你说的这个悬浮之地，也幸好是当时那四位仙帝出手帮你一把，否则，以你之力想要将那悬浮之地收复还有些困难，不过，这也是那四位仙帝对你的一个示好，毕竟，你这逆天的天赋，谁知还会成长成什么样子？”<br/>　　<br/>　　    闻言，凤九轻笑着：“世人说我是鬼才，你也说我天赋逆天，但我觉得，若论起来，我还是远远不及你。”<br/>　　<br/>　　    他的强大，每一次都让她惊讶，每一次，都有一种惊喜与不可思议的感觉。<br/>　　<br/>　　    听到她的称赞，他的心情似乎很好，唇角都勾了起来，深邃的目光落在她绝美的容颜上，低沉的声音带着磁性的道：“我若不强大，如何保护得了你？”<br/>　　<br/>　　    身为男人，尤其是她凤九的男人，岂能弱小？他无心争这天下霸主，无心成为这世间王者，他只愿守护在她的身边，在她需要他保护时，可以保护得了她。<br/>　　<br/>　　    因此，哪怕他无逐鹿天下的雄心，但也会提升自己的实力，这不为别的，只为，保护他所想保护的人。<br/>　　<br/>　　    两人在院中闲聊着，一边喝着酒，享受着这难得的宁静与清幽，要知道，他们两人已经许久没这样在一起聊天了。<br/>　　<br/>　　    直到夜色渐深，微醉的两人这才往房中而去。<br/>　　<br/>　　    卧室中，冷霜早已命人备了热水供两人沐浴，房门关上后，冷霜便也退到了院外。<br/>　　<br/>　　    他一手搂着她的腰，看着她微醉泛红如桃花的面容，娇艳欲滴的朱唇，眸光不由的渐渐幽深。那搂在她腰间的手也开始不规距起来。<br/>　　<br/>　　    “阿九，我帮你脱衣沐浴。”<br/>　　<br/>　　    他的声音低沉而沙哑，带着几分撩人的意味在她的耳边响起。<br/>　　<br/>　　    温热的气息喷洒在她的耳边，凤九微缩了缩脖子，脑袋在他怀中蹭了一下，半眯着的目光带着些许醉意的瞅了他一眼，轻笑一声，慵懒而带着笑意的声音自口中传出。<br/>　　<br/>　　    “你要帮我脱衣？不行不行，还是我帮你脱比较好。”<br/>　　<br/>　　    她整个人扑进他的怀里，双手环抱着他的腰，一双纤细的手在他的腰间胡乱的摸索着，嘴里一边呢喃着：“嗯？腰带呢？”<br/>　　<br/>　　    “这里。”他牵着她的手来到他的腰带处，低沉的声音带着磁性的说着：“既然阿九要帮我脱，那就顺便侍候我沐浴吧！”<br/>　　<br/>　　    “好啊！”她爽快的应了下来，醉意盈然的眸子带着慵懒而撩人的媚意：“那你可要听话，坐着别乱动，让我帮你擦背。”<br/>
