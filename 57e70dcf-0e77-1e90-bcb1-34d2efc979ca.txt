　　“是我太厉害。”叶少阳有点得瑟。
<br/>　　“别臭美了，”谢雨晴白了他一眼，道：“我意思是，这东西一定是什么人放出来对付你的吧，既然人间知道你这么厉害，为什么就弄只这个来对付你，总不会是为了来恶心你吧。”
<br/>　　她话没说完，两人都听见山洞他们刚才走过来的方向有悉悉索索的动静，叶少阳拿出一把草纸，在灯笼的蜡烛上点燃，朝前方丢过去，火光暂时照亮了山洞。
<br/>　　两人顿时倒吸一口冷气：
<br/>　　在距离他们十几米远的山洞里，到处都是跟他们刚弄死这只一模一样的人形妖尸，有的趴在地上，有的贴在岩石上，层层叠叠的，粗看下去至少有上百只，一个个都趴着不动，上身支撑起来，睁着一双死鱼眼看过来。
<br/>　　“少阳……”谢雨晴声音颤抖。
<br/>　　“刚说什么来着，看到了吧，不是一只，人家准备了一群！”
<br/>　　“怎、怎么办！”
<br/>　　“你拿好灯，跟着我，身后路不通，只能杀回去看看。”
<br/>　　“你不说这是幻境吗，如果来路也被封上了呢？”
<br/>　　“你问的真多啊！总得试试吧！”
<br/>　　叶少阳握紧勾魂索，冲着这群人形妖尸冲过去。
<br/>　　“哇……”这群妖尸也朝他们爬行起来，一边纷纷发出那种诡异的叫声，此起彼伏，并且口中吐出那种能化成雾气的粘液。
<br/>　　来不及多想，叶少阳手起鞭落，一路抽下去，这一次没用什么手段和技巧，就是仗着自己深厚的罡气硬来，这些妖尸纵然实力不弱，但哪里是叶少阳对手，全是一鞭子抽飞，血肉模糊，再也爬不起来。谢雨晴紧紧跟在他后面，提灯为他照亮。
<br/>　　上百只妖尸，就让叶少阳这么一路抽了过去，留下一地碎肉。
<br/>　　闯过了这一段，前面再没有妖尸了，叶少阳拉着谢雨晴跑了一段才站住，用灯笼往身后照过去，有一些没死的还在追赶他们，叶少阳又上去一通乱杀，全给弄死了。
<br/>　　这时候，透过灯笼的光看过去，整个洞穴都弥漫着妖尸吐出来的黄色雾气，只能感觉到妖气弥漫，吸入之后也没任何感觉，但越是这样，叶少阳更加不敢掉以轻心，拉着谢雨晴继续沿着洞穴前进。心中犯起了嘀咕：如果这场埋伏真是地藏菩萨安排的，那也太小儿科了吧？
<br/>　　或者，这才刚刚是热身，后面还有别的手段？
<br/>　　不过很快叶少阳就发现，是自己想多了，沿着洞穴走了大概十几分钟，便看到了外面的亮光，小心翼翼地走完最后一段，便出了洞穴，回到地面。
<br/>　　一切还是从前模样，连路边的刷着红漆的“水帘洞”三字的石碑还在，前面有稀稀拉拉的游客在拍照。
<br/>　　两人面面相觑，意思再明显不过，他们……就这么走出来了？
<br/>　　“少阳……”谢雨晴睁大着眼睛。
<br/>　　叶少阳明白她的意思，道：“我也不知道。”
<br/>　　他是真的迷糊了。
<br/>　　就算对手不是地藏菩萨派来的，不管是谁，既然敢袭击自己，都不会让自己这么容易就走出来。那上百个人形妖尸虽然吓人，对付一般法师也足够了，但在自己跟前就是菜啊。
<br/>　　叶少阳百思不解，甚至想要重回山洞里，再从头走一圈，但谢雨晴担心他出事，死活不让，叶少阳一想，这样也好，见怪不怪其怪必败，自己就当没这回事，只是当心些，假若对方真还有什么手段，一定还会出手的。于是拉着谢雨晴慢慢地往回走。
<br/>　　路上遇到几个游客，嬉笑着从他们身边走过。叶少阳也没想什么。
<br/>　　来的时候是下坡，回去就是上坡了。两人默默爬山，爬了有十几分钟，叶少阳气喘吁吁了。
<br/>　　“你受伤了吗？”谢雨晴发现之后，关切地问道。
<br/>　　叶少阳体力很好，她平时是知道的。
<br/>　　被她这么一问，叶少阳才觉得有问题，自己并没受伤，但就是感觉浑身没力气，这时候他们已经爬到山顶，往前慢慢地走着，结果就算是平路，叶少阳走了一会之后，更加感觉累，几乎到了走不动的地步，尝试着运气，这才惊讶地发现，体内罡气竟然流失了许多！
<br/>　　糟了！
<br/>　　叶少阳懵逼地站着，不知道这是怎么一回事，心下惶惑，谢雨晴在一旁疑惑地看着他，一个劲地问他怎么了。
<br/>　　“你没有任何不对劲的感觉吗？”
<br/>　　谢雨晴迷茫地摇摇头。
<br/>　　叶少阳再次尝试运气，这才发现，自己体内的罡气正在快速得流失，怪不得自己越走体力越不行了。
<br/>　　叶少阳心中震骇，不知道是什么吸走了自己的罡气，也找不到目标，就好像四周的空气中自带一种磁场，能吸走身上的罡气。
<br/>　　特么的，这什么情况？
<br/>　　叶少阳强迫自己冷静下来，这时候谢雨晴还在一个劲地问自己，叶少阳目光从她脸上扫过，突然发现了什么，心中咯噔了一下，一个大胆的猜测浮上心头。
<br/>　　难道是这样？
<br/>　　正好这时候，附近有几个游客走过。
<br/>　　叶少阳突然冲上去拦住他们。几个人一愣，叶少阳抓住一个姑娘的胳膊，用力扯了一下，只听咔擦一声，胳膊被自己扯下来，鲜血狂飙，但是很快，被他抓在手中的胳膊就化成了灰烬……
<br/>　　对面几个人还在尖叫。谢雨晴也捂着嘴叫。
<br/>　　叶少阳冷笑着，望着他们的表演，几秒钟后，他们都从下而上地粉化成了灰烬，包括谢雨晴也是，又过了一会，整个空间都扭曲去来，天旋地转。
<br/>　　幻境。
<br/>　　果然是幻境。
<br/>　　叶少阳深吸一口气，抱元守一，暗道好险，自己他妈的差点上当，能发现奥秘，纯属偶然：方才谢雨晴望着他的时候，他无意中发现，谢雨晴的瞳孔上，没有自己的倒影！
<br/>　　这件事情虽小，毕竟是违反了物理规律。
<br/>　　要是早先，就算发现这一点破绽，他也不会怀疑世界是假的，但几次经历幻境危机的他，算是一朝被蛇咬十年怕井绳，立刻就想到了幻境上面，但为防万一，他不敢拿谢雨晴去试，正好有路人经过，就这么试了一下——假如不是幻境，也最多被人当成流氓暴打一顿，但一下子就把人胳膊扯下来，却是作证了叶少阳的判断。
