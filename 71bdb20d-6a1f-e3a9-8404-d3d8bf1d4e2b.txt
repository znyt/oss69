　　时间慢慢的走着，转眼间又是几周过去，荣耀白队依旧是一场未败，而蓝队也只是输给白队那一场而已。
<br/>　　而在这一段时间的修养，叶帝的身体也是逐渐恢复到了正常水平，每周都去医院接受治疗，而且还是家里找的最好的医生过来给他看，自然是不会出现什么问题。
<br/>　　“下周就要开始季后赛了，大家都准备好了么？”基地内，大家的情绪都很高。
<br/>　　“哈哈，我们蓝队出现肯定没问题，就怕你们白队到时候马失前蹄，最后连决赛都进不去。”蒋念笑嘻嘻的说着。
<br/>　　经过这一段时间的调整，大家的状态都已经是越来越好，逐渐到达了顶峰，如果说唯一的一个例外，那可能就是叶帝了。
<br/>　　这一段时间他的身体虽然是恢复了，不过谁也不知道现在叶帝的状态到底是不是巅峰。
<br/>　　“小叶子你给我们透个实话，你现在的状态到没到巅峰？”林萧开口问道。
<br/>　　“想探我的底啊？”叶帝有些好笑的看着林萧，根本不回答。
<br/>　　“小叶子就算是实力不到巅峰，那也是最顶尖的中单，大家别想太多了，好好打就是了。”一旁的李景禹摇了摇头，跟着说道。
<br/>　　“白队快要赶上咱们的克星了，春季赛的时候要不是输给他们一场，咱们可就连胜拿下冠军了，现在夏季赛又是如此，无奈啊。”蒋念有些惆怅的说着。
<br/>　　“夏季赛你们就没那个机会了，坐等被我们白队血虐吧，夏季赛的冠军已经被我们给预定了。”香蕉拍着自己的胸口自信满满的说着。
<br/>　　闲聊了一阵后，众人便是再次开始了自己的训练，下一周叶帝他们迎战的是A组的EG战队，皇族则是迎战的另外一支战队。
<br/>　　总体来说A组的实力要略微落后B组的，毕竟B组可是有着雷霆这王朝这两支世界有名的队伍呢，他们这边除了皇族之外，其他几支队伍虽然实力有，但是毕竟和叶帝他们还有一段差距。
<br/>　　网上无数的粉丝这个时候也都是在激烈的探讨着关于季后赛的事。
<br/>　　“感觉这个季后赛阵容我见过，去年好像也是这几支战队。”
<br/>　　“老夫今夜耗费无穷法力来给大家推算一下结果，半决赛是白队对阵皇族，雷霆战队对阵蓝队，最终白蓝两队成功晋级决赛。”
<br/>　　“卧槽，楼上的你继续啊，我还想听你后面的推算呢。”
<br/>　　“法师因为泄露天机过多，已经吐血身亡了，大家就不要为难他了，2333333333.”
<br/>　　“傻子都能看出来，今年的决赛肯定又是蓝白两队了，就是不知道最后会是哪支战队能拿下冠军。”
<br/>　　“雷霆、皇族真是生不逢时啊，这两支队伍除了在华夏和韩国之外，无论放到哪个赛区，都绝对是种子战队，可是在华夏却是只能争夺最后一个出线名额了。”
<br/>　　“已经迫不及待的想要看京都最终对决了，一想到芷澜女神要过来开唱，我的心都要酥了。”
<br/>　　论坛上无数的粉丝在那里议论着，而且越到临近比赛的时候，这些人的情绪便是越发的高涨。
<br/>　　而在基地内的叶帝等人，此刻虽然也都看到了论坛上的议论声，不过他们这些人倒是还都保持着一颗平常心，依旧在那里抓紧训练着。
<br/>　　一转眼的时间，周六这一天便是到来，这一周是A组的对决，周六是白队和EG的对战，周日则是皇族的对战。
<br/>　　时间才刚刚达到12点钟，现场便是已经座无虚席，很多人都在那里拉着横幅。
<br/>　　如果自信看的话，你就会惊讶的发现，整个场馆九层九的地方都是挂着白队粉丝的标志。
<br/>　　“今天想必EG队员的压力肯定会非常的大，不考虑实力的因素，单单就是这现场的因素就能够将一些心理素质不够好的队员给压垮了。”看着现场这个情况，后台的米粒对着一旁的林雅楠说道。
<br/>　　“是啊，不过这也是没有办法的事情，现在蓝白两队的势头实在是太强劲了，像是EG这种实力不强的队伍，真的很难啊。”林雅楠跟着开口说道。
<br/>　　时间到达12点40分钟的时候，叶帝他们一行人率先登场。
<br/>　　看到一身黑色队伍帅气登场的叶帝等人，场下顿时响起了震耳欲聋的欢呼声。
<br/>　　虽然已经不是第一次听到这个欢呼了，但是叶帝他们内心都还是非常的开心的。
<br/>　　对着场下的粉丝们示意了一番后，他们便是向着比赛室内走去。
<br/>　　后台还没有登场的EG战队的队员，在听到场下的欢呼声的时候，一个个更加的紧张起来了。
<br/>　　“教练咱们能不能弃权啊？”一个有些瘦弱的男生对着教练说道。
<br/>　　“顶住压力，这一局你们只要搞死大魔王一次就算是赢了，多杀都是你们赚，相信自己！”教练对着几名队员打气道。
<br/>　　到了这个时候就是连教练都没有一点底气，实在是现场的气氛太热烈了，而且还都是为白队欢呼的，这种压力实在是太大了。
<br/>　　又对着自家的几名队员说了两句话后，教练便是带着五人向着场上走去，这个样子就像是上断头台一般，每一步他们走的都很沉重。
<br/>　　场下不知是谁先开口喊了一句‘EG加油’，紧接着在EG战队众多队员一脸诧异的目光中，现场的十万名观众整齐的响起了为EG战队加油的声音。
<br/>　　“这是什么情况？”EG战队的队员有些摸不清头脑，他们有多少斤两，几人都是知道的，别说十万粉丝一起为他们欢呼了，就算是有一万，那都够他们吹一年的了。
<br/>　　当然了这个是指来到现场的，可是现场的欢呼声却是实实在在的映入了他们的耳畔，这下子他们更加的懵了。
<br/>　　带着这种懵懵的又有点兴奋、激动的情绪，EG战队的六人昂首阔步向着训练室走去，这一刻他们比先前的气势不知道强了多少。
<br/>　　EG战队的队员虽然是懵的，不过在观看直播的众多观众这一刻却是清楚的很，荣耀战队的粉丝怎么可能欺负别人，不要忘了，我们可是最有素质的粉丝。
