        “当年偷走童筱颖的人，其实就是白薇她自己。她终究是个母亲，不忍心看着自己的女儿成为实验品，也是人之常情。”裴医生黯然神伤地感慨道，脑海里忍不住想起了裴金澈。
<br/>
<br/>        大概，是母性泛滥了吧！
<br/>
<br/>        果然，母爱比父爱更伟大。
<br/>
<br/>        “裴冉，你瞒了我这么多年，是为了什么？”连中凯凝眉质问。
<br/>
<br/>        裴医生苦笑：“是白薇不让我说的，不然，你以为我为什么要对童筱颖这么好？”
<br/>
<br/>        她和白薇的关系比亲姐妹还亲，当她得知童筱颖就是他们要找的“解药”时，内心开心的同时又替白薇感到忧伤。

<br/>        事已至此，她也没必要再隐瞒下去了。
<br/>
<br/>        童筱颖再次醒过来的时候，发现自己正躺在一处弥漫着福尔马林气味的房间里。
<br/>
<br/>        她感觉自己做了一个很长很长的梦……
<br/>
<br/>        梦到自己变成了另一个女人……
<br/>
<br/>        “你醒了？”忽而，一个女音的关切声传了过来。
<br/>
<br/>        童筱颖抬眸循声望去，只见一个穿着大白褂，剪着短发的中年女子，向这边走了过来。
<br/>
<br/>        “你还记得我吗？”裴冉一脸无奈地问道。
<br/>
<br/>        童筱颖微微摇了摇头，又连忙点了点头，回答道：“我在梦里见过你。”

<br/>
<br/>        童筱颖正一头雾水时，门口又走进来另一个中年男子。
<br/>
<br/>        他手里端着托盘，托盘上有色香味俱全的饭菜，面带笑容，大步流星地走了进来。
<br/>
<br/>        “来来来，宝贝女儿，尝尝爸爸给你做的晚饭。”连中凯笑容格外灿烂地将托盘里的饭菜，一一端上了桌。
<br/>
<br/>        童筱颖惊愕地看着连中凯的一举一动，怀疑这个老男人是不是个疯子。
<br/>
<br/>        裴冉见状，不得不把事情的经过跟童筱颖解释了一遍。
<br/>
<br/>        童筱颖得知自己的身世后，脸色都变了。
<br/>
<br/>        为什么自己的亲生父亲是个“医学变态”？！
<br/>
<br/>        裴冉觉得，她应该给这对父女一点时间来缓冲一下彼此的关系，于是起身独自离开了。
<br/>
<br/>        连中凯面对童筱颖时，心里也有些尴尬，但更多的是喜悦之情。
<br/>
<br/>        DNA比对结果出来了，她童筱颖，就是他连中凯的生物学女儿，他们是血浓于水的父女关系。
<br/>
<br/>        “我想杀傅夜沉，是因为你妈妈当年是为了救傅夜沉的父亲中弹身亡的，我可是因爱生恨。现在，他是我女婿了，我自然是不会对他起杀心了。”连中凯弱弱地解释道。
<br/>
<br/>        童筱颖往床上一躺，侧过身去，背对着连中凯，没好气地回答道：“你别指望我会认你。”
<br/>
<br/>        抽她的血制药不说，还利用药物与催眠结合来控制她，甚至还想要借她之手去杀她心爱的男人。
<br/>
<br/>        如此狠毒的人，为什么会是她的父亲？
<br/>
<br/>        童筱颖不想接受这个事实，比起这样的事实，她宁愿自己是个被父母抛弃的孤儿。

<br/>        “那这样，好不好？我不再搞那种研究了。你放心，有爸爸在，没人敢再抽你的血！”连中凯信誓旦旦地保证道。

<br/>        童筱颖皱着眉头，依旧不领情。

<br/>        “就是华生啊！你不是认识他吗？你和他同父异母。”连中凯一本正经地回答道。

<br/>        童筱颖顿时尴尬了。
<br/>
<br/>        连华生竟然是她同父异母的亲哥哥？！

<br/>        “咚咚咚——”一阵急促的敲门声响起。
