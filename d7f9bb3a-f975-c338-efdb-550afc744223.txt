　　苏莫面现沉吟之色，他最想加入的是皇祖圣朝派系，但苍云圣宗派系却是拿出了足够的诚意，倒也值得考虑一番
<br/>　　“周方，你们未免太小气了吧！”
<br/>　　“哈哈！这么小的手笔，还想要拉拢苏莫？”
<br/>　　“你当我们不存在吗？”
<br/>　　道道朗喝之声响起，十几座岛屿之上，又有十几人飞身而起，来到高空中。
<br/>　　这些人大多数都是内府弟子，修为高达武尊之境，只有寥寥两三人，是武帝境的修为。
<br/>　　“苏莫，只要你加入了我们苍海圣殿派系，灵晶不再话下，我们首先就送你一门尊级上品功法！”一名高大的青年沉声说道。
<br/>　　下方岛屿上的不少人，闻听此言，顿时倒吸一口凉气。
<br/>　　苍海圣殿居然如此大的手笔，拿出尊级上品功法。
<br/>　　要知道，无论是武学还是宝器，达到尊级以后，每高一个级别都是天差地别。
<br/>　　一门尊级上品功法，比一门尊级中品功法加一件尊级下品宝器，还要珍贵数十倍。
<br/>　　苍海圣殿愿意拿出尊级中品功法，这真是下了血本了。
<br/>　　“苏莫，我们赤霞洞天派系，也愿意拿出尊级上品功法，另外还愿意拿出一门尊级上品武技！”一名年轻女子郑重的说道。
<br/>　　“苏莫，我们无极圣山派系，也愿意拿出同样的武学！”
<br/>　　“我们广云天宫派系圣子说了，愿意赐你一枚圣果，瞬间就能提升一重修为！”
<br/>　　“圣果算了屁，加入我们星象圣门派系吧！修周天星辰之力，保准你的战力再提速一个档次！”
<br/>　　众人七嘴八舌的开口，每一个人都许诺了天大的好处。
<br/>　　下方十几座岛屿上，观战的众人个个目瞪口呆，这些派系真是一个比一个手笔大，这是要下血本争抢苏莫了啊！
<br/>　　很多人都满脸羡慕，真是人比人气死人啊！
<br/>　　他们有的人，动用各种关系，都不能加入一个派系，但现在，却是十几个派系争抢苏莫一人。
<br/>　　尤天韩等太史世家派系的人，个个脸色难看，这些派系的人，居然在这个时候拉拢苏莫，完全不顾及他们太史世家派系的脸面。
<br/>　　这要是苏莫被拉拢了，加入了其它派系，他们再想要对方，就很麻烦了。
<br/>　　但是，他们此刻却是毫无办法。
<br/>　　苏莫面色平淡的打量着这些人，这些人许诺的好处，他并没有太在意，因为这些虽然珍贵，但对他来说并没有什么大不了的。
<br/>　　他自己有足够的财富，至于功法，有比他混天魔功更强大的吗？
<br/>　　他需要的是诚意，足够的诚意。
<br/>　　“阁下是哪个派系的人？”苏莫目光落在了一名消瘦的青年身上，抱拳问道。
<br/>　　他虽然是询问，但已经知道对方是皇祖圣朝派系的人，因为拉拢他的派系之中，唯缺皇祖圣朝派系。
<br/>　　此人面容普通，但眸光锐利如鹰，虽然修为并不是武尊境，只是武帝境九重，但明显不是简单人物。
<br/>　　“在下任云都，乃是皇祖圣朝派系的人！”消瘦青年沉声说道。
<br/>　　苏莫点了点头，问道：“不知你们愿意出什么代价？”
<br/>　　他并没有妄自菲薄，既然对方也来到了他的面前，自然是要来拉拢他的。
<br/>　　“这个……我们圣子没有明确交代，但绝对不会比他们差！”任云都言语略显迟疑，沉声说道。
<br/>　　他之所以没有像别人一般，上来就许诺好处，就是因为圣子古天意没有明确传下话来。
<br/>　　这让他难办不已，并不敢向苏莫许下大的海口。
<br/>　　苏莫闻言点了点头，随即扫视了众人一眼，略作沉吟，道：“各位给我一点时间考虑如何？容我先将眼下的事情做完！”
<br/>　　他没有答应任何人，而是将事情拖后，先将眼下的事情解决。
<br/>　　“嗯！”
<br/>　　“可以！”
<br/>　　众人纷纷同意，没有太过强求。

<br/>　　“尤天韩，你都出手了，是你们太史世家派系无人了吗？”目光望向尤天韩，苏莫冷笑着说道。
<br/>　　他已经将对方列入了必杀名单，待实力足够以后必杀对方。
<br/>　　之前若不是其他人出手的话，他还真的有可能栽在对方的手中。
<br/>　　现在，他底气很足，所有派系都想拉拢他，必然会保证他的安全。
<br/>　　“苏莫，你杀我们这么多人，我们太史世家派系，绝对不会放过你！”尤天韩阴沉着脸说道，现在他的确是无人可派了。
<br/>　　武帝境四重和五重的弟子之中，根本就没有人是苏莫的对手，除非派出武帝境六重的人。
<br/>　　“不会放过我？”
<br/>　　苏莫闻言嗤笑一声，朗声道：“那好，我现在就给你们一个机会，你们太史世家派系，有多少武帝境四重和武帝境五重的弟子，全部都一起上吧！”
<br/>　　苏莫的声音传遍全场，虽然不大，却在所有人的脑中回荡。

<br/>　　很多人都张大了嘴巴，面色凝固，满脸不可思议的盯着苏莫。
<br/>　　就连尤天韩，以及一众太史世家派系的人，都愣住了。

<br/>　　“什么？刚才苏莫说什么？”
<br/>　　“我没有听错吧！他要让太史世家派系，所有的武帝境四重和五重的人一起上。”
<br/>　　“天呐！他疯了吗？”
<br/>　　“自信心膨胀道爆炸了吧！”

<br/>　　“怎么？不敢吗？”
<br/>　　苏莫眸中尽是不屑之色，面色张狂不已，突然暴喝道：“今日我就放下话了，你们太史世家派系的弟子，武帝境四重和五重的弟子，来一个我杀一个，来十个我杀十个，来一千我杀一千！”
<br/>　　苏莫的声音越来越大，如同滚滚雷霆，响彻在众人的耳中，尽显张狂霸道之姿。
