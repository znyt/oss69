　　　　白楚艳的担心其实完全是多余的，因为涂小安将自己的话说完，扭头就走，在也没有回头多看她一眼。
<br/>　　只是走的时候幽幽然的丢了一句：“手感不错”
<br/>　　气的白楚艳要咬牙切齿。
<br/>　　走到猫精逃跑的那扇窗户，涂小安翻窗就直接跳了下去。
<br/>　　这一幕，将白楚艳看的差点尖叫出来，要知道这里可是八楼呀。
<br/>　　他就这样跳下去？
<br/>　　这人到底恐怖到何种地步。
<br/>　　涂小安可没心思去想白楚艳的内心阴影，翻窗跳去，直接运用飞行术，飘飘然的当起了空中飞人。
<br/>　　此时繁星点点，夜色如水。
<br/>　　这里是温城大学的大后方，按道理说应该是要脱离了学校的范围，但涂小安意外的发现这里居然还有不少建筑物。
<br/>　　前方有扇古朴的大门，青色的石阶，雕刻着岁月的沧桑，既然有一股庄严、神圣的气息弥漫于其中。
<br/>　　别有洞天呀。
<br/>　　没想到温城大学的后方还有一座隐藏的学府，看上去这好像是一个古学院，蕴含千载岁月般的沉淀。
<br/>　　莫非这是修武者专属的学院？
<br/>　　涂小安微微的思忖，他落下来站在了这栋隐藏学院的大门之外，随即感受到了一丝异样的气息，他心中涌起一股复杂莫名的情绪。
<br/>　　“好强的灵气”
<br/>　　没错，是灵气，当代社会，灵气早就非常的枯萎，几乎不太存在，只有特殊的地点，比如一点大山的深处。
<br/>　　但涂小安现在站在这座学院的门前，能很明显的感受到里面的灵气，就好比空气一样，看不到，但却能深深体会。
<br/>　　这个地方可谓是钟天地之灵气，看来是觉醒修武者修炼的圣地呀。
<br/>　　怪不得温城大学首屈一指，原来背后还隐藏着一栋学府。
<br/>　　之前他就听李清照说，这栋大学是一些大家族，富二代，官二代的首选。
<br/>　　现在涂小安才明白，温城大学应该是与修武界合作的大学，前面是普通人的学校，后方是修武者的学府。
<br/>　　有意思，果然不愧是大城市，还有专门的无修武者学府。
<br/>　　在白镇，连觉醒者都不可见。
<br/>　　随后，涂小安发现了猫精散发的蛇类气息就在里面，想必也是误打误撞的跑了进去。
<br/>　　既然这样，那我就进去走一趟。
<br/>　　涂小安纵身攀上了一面高大的院墙，仔细观察了一会儿，没有发现里面异常，就立刻飞身飘了进去。
<br/>　　这座学院静悄悄，在层层院落中，只有几个房间还亮着灯，他如午夜幽灵一般在学院各处游荡。
<br/>　　按理说，人类身体想觉醒，也不是那么简单，所以学府的学子也应该不会太多，而且现在还没到真正开学的时候。
<br/>　　涂小安路过一个巨大的演武场，这应该是修武者们比武的地方。
<br/>　　其实涂小安不知道，温城大学有一门关于人类身体觉醒的课程，每一个刚进入学校的人，都会被做全身检查，抽取你的血液。
<br/>　　要是你的血液显示，你有修炼的天赋，那么就会被学校特别安排，来上这堂身体觉醒课程。
<br/>　　如果你那天身体觉醒了，就会被直接安排在后方学府。
<br/>　　当然，你要是本身就是觉醒者，就直接到后方学府报道。
<br/>　　因为后方学府毕业的学生，很多都是被国家吸收，委以重任，也有隐藏家族的子弟，回去效力。
<br/>　　白楚艳就是后方毕业的，然后直接到了警察局，上任就是队长。
<br/>　　“要是在这种学府上学，应该是有点意思的”
<br/>　　涂小安没有上过大学，所以来看看，已他现在的身份，其实是不可能完美的融入普通人的生活。
<br/>　　更别说去当一个大学生。
<br/>　　但是这种修武者的学府，到是适合他，可以学习一些功法，不断的强化自己的实力。
<br/>　　当然，他也就想一想，他现在的实力直逼三阶生灵，将这座学府给一手毁灭都不是问题。
<br/>　　谁有资格教他。
<br/>　　涂小安四处看着，暗地中，不断的感应猫精的蛇类气息，慢慢的靠近。
<br/>　　转来转去，他来到了学院的深处，一个美丽的湖泊出现在他的眼前，湖水如镜，映照出天上点点繁星，让涂小安微微一惊。
<br/>　　不愧是提供修武者的古学府，还真有那么点古味。
<br/>　　此时一缕悠悠笛音在湖面轻轻飘扬，一股淡淡的哀伤藏蕴笛音之中。
<br/>　　涂小安微微蹙眉，仔细搜索了半天，也没有发现吹笛之人，他有些心惊：“这笛声何处而来？”
<br/>　　说起笛声，涂小安系统传出一阵波动，意念一动，手中出现了根玉笛，晶莹剔透，散发光泽。
<br/>　　这玉笛在微微的颤动，因为是跟这股笛声产生共鸣。
<br/>　　这玉笛是宝物，用意识力吹奏，可惜涂小安不会吹，在他的手中到有点暴残天物了。
<br/>　　那股笛声时近时远，最后涂小安发现笛声就在湖泊的下方传出，很是吃惊。
<br/>　　湖泊的水下，怎么会有笛声呀，莫非水下还藏着一个人？
<br/>　　看来这古学府的水有点深呀。
<br/>　　先不管，找到猫精在说。
<br/>　　涂小安悄悄的绕过了湖泊，继续向前走去，距离猫精海棠的位置越来越逼近。
<br/>　　不仅远处有一片花圃，花圃旁一对石椅上坐着年轻的一男一女，正在轻轻私语，状态亲密之极。
<br/>　　总算看到人了。
<br/>　　从他们的身上，涂小安感受到了觉醒者的非凡气息，生命力都比普通人要旺盛的多。
<br/>　　一个意识洞察丢过去，他们的等级都在十一级，十二级左右。
<br/>　　想必都是刚觉醒的修武者。
<br/>　　涂小安是偷偷溜进来，不想被人发现，刚想绕过去，但男女的对话声引起了他的注意。
<br/>　　只听那名女子道：“这笛音好伤感啊，凄婉而又哀伤，每次一到夜晚就出现，白天却没有，真是奇怪”
<br/>　　那名男子道：“这笛音在学院初建时就已存在，有种传说，湖底镇压一只女妖，是修成证道的三阶生灵，每当星光璀璨时，她就会吹响袅袅笛音”
<br/>　　“三阶生灵？是什么生灵呀，是人还是动物”
<br/>　　“应该是动物，至于是什么动物，我也不清楚，这只是传说，当不得真”
<br/>　　男的继续道：“还有另外一种说法，说湖面之下是一位修武者高手留下的古阵，那笛音不过是古阵运转时发出的天地妙音，也正因为有了这古镇，这学府的灵气才会那么的浓郁”
