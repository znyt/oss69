　　萧逸的修为虽然暂时只有后天一重，但他的实力却一直都不弱。
<br/>　　越阶战斗对他来说，已是家常便饭。
<br/>　　特别如今手上还有灵器。
<br/>　　烈焰手套一直都戴在他手上，虽没有用真气激活显露过，但仅仅是戴在双手上，已经给予了他很大的增幅。
<br/>　　这也是为什么他之前能在外头秒杀那些普通的荒鳄。
<br/>　　一旦完全激活，配合上升龙和武技，他有把握击杀这巨斧。
<br/>　　如今整条长廊上，无数荒鳄肆虐。
<br/>　　一旦铁达大叔受伤或者死亡，整支猎妖队都会变得相当危险。
<br/>　　仅凭萧逸自己一人，是绝对无法保护其余四人的，所以铁达大叔绝对不能有事。
<br/>　　萧逸不介意现在击杀掉巨斧。
<br/>　　然而，所有人都同仇敌忾，愿意站在铁达大叔身边。
<br/>　　只有杜新，在背后捅了捅萧逸，低语道，“易霄，你自己找死好了，别把我们也搭上去。”
<br/>　　铁达大叔也沉声说道，“易霄兄弟，我承你的情，但我这是我巨斧间的仇怨，你们先走，别把命搭上了。”
<br/>　　“呵。”萧逸笑笑，“铁达大叔，大伙就交给你保护了。”
<br/>　　话音刚落，嗖的一声，萧逸消失在原地。
<br/>　　当他再次出现时，已经在巨斧面前，并一剑劈出。
<br/>　　“易霄兄弟，快回来。”铁达大叔和林飞大惊失色，暴喝一声。
<br/>　　然而，这时巨斧已经一斧头轰出。
<br/>　　“哼。”萧逸冷哼一声，正准备使用烈焰手套时...
<br/>　　霎那间，整条长廊狂风大作，电闪雷鸣。
<br/>　　无数条深蓝色雷电凭空降落，凡是被打中的武者，瞬间化作一阵青烟，尸骨无存。
<br/>　　连荒鳄被劈中了，也是当场死亡。
<br/>　　轰..轰..轰...无数雷电让得长廊仿佛成了一个危险的雷电世界。
<br/>　　很多猎妖师甚至还未反应过来，便已经被劈死，连同身边的荒鳄也一样。
<br/>　　一切都在瞬间发生。
<br/>　　恰在此时，一条狂雷在萧逸与巨斧中间劈落。
<br/>　　“不好。”
<br/>　　两人同时一惊，立刻跳开。
<br/>　　但，雷电的速度太快了。他们两人还未来得及跳开，狂雷便将地面劈出一个深不见底的大坑。
<br/>　　两人瞬间掉落。
<br/>　　而后，长廊上一股诡异的力量，将大坑瞬间修补，化为完好。
<br/>　　“易霄兄弟。”
<br/>　　“队长。”
<br/>　　铁达大叔等人，巨斧猎妖队的人，同时大喝一声。
<br/>　　没等他们来得及深究，密布的雷电已经朝他们劈去。
<br/>　　“该死，先离开这里。”铁达大叔喝了一声。
<br/>　　虽然担忧萧逸的安危，但他身为队长，也要顾及其他队员的安全。
<br/>　　倒是巨斧猎妖队的人，见自己的队长跌入大坑，仅仅叫了一声，没有丝毫犹豫便跑了。
<br/>　　......
<br/>　　踏...踏...
<br/>　　萧逸和巨斧同时站稳了身子。
<br/>　　周围一片漆黑，伸手不见五指。
<br/>　　下一秒，两人同时迸发出一身真气。
<br/>　　“小子，区区一个后天一重，敢在我面前放狠话，真真找死。”
<br/>　　巨斧一斧头劈出。
<br/>　　他的武魂是一把巨斧，但到底什么属性或者阶级，倒暂时不知。
<br/>　　“我听说过你，破玄城名声最差的猎妖队，为了钱，无所不作。虽然我不是爱管闲事的人，但惹了我，我也不介意送你下黄泉。”
<br/>　　萧逸冷笑一声，准备战斗。
<br/>　　然而，正当两人准备大战一场时。
<br/>　　漆黑的周围，忽然露出了无数双猩红的眼睛。
<br/>　　“什么东西？”两人瞬间一惊，并闻到了一股危险的味道。
<br/>　　“吼”，一阵阵骇人的兽吼声出现，之后便是一道道黑影快速袭来。
<br/>　　武者的身体受天地灵气洗涤过，也被自身真气淬炼过，六识过人，眼睛渐渐适应了这里的环境，并看清楚了周围。
<br/>　　这里是一个巨大的房间，房间内，无数条荒鳄发起了进攻。
<br/>　　“又是你们这群孽畜，找死。”巨斧暂时停下了对萧逸的攻击。
<br/>　　因为，数十只荒鳄已经包围了他。
<br/>　　他乃是先天四重武者，就算这里的荒鳄有些特俗，但也不过是后天九重的实力，数量再多也奈何不了他。
<br/>　　但，令人惊骇的事在一瞬间发生。
<br/>　　巨斧的斧头劈出，并没能杀死荒鳄，仅仅是将一头荒鳄轰飞。
<br/>　　“什么？”巨斧大吃一惊。
<br/>　　剩下的荒鳄一拥而上，彻底淹没了他魁梧得身体。
<br/>　　不过数十秒时间，当荒鳄散开时，巨斧已是一具被啃噬得体无完肤的尸体。
<br/>　　另一边，同样有荒鳄攻击萧逸。
<br/>　　但萧逸没有大意，劈了一剑后，立刻借着反冲力后退。
<br/>　　仅仅一剑，他已经试出了这些荒鳄的实力。
<br/>　　“怎么可能？竟然有先天一重的实力。”萧逸后退，感受到自己发麻的手臂，惊骇至极。
<br/>　　再看了眼周围，这巨大的房间中，竟有上百头荒鳄。
<br/>　　先天实力的妖兽，已是三级妖兽。
<br/>　　上百头三级妖兽，且荒鳄本身凶残至极，肉体力量过人，怕是先天九重在此也难逃一死吧。
<br/>　　难怪那巨斧还没反应过来就被杀死了。
<br/>　　“遭了。”萧逸也不敢再大意，立刻催动烈焰手套。
<br/>　　一股真气凝聚在手上中，顿时，烈焰手套出现，一股炙热火焰自动围绕在萧逸的双手。
<br/>　　这些火焰，属于烈焰手套本身的力量，并非萧逸凝聚出来的。
<br/>　　“吼”数头荒鳄嘶吼一声，快速攻来。
<br/>　　萧逸不敢托大，一记噬火百刃劈出。
<br/>　　只是，让萧逸惊讶的是，发出的火刃，竟比以前巨大了十倍不止，火焰的温度更是高得骇人。
<br/>　　嘭的一声，荒鳄直接被噬火百刃分尸，而后轰然爆炸。
<br/>　　“竟然秒杀了这头荒鳄？”萧逸瞳孔一缩。
<br/>　　这是他第一次使用灵气，没想到威力竟强悍如斯。
<br/>　　他明显感觉到，自己施展噬火百刃时，真气传递到烈焰手套上，真气凭空被增强了数十倍不止。
<br/>　　灵器能增幅武者的实力，但他没想到竟然增幅程度如此之大。
<br/>　　其实，灵器内本身就蕴含庞大的力量，武者使用时，就是在利用这股庞大的力量。
<br/>　　也难怪灵器是所有武者都梦寐以求的宝物。
<br/>　　这时，萧逸不断地释放火刃。
<br/>　　无数火刃成为密集的攻击，每打中一头荒鳄，便预示着一头荒鳄的死亡。
<br/>　　漫天火焰，将原本漆黑的房间照耀得火红一片。
<br/>　　连绵不断的爆炸，震得人耳朵轰鸣。
<br/>　　不多时，整个房间的荒鳄被秒杀一空，只留下一地妖兽尸体。
<br/>　　“嘶。”萧逸倒吸一口凉气，“这就是灵器的威力吗，竟强悍如斯。”
<br/>　　上百头荒鳄悉数死亡，但萧逸消耗的真气，却不足一成。
<br/>　　释放噬火百刃，大多是借助烈焰手套中的力量，萧逸自己的真气并没有消耗多少。
<br/>　　萧逸看了眼自己双手上仍旧环绕的绚丽火焰，感觉到一股骇人的力量在自己手心凝聚着。
<br/>　　看着满地的尸体，微微感到有些自豪。毕竟，就算是一个先天九重，也无法做到这个地步。
<br/>　　但，这并不代表着他比先天九重强。
<br/>　　若是一个普通的先天九重武者在这，同样也能秒杀荒鳄。
<br/>　　只不过，他会在杀光所有荒鳄前耗尽真气，最终等死。
<br/>　　萧逸释放的火刃，大概只在先天六重的威力，毕竟他现在只有后天一重的修为。
<br/>　　但他胜在使用武技几乎不用消耗真气。
<br/>　　萧逸看着满地的妖兽尸体，将内丹全部取出。这可是三级内丹，价值不低。
<br/>　　而后，取了巨斧的乾坤袋，用于装妖兽尸体，能装多少是多少，装不完就算了。
<br/>　　反正妖兽最值钱的只是内丹，其它的皮毛、爪子之类的全部加起来，也抵不过内丹十分之一的价值。
<br/>　　萧逸现在不缺钱。
<br/>　　忙完一切，萧逸才有时间认真地观察周围。
<br/>　　他万万没想到，从大坑中掉下来，会掉到一个房间中，而且还困满了先天层次的荒鳄。
<br/>　　再另一头，有一门。
<br/>　　萧逸尝试打开，但门上有一股力量在阻碍着，那是灵气屏障。
<br/>　　灵气屏障有很多用途，或用于隐藏事物，或用于围困事物。
<br/>　　这一个灵气屏障，显然是用于围困事物的，防御极其强大。
<br/>　　噬火百刃、崩山斩，乃至加上灵器都撼动不了这个灵气屏障分毫。
<br/>　　最终还是凝聚出冰鸾剑，借助其无坚不摧的锋利，才轻松破了屏障。
<br/>　　萧逸不禁摇了摇头，自语道，“若不是我有冰鸾剑，就算杀了那些荒鳄，也会在这房间中困死。”
<br/>　　推开房门，走出去，竟是一条长廊。
<br/>　　萧逸大惊，皱起了眉头。
<br/>　　难不成，这宫殿分地上地下两层？上面那一层，是猎妖师们战斗的地方，而自己跌落大坑后，来了这下面的一层。
<br/>　　只不过，这条长廊中，并没有任何荒鳄。
<br/>　　萧逸怀着疑虑，小心翼翼地走着。
<br/>　　在穿过长廊后，入目所视，竟是一间古朴的房间。只不过，这房间比之其他房间更大。
<br/>　　房间内，两旁是密密麻麻的黑盒子。
<br/>　　正中央，有一卧榻，卧榻前，有一蒲团，并不怎么起眼。
<br/>　　但，卧榻上，显然传来了一股让人无法想象的恐怖气息，远超萧逸曾经感受过的慕容墨的洞玄境气息，强了百倍不止。
<br/>　　“难道，这里才是狂血玄君真正的洞府？”萧逸脸色一喜。
