　　清长风淡淡一笑，“我本以为，这一战要等很久，却没想到，这么快就等到了你，今日你我放开一战，却是一大快事！”
<br/>　　“你下来单挑吗？”
<br/>　　“不忙，道风，这可知这石碑是何物？”
<br/>　　经他这么一提醒，道风注意观察着那些围着自己飞转的汉字，在脑海中组合起来。
<br/>　　“高上洞元，兀生九天，炁祖太元，众风乱玄，玄曾绝散，四清抚闲，帝一上景，连众摄烟，长契一运，七世投闲……这是上清《大洞真经》。”
<br/>　　边上的上古邪神替道风说了出来。
<br/>　　清长风有些意外地看了他一眼，道：“你却也有些见识，修为也不错，去何苦跟了道风，不如归顺我北斗观，在这仙源乐土之中，做一个长生灵物，岂不是好。”
<br/>　　上古邪神没回答，清长风当然也不期望他回答，接着说道：“不错，这便是大洞真经，人间道门奉若经典，但你们不知，这块石碑，便是大洞真经首篇真迹，许真君当年用来镇压铁背虬龙所用，后为我门人所得，贡献于我。被我用百年祭炼成器，其中神妙，你们即刻便知！”
<br/>　　道风心中一惊，这种经文之类的，如果镌刻在物体上，所谓文通鬼神，长年累月吸收灵气，能成为十分了得的法器，如天下九鼎，便是国之神器，甚至能影响国运。
<br/>　　这大洞真经的石碑，当年是许真君用来镇压妖邪，灵力自然更强，又被清长风祭炼百年……自然强大无匹，只是这种法器不方便携带，发动起来也很费劲，但威力却是巨大。
<br/>　　道风和上古邪神互相看了一眼。
<br/>　　“顶住。”道风在神识中对上古邪神说道。
<br/>　　“阿弥陀佛……”
<br/>　　一声佛音，从远处传来，两人定睛望去，是一个身材高大的胖和尚，从山脚下走来，光着双脚，手里捏着一串佛珠，低着头缓缓走来。
<br/>　　盘古僧三大神僧之一，山山禅师！
<br/>　　道风一眼便认出他来。
<br/>　　山山禅师走到山下，站住，望着道风，微微躬身，又宣了一声佛号。
<br/>　　道风冷笑道：“大和尚你也是来报仇的吗？”
<br/>　　“老僧出家之人，哪有什么仇怨，但几位施主擅闯轩辕山，老僧职责所在，不可不问，还劝诸位速速离去，休生事端。”
<br/>　　“大和尚你说的好听，我们要走，现在能走得了吗？”
<br/>　　山山禅师沉默半晌，缓缓说道：“元神可走，魂魄留下。”
<br/>　　“道貌岸然。”道风淡淡说道。
<br/>　　这时，彩云仙子弹奏琵琶的速度猛然加快起来，那些云朵旋转的速度也加快，朝着两人所在位置收缩起来。
<br/>　　“道风子，你可看好了！”
<br/>　　清长风站在岩石上，双手结印，操控着那些闪闪发光的金字，组成了一副北斗七星的星象图，朝道风兜头砸下来。
<br/>　　道风祭出三花聚顶，护住自身，上古邪神与他背靠着背，也释放出了自身修为，与道风呼应，一起抵抗碑文的强大力量……
<br/>　　“怎么什么都没有！”
<br/>　　洞穴深处，只有一汪清泉，在没有光源的情况下，水面很奇怪地泛着波光，能大约照见洞穴里的情形，整个洞穴也都笼罩在一片水汽之中，走在这里，芮冷玉能明显感觉到这里灵气充沛，甚至比空界某些灵山还要高得多。
<br/>　　但现在芮冷玉关心的不是这个，站在洞穴深处，她东张西望半天，只看到一张寒玉床，上面还没人。
<br/>　　床边上有一只香炉，边上摆着一堆香，还有两只蜡烛。
<br/>　　芮冷玉一眼便知，这种地方，八成是某个大佬的修所，但是除了这些，这里一目了然，什么都没有，更不要说星月奴的肉身了。
<br/>　　她质疑地朝叶少阳望过去，却发现叶少阳站着不动，表情非常古怪。
<br/>　　芮冷玉一下子怔住，皱眉说道：“怎么了？”
<br/>　　叶少阳摇摇头，朝她走过去，到面前的时候，突然抬手，一掌朝她拍下来。
<br/>　　芮冷玉一惊，本能地举手反击，她实力倒也不错，在人间也算是一流强者，但跟叶少阳相比，还是差了一个等级，又是被动抵挡，没有准备好。
<br/>　　不出几个回合，她就被全面压制。叶少阳抓住一个机会，破了她创造的防御结界，勾手一挑，一张灵符贴在了她后脑上。
<br/>　　定魂符！
<br/>　　叶少阳的实力高出她太多，她冲不破这定魂符的约束。
<br/>　　她傻了一样看着叶少阳。
<br/>　　“你就待在这里，你放心，没人能伤害你，事成之后，你可以留在轩辕山，做我的妻子。
<br/>　　“事成，什么事？”虽然魂魄被封，但芮冷玉还是可以说话的。
<br/>　　叶少阳望着她，缓缓吐出了四个字：“等道风死。”
<br/>　　芮冷玉不敢相信地看着他。
<br/>　　突然，她一咬牙，开口说道：“你不是叶少阳！”
<br/>　　“我是。”
<br/>　　“不，你肯定不是！”芮冷玉不为所动，很简单的一点，在叶少阳心中，道风几乎比自己还重要，他断不可出卖道风。
<br/>　　叶少阳望着她，笑了笑，“我是叶少阳，但……不是你认识的那个。”
<br/>　　芮冷玉傻傻地看着他。
<br/>　　叶少阳叹了口气，摇头说道：“这个世界的叶少阳，命真好……好到让我嫉妒，在我的那个世界，我已经失去了一切，幸好，还有机会让我从头来一遍，这一次，我不会再失败。”
<br/>　　听到这里，芮冷玉大致明白了，深深吸了口气，颤抖着说道：“你是从别的时间穿越过来的叶少阳！”
<br/>　　“你只说对了一半，不是别的时间，我，是从恶灵空间来的。”
<br/>　　恶灵空间！
<br/>　　芮冷玉懵了。
<br/>　　这个空间是人间的镜像，如果人间为阳，那里就是阴，如果人间是善，那里就是恶。恶灵空间的时间过的更快，但一切都是从人间镜像过去，包括生灵也是，只不过命运完全是自己掌控，可能一个人会有截然不同的命运。
<br/>　　有的人在人间活着，在那里可能已经死了。
<br/>　　而且恶灵空间一共有四维，时间线也不一样，对人间来说，那里时间错乱，恶灵遍地，根本就是一个不该存在的地方，但正是因为恶灵空间容纳了人间所释放的“熵”，人间才能万古不变，长存下来。
