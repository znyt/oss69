<br/>    凌仙和李素衣站在山脚之下，看向了山顶一眼，凌仙道：“三师弟，你猜这次大师姐叫我们过去是什么事？”
<br/>    
<br/>        李素衣道：“许是大师姐无聊了，所以叫我们两个去聊天呗。”
<br/>    
<br/>        这个时候，两人忽然感觉脚下吹来一阵寒风，让人遍体生寒，凌仙道：“我忽然有一种不祥的预感。”
<br/>    
<br/>        李素衣也是有些害怕的看向山顶道：“我也有同感。”
<br/>    
<br/>        两人迟疑了一阵，凌仙道：“还是去见大师姐吧，不然，后果不堪设想。”
<br/>    
<br/>        两人上山，不过一刻钟之后，便传出了两声惨呼。
<br/>    
<br/>        三个时辰之后，凌仙和李素衣已经浑身是伤，卓青瑶看向两人，问道：“怎么，二师弟，三师弟，你们这段时间是在摸鱼吗？实力竟然没有半点长进。”
<br/>    
<br/>        “算了，先下山恢复伤势，明早再来。”
<br/>    
<br/>        凌仙和李素衣拱手道：“是，大师姐！”
<br/>    
<br/>        凌仙和李素衣下山，各自休息了一夜，第二天早晨，两人站在山脚下对视一眼，各自苦笑，凌仙道：“我还以为你不来了呢！”
<br/>    
<br/>        燕山有很多酷刑，其中在凌仙和李素衣心中排名第一的，叫做大师姐教你修行。
<br/>    
<br/>        两人昨天都是被卓青瑶揍得够呛，有些难受。不过知道了这神山是师父的无字劫之后，两人都是没有什么二话，再次来到卓青瑶画地为牢的山峰之上。
<br/>    
<br/>        李素衣道：“养徒千日，用徒一时。若是这真的是师父的无字劫，这人间劫，说什么我也要插上一手！”
<br/>    
<br/>        凌仙道：“到时候看看，这世间的天骄，和我们比到底有什么差距。”
<br/>    
<br/>        李素衣道：“师父渡劫那一日，燕山将会被整个星空知晓，而且，闻名天下。”
<br/>    
<br/>        凌仙笑道：“师父说过的，人这一辈子，就这么长的时间好活，总要做些自己喜欢的事情，天下这么大，我想打打看。”
<br/>    
<br/>        李素衣和凌仙抬头，看向天空之中的那些所谓的天才，似乎，在大师姐的凶威面前，也就那样。
<br/>    
<br/>        话分两头。
<br/>    
<br/>        陈明和九太子鏖战了一天的时间，陈明身上有着些许的伤痕，但是并无伤大雅，并不会影响陈明的战斗力。
<br/>    
<br/>        而陈明如今做的，便是想要看看九太子的涅到底是不是真的打不死。
<br/>    
<br/>        至少在目前看来，对方根本没有半点伤势，按照的描述，是在一定程度上拥有不死之身，并不是真正的不死。
<br/>    
<br/>        至少陈明发现了一点，那就是对方伤势恢复的速度，似乎是比之前慢了一些。
<br/>    
<br/>        至于周围的修行者，也是看了半天一夜，对于陈明和九太子的战斗持久程度，也是惊叹不已。
<br/>    
<br/>        这样的对手要怎么打？
<br/>    
<br/>        若是普通的地仙，就算来一千个也没有一点用啊！
<br/>    
<br/>        根本不知道累字怎么写。
<br/>    
<br/>        而且最为离谱的是这还是两个法修肉搏了半天一夜，那么无冕者议会的那些苦修，又强大到了什么程度？
<br/>    
<br/>        强大的法修，一般都有着保护自己的手段，否则有可能直接被苦修近身一招解决掉。
<br/>    
<br/>        至于像是陈明和九太子这种，都是修行的极为强大的法修功法，法力属性霸道之极，淬炼肉身，同样达到了苦修的程度。
<br/>    
<br/>        两人都没有施展真正的最后手段，就好像陈明，周天星辰大阵完全没有动用。
<br/>    
<br/>        而九太子修行的十大仙术之一太阳神光，也是完全没有动用。
<br/>    
<br/>        所谓功法，便是用来提升修为的境界的，功法所能提升的，便是肉身的强大程度以及更多的法力和仙力。
<br/>    
<br/>        而法术，则是交战之时分出高低的手段。交战的结果，更大的程度上还是取决于术法的强大与否。强大的修行者，都有着杀招，那就是一种能够在一招之间将自己大量的法力或者是全部法力全部挥霍一空的术法。
<br/>    
<br/>        战斗并不是你一招我一招慢慢过招，真正的胜负，永远都在一招之间。
<br/>    
<br/>        那么问题来了，按照这个说法，那不是说修行者只需要修行一种术法就够了，而实际上，每一个修行者，总是修行了各种各样的奇奇怪怪的术法。
<br/>    
<br/>        这是为了什么呢？
<br/>    
<br/>        这个问题大概可以归结为：打偏了。
<br/>    
<br/>        一次性将自己大量的法力，或者说全部法力打空，若是这一招打偏，那事情就大条了。
<br/>    
<br/>        那会极其尴尬，因为可能会危及自己的性命。
<br/>    
<br/>        所以修行的其他，术法，便是用来寻找那个最为合适的机会的，一个不会打偏的机会。
<br/>    
<br/>        所以，这种方式并不是最常见的战斗方式，这种战斗方式只适合天才，真正的天才，真正的天才才敢这么去赌，他们有着充足的自信，对方无法躲开。
<br/>    
<br/>        仙术分为普通的仙术，以及大仙术，大仙术便是仙王才能发挥到极致的仙术，大罗金仙，普遍也会，但是无法发挥到极致的威力，而十大仙术，便是所谓的能够一招灭杀敌手的招数。
<br/>    
<br/>        像是天才，普遍都是修行的大仙术，只是实力低的时候无法完全发挥出来。
<br/>    
<br/>        同样的法力，施展十大仙术的威力会大于大仙术，大仙术大于仙术。
<br/>    
<br/>        当然，这说的是同阶的情况。
<br/>    
<br/>        要是境界不一样，那问题就大了。就好比一个地仙和超脱，地仙可以仙术随便甩，很多次机会，超脱最多用一次，而且超脱还不一定会仙术.......
<br/>    
<br/>        有修行者看向陈明，笑道：“星龙太子也不过如此啊，居然拿九太子一点办法都没有。”
<br/>    
<br/>        “什么星龙，根本比不上九太子。”
<br/>    
<br/>        “星龙呢，难道是战斗太久无法施展吗？”
<br/>    
<br/>        “九太子，可是连太阳神光都没有使用！”
<br/>    
<br/>        在陈明的背后，一颗颗星辰接连点亮，一条由星辰组成的大龙，从陈明的背后升起，完整版的周天星辰大阵，终于第一次露出了水面。
<br/>    
<br/>        星辰大龙腾空而起，陈明周围，方圆三里的大地，凭空下降一尺，活生生被这星辰大龙压得下陷。
<br/>    
<br/>        这可是媲美仙金的大地，都变成了如此的模样，周围的地仙，浑身遍体生寒，他们若是在这星辰大龙之下，怕不是要被瞬间压成肉泥！
