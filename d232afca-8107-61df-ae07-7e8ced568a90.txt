　　迷宫之外，龙腾、龙洌、龙烨、龙城和龙双双五人，伫立在一起，一直在观察着迷宫内的情况。
<br/>　　苏莫抢到了一枚神果，击败了三名奴仆，自然没有瞒过众人的眼睛。
<br/>　　“这小子的实力，貌似也很惊人啊！”龙烨面带惊讶之色，他可是看的清清楚楚，苏莫一招擒拿三人。
<br/>　　要知道，能成为龙族奴仆之人，天赋都不错，能一招擒拿三人，这等实力很惊人了。
<br/>　　“还行吧！”龙腾淡然一笑，笑容显得极为神秘。
<br/>　　“呵呵，龙烨，我的冰儿也厉害吧！”这时，龙洌轻笑出声，就在刚才，他的奴仆冰儿，也击败了两名奴仆，并且得到了一件宝物。
<br/>　　“也不错！”龙烨笑了笑，他们几人的奴仆，没有一个差的。
<br/>　　“龙城，你那个奴仆在干什么？”龙城开口了，面带疑惑之色。
<br/>　　因为此时此刻，苏莫不向迷宫内部搜索，而是不断来到了最外围，并且，不断的顺着迷宫外围搜索。
<br/>　　“他好像是要出来吧？”龙洌道
<br/>　　“他是在找出口！”龙烨疑惑的说道。
<br/>　　众人见此，均是有些惊疑，龙腾的这个奴仆实力很强，挥手间就能击败三人，为何要找出口？
<br/>　　难道是要出来，只得了一件宝物，并且已经服用了，为何还要出来。
<br/>　　就连龙腾都有些疑惑了，不明白苏莫为何要找出口。
<br/>　　“难道是？”转念一想，龙腾便猜到了苏莫的意图，顿时心中无语了。
<br/>　　苏莫这难道是要堵在出口，然后抢夺所有人的宝物吗？
<br/>　　龙腾不怀疑苏莫的实力，他主要是担心，苏莫别把事情搞大了，最后暴露出身份。
<br/>　　若是身份暴露，他可护不住苏莫，龙族很可能将苏莫交出去。
<br/>　　不过，他转念一想，苏莫不是鲁莽之人，应该会注意分寸。
<br/>　　只要注意好分寸，以苏莫的实力，这个办法倒也可行。
<br/>　　毕竟，大多数人得到宝物之后，若是神果神药的话，也不会立刻吞服。
<br/>　　比如，苏莫刚才得到的那枚神果，其实药效并不长，若不是在冲击境界瓶颈之时服用，那就是浪费了。
<br/>　　迷宫之中。
<br/>　　苏莫按照进入迷宫之时的记忆，在寻找出口，因为他知晓迷宫城最外围城墙的方向，所以这个并不难。
<br/>　　遇到岔路，他转变方向，但是再遇到岔路，他会再把方向调正。
<br/>　　仅仅是片刻的功夫，他便来到了迷宫城外墙的墙下。
<br/>　　不过，迷宫城的四方外围城墙，非常的长，想要找到出口，也不容易。
<br/>　　最主要的是，通道四通八达，一堵堵高墙隔离，无法畅通无阻的顺着外围城墙寻找出口。
<br/>　　“按照之前观看，出口是在不断的移动，以那种移动的速度，半个时辰差不多就能绕迷宫一圈。”苏莫心中暗道。
<br/>　　他未进迷宫之前，已经仔细的观察了一番迷宫，所以这些情况都了解。
<br/>　　所以，他决定不走了，就在此等待，等待出口自行到来。
<br/>　　不然的话，他想要找到出口，还不知道要花费多少时间，很可能远超三个时辰。
<br/>　　毕竟，他就算要顺着外墙寻找出口，也要不远的饶过一堵堵内墙。
<br/>　　所以，等待才是最好的方法。
<br/>　　反正，出口是唯一能出去的地方，而那些被淘汰出去的人，肯定也带不出去宝物。
<br/>　　所以，他守住出口，便能抢夺到大量的宝物。
<br/>　　随即，苏莫就在外墙边，就地盘坐了下来，开始了修炼。
<br/>　　因为，他吞服了那枚神果，药效还在，他不想浪费了药力。
<br/>　　果然，他一运转体内源力，药力的效果大大的显现了出来。
<br/>　　修炼的速度非常的快，源力快速的凝练，是平常的八九倍以上。
<br/>　　他也没有吸纳源石，就只是单纯的凝练源力，效果已经非常好了。
<br/>　　苏莫在安安静静的修炼，但是迷宫城各处，却是血雨腥风，宝物的争夺极为惨烈。
<br/>　　五百多件宝物，几乎是不断的出现，每一件宝物都不可避免的要遭到抢夺。
<br/>　　几乎只要有两个人相遇，便会出手，毕竟，任何一个对手，身上都可能有宝物。
<br/>　　所以，在迷宫城之中，不断的有人被淘汰，也不断的有人陨落。
<br/>　　更有一些相熟的奴仆，三三两两的联合起来，实力大增。
<br/>　　几乎只是一刻钟的时间，迷宫城之中的奴仆，便被淘汰了大约三分之一。
<br/>　　也有个别几个奴仆，得到了宝物之后，找到了出口，从而离开了迷宫城。
<br/>　　但是，这种人屈指可数，也就三两人而已。
<br/>　　迷宫城之外。
<br/>　　龙腾几人伫立在一起，时刻关注着迷宫城内的战况。
<br/>　　“哈哈！凝雪干的不错！”龙洌大笑出声，因为就在刚才，凝雪击败了两位奴仆的联手，成功抢到了一件宝物。
<br/>　　而他的另一位女奴冰儿，早已抢到了一件宝物，两人虽然不在一起，但表现都是不错。
<br/>　　“巫铨也不错！”龙城脸色冷漠的说道，虽然巫铨之前败给了那个炎空，但是，实力还是非常强的。
<br/>　　不知道是不是受了炎空的气，巫铨出手非常狠辣，已经击败了近三十人，其中十人被杀。
<br/>　　而巫铨击败了三十人，也成功得到了一件宝物。
<br/>　　“石崐也不差！”龙烨笑着说道，他的奴仆石崐，同样得到了一件宝物，而且比巫铨等人得到的还要早。
<br/>　　石崐运气比较好，不是靠抢夺，而是自己寻到了一件宝物。

<br/>　　要知道，在这么大的迷宫城之中，二十万人的争夺之中，能这么快得到一件宝物，已经非常的难得了。

<br/>　　这简直是奇葩，迷宫之中，所有人都在快速的争夺宝物，而龙腾的那位奴仆，居然在修炼，简直是不可思议。

<br/>　　“开什么玩笑？”
<br/>　　龙城闻言嗤笑一声，摇头说道：“他目前在修炼，等彻底消耗了神果的药力，迷宫争夺都会结束了，你告诉我，他怎么成为最大的赢家？”
