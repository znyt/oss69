<br/>　　“玩？”

<br/>　　涂小安这一抽，可不是闹着玩，抽到地面，绝对是山崩地裂。
<br/>　　他也不怕一鞭子能把这孔雀给抽死。
<br/>　　要是就这样的死了，也太辱没高等生物的称号。
<br/>　　没一会儿，一道狼狈的身影从凹凸的坑中爬了出来，只见青枭慌忙的梳理自己的羽毛，一边还哭泣：“呜呜呜，我的羽毛，我的羽毛”
<br/>　　涂小安一头黑线，旋即眼睛一亮：“臣服于我，不然我拔光你的毛”
<br/>　　此话一出，对于这只美爱的孔雀来说，简直就是末日审判的号角，吓的三魂不见了七魄。
<br/>　　“意识笼罩，给我收”
<br/>　　涂小安看到孔雀乱了心神，立刻决定收服。
<br/>　　意识一笼罩，孔雀浑身一颤，一股窒息感侵袭着它，压迫挤压，它用惊骇的眼神看向扶摇直上的巨蟒，充满了畏惧起来。

<br/>　　涂小安大喝一声，无法强制性收服真是麻烦，他必须不断的意识力给孔雀压力，逼着他臣服。
<br/>　　一分钟。
<br/>　　三分钟。
<br/>　　五分钟。
<br/>　　孔雀青枭还在拼力挣扎，它深深的明白，要是臣服了，它就沦为了这条蛇的宠物。

<br/>　　它可是鸟类之王，高贵不凡，当一条蛇的宠物，还不如死去。

<br/>　　涂小安吃喝一声，青枭又委屈又后怕，活脱脱一位被怪蜀黍欺负的小女孩。
<br/>　　“看来要换一个政策，威逼还要利诱”涂小安灵光一现，意念一动，一片蛇皮出现在了青枭的面前。
<br/>　　这片蛇皮散发着异香，又蕴含能量，顿时吸引了孔雀的注意力，它的眼睛亮晶晶起来，恨不得立刻将蛇皮吞到肚子里去。

<br/>　　让它根本抵御不了诱惑，长长的嘴一啄，蛇皮已经被它吃到嘴巴里去。
<br/>　　这一吃，也就忘记了反抗笼罩在自己身上的那股压力。

<br/>　　涂小安眼睛明亮起来，得了一只百鸟之王，还有十万能量值，简直不要太爽。
<br/>　　忽然，他转念一想，要是吞了这只孔雀，能量值会不会更多一点。
<br/>　　不过这只频临绝种的绿孔雀，血统高贵，又是觉醒生物，收了比吞了有价值点。
