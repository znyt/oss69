    <br/>    <br/>    鹿城郡，水仙镇。<br/>    <br/>    鹿城郡的百姓们都很富足，如果以南平江来划分的话，南平江往南的百姓生活比江北的百姓生活要更好一些，鹿城在这繁华之中也称得上是一颗璀璨的明珠。<br/>    <br/>    鹿城和距离只有几百里之外的临江城被人称为一道双城，临江城在楚时候名为泉亭城，就是唐匹敌带着十三盟的降兵千里突袭的那座城市。<br/>    <br/>    大宁立国之后不久，泉亭城就被定为道府所在，所以相对来说在发展上占了些便宜，大宁初期改名为杭城，后又改名为临江。<br/>    <br/>    鹿城能与临江城齐名，可见其繁华。<br/>    <br/>    水仙镇是典型的江南水乡风格，白墙碧瓦小桥流水，水仙镇的人大部分都经商，一年四季男人们在家的时间都不多，大部分都是女人操持家务。<br/>    <br/>    富户家里的田基本上都租种给农户，当然也不在乎那几个租田的小钱，大抵上每年多给一些粮食和蔬菜更让他们开心，开心起来连田租都可能免了。<br/>    <br/>    水仙镇最外边一排房子，村子的一角，那一户人家常年在外，据说定居在了杭城，半年多前，有人来把这宅子买了下来，也只是买房子的时候来过，之后两三个月也不见人回来。<br/>    <br/>    三个月之前，这户人家回来了，深居简出，村子里人也没什么多关注，这种事寻常之极。<br/>    <br/>    两层的白墙小楼，左右各有厢房，院子不大，中间有一座水池，四周铺着石子，水池一侧种了些竹子，看起来绿意盎然。<br/>    <br/>    陈三阳坐在院子里的摇椅上闭目养神，不得不说，鹿城这边的气候真的太好了，虽然他老家江南道鱼鳞镇的气候也不错，可是临江显得太过潮湿，而且鱼鳞镇其实还算是北方，冬天也冷的出奇，下雪是常事。<br/>    <br/>    而鹿城这边，虽然夏天显得热了些，可是冬天也不冷，风吹在人脸上和江边的风完全不一样，是那种清爽的让人觉得无比惬意的感觉。<br/>    <br/>    他已经在这躲了三个月，似乎已经习惯了安宁。<br/>    <br/>    两个手下从外边急匆匆的回来，进了院子之后俯身说道：“东家，出了点事。”<br/>    <br/>    陈三阳睁开眼睛，立刻坐直了身子，看起来很享受的他瞬间就紧张起来。<br/>    <br/>    “怎么了？”<br/>    <br/>    “两个伙计失踪了。”<br/>    <br/>    其中一个手下说道：“前天出去办事，到现在也没回来，打听到了他们曾经去了鹿城的赌场，在里边用了官银。”<br/>    <br/>    “这两个废物！”<br/>    <br/>    陈三阳猛的站起来，在院子里来来回回踱步，片刻之后朝着屋子里喊了一声：“收拾东西，准备搬家。”<br/>    <br/>    他老婆抱着孩子从屋子里出来，脸色难看起来：“又搬家？还能去哪儿？这样提心吊胆的日子还要过多久啊。”<br/>    <br/>    “别废话！”<br/>    <br/>    陈三阳瞪了她一眼：“我想这样搬来搬去的？”<br/>    <br/>    他看向那两个手下：“去套车，立刻就走。”<br/>    <br/>    然后又朝着他老婆喊：“带金银细软，收拾几件衣服就行，其他的东西都不要带了。”<br/>    <br/>    他夫人叹了口气，抱着孩子回屋吩咐丫鬟收拾东西。<br/>    <br/>    那两个手下人连忙跑出去套车，不多时马车就在门口停了下来，两辆车，住在后院的伙计也过来十余个，看起来也都跟着紧张起来。<br/>    <br/>    “听我说。”<br/>    <br/>    陈三阳把伙计们召集起来：“我们这次得分开走......我把家人托付给你们了。”<br/>    <br/>    他回头看了看屋子里，然后压低声音继续说道：“我知道你们也怕，可当初我们也别无选择，要么死，要么合作，我黑了他们的银子分给大家伙，足够大家安身立命......他们的目标是我，所以我得和你们分开走，你们护送我的家眷一路往北，走水路，但不要走大运河，直接回安阳郡，他们想不到你们会回去，过了南平江之后再往北走三百里有个地方叫清隽山，那山下有个镇子，我在那置办了一座宅院，你们直接过去。”<br/>    <br/>    他缓了一口气后说道：“我先去鹿城晃一圈，不管是不是宇文小策的人，发现我之后会跟上我，我尽量把人给你们带开......”<br/>    <br/>    他说到这，撩开衣袍单膝跪下来：“我的家人，就拜托你们了。”<br/>    <br/>    几个伙计连忙过来把他扶起来，其中一个说道：“东家，不然咱们去投靠票号吧，票号不会杀了我们，可是那些人会。”<br/>    <br/>    “我不是没有考虑过。”<br/>    <br/>    陈三阳道：“我得先探探情况......这样，如果我这次没出事，我会想办法联络票号的人，这件事和你们其实关系不大，票号不会太为难你们......”<br/>    <br/>    他沉思片刻后说道：“谁愿意跟我走？跟我走的，九死一生。”<br/>    <br/>    那些伙计互相看了看，有人说道：“东家，我们都愿意。”<br/>    <br/>    “抽签吧。”<br/>    <br/>    陈三阳道：“跟我走两个人，其他人去清隽山。”<br/>    <br/>    几个月之前，重安郡费县的县丞劳崖派人找到他，说是县令大人的一个亲戚第一次做生意，进了一大批海货，可是根本不懂得怎么经销，眼看着要亏钱，所以着急把货出手，县令大人的意思是，如果票号看着货还可以的话直接收进去，就按照进价给，运费都不要。<br/>    <br/>    做生意的人，不可能不给官府面子，所以陈三阳带着手下伙计和船队去了大开山，可是到了那就被水匪扣住了，水匪留下了一半人，逼迫他们想办法把官银换了，不然的话就把扣下的那一半伙计都杀了。<br/>    <br/>    陈三阳本来不想答应，可是水匪们递给他一件东西，他看了看，竟是自己儿子平日里玩的一件木头玩具，是一辆小木车，他看到之后就知道，没得可选了，更主要的是，他不答应的话，那些人就说要把他做过的龌龊事告诉伙计们，让那些伙计们知道他的真面目，那龌龊事，也是因为票号的银子。<br/>    <br/>    也是因为做过亏心事，所以陈三阳才会谋后路，在远处置办了两处房产，准备着以后藏身用，这次就用上了一处。<br/>    <br/>    但是陈三阳也不想就这么任人摆布，他悄悄派人想去长安报信，他信不过重安郡的官员，一个县令和水匪勾结，那么重安郡郡治府里的官员也没准是他们的人。<br/>    <br/>    可是派出去的人，第二天就被带回来，还是两具冰冷的尸体。<br/>    <br/>    陈三阳无奈，协助大开山里的水匪偷换了官银和粮食，那些水匪押运着银子走，为了稳妥起见不敢一批都带走，所以分成两批运回大开山。<br/>    <br/>    银子起运之前，陈三阳要求大开山的人把他的伙计放了，可是那些人却说被扣下的伙计已经都死了。<br/>    <br/>    陈三阳心中悲愤，所以又悄悄把其中一批官银换了，第二批银子运往大开山的银子是假的。<br/>    <br/>    之后就开始了一路逃亡。<br/>    <br/>    见到宇文小策的时候，宇文小策对他说过，在做些事之前，他就已经派人去了长安百晓堂，因为百晓堂和廷尉府关系密切，他要借百晓堂的口把这案子通知给廷尉府，宇文小策告诉他，只能配合，不然的话他就是主犯。<br/>    <br/>    只是谁也没有想到，李百晓居然把那个人打死了。<br/>    <br/>    这有些脱离了宇文小策的预计，然而并不影响大局。<br/>    <br/>    逃亡之后，陈三阳又派人去长安打听一下消息，他不知道情况到底怎么样了，所以告诉派去的人，直接去百晓堂买消息，看看百晓堂知不知道些什么，也是为了试探。<br/>    <br/>    “兄弟们。”<br/>    <br/>    陈三阳道：“这件事如果有个结果，我会向你们谢罪。”<br/>    <br/>    他张了张嘴，有些话没有说出来。<br/>    <br/>    之所以逃亡之后也没有联络天机票号的人，其实是因为他心虚，他确实也做过一些对不起票号的事，作为重安郡票号的分号大掌柜，他曾经不止一次挪用过票号的银子。<br/>    <br/>    长安城总号被查之后，高小样去了东疆，每三个月向总号递交一次账本的事也就停了。<br/>    <br/>    在这期间，陈三阳确实动了歪心思，他做了一些假账，挪用了大概几万两银子，而之所以一开始动了歪心思，其实......也和宇文小策的人有关。<br/>    <br/>    陈三阳因为生意上的往来，一个老客介绍了别人给他认识，当天夜里去了重安郡郡城里的青楼，在那家青楼里，陈三阳和那些人赌钱，一夜赢了一万四千两，当时开心的已经要飞升了一样。<br/>    <br/>    三天后，他又和那群人聚在一起赌钱，同样的那家青楼，他一夜输掉了三万多两，然后噩梦就开始了。<br/>    <br/>    他后来才醒悟过来，这个局就是宇文小策的人给他设的，让他心里有鬼，关键时候就不敢直接去找票号的人。<br/>
