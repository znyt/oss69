        &lt;h3&gt;第三卷第四百零四章  犹豫的夏颜&lt;/h3&gt;
<br/>
<br/>        夏颜的人生真的是开了挂了，从一个信息老总调到掌管西省整个经济的地方，以后有什么项目都会找到她，比信息老总还要风光。
<br/>
<br/>        她没有想到洛城的事情不但没有动了她反而让她更进一步。
<br/>
<br/>        在她离职之前她需要找一个可靠的接班人，方便以后掌握大量的资源。
<br/>
<br/>        而莲鱼的项目超出了他们的控制范围被易倾秋她们拿到了，这让夏颜大为恼火，将孙毅狠狠的教训了一通。
<br/>
<br/>        最为让她感觉憋屈的是，顾恩赐提出来的几家中标商联合起来，准备搞一个仪式，宋长生已经策划好了，如今却要让陈子昂的公司参加，让她像吃了一颗苍蝇般难受。
<br/>
<br/>        她将宋长生、顾恩赐、许弋澄召集到一起，也将孙毅叫到一起，她要听听他们是如何配合的。
<br/>
<br/>        “你们给我说说为什么到手的鸭子会让你们煮飞了？”夏颜看着眼前的几个人，真的是怒其不争。
<br/>
<br/>        “这一次招标过程中很严厉，我们在开标之前根本就不知道有谁参加，还有抽专家的时候我们只抽了俩个人，从专家人数上来说，我们就失去了先机，虽然他们几个进行围堵，但是这一次有十来家参加投标，而无法辩驳的是易倾秋他们的方案和标书是做得非常好的。有些专家基本都给到了满分，最差也是九点几分。就算我们这边的人给他们打低分，也无法改变这个事实，况且我们不能给太低的分。”孙毅解释着。
<br/>
<br/>        “我们也商量好了价格，顾总是第一，我是第二，许总是第三。”宋长生幽幽的说，他们都不知道为什么会脱标。
<br/>
<br/>        “已经这样了，就不要抱怨了，这个世界上不可能什么都被你掌控在手上，失去了未必就是最好的，我还有事情要做，如果没有别的事我就先行告辞。”许弋澄原本就没有打算和他们几个联手，他手中的两个项目已经在开始了，这一次没有中标他还暗暗欢喜。
<br/>
<br/>        “弋澄，我也不知道事情会这样，最近有个智慧城市的项目，你看看你能不能参加？原本打算做一个包，这样吧，你们都回去做个方案，分上几份拿来我看看。”夏颜吩咐他们。
<br/>
<br/>        智慧城市的项目不是一个小项目，这个设计的范围很广泛，夏颜竟然要给他们几个行外之人做，孙毅张了张口，正准备说话。
<br/>
<br/>        “智慧城市的项目很大，如果你要分的话，只能错区来做，也不能同时来做，需得做一个试点。”宋长生抢在孙毅前开口了。
<br/>
<br/>        “那你们就错区写方案吧！没事了就走吧，长生你留一下，给我说说你们联合会议的事情。”夏颜其实很聪明，她基本不单独见他们，几个人一起来给外人也是他们正常的工作交往，在他们面前让宋长生留下来也是让大家认为她是出于工作的关系。
<br/>
<br/>        大家都走了，留下宋长生。
<br/>
<br/>        “颜颜，你想说什么？”
<br/>
<br/>        “我过几天就要调走了，在推荐人选的事情上我一直犹豫不定，现在孙毅将莲鱼的项目又弄飞了，让我对他的工作能力有了质疑。”
<br/>
<br/>        “颜颜，孙毅的工作能力是不用质疑的，他确实比陈伟冯要强很多。这一次莲鱼的事情不能怪他，况且他做得没错，如果大家都认为陈子霖他们的方案做得好，他们偏偏就给低分的话，那就是让他们暴露在大家的眼前，以后做事情不好再说话，最起码在专家哪里就失去了信任，你想一想？”宋长生的话说得比较隐晦。
<br/>
<br/>        “马总问我有没有合适的人选，如果没有合适的人选他就从别的地方调一个，我说我在考虑考虑。”
<br/>
<br/>        “陈伟冯除了磨叽以外，其实也不错，你现在要做的就是找一个你能靠得住的，知恩图报的，你觉得陈伟冯？”
<br/>
<br/>        夏颜心里一紧，知恩图报是不可能的了，如果陈伟冯知道冯睿的事情恐怕会送她进去，怎么可能会报答她。
<br/>
<br/>        “那就孙毅吧，你最近多和孙毅走动走动，多联络联络感情。”夏颜看了一眼宋长生，最近这一段时间她特别依赖宋长生。
<br/>
<br/>        “行，你说怎么办就怎么办。”宋长生对夏颜言听计从。
<br/>
<br/>        “我大概过一段时间就要走。”夏颜靠在座椅上，摇了摇头。
<br/>
<br/>        “你要调到哪里去？”宋长生走过去给夏颜捏着肩膀。
<br/>
<br/>        “经济部门，陈子乔要调走了，没有人能接手她的工作。”
<br/>
<br/>        “陈子乔工作得好好的，为什么要调走？”
<br/>
<br/>        “原因很可笑，说是家里的俩个孩子要毕业了，家里和工作不能兼顾，找了马总很多次。”
<br/>
<br/>        “哈哈哈，那还真是便宜你了，你去这个部门，我的天，不得了呢。”
<br/>
<br/>        “长生，等我去那个单位稳住了，我就回去和傅海生离婚。”夏颜抓住了宋长生的手，宋长生年轻帅气最重要的是懂她。
<br/>
<br/>        宋长生沉默了一下：“傅海生最近在西城。”
<br/>
<br/>        “我知道！”
<br/>
<br/>        “你们一直都没有联系吗？”
<br/>
<br/>        “没有。”夏颜不愿意多说。
<br/>
<br/>        “你和马总来玩往密切吗？”
<br/>
<br/>        “工作上的事情肯定会有所来往的，平时不怎么联系，其实我真的很感谢冯妈，是她将我介绍给弥青云的，然后弥青云就让马总多照顾我点。”
<br/>
<br/>        “陆战君的妻子？”
<br/>
<br/>        “嗯，她将我认成干女儿了，所以我才能从这个位置到那个位置。”
<br/>
<br/>        夏颜这一解释消除了宋长生心里的疑虑。
<br/>
<br/>        宋长生在和夏颜交往的这一段时间里，夏颜就像一个小女人一样，事事和他商量，并且很依赖他，让宋长生的心里起了莫名的变化。
<br/>
<br/>        原本宋长生以为夏颜和老马有一腿，夏颜这样一解释，再加上这一段时间也并没有发现夏颜有任何不妥的地方，而夏颜有什么事情都会给他说，这无形之中让他最初的目的已经慢慢发生了变化，夏颜在工作上表现得很努力，在生活上表现得很干净，而身份又发生了变化，而且夏颜脖子上的心形吊坠一直被夏颜带着，他并没有得到实质性的他想要的东西，也没有发现夏颜和老马有什么问题，宋长生动心了。
<br/>
<br/>        其实宋长生不知道的是，夏颜只有见到他的时候才会带着他送的项链吊坠，因为她的首饰大多都是马总买的。
<br/>
<br/>        夏颜的包里总有不同款式的项链，她会根据她要去的场合适时的换掉她的首饰。
