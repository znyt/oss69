　　叶尘的办法很简单，就是将自己炼制的丹药，拿到拍卖场去卖。
<br/>　　虽然云丹城最不缺的就是丹药，但是完美品阶的丹药，无论在哪里，都是十分稀少的资源。
<br/>　　否则之前在考核大会上，就不会有人愿意出十几万灵石，来购买那颗完美品阶的筑基丹了。
<br/>　　更何况，叶尘的手中，除了丹药之外，还一些法宝，若是卖出去的话，应该也能卖不少灵石。
<br/>　　离开事务处之后，众人先在附近找了一处普通的客栈，暂时安顿了下来。
<br/>　　按照云丹城的规定，像他们这样的临时居民，可以在城中居住十天，时间倒还算充足。
<br/>　　不过，叶尘并不想在云丹城多待，他现在只想尽快安顿好众人，然后前往迷罗星海。
<br/>　　进入自己的房间之后，叶尘随手布置下了几道禁制，然后便将乾坤镯里储存的药材，尽数取了出来，开始着手研究，炼制何种丹药比较好，
<br/>　　“我如今手上的药材不多，若是用来炼制二品、三品的丹药，即便是完美品阶的丹药，恐怕最多也就只能卖几十万灵石而已！”
<br/>　　“如果想要卖到百万之上的天价，恐怕至少也得是四品灵丹，而且还得是完美品阶！”
<br/>　　“这种丹药必须是实用性很高，而且难度不能太大，否则以我如今的炼丹水平，恐怕很难百分百炼制出完美品阶的丹药……”
<br/>　　叶尘寻思了片刻，将眼前药材，可能炼制的四品丹药，在脑海之中，一一过了一遍，忽然两眼一亮，
<br/>　　“有了！化婴丹！”
<br/>　　所谓的化婴丹，顾名思义，便是能够帮助金丹境巅峰境界的修士，突破到元婴境的灵丹。
<br/>　　普通品阶的化婴丹，可以提升两成左右的突破概率。
<br/>　　而精品级别的化婴丹，可以将概率提升到六成。
<br/>　　至于完美品阶的化婴丹，则是九成以上！
<br/>　　在诺大的云丹城之中，那些被困在金丹巅峰境界的修士，肯定不在少数。
<br/>　　普通品阶的化婴丹，对于他们来说，并没有太大的用处，可能要一连失败数次，才能成功突破。
<br/>　　而一颗完美品阶的化婴丹，足以将他们突破到元婴境的时间，提前数年乃至十几年！
<br/>　　要知道，金丹境修士的寿元只有四、五百年，而元婴境修士的寿元，则提升了足足一倍，可以达到千年以上！
<br/>　　或许对于那些年轻的修士而言，早几年突破到元婴境，并不算什么大不了的事情。
<br/>　　但对于那些，已经走到生命尽头的金丹境修士来说，
<br/>　　突破，
<br/>　　就意味着延长寿命！
<br/>　　他们肯定不惜一切代价，也会把握这样的机会。
<br/>　　这些念头在叶尘脑海中闪过之后，叶尘顿时便下定了决心，
<br/>　　“好！就选化婴丹！”
<br/>　　当下，
<br/>　　叶尘说干就干，迅速将炼制化婴丹所需要的药材，全部整理了出来。
<br/>　　一切准备就绪，叶尘微微调整了一下自己的气息，将状态调整到最佳，这才取出吞天鼎，然后祭出五昧真火，开始正式炼丹。
<br/>　　化婴丹在四品丹药之中，属于中上等，即便叶尘动用了吞天鼎，拿出了自己最好的状态，也没有绝对的把握，可以炼制出完美品阶的丹药。
<br/>　　而且，叶尘估算了一下药材的份量，最多仅仅只能供他炼制三次而已。
<br/>　　呼呼！！
<br/>　　随着猩红色的火焰，在吞天鼎中升腾起来，叶尘开始将药材，按照顺序，缓缓投入鼎炉之中。
<br/>　　大约半个小时之后，
<br/>　　“滋滋滋~~”
<br/>　　伴随着一股浓郁的丹香，在整间屋子里飘荡开来。
<br/>　　化婴丹，顺利炼制成功！
<br/>　　然而，
<br/>　　叶尘却一点也高兴不起来，
<br/>　　“只是精品品阶吗？果然，想要炼制出四品完美品阶的丹药，绝对不是一件容易的事情啊！”
<br/>　　如果这番话，要是让其他四品炼丹师听到，估计连杀了叶尘的心都有了。
<br/>　　那可是四品精品品阶的化婴丹，他居然还不满意！
<br/>　　叶尘将炼制出的四品化婴丹，像丢垃圾一般，随手丢在一旁，然后便再次祭出火焰，开始了第二次尝试。
<br/>　　一晃眼，
<br/>　　又过去了半个多小时。
<br/>　　“滋滋滋~~”
<br/>　　化婴丹再次顺利成型，叶尘的眉头却皱的更紧了，
<br/>　　“还是差了一点！难道以我如今的炼丹水平，真的无法炼制出四品完美品阶的丹药？”
<br/>　　眼看着眼前的药材，只能再炼制一次化婴丹，叶尘不由得有些沮丧，
<br/>　　“罢了！实在不行的话，只能炼制其他四品丹药试试了！”
<br/>　　一念及此，叶尘几乎已经不再抱希望，随手将剩下的药材，全部丢入鼎炉之中，开始了第三次化婴丹的炼制。
<br/>　　这一次，不知道是不是因为，叶尘没有了之前的必胜之心，反而在冥冥之中，触摸到了一丝天道……
<br/>　　“滋滋滋~”
<br/>　　“轰隆隆！”
<br/>　　就在这第三颗化婴丹，成型的那一瞬间，周围的空间，忽然一阵剧烈晃动！
<br/>　　竟然连叶尘布置在周围的禁制，在那道天地之力的攻击下，似乎随时都要崩溃。
<br/>　　而与此同时，
<br/>　　客栈之外，天空之上，风云变色，雷音滚滚，足足持续了好一会，方才渐渐散去，又恢复了之前的晴朗，仿佛什么事情都没有发生过一般。
<br/>　　一时间，周围过往的行人，纷纷驻足，个个惊呆。
<br/>　　而此时，
<br/>　　云丹城四面八方，一股股强横的气息，突然冲天而起，伴随着一道道惊呼声，
<br/>　　“异象生！天地变！这是四品以上宝丹出世的征兆啊！！”
<br/>　　“难道是宗主大人又开炉炼丹了？”
<br/>　　“不对！这道气息的方向，并不是来自云丹宗！”
<br/>　　“莫非我云丹城，又来了哪位丹道高人？”
<br/>　　“走！去看看！”
<br/>　　……
<br/>　　那一道道强横的气息，纷纷朝天地异象出现的方位赶了过来。
<br/>　　叶尘并不知道，他炼制出了一枚完美品阶的化婴丹，竟然弄出这么大动静，甚至将整个云丹城的高手，全都吸引了过来。
<br/>　　此刻，他已经将丹药收入玉瓶之中，连同之前那两枚精品级别的化婴丹，一并收入乾坤镯。
<br/>　　出了房间之后，叶尘先嘱咐了灵山等人，不要离开客栈，然后便出了门，雇了一辆车，直奔云丹城最大的拍卖场而去。
