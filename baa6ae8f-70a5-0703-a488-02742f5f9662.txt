<br/>				陈默手里拿着两只鸡子，然后施施然的找到了小舅。<br/>　　<br/>　　大黄和大灰一见到陈默，顿时欢喜的跑来跑去。两只小狗又长大了一圈。这家伙，可是眼看着就渐渐变大，溪水的能力还真是强，这小狗长的非常壮实，毛色还油光锃亮，毛色顺滑。<br/>　　<br/>　　“小舅，将这两只鸡子做了，就做清炖的就成。一只给两个老头吃，一只咱两吃。鸡子的内脏什么的，都喂狗，不过一定要煮熟啊！”陈默蹲下来摸着两只小狗，边和小舅交代着。<br/>　　<br/>　　“好！”小舅接过鸡子，就动手宰杀。本来中午饭都做好了，不过加个炖鸡也没有什么，耽误不了多长时间。<br/>　　<br/>　　村子里要是吃鸡的话，基本都是要拿高压锅炖的，因为都是柴鸡，普通的锅没有一个小时，炖出来的鸡根本咬不动。所以高压锅炖鸡就成了必须，只要半个小时就好了。<br/>　　<br/>　　陈默将鸡子给了小舅后，也顺便看了看这个厨房。集装箱制作的厨房送过来的时候自己不在，等到现在再见到，不过厨房的东西很全，而且都是配置的不锈钢，还不错。<br/>　　<br/>　　“陈小子，你回来了？”何老笑着问道，在陈默转身回屋的时候，吴老和何老从外面走进院子。<br/>　　<br/>　　“嗯！回来了，今天中午的饭要稍等片刻，我抓了两只鸡子让我小舅炖上了，看看样的鸡子味道如何。”<br/>　　<br/>　　“好！没有问题，早就看着那些鸡子想吃了！”吴老也是个吃货，来到陈默这里后，发现这里的饭菜实在是太合自己胃口了，不仅吃的好，也睡的好，每天精神焕发，都不像是有心脏~病的人。<br/>　　<br/>　　“对了，先不谈鸡子的事情。那个我找你有点事情。”何老说道。<br/>　　<br/>　　“你说！”陈默边听着，边去了房子里，将里面的茶壶什么的，还有水都提了出来，放到自己弄得一个小凉亭里。凉亭里有电源，插好后和两个老头一起喝茶。<br/>　　<br/>　　“你的茶不好，先等等，我去拿我的茶过来！”吴老看到陈默要烧水喝茶，就说道。<br/>　　<br/>　　陈默的茶叶就是茉莉花茶，所以没有啥好喝的，还不如自己带来的茶叶呢！<br/>　　<br/>　　等吴老将他的茶叶拿过来，水也差不多快烧开了，此间本来又是的何老，也不说话了，都在等茶水。<br/>　　<br/>　　由于有外人在，所以陈默并没有用溪水，而且使用的过滤水。烧开后。吴老接过茶壶开始泡茶，他对陈默的泡茶手段实在是看不上，还是他自己来的好。<br/>　　<br/>　　喝茶虽然南方和北方都有区别，但是泡茶的水温讲究还是差不多的。有些茶不能用滚水，有些茶必须用滚水等等。甚至还有些茶水烧开以后，要稍稍静置片刻，等水温下降几度后，才能泡茶。<br/>　　<br/>　　对于这些，陈默是一概不懂。他讲究的是渴了喝茶，能解渴就成。另外就是有茶的香味和苦味就成。至于说回味甘甜还是回味悠久，都没有任何的印象。<br/>　　<br/>　　吴老拿过来的茶叶，属于绿茶，所以要等水温降到八十度左右泡茶，这样泡出的茶叶回味甘甜，没有太大的苦味，而且茶叶的清香也能够泡出来，还不损茶叶中的各种有益成份。<br/>　　<br/>　　不过看着陈默喝茶的动作，吴老的眼角就抽抽，实在是浪费了自己的好茶和泡茶手艺。陈默是待茶温差不多的时候，拿起玻璃杯就大口的喝完，跟平时喝水解渴般。<br/>　　<br/>　　“我看还是茉莉花茶适合你！”吴老实在看不下去了，郁闷的说道。<br/>　　<br/>　　“还好。不过吴老你泡的茶比我的茶叶好喝多了！”陈默喝完茶水后，感觉这个茶叶比自己喝的茉莉花茶要好喝的多。<br/>　　<br/>　　“废话！老吴的那点茶，可是一两上万元，当然好喝了！”何老静静的品着茶，看到陈默如此喝茶，他也是眼角抽抽，真是不当人子啊！<br/>　　<br/>　　“嘿嘿！”<br/>　　<br/>　　陈默知道自己喝茶的动作让俩老头看不惯！但是看不惯就看不惯吧，反正自己只要喝着爽就成。看看两个老头对这点茶叶珍惜的样子，等乾坤珠内的茶叶出来后，看看这两个老头还想说什么？哼哼！<br/>　　<br/>　　“对了，何老！你不是有事情找我么？”<br/>　　<br/>　　“不错！”<br/>　　<br/>　　“你说！”<br/>　　<br/>　　“那个，我昨天转到你盖房子的哪里。”何老用手指了指隔着马路的那个工地，接着说道：“你堆在哪里的旧木头卖给我怎么样？”<br/>　　<br/>　　“哪里堆放的旧木头？”<br/>　　<br/>　　“不错！”<br/>　　<br/>　　“那你出多少钱？”<br/>　　<br/>　　“你的眼力劲也不错，所以我也不哐你！哪里大梁5根，小梁总共有十几根，我估算了一下，大概有个4吨左右，总共给你四千万怎么样？”<br/>　　<br/>　　“四千万？”陈默一时间有些无语，那些木头价值四千万？而且夸自己眼力劲不错？呵呵！过誉了啊，自己不过是凭借神识，要不然还真没有什么眼力劲的。就像上次一样，那是什么木头，自己是不可能分辨出来的，虽然自己是学生物的，但是做成木头盒子以后，这木头是什么鬼，哪里能分辨的出来？<br/>　　<br/>　　“陈小子，四千万已经很好了，虽然有所出入，但是也差不离！”吴老也接口说道。<br/>　　<br/>　　那些废旧木头价值四千万，陈默听后也是懵比的。这些木头拆下来的时候，都被熏的黑黑地，他也就没有仔细观看，不过看到这些木头没有什么开裂或者虫蛀等，所以就堆放在一起，准备以后盖好院子后搭建个凉亭什么的能够用到，没有想到的是现在这么值钱。<br/>　　<br/>　　“何老，那些木头拆下来后，我都没有仔细看，就是让人堆在一起，我还不知道它们这么值钱！”<br/>　　<br/>　　陈默笑着说道，不过在心里倒是想到了，这些老宅是在特殊时期盖的房子，当时听说是拆除了一家地主家里的老房子，然后一些村民盖房子的时候，就拉过来用了。<br/>　　<br/>　　不过这个事情也是年代太久远了，自己也是听村里的老人说的，至于说是不是这么回事，还真不好估计。不过现在看来，可能就是真事了。<br/>　　<br/>　　俩个老头面面相觑了一下，没有想到陈默连看都没有看过，真为那些木头不值！<br/>　　<br/>　　“陈小子，实话说吧，那些都是海黄，你明白了吗？”何老喝了一口茶，压了一下心脏咚咚挑的痛感觉，这小子绝对是来气咱们的。<br/>　　<br/>　　陈默听见是海黄，顿时就明白这个价格还真是差不多，可能还高了点。现在海黄在市场上，已经卖到了800万到一千万每吨，这还是有价无市的结果，随着时间的推移，价格还在涨。<br/>　　<br/>　　不过陈默倒也不没有说贪心什么的，要是没有何老看出来的话，那么这些木头依然会让陈默使用掉，至于使用的时候会不会察觉是海黄，还真不好说。<br/>　　<br/>　　虽然陈默是学生物的，也知道海黄的一些特点，但是对于那些旧木头，他会不会去看都是个问题，而且就算是去看了，能不能看出来也是个问题。最有可能就是找个木匠，然后让他们直接制作凉亭了。<br/>　　<br/>　　好在现在何老将这些木头发现，并且要出四千万买来，这就好了，自己葫芦谷的开发又有些钱可以投进去了。<br/>　　<br/>　　“行！既然何老想要，那就拿去好了。”陈默大方的说道。<br/>　　<br/>　　“好，我现在就找车来啦!”何老听到陈默开口答应后，顿时高兴的说道。<br/>　　<br/>　　“那是咱两个人的，可不要都成你的了！”吴老在旁边赶忙喊道。<br/>　　<br/>　　“放心！东西先拉回去放到我哪里，等回去以后，你在把你的那一份拉走就成。”<br/>　　<br/>　　看来这两个老头都将东西分好了，就差陈默答应了。<br/>　　<br/>　　“何老，你来啦木头的时候，千万不要说是四千万买的，就说是这些木头你拉回去当柴烧的。”<br/>　　<br/>　　陈默对何老说道，如果消息走漏的话，村里的人可能会所三到四，或者眼红也说不定。<br/>　　<br/>　　“好，没有问题！”何老点头答应道，对于陈默的担心，他一下就能想到，所以也就在电话中叮嘱了几句。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
