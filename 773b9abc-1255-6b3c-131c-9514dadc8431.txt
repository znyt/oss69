　　苏莫找到了宫务大殿，正是上次入门之时所去的大殿。
<br/>　　在大殿之中，他顺利的完成了进入内宫的手续，成为了内宫弟子。
<br/>　　人王宫弟子，只要有真神境的修为，便能直接进入内宫之中。
<br/>　　当然，苏莫办理入住内宫的手续，报出段惊天的名号之后，不免引起了很多弟子的关注。
<br/>　　毕竟，他现在不再是无名之辈，即便过去了几年时间，也有很多人记得他这个妖孽。
<br/>　　内宫很大，弟子居住的区域，是一片连绵的院落群。
<br/>　　苏莫轻易的找到了自己的院落，占地颇广，大门之上的匾额，空无一字。
<br/>　　唰！
<br/>　　苏莫取出长剑，连连挥动，立刻之匾额之上，留下了三个龙飞凤舞的大字——惊天府。
<br/>　　他知道，每一个内宫弟子，都有一座自己的院落，由自己命名，所以，他便取了个名字。
<br/>　　院落很大，其内有几个房间，有花园、有池塘，如同一个小型的府邸。
<br/>　　苏莫找到了一个修炼室，再次准备修炼。
<br/>　　不过，内宫弟子的修炼室，果然不是外宫弟子可比。
<br/>　　修炼室之内，有强大的阵法，能够辅助武者吸纳天地源气。
<br/>　　苏莫取出一些源石，也没有用吞噬源神，就这样缓慢的吸纳了起来。
<br/>　　他的源石，还有四五十万，足够再修炼一段时间了。
<br/>　　他准备将源石用完，在出去赚取资源。
<br/>　　不过，修为达到真神之境后，下品源石的效果大大的降低了。
<br/>　　因为，他不再是单纯的炼化源气，而是要提炼其中精纯的本源力量。
<br/>　　“不知道人王宫，能不能弄到混沌源石？”苏莫心中暗道，混沌源石的效果，肯定要比五行源石要好。
<br/>　　咚！咚！咚！
<br/>　　就在此刻，院落之外，突然传来了一阵敲门之声。
<br/>　　苏莫神识一探，顿时双眉一挑，门外是两个女子，一个娇小灵秀，一个美艳动人，正是沁儿和骆妤。
<br/>　　“居然又来了！”
<br/>　　苏莫深深的吸了口气，他知道他在闭关的时候，两人来过一次，没想到他才刚出关，换了住处，两人居然又找来了。
<br/>　　随即，他只能暂时停止了修炼，出去打开了院门。
<br/>　　“两位大驾光临，有失远迎！”苏莫看向门外的两位女子，微笑着说道。
<br/>　　沁儿穿着一身粉红色的皮裙，显得俏皮可爱，而骆妤则是一身白色长裙，显得纯洁高雅。
<br/>　　“段惊天，恭喜你达到了真神之境！”沁儿微微一笑，上下打量着苏莫。
<br/>　　“也恭喜你突破了修为！”苏莫同样微微一笑道，他能清楚的看到，沁儿姑娘的修为是一品高级。
<br/>　　而在几年前的时空界域，对方只是一品中级的修为。
<br/>　　“嘻嘻！本小姐的天赋，突破修为不是很正常嘛？”沁儿得意一笑，显得志得意满。
<br/>　　“段惊天，你不请我们进去坐坐？”
<br/>　　这时，一直没开口的骆妤淡淡的说道。
<br/>　　“两位，请！”
<br/>　　苏莫摆手虚引，示意两人进入院中。
<br/>　　少倾，三人在院中的石桌前，围坐了下来。
<br/>　　“段惊天，你跟我说说，你是怎么闯到熔岩窟第十八层的？”
<br/>　　沁儿如同好奇宝宝一般，凑到苏莫的面前，睁着一双闪亮的大眼睛，好奇的问道。
<br/>　　骆妤虽然没问，但亦是目光凝视着苏莫，显然也是心中好奇。
<br/>　　“呵呵，实力够了，自然便能闯到十八层！”苏莫笑了笑，他不可能暴露出自己的底细。
<br/>　　“切！你这不是废话吗？”沁儿闻言，顿时瞥了撇嘴。
<br/>　　“对了，沁儿姑娘，你是如何让宫中执事找我入宫的？”苏莫沉声问道。
<br/>　　他知晓，此女身份估计不简单，所以想要弄清楚。
<br/>　　“嘻嘻，以本小姐的身份，让你入宫还不是一句话的事情！”沁儿得以的说道。
<br/>　　“哦？沁儿姑娘是何身份？”苏莫问道。
<br/>　　“沁儿的父亲，是宫内十大长老之一，玄天神王！”骆妤道。
<br/>　　“原来如此！”苏莫闻言恍然，原来父亲是神王，这就难怪了。
<br/>　　“爷爷是宫主！”紧接着，骆妤又说了一句。
<br/>　　闻听此言，苏莫顿时心中一震，眸中闪过一丝微不可查的精光。
<br/>　　人王宫宫主吗？
<br/>　　苏莫心中微微激动了起来，沁儿如此身份，会是他实行计划的切入点。
<br/>　　“没想到沁儿姑娘，身份如此不凡，在下失礼了！”
<br/>　　苏莫急忙起身，向沁儿抱拳，继续道：“在此，感谢沁儿姑娘，让我得以拜入人王宫！”
<br/>　　“好啦好啦！感谢就不必了！”
<br/>　　沁儿不在意的摆了摆手，好奇的问道：“段惊天，听说你有万化神剑元神，那你还有什么其它过人之处？”
<br/>　　沁儿心中好奇不已，单凭一个万化神剑元神，绝对不可能以虚神境四变的修为，闯到熔岩窟第十八层。
<br/>　　所以，他非常好奇，苏莫到底有什么厉害之处？
<br/>　　“我血脉之力比较强？”苏莫无奈，只能这样说道。
<br/>　　“你是什么血脉？”沁儿和骆妤同时问道。
<br/>　　“一种从未闻名的血脉，主要是防御力强，所以才能抗住熔岩窟十八层的火属性之力！”苏莫道。
<br/>　　“原来如此！”
<br/>　　沁儿和骆妤点了点头，原来是一种偏向防御的超强血脉，难怪能挡住熔岩窟十八层的火属性之力。
<br/>　　“沁儿姑娘，我对人王宫不熟，以后，还需要你多多关照！”苏莫笑着说道，他还需要和此女套套近乎。
<br/>　　毕竟，此女的爷爷可是宫主，是他来到人王宫，最需要搞定之人。
<br/>　　“行，我会关照你的！”
<br/>　　沁儿点了点头，道：“以后有机会，我们可以一起历练！”
<br/>　　“如此甚好！”苏莫笑了笑，他心中却是感慨，这个沁儿姑娘，实在太单纯了，完全没有什么心机啊！
<br/>　　一旁，骆妤静静的看着苏莫和沁儿，柳眉微皱。
<br/>　　不过，她并未开口，只是在一旁静静的坐着。
<br/>　　沁儿和苏莫闲聊了片刻，便告辞了，带着骆妤，离开了苏莫的院落。
<br/>　　苏莫独自坐在院落之中，低头沉思，他准备日后私下里多接触沁儿。
<br/>　　先了解一些宫主和宫内各大长老的情况，尤其是对于帝一魂和拓跋青云的看法。
<br/>　　此女，就是他展开计划的第一步。
<br/>　　静坐了片刻之后，苏莫再次进入修炼室，开始了修炼。
<br/>　　……
<br/>　　月沁园。
<br/>　　这里便是沁儿的院落，其原名姬月沁，便以自己的名字命名。
<br/>　　院落中，骆妤站在沁儿的身前。
<br/>　　“沁儿，你以后还是少和那个段惊天接触！”骆妤告诫道。
<br/>　　“怎么了？”沁儿疑惑的问道。
<br/>　　“段惊天此人，天赋是不错，但只是血脉防御强而已，修炼天赋还没到妖孽的程度，我觉得他接近你，是有企图的？”骆妤道。
<br/>　　“什么企图？”沁儿微微皱眉说道。
<br/>　　“答案很明显，你如此身份，哪个男人不想高攀？”骆妤郑重的说道。
<br/>　　作为人王宫宫主的孙女，玄天神王的女儿，沁儿是很多人心仪的对象。
<br/>　　因为，傻子都知道，谁若是摘得沁儿这朵花，将成为人王宫最有权势的人之一。
<br/>　　所以，骆妤猜测，苏莫一直都是有目的的，也早就知晓沁儿的身份。
<br/>　　“应该不会吧？”沁儿有些迟疑的说道，平常有很多弟子主动接近她，这点她是知道的。
<br/>　　但是，她不太相信，在时空界域随便碰到的一人，就有如此深远的企图。
<br/>　　“以后你就知晓了！”骆妤冷笑着说道，那段惊天是不是有企图，要不了多久就会显露出来。
<br/>　　沁儿点了点头，不过，他也不在意，平常这么多人接近她，也不会对她有什么影响。
<br/>　　就算那段惊天，真的对她有企图，也不可能得逞。
