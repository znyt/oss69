　　果然不是蛇博士。
<br/>　　当初美林姐发给涂小安的视频中，吸血蛇横冲直撞的背着一个人离开了山野集团。
<br/>　　吸血蛇背上的那个人是谁，涂小安一直很好奇。
<br/>　　除了蛇博士，当今世上，还有谁能控制的住吸血蛇。
<br/>　　新帐旧账？
<br/>　　涂小安留意黑袍男子说的话，在看这丑陋的人对自己恨意滔天，莫非他跟自己有仇？
<br/>　　不会呀，那么丑的人，要是跟自己有仇，涂小安不可能不记得。
<br/>　　一般给人影响比较深刻的有两种，要嘛美如天仙，要嘛奇丑无比，显然，这个人是后者。
<br/>　　悬浮在空中的涂小安极为嫌弃的看着黑袍男子，越看越丑，丑的让他这条蛇都心生绝望。
<br/>　　金银蛇嫌弃的眼神，被江行泽读懂了，他整个人犹如一头愤怒的狮子，仰天咆哮：“该死的蛇，我江行泽要喝你的血，吃你的肉”
<br/>　　咦？
<br/>　　江行泽？
<br/>　　他说他是江行泽，他没死？
<br/>　　涂小安更为的惊讶，当初在实验室，他控制数千蜜蜂同时蛰咬江行泽，这样都能不死吗？
<br/>　　这命也太大了点吧，都说好人不长命，祸害遗千年。
<br/>　　看来，这句话是有点道理的，古人诚不欺他。
<br/>　　如果这丑鬼是江行泽，那么还真是熟人了。
<br/>　　他如今变成这般丑陋，皮肤千疮百孔，莫非是数千蜜蜂蛰咬后的后遗症？
<br/>　　要是这样，涂小安心里就平衡多了，不然他数千只蜜蜂是吃干饭的？
<br/>　　“滚开，你恶心到我了”
<br/>　　戏谑带着极度的嫌弃，悬浮的金银蛇淡淡的对着江行泽开口。
<br/>　　恶心？
<br/>　　江行泽浑身颤抖，整个人怒火中烧，他变成这般人不人鬼不鬼的样子是谁害的。
<br/>　　任何人都有资格嫌弃他的丑，唯独这条该死的蛇没资格。
<br/>　　想当年，江行泽读书的时候，还是校草级别的，如今，沦落到如此丑陋的地步。
<br/>　　那个人不爱美。
<br/>　　“该死的蛇，今天我江行泽发誓，一定要将你挫骨扬灰”
<br/>　　江行泽瞪起眼，狰狞的脸上暴起了一道道青筋，更丑了，丑的日月无光。
<br/>　　他现在什么都不在乎了，只要杀了这条该死的蛇就好，丑怎么了，自己又看不到。
<br/>　　要是能将这条蛇给恶心死，江行泽到是非常的开心。
<br/>　　涂小安揶揄的看到江行泽大言不惭的话，就跟看傻子一样的看着他，这个人怎么就不长记性呢。
<br/>　　无论当初的佘山，还是在实验室中，他都被自己整的很惨，之前涂小安一度以为这个人已经去阴间投胎转世去了。
<br/>　　没想到是他带着吸血蛇离开了山野集团。
<br/>　　可不管怎样，被数千蜜蜂蛰咬而不死，也是他的造化，但他为什么不知道珍惜生命，还敢在自己的面前叫嚣，记吃不记打？
<br/>　　他应该知道，自己想杀他，太容易的，莫非变的如此丑，已经不怕死了？
<br/>　　涂小安恶趣味的胡思乱想着。
<br/>　　“给我跪下”
<br/>　　涂小安的眼神似是冷淡无波又含着高高在上的鄙夷，对着江行泽命令似的开口。
<br/>　　就好像这个人是他的奴才。
<br/>　　跪？
<br/>　　一听跪字，江行泽就要原地爆炸了，当初在佘山，这条蛇就逼着自己下跪，丢尽了脸面。
<br/>　　此时，再度相遇，又让自己跪？
<br/>　　现在他江行泽可是今非昔比，不但成为人类觉醒者，更是得到了吸血蛇为宠物。
<br/>　　他早就不是昔日的阿斗，他有足够的底气面对一切。
<br/>　　“你算什么...”
<br/>　　江行泽刚想破口大骂，倏然地，只见悬浮的金银蛇蛇眸对他微微一瞥，顿时间他的脸色猛然的一变，整个人动弹不得。
<br/>　　一股如山般的压力侵袭而来，压在他身上重如千斤，他的膝盖承受不住的开始弯了下来。
<br/>　　他咬紧牙关的坚持，不让自己跪下，一个余光下，他看到那条悬浮的蛇露出了极为人性化的轻蔑表情。
<br/>　　自己在这条该死蛇的眼中，就好像是蝼蚁，不值一提。
<br/>　　“混蛋...”
<br/>　　又想让自己跪在他面前，不可能的，江行泽现在可是人类觉醒者，一拳轰出去，不下千斤，他岂能在承受这种耻辱。
<br/>　　他死死咬着牙，用尽浑身力气反抗，不能跪，打死也不能跪下。
<br/>　　可...轰..．
<br/>　　江行泽整个人傻眼了，他已然双膝跪地，根本不受自己的控制，而且最可恶的是想站都站不起来，犹如身上压着一座五指山。
<br/>　　“跳梁小丑，我想杀你，就捏死一只蚂蚁一样简单，你知道吗”
<br/>　　涂小安满是嫌弃的看着跪在地上的江行泽，不得不说这人现在真丑。
<br/>　　他到底有什么勇气活下来，都这样了，活下来准备装鬼出去吓人吗。
<br/>　　好吧，他承认自己被吓到了。
<br/>　　“喊蛇王万岁，我或许会跟当初一样饶你一命”
<br/>　　仿佛是旧日的场景再现，江行泽依然被这条蛇逼着跪地，逼着高呼蛇王万岁。
<br/>　　该死，混帐，江行泽气疯了，他有无数次幻想遇到这条该死的蛇，自己是如何快意恩仇的报复，可现如今为什么跟他想象的完全不一样。
<br/>　　简直南辕北辙。
<br/>　　对方一个眼神，自己就只能跪地，完全不能挣扎，难道他这个人类觉醒者是假的？
<br/>　　“吸血蛇，你死了吗，你要干嘛，快帮我”
<br/>　　江行泽忽然想到自己现在有吸血蛇，他被一股无形的力量压的不能动弹，慢慢的，他发现自己快不能呼吸了。
<br/>　　一股熟悉的死亡恐惧感笼罩全身。
<br/>　　他身后的三十米庞大吸血蛇一直眼冒凶光的看着，随后江行泽一声咆哮，此蛇眼中的凶光更甚，有一股骇人的气息迸发出来。
<br/>　　砰..．
<br/>　　涂小安笼罩在江行泽身上的意识力骤然溃散。
<br/>　　江行泽如释重负的赶紧站起来，也不在逞口舌之能，他深深的明白，纵然自己变成人类觉醒者，在这条会飞，会掌控雷电的金银蛇面前，就是渣渣。
<br/>　　太恐怖了，这个世界是变了吗，为什么蛇都那么的牛逼，可他堂堂一个人类，却跟蝼蚁一样的卑微，弱小。
<br/>　　江行泽内心受到了一万点的伤害，这条蛇就好像是他的克星，对它恨意滔天，却拿它一点办法没有。
<br/>　　如今在见，结局还跟当初佘山一样，貌似并没有什么区别，还是一样被蹂躏，被羞辱。
