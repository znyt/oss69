　　山巅之上，白妖王看着天劫结束，脸上挂着淡淡王者的笑容，看来他今天来的真是有点多余了。
<br/>　　不过无妨，他到是庆幸自己来了，这等蛇精要是真被渡劫成功，那么他丝毫不怀疑玄龟说的话，或许真的会对他白妖王产生有一丝威胁性。
<br/>　　但有一点，白妖王想不明白，自己与这条蛇精素未谋面，就算这蛇精血脉高贵，潜力非凡，跟自己有何干系？
<br/>　　难不成这蛇精会来找自己的麻烦？还是跟自己有什么仇恨。
<br/>　　应该有可能吧，他白妖王一路修炼走来，统治一方区域，不知道多少精怪妖灵死在他的手上，直接的间接的，数不胜数。
<br/>　　但无碍了，不管有什么仇什么恨，这条蛇精现在都已经灰飞烟灭了，白妖王也懒得多想。
<br/>　　“大王，那条还算识相，死的透透的，没让大王你出手”一位妖灵讨好的说。
<br/>　　“区区一条蛇精，就算证道了，也配大王出手吗，我等出手都是给他脸了”
<br/>　　两位妖灵奉承着，白妖王呵呵一笑，准备打道回府，妖气冲天的身形刚要消散，可忽然..．
<br/>　　忽然他的一双锐利的眼神猛然朝着山谷看去，眼瞳收缩起来，杀气顿现。
<br/>　　..........．
<br/>　　山谷中，牛妖王跟树精身上笼罩的光芒消失，相互一脸的懵逼，那是既惊既喜，就跟吃了蜜蜂屎一样，跟傻子一样杵在那里一动不动，不断的傻笑着。
<br/>　　这时，牛妖王回神了，他察觉到自己的脚底下好像有什么东西在动，好像被他踩到了什么东西。
<br/>　　“死牛，把你的牛蹄子给我拿开，你想弑主不成”
<br/>　　一道虚弱的声音从牛妖王的脚底下悄然响起，空气都好像被吓得波动涟漪了一下。
<br/>　　卧靠..．
<br/>　　牛妖王几百斤的身子哆嗦了一下，茫然若失，看了看树精疑惑道：“你有没有听到什么声音”
<br/>　　树精沉寂在巨大的喜悦之中，根本没工夫搭理牛妖王。
<br/>　　“一定是幻觉，不可能，不可能的，主人在天劫之下，死的连渣渣都没剩下”牛妖王自言自语，摇了摇头。
<br/>　　霎时，地面一道无比阴冷的蛇类气息蔓延出来，带着让人难以抗拒的威压，顿时间，牛妖王跟树精不禁就诚惶诚恐了起来，一种来自灵魂深处的恐惧油然而生。
<br/>　　“主人？”
<br/>　　“主人，主人？”
<br/>　　牛妖王跟树精那里还能不明白，这股气息太熟悉了，灵魂中的害怕假不了。
<br/>　　牛一脸震惊的将自己一只脚给抬了起来，放眼过去，什么都没看到，到是在土缝中有一抹小光点闪烁着。
<br/>　　仔细看去，这小光点分明就是一条小小虫子，小的几乎可以让人忽略。
<br/>　　这条虫子实在太小了，太纤细了，就跟一根银针似的，不过这纤细的小虫子通体却亮的出奇，若不是这光亮，压根让人捕抓不到。
<br/>　　“主人我证道三阶，现在要进行蛇蜕皮，完成最后阶段，成为三阶蛇王，统领天下群蛇，现在我命令你们两个，方圆五公里内，不需要有任何生物靠近，否则格杀勿论”
<br/>　　这道声音非常的虚弱，但是话语却带着毋容置疑，盛气凌人，傲睨一切，霸气无边。
<br/>　　“是，牛妖王领命”
<br/>　　身为三阶妖灵的牛妖王岂能不明白，但凡证道渡劫成功，身体都会非常的虚弱，自己的主人五十米巍峨的蛇躯被天劫摧残的只有那么一丢丢了，现在又要蛇蜕皮，彻底从二阶精怪层次，成为三阶证道蛇王。
<br/>　　现在至为关键，绝对不能被打扰。
<br/>　　尤其是猴儿山的两位猴大王，它们要是知道蛇精还没死，一定会大打出手，不死不休，前来报复。
<br/>　　须根之间，牛妖王的脸色凝重起来，全身立刻爆涌出恐怖如斯的残暴气息，直接进入战斗状态，成为高大威猛的牛人头，头上崭露牛角，威势赫赫，好似能将天都捅破一个窟窿，气吞山河，一根铁棍就地显化的出现在他的手中，严阵以待，不管是谁来了，都当头棒喝，当场诛杀。
<br/>　　“查理遵命”
<br/>　　只见本就高大挺拔的大树原地就地拔高，成为了遮天蔽日的苍天大树，枝干虬曲苍劲，枝繁叶茂，郁郁葱葱，方圆一公里内全部笼罩在它的树阴影之下，几十根苍穹有力的腾条如触手怪一样的可怕，不容许任何生物靠近，哪怕连一只蚊子都不行。
<br/>　　牛妖王跟树精的举动惹的无数生物都诧异非常，震惊四座。
<br/>　　更让无数生物惊奇的事情陆续发生。
<br/>　　砰..．
<br/>　　山谷中空间波动涟漪起来，一道好似嘹亮的凤鸣声冲天而起，一只浑身散发九彩神光的仙灵孔雀凭空出现，直接落在了苍天大树的树顶，就好像凤凰落到梧桐树上，神圣不可侵犯，举世无双。
<br/>　　嗷..．
<br/>　　山谷地面的空间再次激荡起来，拥有老虎之威，狮子之势的黄金瞳睥睨万物的出现在苍天大树的东面，一出现体形就地暴涨，直接化为猛虎般的巨兽，威风八面，虎啸声震天，惹的无数生物诚惶诚恐，一双阴阳眼光芒大亮，鬼魅皆无。
<br/>　　嘶..．
<br/>　　与此同时，一条三十米巨大，浑身充满邪气的三头蛇显化，三头六眼，凶煞无比，通灵摄厉，直接盘踞在了苍天大树的正前方，一副遇神杀神遇佛杀佛的可怕模样，让人看一眼，就害怕莫名。
<br/>　　猴儿山的无数生物彻底惊呆了，这尼玛什么情况啊，已牛妖王这只三阶妖灵为首，一颗苍天大树试比天高，另外出现三头气息恐怖的超等生物，那种凌驾普通生物之上的气息无与伦比。

<br/>　　一股股超强生物的气息侵袭了整个山谷，山谷的天又要塌陷了一样，空气爆响声连连，空间被这些气息搅的扭曲起来。
<br/>　　它们都在干什么，严阵以待，杀气腾腾，好像跟妖灵山脉的所有精怪妖灵都有生死大仇一样。
<br/>　　妖灵山脉任何生物要是敢踏入山谷，那么就会立刻被攻击，死无葬身之地。
